Hi 大家～ 我是 TonyQ ，算是長期以來在這裡參與討論的技術人， 板上可以查到不少我過去寫的一些不堪入目的文章。XD 因為 JS 其實一直以來都是披著一層面紗的語言， 很多人可能不知道自己到底會多少，會的也不一定很深入， 加上大家用的伺服器端語言都不太一樣，溝通、討論上可能會有些困難， 不太確定怎麼跟其他語言的使用者溝通，加上範例難以單元化， 所以網路上的討論其實很有限，有個好的交流的管道是重要的。 小弟這幾年來也算是略有一點小小小心得， 我在想可能從二月開始在台北弄個 js developer 的討論聚會， 就一個簡單的讀書會，以 JS 開發的經驗分享為主。 大概兩週到三週一次，會有一個週間的晚上，來聊聊技術。 大概是七點開始到十點左右，約 2-3 個小時進行討論、交流。 我自己會準備一些 10 ~ 20 min 的小講題作為資訊分享， 剩下的時間就是跟大家閒聊技術上的東西， 也歡迎大家帶自己的 project 來討論或者分享經驗。 我們歡迎學生也歡迎相關工作從業人員來聊天。:) 具體的時間第一場應該會先訂在 2/21 ~ 2/23 之間的某一天晚上， 基於人數管控的理由我們會採取報名制， 考慮到讀書會的本質跟我們交流的效率， 大概只會找 15~25 人的小眾討論， 等我敲定好場地的問題之後會再通告報名機制跟場地。 原則上第一場的主題我想我會分享 jQuery 的議題， 帶大家回顧一下這幾年 jQuery 到底解決什麼問題， 這幾年網站開發的變化，套件跟著發展了什麼跟其中有趣的事情。 當然，都會是跟專案開發有關的知識，我們畢竟不是考古學家。（笑） 這是個人自發性的活動，活動除了場地可能會有的低消以外， 基本上是免費的活動，也會一直辦下去。:) 回顧看板文章列表，從剛來這裡，在這裡一起討論，已經走過五個多年頭， 現在也是專職的前端工作者，專責 JS / Java 互動網頁框架設計(ZK)， 很高興看到 JS 這幾年來的越來越盛行， 但也很憂心對 JS developer 而言，知識仍然很難累積。 希望從這樣的小聚會開始拋磚引玉。 當然，希望有興趣參加的人至少知道什麼是 html， 知道 html 有哪些 tag (div/span...etc)， 知道 css 的一些屬性，像是 display/position/top/left 啦。 因為這些都是 JS 操作的好夥伴， 如果是完全沒聽過 html跟css 是什麼的人， 那我們可能討論會有一些辛苦這樣， 但是你不必是個 css 或 html 強者，只要「知道」就好！ 所有與 javascript 有關的東西，不限語言，都歡迎來一起討論， 有會的我們可以交流，有不會的我們可以分享跟一起學習。:) -- 網頁上拉近距離的幫手 實現 GMail豐富應用的功臣 數也數不清的友善使用者體驗 這就是ｊａｖａｓｃｒｉｐｔ 歡迎同好到 AJAX 板一同討論。 -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 111.81.139.152 ※ TonyQ:轉錄至看板 Web_Design 02/07 00:15
