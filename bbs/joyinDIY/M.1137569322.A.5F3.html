: 感謝分享 : 但因為我未嘗試過DIY 所以看得不是很懂 : 以下是根據上文所想像的執行步驟 請幫忙檢查 謝謝 : 一.先取出玫瑰花泡製 : Q1:請問是用冷水 溫水 還是熱水呢 熱水 : Q2:哪種水泡製都可以嗎 需要用蒸餾水或是海洋深層水嗎...我不相信台灣的自來水 原則上是蒸餾水，我沒試過海洋深層水，你可以直接用蒸餾水煮玫瑰花 （記得要加蓋子，不然玫瑰的香氣跟精華會跑掉） : Q3:需要泡多久的時間呢 在加蓋狀態下自然放到涼 : 二.以蜂蜜加伏特加代替甘油 : Q1:蜂蜜與伏特加的比例大約是多少呢 大概是一比一吧 : Q2:敏趕型肌膚需要先讓伏特加的酒精揮發嗎 敏感型肌膚不加酒精類的比較好吧！ 加伏特加是為了收斂跟防酸敗，溫和的抗菌劑也可以防腐敗 或是只加蜂蜜或甘油，可是冰在冰箱裡 : Q3:若無法使用酒精 除甘油外 還有其他替代品嗎 兩個的功用不同唷！酒精的功用如上述伏特加 甘油是為了保濕，蜂蜜，甜菜精華也可以代替 : 三.玫瑰花水:蜂蜜加伏特加 = 9:1 我大概是90cc：10cc : Q1:搖一搖就可以了嗎 對呀！乾淨玻璃棒攪一攪也行吧 : 接下來是我瑣碎的問題 : 1.保存期限大約多久呢 看天氣，冬天的話一個月沒問題，花瓣渣渣濾乾淨也會保存較久（個人心得） : 2.需要用玻璃瓶裝嗎 乾淨瓶子都可以吧！不過玻璃瓶比較有質感 瓶子最好用熱水洗淨烘乾，噴點藥用酒精消毒，等全乾後再用 : 3.我喜歡涼涼的感覺 可以加點精油嗎 精油不溶於水，要用前要記得把化妝水搖一搖，不然會有灼傷的可能 : 對不起 問了一大堆問題 做好的成品記得要先試在手腕內側看會不會過敏唷！ 畢竟不是每一種作法都適合每種皮膚 如果顏色味道怪怪的了就倒掉重做吧！別拿自己皮膚開玩笑 你可以多參考一些書，牛爾的第一本DIY書籍就有教做玫瑰花跟蜂蜜化妝水 培根的（家傳美容玅方）我也很喜歡，多參考一些書籍 以後就可以自己嘗試新配方了 -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 163.23.142.253
