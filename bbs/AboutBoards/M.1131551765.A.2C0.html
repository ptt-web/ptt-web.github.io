標題請求群組長檢具其對基督信仰認知之客觀證據
時間Thu Nov 10 00:06:55 2005
本人請求群組長提出組務人員對基督信仰認知 足以判定立場偏激或不實污衊與否的客觀證據 若組務人員認為本人對板友文章給予劣文數過多 請先檢具自身對基督信仰之認知，具有足夠判定能力的證據 若組務人員對基督信仰之認知，不足以判定是否符合基督信仰要旨 本人對於群組長於本次事件的任何判定，皆視為不當干涉板面自治 將上訴至PttLaw ※ 引述《zoea (quit...)》之銘言： : 以下為刪除文章之參考原則 : Ⅰ.蓄意持偏激或歧視立場，或不實污衊 : Ⅱ.經板主刪除且公開於板面勸告後，仍重覆張貼 : Ⅲ.未註明轉載來源，或未經原作者同意而逕自轉載 : Ⅳ.無關基督信仰或其信徒聚會的廣告 : 本板板主於板面刪除之文章，暫收錄於「待刪區」至少一個月 : 若逾期未於板面公開表達異議，視為該文作者同意板主對該文之處理 : 被刪除文章 : 如被判定符合以上刪除文章之參考原則Ⅰ或Ⅱ項，不僅該文被刪除，並予以劣文處分 : ----- : 上列為Christianity板版規 : 申訴人於本版申訴文章經與版主各述理由 : 本組認以達可審酌階段 : 判決如下 : 主文: : 版主處置並無不當且認符合該版基本規定 : 惟劣文發給過當 發給四篇劣文請於BM板申請消除兩篇 : 理由: : 版主所持刪除文章理由 雖有致該版討論範圍限縮之可能 : 然各版版主對於該管看板本就有使其導至某一討論方向之情況 : 對於所持信念自難與所有使用者同一 : 該版版主對於其所認定不適當文章 : 首次給予刪除文章後 使用者應對於版主之管理方向有一定體認 : 該種類型文章就版主認定為違規文章應屬明顯 : 惟申訴人於首次刪除文章後 繼而重複發表類似文章 : 版規規定 : Ⅱ.經板主刪除且公開於板面勸告後，仍重覆張貼 : 組務觀看該版似未發現有公開勸告等情事 : 以據此項給予劣文四篇應屬過當 : 即使僅以 : Ⅰ.蓄意持偏激或歧視立場，或不實污衊 : 而逕行發給劣文 : 就版主應有基本照料義務下 自應於發給一至二篇劣文後於該版公告說明 : 單就本項劣文發給而論 組務認為仍有過當 : 煩請該版版主盡速於BM版聲請取消兩篇劣文 : 本件判決如上 : 若兩造對本件判決不服 請逕向國家研究院群組[aboutboards版亦即本版]上訴 : 小組申訴討論到此為止 若有其他意見或建議 請另開標題 : 人文宗教研究院小組長 zoea 94.11.9 -- 不是因為自己一個人 而是── 因為妳不在身邊才感到寂寞 -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 59.112.43.246 ※ 編輯: rehoboth 來自: 59.112.43.246 (11/10 00:17) ※ 編輯: rehoboth 來自: 59.112.43.246 (11/10 00:24)
