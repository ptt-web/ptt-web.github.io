※ 引述《chaoi (高興的蹦蹦跳)》之銘言： : 我個人覺得我們在討論這個話題的時候都不夠客觀 太過衝動 : 變成今天互罵的 這個局面 : 船沒有在往前滑 反而在原地停留 或倒退 : 大家都沒辦法接受對方的見解 : 我了解版主會不服他的版被廢 畢竟這是他付出很多心血弄出來的版 : 但回話得時候還是要小心一點 不要造成互罵的混亂 這裡是你對他處理板務的建議, 我就不回了. : 1. 標準不夠清楚 : 對於如何的板會被廢的標準沒定出來 光靠幾個人投票 是否有個人情續　喜好因素 : 而希望某版被廢　也很難說 : 沒有一定的標準　不夠公平 : 不能說讓一個人努力了那麼久了以後　隨隨便便投個票　就說　"sorry 你要被廢版了" : "你的版沒被通過.." : 這對版主 與該版的版友都是一種不尊重的態度 標準很清楚吧? 至於個人情緒 -- 你要怎麼排除? : 2. 投票的人不夠多 : 當我看投票票數時 覺得這樣的投票方式跟本不夠客觀 : 當然 投票人要有所限制 對歸舉不夠清楚的人還是不能有投票權 : 但當只有少數投票時 我覺得abstain 應該不能被準許 不然對該版的看法還是不夠客觀 小組內大約有六十人以上有投票權, 他們沒投票也是一種表示 -- 你可以考慮選舉 -- 你是不是有聽過 "唉, 候選人都好爛, 我不去投了啦!" 這類話呢? 雖然我個人傾向於把這個當成一種義務, 強迫性要求板主每三次投票至少要投一次, 畢竟大家的板最初也是靠著投票進來的, 要求板主投票並不過份. 但是, 我不是小組長, 這個權力在小組長手中. : 3.通知廢版的形式 : 我各人覺得這樣通知廢版的形式是蠻不正式且不尊重開版人的 : 我所只的形式是讓一個版試閱 投票後 就直接將沒通過的版廢掉的方式 : 沒有該版主一個警覺 就跟版主說sorry~ 不合格 但為什麼不合格呢? 有幾個人有解釋? : 而且我覺得若該版有意見 應當在 試閱其間就提出不是嗎? 有提出啊. 回去看看之前的文章. : 而不是廢版後來批評該版的缺點 對初次開版的人不太公平 : 我是覺得這個投票的制度應該有所改善 這樣受到傷害的人會比較少 : 1.我覺得入選的模式 和 標準應該要透明化 : 要有一個標準 這樣我們才知道沒有人是因為個人喜好來投票的 你可以制定看看, 我相信你會馬上發現困難度. : 2. 投反對票者應當給該版版主一個解釋 當該版版主要求時 這我反對. 為什麼? 因為我是這次投票, 對我的反對票解釋最多的人, 結果不但有人跑來我的板來鬧, 還有人把 Ancient 板當作罵我的地方 -- 板主也沒處理 -- 甚至, 連板主的同學都放話要來台北揍我. 若是以前我會天真的說, 好啊, 應該解釋的. 但是現在我覺得沒必要, 因為並不是每個板主都有那個雅量聽解釋. : 3. 對於該版的尊重應該要增加 若閱版者對該版有不滿 不以為然的地方應該要在試閱其 : 間發表感言 而不是事後來嘲諷 批叛 對版主 與版友對版的負出都是一種冒犯 有啊 -- 明明就是試閱期間吧, 並不是事後, 你的 timeline 搞錯了. : 4. 不要在發表感言的時候做人身攻擊 那如果板主事後利用他的板做人身攻擊, 而且還拒絕處理呢? : 這是我目前能想的出來的感想 希望它夠客觀 : 如果有意見的話請提出來討論看看 -- ╭─────────────────────────╮ │"To spell out the obvious is often to call it in │ │question." -- Eric Hoffer │ ╰───────────◇ http://dale.rileks.com ◇╯ -- ※ 發信站: 批踢踢實業坊(ptt.csie.ntu.edu.tw) ◆ From: 218.166.74.198
