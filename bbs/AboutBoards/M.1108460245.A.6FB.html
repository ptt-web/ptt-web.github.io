※ 引述《dale (冰城城主)》之銘言： : 審核制、試閱制也沒辦法針對板主的壓力提出解決辦法， : 頂多只是把小組長抓來承擔原本屬於板主的壓力（好可憐）。 : 所以在這裡建議新的制度（在此感謝 HIbaby 那篇給我的靈感）。 : 首先，取消連署機，AboutNew 改成正常看板。 : 然後無論是連署新板、連署板主、罷免板主、申請廢板…… : 一、使用者於 AboutNew 提出申請（格式可以考慮 HIbaby 那篇，但字數不用那麼多）。 : 二、討論（主要讓小組內現任板主和申請人有機會交流），為期一週。 : 三、小組內現任板主與小組長投票。 : 這個制度的好處是什麼？ 這個制度的壞處遠大於好處 : 一、不會發生多數暴力，批踢踢站民藍綠比例根本沒關係。「大不大聲」 : 是看有沒有道理，而不是看有多少個 IP。就算一大票人跟你持相反意見， : 就算對方烙一堆同學來投票，只要你的思維夠清楚，立場站得住腳， : 就有可能會通過。 這使得「少數暴力」更容易達成 現任板主的負荷將會更重 投票意願會更低 每個板主都要在管理自己板板務之餘，思考別的板板務 才能夠使「連署新板、連署板主、罷免板主、申請廢板」這些事項進行 這使每個板主的工作量更大 板主的權力也更大 有心人士聯手勾結的可能也更高 「人數」、「知識」、「經驗」這三者 如果能在投票中用加權的方式處理，可能是比較好的參考 : 二、使用者意見不用被迫陷入二分法思維。有時候想表達的意見明明就不是支持或反對， : 也許支持某些東西但整體上不支持，也許只是沒辦法信任申請人，也許意見沒辦法 : 在一行內表達完整……這些問題在新制度都能夠解決。這個世界上有太多只看連署 : 就硬把人分類的笨蛋，也有太多連署過不了就記恨到其它連署的混蛋， : 針對這些問題，新制度也都比連署機能夠解決問題。 這種事修改投票辦法就好了 不一定要「廢除」連署機 你的思維也是「二分法」（保存或廢除，只有這兩個。） : 三、那些人邏輯不清楚，那些人說話有條有理──時間久了都可以看出來。 : 時間可以很自然地淘汰小白。 一週的時間並不見得夠 哪些人邏輯不清楚，哪些人說話有條有理，時間久都可以看出來？ 那為什麼U黨可以弄成那樣？ : 四、能夠更準確地表達「民意」。目前對於「拉人來連署」這個問題 : 只能靠著「拉更多人來連署」來解決。在新制度下，要表達民意， : 只要在一週內把自己的想法寫出來就行了。 把自己的看法寫出來，也與「廢除連署機」沒有相關 -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 59.112.35.157
