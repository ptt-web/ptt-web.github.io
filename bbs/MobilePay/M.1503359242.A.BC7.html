標題[新聞] 新聞分析－共通QR Code 創三贏局面
時間Tue Aug 22 07:47:19 2017
「工商時報」 連結：http://www.chinatimes.com/newspapers/20170822000122-260205 隨著共通QR Code問世，已可預見，除了先前一度碰到瓶頸的國內行動支付可望加速普及 ，不論是對銀行、消費者、或是店家，都將發揮不同的助益，締造「三贏」局面。 從店家角度來看，其實不少專業人士已發現台灣行動支付推展上的最大問題之一，在於願 意受理的特店家數太少，包括迄今感應式收單機僅大約15萬台左右，這使得即使人手一隻 行動電話隨時可用行動支付，包括Apple Pay、Samsung Pay等國際支付也進軍台灣，但由 於受理店家有限，使消費者「英雄無用戶之地」。 共通QR Code可望改善這個現象，尤其是對於毛利低的大型量販通路及小本經營的網購個 體戶，將能大為提升意願，關鍵在受理行動支付的成本大幅降低。 現在很多商店受限於成本，連信用卡刷卡都僅限於受理所謂的「自行刷卡」，即僅能受理 由收單銀行所發行的信用卡刷卡，原因在於光是店家就得繳付銀行2％的手續費成本，因 此唯一途徑只能藉由自行刷卡這種「左手轉右手」的方式來降低成本，但如此一來，消費 者將非常不便。 但跨行共通QR Code的推出可改變這個局面，透過HCE金融卡，或行動銀行用戶用QR Code 的掃碼來進行無現金支付，比起一般信用卡的刷卡支付，手續費將僅為一般信用卡刷卡支 付的1/4，由於成本大幅縮減的效果顯著，業界也因而樂觀預期未來相關特約店的擴張速 度將遠甚於感應式收單機。 另一方面，用QR Code突破行動支付的瓶頸，連六大國際發卡組織都已採取行動，之所以 在7月宣布將推出統一規格的信用卡QR Code，就是因為發現了感應式收單機在行動支付市 場的使用瓶頸，特別是在發展中國家，信用卡的行動支付也因而無法真正深入到所謂的「 庶民經濟」，因此才擬透過QR Code突破僵局。 對銀行而言，讓行動銀行的用戶，直接以QR Code掃碼來完成轉帳、繳費等功能，將有助 於擴張活存的資金，使資金成本更低；未來共通QR Code的問世，將讓行動支付的風貌出 現更大的轉變，甚有可能顛覆在行動支付市場感應式支付的現有優勢，此消彼長下，也將 促成各家業者的版塊再度大洗牌。 -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 114.136.48.28 ※ 文章網址: https://www.ptt.cc/bbs/MobilePay/M.1503359242.A.BC7.html
