$a=1; $b=3; $result=$a+$b; echo $result."<br>"; echo "1+3=$result<br>"; echo "這例子是讓大家知道 $a+$b=$result<br>"; ## 以上三個 echo 是讓大家知道變數的運用，以及有關加號的運用 ## 第一個echo 會顯示 4 ## 第二個echo 會顯示 1+3=4 ## 第三個echo 會顯示 這例子是讓大家知道 1+3=4 ?> ------------------------------------------------------------------------------------------------------ <?php // 以變數來舉出運用 - 號的例子 // 5-3=? $a=5; $b=3; $result=$a-$b; echo $result."<br>"; echo "<font color=red>5-3=$result</font><br>"; echo "這例子是讓大家知道 $a-$b=$result<br>"; ## 以上三個 echo 是讓大家知道變數的運用，以及有關加號的運用 ## 第一個echo 會顯示 2 ## 第二個echo 會顯示 5-3=2 (此行會是顯示紅字，因為在""內有崁入 html 碼) ## 第三個echo 會顯示 這例子是讓大家知道 5-3=2 ?> ------------------------------------------------------------------------------------------------------ <?php // 以變數來舉出運用 +-*/ 四個符號的例子 // 10*20/2+30-7=? $a=10; $b=20; $c=2; $d=30; $e=7; $result=$a*$b/$c+$d-$e; echo $result."<br>"; echo "<font size=5>10*20/2+30-7=$result</font><br>"; echo "這例子是讓大家知道 $a*$b/$c+$d-$e=$result<br>"; ## 以上三個 echo 是讓大家知道變數的運用，以及有關加號的運用 ## 第一個echo 會顯示123 ## 第二個echo 會顯示10*20/2+30-7=123 (此行會是顯示size為5 的字，因為在""內有 ## 崁入 html 碼) ## 第三個echo 會顯示 這例子是讓大家知道 10*20/2+30-7=123 ?> ------------------------------------------------------------------------------------------------------ % 取餘數 簡單解釋為 若是 $a=10; $b=4; $result=$a%$b; echo $result; // 顯示為 2 ，因為 10除以4，商數2，餘數為2 ------------------------------------------------------------------------------------------------------ ++ 累加 -- 遞減 這二者則是通常運用在迴圈，簡單以 for 下迴圈讓大家了解 先簡單介紹迴圈 Loop (for,while)，後面會為大家簡單介紹 while <?php # for 在()分別需要三個條件 (a1;a2;a3) # a1 為 變數($i) 的開始條件 (以$i 為例) # a2 為 變數($i) 的終點條件 # a3 為 變數($i) 的附加條件 # 附加條件如 $i++ or $i-- # $i++ = $i+1 # $i-- = $i-1 // 輸出結果如下： // Num0 // Num1 // Num2 // Num3 // Num4 for($i=0;$i<5;$i++){ echo "Num".$i."<br>"; } ## 簡單解釋上面迴圈 ## $i=0 則是將0 作為開始條件 ## $i<5 則是數字不得跑超過5，也就是最大結果為4 ## $i++ 則是他的附加條件，為每跑一個迴圈，數字加一，也就是由 0 開始跑會 ## 跑到 4 為止 ?> 小練習題： 利用 for 寫出下方所示的圖形 * * * * * * * * * * * * * * * * * * * * * * * * * * * * -- 討論是你我教學相長的機會 UO-小築 http://uniorg.net/ 讓我們一同喜愛這討論空間 喜歡他的話 別忘了推薦給大家... -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 203.73.23.169
