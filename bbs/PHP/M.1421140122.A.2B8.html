※ 引述《kingoface (Vince)》之銘言： : 嗨,大家好 : 請問PHP可以用條碼機來輸出貼紙嗎? : 比如我建立了一個 料號 : 然後系統產生了一個barcode : 接下來想把他利用條碼機印出來. : 假設我去買了一台條碼機,驅動都灌好的情況下. : 是直接右鍵點列印,就會出來嗎? : 還是我要去弄個API接條碼機之類的? 以下是我的方法,不是很高明方法,但你可以參考一下 我是使用bartender排版(舊版7.X),因為紙張尺寸小張容易對不準 而且用網頁的話,應該容易因為不同瀏覽器,不同版本,不同設定,而造成排版差異 首先就是將要列印的參數資料塞進伺服器的TXT暫存檔,再將區網共享此檔(塞進Client端 我不知道怎麼寫) Client端將共享的資料夾設成網路磁碟機 再來你會遇到一個問題就是網頁如何啟動應用程式? 將以下登錄檔匯入就能辦到了(在網路上找到的,忘記在哪個網頁了) =========================== Windows Registry Editor Version 5.00 [HKEY_CLASSES_ROOT\GT] @="URL:GT Protocol" # 註冊 protocol "GT" "URL Protocol"="" [HKEY_CLASSES_ROOT\GT\DefaultIcon] @="C:\\bartender\\bartender.exe C:\\bartender\\11DRIVE.btw /p /x" # 是 GT.exe 的路徑 [HKEY_CLASSES_ROOT\GT\shell] @="" [HKEY_CLASSES_ROOT\GT\shell\open] @="" [HKEY_CLASSES_ROOT\GT\shell\open\command] @="C:\\bartender\\bartender.exe C:\\bartender\\11DRIVE.btw /p /x" # 這裡會打開 shell 執行命令，"%1" 可以將URL本身傳為 GT.exe 的外部參數 =========================== C:\\bartender是bartender路徑位子 11DRIVE.btw是bartender排版樣版 可設定紙張大小,印表機,將資料來源指定網路磁碟那個檔案 塞資料的那個PHP檔最後請加上轉址 echo "<script>location.href='GT://WhateverYouWriteThisContent'</script>"; WhateverYouWriteThisContent記得隨便打就行了,主要是gt字眼 當然如果大家有更高明方法 希望大家互相切磋一下 -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 118.163.203.130 ※ 文章網址: https://www.ptt.cc/bbs/PHP/M.1421140122.A.2B8.html
