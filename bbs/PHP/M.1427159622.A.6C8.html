"; } ....略 $q = $pdo->prepare($SQL); for($i=0,$max_i= COUNT($m_SERACH_CODE); $i < $max_i ; $i++) { $q->bindParam(($i+1), "%{$m_SERACH_CODE[$i]}%"); } $q->execute(); ... 大概是這樣吧 :) -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 140.134.48.253 ※ 文章網址: https://www.ptt.cc/bbs/PHP/M.1427159622.A.6C8.html
