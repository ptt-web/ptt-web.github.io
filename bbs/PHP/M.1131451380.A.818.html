※ 引述《xamous (一天死去一點)》之銘言： : 我有一個樹狀的資料儲存在資料庫裡，結構大概是這樣 : id, title, parent : root 的 parent 是 0，id 是 1 你可以將它簡化為 parent 0, id 0, 就是自己當自己的 parent, 另外, 0 也是唯一會被當成 false 的數字 (應該啦), 所以這能夠讓程式碼也更加簡潔 : 1 root 0 : 2 同學 1 : 3 朋友 1 : 4 大學 2 : 5 高中 2 : 我希望他在網頁上能夠顯示成 : root : * 同學 : * 大學 : * 高中 : * 朋友 : 可是就是想不到有什麼簡單有效的方法來處理這樣的資料，請問各位前輩有 我最近做過類似的東西, 提供給你參考. 不過只是跑跑遞迴或迴圈而已. 記得要將 id 設為 primary, parent 設 index, 並且這個資料表不要加入非必要欄位 (喔, 除了這三個大概都算非必要了) 1. 建立 $tree = array() ; 取得 root_id (0 or 1) 並取得所有資料至 php 變數中 (例如 $buff) (如果你的資料量很大, 那麼就要以 DB query 次數來當作 trade off, 不過原理是相同的) 對 $buff 這個變數來說, 也許它會長這樣 (也以此為例) $buff[0]['id'] = 1 $buff[0]['title'] = "root" $buff[0]['parent'] = 0 而我希望 $tree 會長成這樣: $tree = array(....) $tree[1]['title'] = "root" $tree[1]['type'] = "node" // 你很可能在枝下又生葉 $tree[1][2] = array(....) $tree[1][2]['title'] = "同學" $tree[1][2]['type'] = "node" 你能夠再使用一個陣列, 包含了 $tree 內的元素之 reference, 以利資料的操作. $myindex = array() 2. 接著開始對 $buff 跑無窮迴圈, 直到 "某一圈跑完都無法丟出新資料" 為止. 迴圈開頭要先 reset ($buff), 之後使用 foreach 去裡頭讀資料 對於 $buff 內的每個元素 $item (這時候的 key 不重要. 那只是被 select 出 的順序而已. 把它叫出來只是刪除方便), 我會先在 $myindex 建立其 reference, 再將其 ref 到 $tree 內該有的位置, 最後將原資料 unlink, 並 break foreach 迴圈, continue 外部 while 迴圈. 大致程式碼如下: $buff ; // you should prepare this. $myindex = array() ; $tree = array() ; // building root reset($buff) ; foreach($buff as $key=>$item) if ($item['id'] == 1){ $myindex[1] = &$item ; $tree = &$item ; unset($buff[$key]) ; } do { $noNew = true ; reset($buff) ; foreach($buff as $key => $item) if (isset($myindex[$item['parent']])){ $myindex[$item['id']] = &$item ; $myindex[$item['parent']][$item['id']] = &$item ; unset($buff[$key]) ; $noNew = false ; break ; } } loop while (!$noNew) ; // ending the main while loop 以上碼未測試, 不過除了打錯字外應該可行. -- 鬼壓床怎麼辦 騎上去啊 -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 140.112.217.134
