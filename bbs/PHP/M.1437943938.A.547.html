※ 引述《SophiaH (Sophia)》之銘言： : 因用圖書館電腦練習 PHP 都得先安裝 XAMPP挺麻煩, : 想試看看用 IIS 環境是否也可練 PHP : 有先爬前文, 在 IIS 跑簡單的 echo 應該 ok : 參照 http://tinyurl.com/lu3l4nz 設定 IIS 的目錄.. : 設定完後將範例檔存在 Website1 或 Website2 都無效 : 想請問如同用 XAMPP, 其 php.ini 檔預設的網頁路徑是 C:/xampp/htcdoc : 不知 IIS 環境預設的網頁路徑 該到哪個設定檔查詢 ? : 謝謝 挖靠你用XP設定阿.... 小魯用的是WIN7的IIS來架設PHP的(我想在這版用這架站軟體會被噓翻orz) 好了那我來說如何用IIS 7來架設PHP吧，畢竟有人發問了(但原PO好像用的是IIS 6?) 首先先到微軟官網下載 WEB PI https://goo.gl/7pRdP8 像這樣 http://i.imgur.com/6iGdRBq.png (SQL隨人喜好安裝拉，畢竟小魯才剛學PHP連
SQL語言都不會) 使用完之後他會幫你把IIS的檔案關聯之類的都設定好 http://i.imgur.com/IfdaJSE.png
在剛剛的 PHP manager 中他會告訴你IIS有哪邊沒有調校好，再把他勾選一下就調整好了 而且還可以 check PHPinfo(確認PHP是否可正常執行) 或是你可以新增一個新文件檔，內容打上 <?php phpinfo(); ?> 然後儲存，之後再把檔名改成index.php(IIS預設為index.xxx) 其實在IIS中有個 Default Web Site，只要在你的首頁中打上 http://localhost 就能執行； 可以先把PHP往這裡塞(預設為：C:\inetpub\wwwroot) 然後再把 Default Web Site 的預設文件改成 index.php http://i.imgur.com/87G09XT.png
記得將PHP的優先權開到最高 http://i.imgur.com/JyJGWkM.png
