時間Tue Feb 21 12:16:17 2006
2006年好用IDE推薦活動 名稱 ：Crimson Editor 作業系統：Win 32 價格 ：Freeware 優點 ：這套是我進公司寫php就一直用到現在的軟體 他支援各種程式的語法顏色 像我常常 php html javascript 混在一起寫 他都能正確的顯示顏色 而且函式 變數 字串 的顏色都分的很清楚 他還完整支援 UTF-8 的編輯 而且可選含不含 BOM 還可加 入右建選單內開啟檔案 IE 原始碼用他來檢視 也有一個代替記事本的功能 自動排版 巨集功能 檔案內的 Tab 跟 Space 互轉 消除程式碼尾端空白 滑鼠拖曳編輯 分割視 窗 檔案總管選單 整個目錄含子目錄文件內容搜尋...etc 功能一大堆 以一個小巧又免費的軟體來說 這樣的功能真的很多又很夠用了 缺點 ：檔案總管無法顯示網路磁碟名稱 一律叫 Network Drive 編輯中文字有時會 出現亂碼 或游標消失(不常發生) 註解掉該行有含 ( 或 { 的程式碼 無法消除判別 還 是會跟下一個做標示對應 單行顯示最大 1024 個字 超過要用鍵盤移過去 無法拉捲軸 無查詢跟自動產生程式碼功能 官網一直沒升級 不支援 php5 語法變色 基本上用起來很順手 該有的功能都有了 一直期望他改版改掉一些Bug 但用起來真的不錯用 又免費 最重要是能正確編輯 UTF-8 的文件 目前只有官方英文版 但我有一次不小心抓到有人中文化的版本 (忘記在哪抓的了..) 如果有需要中文化版本的人在跟我要囉 官方網站: http://www.crimsoneditor.com/ -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 203.75.99.18
