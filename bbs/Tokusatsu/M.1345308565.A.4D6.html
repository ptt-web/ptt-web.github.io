抱歉改了標題 先講54年的水中氧氣破壞劑(Oxygen-Destroyer)好了 它是哥吉拉作品中，唯一能完全殺死哥吉拉的超兵器 是芹澤大介博士在研究氧氣過程中發現的 一般設定資料都是寫，這種特殊物質能將水中的氧氣一瞬間完全破壞 附近的生物會先窒息死亡，隨後被溶解液化 說真的，搞不懂裡頭機制，有興趣更深入研究或許可參考空想科學讀本2的p.76 接著講95年的微氧(Micro-oxygen) 這個就比較好懂些，它是微小化的氧氣分子 用它來養金魚能變超大隻，可促進生物成長 又因為其體積很小，可侵入分子與分子的縫隙間破壞其結合狀態 以其分子結合破壞力粉碎該物體，畫面看起來便是該物體被溶解 是由伊集院研作博士發明，有設定說是為了修復臭氧層而研發 在溫度降到零下183.2度時會液化，失去作用 然後再講世紀終結者(電影上映時譯名，錄影帶不知為啥翻成核能大恐龍) 1954年芹澤博士在東京灣使用了水中氧氣破壞劑 該處化為無氧狀態，使得原本沉眠於海底地層，生活在無氧時代的遠古微生物復活 微生物在海底隧道工程開挖地層時接觸到充滿氧氣的空氣，為了適應急速進化 (該微生物擁有補食其他生物攝取其遺傳情報以讓本身進化的能力) 變成Crawl體，進化出能吸取二氧化碳並轉化成微氧分子的肺 並以微氧溶解了工程機具、燒瓶及水族館的魚隻 之後再進一步進化成兩公尺大的幼體，在臨海副都心與特殊部隊發生激戰 由於受到火燄放射槍攻擊影響又急速成長為18公尺大小(不過還是叫幼體) 伊集院博士發現該生物是以微氧為能量動力，要求自衛隊以冷凍兵器攻擊 幼體們雖有受創，但此威脅反而讓牠們更進化，合體為全長60M，高40M的集合體 能製造更多的微氧分子，更多的能量供給也讓其戰鬥力大增 更進化出將體內微氧分子經由體內加速管轉化為氧氣破壞劑從口器噴射出的力量 所以沒錯，微氧可以算是氧氣破壞劑的未完成型! 片中參與了聯合國G對策總部的山根健吉曾請求伊集院博士 以微氧為基礎進一步開發出氧氣破壞劑來消滅將面臨體內核爆發的哥吉拉 不過分析過東京灣土壤，驚覺於氧氣破壞劑恐怖威力的伊集院拒絕了 話再說回世紀終結者，集合體擁有了製造出氧氣破壞劑的能力， 並因其所擁有的多樣他種生物遺傳情報，還能變身成飛翔體。 被哥吉拉Jr.擊落在品川火力發電廠而吸收了大量能源的飛翔體因吸收了Jr.的DNA 進化成兩足步行形態，全高120M的完全體。 體內有兩組製造微氧分子的肺，還能將大量的微氧分子壓縮成更精純的能量儲存， 此高能量化的微氧分子，能經由更加巨大化的加速管化為氧氣破壞劑射線從口中射出 其微氧壓縮器的位置在完全體胸爪之下，有時候胸爪會打開釋放過多的壓力，畫面有帶到 話說原案中有讓那打開的胸爪發射光線的設定圖。 完全體的頭角可放出微氧分子切斬敵人，還能分離成好幾隻集合體群起攻擊哥吉拉 加上氧氣破壞劑射線，真的是哥吉拉宿命中的最後大敵 不過哥吉拉最後爆氣使出的HYPER熱線還是重創了完全體 完全體逃至天空時又被自衛隊冷凍兵器集中攻擊 電影版的話牠是掛在自衛隊手下 不過DVD影像特典中，牠掉下來後又被哥吉拉補了好幾刀才死 (很久以前有看過，目前網路上找不太到....，之前也沒抓下來的樣子) 期待這隻的SHM模型 最後補個恐龍帝國當時台灣上映的廣告影片 http://www.nicovideo.jp/watch/sm16461220 印象中報紙的廣告把哥吉拉的熱線講做超光速核武金光，好帥的感覺 ※ 引述《kuja (庫賈)》之銘言： : 恐龍帝國的經典性 : 我就不贅言了 : 只是還是會感嘆 : 61年芹澤博士認為殺死所有生物的武器 : 竟然會誕生毀滅性生物 : 95年微小氧氣素開發時 : 山根小姐提出 : 兩者很相像 : 雖然我不是理工出生 : 但這讓我想到無氧酵素和微小氧氣的關連 : 會不會某種原理相同呢 : 或者無氧酵素其實是微小氧氣的應用版 : 因為 : 無氧酵素有提到 核能 的概念 : 就像核子分裂一樣 : 如果縮小的氧原子衝撞破壞水中氧氣 : 依照氧原子的活性 : 的確可以造就超級大的破壞 : 哥吉拉屍骨無存 : 難怪芹澤博士只敢在水中使用阿 : 畢竟海水的氧氣真的很少 : 但我想他也想不到 : 或者昭和時期的科學沒有探究到 : 不用氧氣生存的生物 : 結果反而取得使用氧的能力 : 雖然說世紀終結者有點抄襲異形 : 但她還是獨樹一格的怪物 : 氧的活性讓她接近不死的狀態 : 可以快速繁殖 合體 進化 : 噴出的氧氣可以持續性的爆炸 : 當時哥吉拉已經是紅蓮狀態 : 紅色火燄可以輕易打倒機械二代 太空哥吉拉 : 但想不到這些高溫都是助長氧氣的幫兇 : 完完全全克制紅蓮 : 若非紅蓮快溶解時噴發最強的火燄 : 加上人類的冷凍武器 : 這個世紀終結者真的就是世紀終結了 -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 114.39.102.10
