※ 引述《sky220 (天空之城)》之銘言： : 看戰隊系列也有一段時間了 : 從太陽戰隊開始就有在接觸(不過因為那時還小~所以總是斷斷續續) : 中間有幾部沒機會看 : 真正有看完要從百獸開始 : 不知道大家心目中的最愛有哪些呢? : 最愛的機器人 : 第一名 五星戰隊 : 這部的造型是我非常喜愛的一部 : 我覺得同樣是用中國拳法~我愛這部比獸拳多多了 : 機器人具武士風格超帥 還有之後的第二機白色的獅子(我小時候還有玩具) : 超帥!! : 第二名 魔法戰隊 : 可能比較喜歡擬人型態的機器人 魔法戰隊的機器人我也很愛 : 兼具帥氣和奇幻的風格~感覺起來又很靈活~傳說合神就很棒 : 其他很多機器人看起來都很笨重 : 第三名 特搜戰隊 : 再用汽車飛機組合的機器人中我比較愛的一部 蘿蔔部份： 1.魔法戰隊 　 神話風格（美人魚、精靈、鳳凰等）的蘿蔔很有味道，魔法王合體方式也很特別 傳說合神也不錯，但魔法鐵神就不予置評XD 但這是近年來難得不玩蘿蔔手腳交換的一代 2.高速戰隊 高速第二的理由.... 超空合體和五連合體是我最早買的兩個戰隊蘿蔔:P 3.轟轟戰隊 轟轟發表到終極大冒險以前，給我又跟以前一樣只換手換腳的感覺 但終極大冒險登場時，覺得原來換手換腳也可以這樣玩啊:P 十合一的感覺也不會有過度堆積的感覺，相較於炎神戰隊.... : 最愛選角 : 第一名 轟轟戰隊 : 這是我最愛的一部選角 隊長實在是太帥 : 否則近幾年紅連者我都不愛(轟音我還差點因為這樣看不下去 = =) : 還是早期的紅連者比較帥氣 : 另外就是末永遙完完全全就是我喜歡的類型阿>///< : 而且其他的選角也不落俗套 : 我只能說這真的是我心目中最完美的選角 : 第二名 忍風戰隊 : 鷹介雖然也是走熱血路線~可是他的熱血讓我看起來很舒服 : 像魔法紅的熱血有時候就會讓我很囧" : 七海和芙拉畢裘很可愛 >///< : 另外天空忍者的設定很妙~(讓前幾任的人再出現扮演這點我還滿愛的) : 第三名 從缺 : 其他不是年代太久就是沒什麼特別的感覺了 選角的部份.... 1.轟轟戰隊 沒辦法，隊長有昭和的味道，而且是個陽光帥哥XD 加上聰慧又一整個是菜的的小遙和可愛的中村 轟轟銀和轟轟藍也選的不錯（轟轟黑那個小鬼就....），反派也有味道 壓倒勝XD 2.特搜戰隊 雖然不欣賞戴寧那個熱血呆瓜隊長， 但熟女石野阿姨跟超可愛的美香以及冷艷的木下都很優 特搜藍、特搜綠和小鐵選的倒是蠻不錯的 女性角色小輸轟轟的一代:P 2.超獸戰隊 嶋、西村、森，當年可是夢幻偶像組合:P 可以說是最早集俊男美女的戰隊作品吧 雖然後來加入超獸綠、黑有點破壞超獸的完美度 但當年帥氣的嶋和西村（西村老大現在還不錯，嶋老爹就XD） 還有急流勇退的森惠的身影還是常駐在我大腦中:P 3.救急戰隊 救急說起來沒什麼俊男美女，可是兄妹熱血的羈絆真的好看（比起魔法....） 惡役也是兄弟軍團，很特別。 只能說這代的人物設定的很成功，所以才幾乎沒有用第六人來增加戲份。 3.地球戰隊 也是一部兄弟戰隊（外加老師戰隊） 敵方用假媽媽對付他們那篇很催淚， 兄弟姐妹間團結的感覺也做的很好（魔法....好像只有冥府不錯，其他都XD） 很懷念的一部老戰隊:) : 最愛第六人 : 第一名 恐龍戰隊 : 龍連者雖然一開始是壞人~不過真的很帥!!! : 我超愛他吹笛叫龍凱薩出來的時候 音樂也很棒 : 之後燃燒生命蠟燭那段更是帥到爆炸!! : 第二名 百獸戰隊 : 閃耀的銀狼~ : 其實沒有什麼特別的理由~ : 因為整個百獸戰隊我只喜歡他 相較之下就很突出 : 看百獸都在期待他出場XD : 第三名 轟轟戰隊 : 其實我比較喜歡映士一開始的衣服 : 換了制服後反而變不帥了 XD : 一開始的服裝和禪杖的帥氣度百分百 第六人 1.恐龍戰隊 幾乎是共識了吧XD 2.忍風戰隊 忍風綠的本尊設定太特別了，說話也很有趣 3.未來戰隊 在未來火燄和未來紅之間鋪下伏筆，個人覺得蠻特別的 未來火燄之死也讓人有點小小哀傷:) : 最愛女性腳色 : 第一名 末永遙 (冒險粉紅) : 冷靜的頭腦加上會自己跑去吃甜食的可愛個姓 : 另外就是完全正中紅心的外型 : 除了她不做其他選擇 : 第二名 以後全部從缺!!!
