先說 漫畫版（石森版 不是spirits）跟電視版強調的不一樣 漫畫版強調的其實是面具(臉) 也就是改造人"是人非人"的矛盾 電視版比較不強調改造人的陰暗面 (還是有就是了) 為了代替"戴上面具"這個很難強而有力表現出來的描寫 (看過紅蓮女應該就知道吧XD) 因此電視版才會有漫畫版沒有的變身動作跟腰帶特寫 ※ 引述《zyuranger (獸連者)》之銘言： : 剛看了2756篇i大的推文 : 有了個疑問 : 到底假面騎士是怎麼變身的？ : 昭和騎士應該都是用腰帶變身的吧 : 雖然他們大多是改造人，沒變身也會有改造人的力量 : 不過要變身才有假面可以帶吧 : 除了亞馬遜外 : 1、2、V3、RIDERMAN、X、SUPER 1都是用風車腰帶 : 然後在經由特殊的變身動作來啟動腰帶的變身裝置變身 : 至於SKY跟ZX跟STRONGER就不是風車腰帶，不過還是有特殊的變身動作再來啟動腰帶 : 而光太郎也是需要用變身動作來啟動KING STONE，KINGSTONE變腰帶 : 所以腰帶是昭和騎士的變身來源吧？應該是吧@@ 其實sky也是風車 強人則是電氣腰帶 除了amazon是用腕輪外 其他9個騎士有6條風車腰帶 2條電氣腰帶 1條不知道幹嘛用的 (就是ZX 樣本太少) (剛剛忘了riderman其實是電氣腰帶 但是他的腰帶其實沒用) 風車腰帶在昭和騎士年代的確是變身的能量來源 機制大概是風車轉動→發電啟動變身裝置→變身 （強人則是摩擦雙手→通電→啟動腰帶→變身） 所以常常會有腰帶被封印 冷凍 破壞 或是像強人放完電後 耗盡能源無法變身的情境 而昭和騎士不變身 能力就有限 除了特殊能力無法發揮 (比方說V3 hopper或是sky的滑翔能力) 電視劇中常會有沒辦法變身 用肉身對敵時險狀百出的描寫 顯示變身後的戰鬥能力其實也高於平常 這跟初代漫畫的描寫有很大的出入 所以我認為漫畫跟電視版不應該混為一談 至於腰帶的特寫為什麼一定會存在 一開始只是為了帶出變身而已 （舊一號用機車變身時就有風車特寫 是因為為了跟漫畫區隔 電視版的設定是"用高速轉動風車發電變身" 所以風車的描寫是必要的 而這個設定也延續到後來 成為支持"風車＝變身能量來源"設定的重要設定） 後來二號的變身動作也確立以後 變身動作跟腰帶就變成假面騎士的兩個招牌 簡單的說 不管觀眾認不認識這個英雄 只要看到特殊的變身動作＋腰帶特寫就能知道這不是宇宙刑事 不是超人 是假面騎士 我覺得這算是連結昭和假面騎士系列的重要因素 所以騎士迷就算會有偏好 也比較少出現超人迷「7號之後的都不算超人」「太郎是幼兒看的」之類的激烈情結 （※這點勿戰 單純陳述 我認識好幾個喜歡超人跟7號的日本大叔就是這麼說的Orz） 因為都是騎士 沒什麼好戰的 同時 最重要的就是要賣那條"會亮 會叫"的騎士腰帶啦 XD : 不然每次變身都帶到腰帶的鏡頭該不會只是為了聲光效果吧.....囧 : 至於平成騎士 : 變身方式就變的很明顯了 : 除了KUGGA(跟光太郎一樣用石頭腰帶變身)跟AGITO(人家是神人，也還是要用動作 : 啟動腰帶) : 還有響鬼有鍛練過的身體再用音波去突變(這我喜歡) : 剩餘的很明顯，都是外掛......囧(劍應該算外掛吧，還沒看過) : 延續2756篇的主題 : 假面騎士對我來說 : 應該是個改造人，變身只是為了戴假面 : 而不是像戰隊一樣需要靠變身器來獲得超力量(戰隊幾乎都是外掛沒錯吧....) 假面騎士的電視跟漫畫版不一樣 需要腰帶變身 否則就無法發揮完全的力量 不知道為什麼他們就算是改造人 沒變身還是常常被打得抱頭鼠竄 Orz : 打完這篇小弟有個疑問，看完討論串還是不太知道 : 到底KIVA是外掛還是本身就有超能力@@ 我也不知道... 因為我還沒時間看 Q_Q -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 218.167.73.80
