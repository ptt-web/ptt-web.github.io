: 只有小花?還是小花、櫻井、姐姐？依照劇情來推斷分歧點應該只有小花一人，可是Kai : 在那天出絕招殺了櫻井跟姐姐(在這裡我理解為記憶抹殺)之後，未來才出現異魔神世界 : ，但未與現在世界相連，這會不會表示Kai只殺了一半，其實櫻井跟姐姐也是分歧點? 《電王》的世界裡，時間像是鐵道，每個特異點的存在就像是不同的車站，時間的運行 就像電車行駛；另外還有分歧點，通過分歧點後會有通往不同的時間。用台灣鐵路來 說的話，西部幹線南下列車通過竹南站後，就分為山線跟海線，一班列車只能選擇一 條鐵道行駛，但是即使這班列車走海線，山線的車站還是存在。 從主角良太郎存在的「現在」開始，未來至少有兩個車站： 1. 人類的未來，這個未來裡存在Hana這個特異點。 2. 異魔神的未來，這個未來的特異點是Kai。 影響這兩種未來走向的分歧點是野上愛理──愛理的小孩Hana是「人類未來」的特異 點。 而良太郎所存在的這條時間線上，未來是原本應該是通往「人類未來」。身為「異魔 神未來」的特異點Kai希望過去可以和他的未來銜接，所以來到了「現在」，意圖殺 了分岐點，將未來導向他的未來。用鐵路來說，就是原本這班車應該走海線，結果 山線的站不甘心，跑到竹南站去，希望找到分歧點好讓讓電車開到山線去。 但是由於櫻井和愛理故意誤導，使得Kai以為櫻井才是分岐點。 : 2.Kai那天的絕招天降殺 (誤)範圍有多大? : 若是範圍大到全球記憶抹殺，良太郎應該也消失了才對，那靠誰的記憶來修復世界?隔 : 天的良太郎?若是範圍僅限那片沙灘，也就是只有櫻井跟姐姐消失，那麼那天的姐姐對現 : 在的良太郎說只有靠你的記憶才能修復世界的話就破功了，因為其他人還活著，可以靠 : 其他人的記憶修復當天的世界。 全世界都毀滅吧。但是因為這是時間被破壞，所以那些維護時間運轉的人可以用「現 在」的特異點良太郎的記憶修復。假設良太郎記得10個人，那麼這10個人就可以被修 復，然後在用這10個人的記憶去修復他們記得的人，以此類推，直到整個世界被修復。 : 3.特異點殺不殺的死? : 爬之前Momogoter大的文寫道，所謂的『特異點』就是，不會受到任何過去而影響，絕對 : 存在的人類。依照Kai跟良太郎的情況，特異點應該會永遠存在才是，小花也是特異點， : Kai怎麼殺也殺不死小花，特異點絕對存在，這樣異魔神世界永遠不能與現在世界連結 : 吧? 又因為小花是特異點，她為什麼不能像良太郎修復消失記憶的那一天一樣，用自己 : 的記憶修復她本來的未來世界? 因為被修復的世界中，良太郎刻意遺忘了愛理懷孕的事，被修復的愛理沒有懷孕， 導致Hana存在的未來和過去（也就是良太郎所存在的「現在」）的關連斷了，所以 Hana的世界在某一天突然消失。Hana是特異點，所以她的存在不受影響，但那個未來 因為失去了和過去的連繫，所以無法修復。 : 4.櫻井的存在? : 雖然說最後給予了姐姐希望，在未來一定會相遇，依照前面大大的討論，小花排除"人為 : 因素"以外，合理的解釋是姐姐晚十年碰到櫻井，晚十年生小花(= =)；但是這裡有BUG吧 : ，Kai被消滅之後世界重新連結，加上良太郎記憶修復消失的那一天，姐姐對櫻井的記憶 : 恢復，到這裡是沒問題的，但是櫻井用完最後一張卡片照理說包括姐姐在內所有人都會 : 忘記他才對，怎麼姐姐還記得?還在病床旁邊對完話才消失，良太郎記得櫻井是因為他是 : 特異點，可姐姐不是吧?
