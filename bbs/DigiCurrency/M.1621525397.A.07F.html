標題Re: [閒聊] 對這幾個加密貨幣的負面看法，怎麼駁斥？
時間Thu May 20 23:43:15 2021
要回答這個問題, 我們可以先思考一下黃金為何會有價值? 許多人會跟你說是因為"有共識", 所以才有價值, 其實這個說法是倒果為因了 它是因為有其內在價值, 所以才讓人們產生共識, 這樣也才能解釋為什麼在千年以前交通不發達的年代, 全世界都"不約而同"地對黃金的價值產生共識, 這絕不是巧合. 而黃金的核心價值就在於它物理上的穩定性, 你要產生新的黃金原子, 是需要有超新星爆炸等級的能量才能辦到, 也就是說黃金的生成或銷毀, 基本上是無法透過人為的方式控制的, 換句話說, 把黃金當成交易媒介/記帳系統時, 我們可以說這個系統的運作, 完全不需要仰賴對特定人/組織的信任, 它不會因為發行機構倒了就失去價值, 即使整個地球爆炸, 人類完全消失, 原本的黃金還是在那裡, 這就是它的價值. (很多人會說黃金是因為稀缺所以有價值, 這也是錯誤的說法, 稀缺但沒價值的東西太多了, 稀缺性只是讓本身有價值的東西變得供不應求, 價格自然就高了, 而不是因為稀缺所以產生價值, 或許有人會以稀缺為招牌炒作某些商品, 但若商品本身沒有內在價值, 泡泡終究會破的) 而加密貨幣說穿了其實也就是一套記帳系統, 基於區塊鏈網路的記帳系統. 而這套系統的內在價值就像黃金一樣, 奠基於其去中心化的穩定性, 這個去中心化的網路, 使其運作不再需要仰賴對特定人/組織的信任. 而和黃金不同的是, 確保加密貨幣這套系統安全正常運作的基礎並非物理上的穩定性, 而是去中心化程度以及算力規模大小, 這也是加密貨幣是否有內在價值的重要衡量指標. 雖然任何人都可以輕易創造出一套加密貨幣, 但缺乏足夠的去中心化算力保障其運作的話, 這個系統是很容易崩潰的, 這樣的系統也不會讓人敢把身家財產放在上面, 也就是說你隨便創建的加密貨幣並沒有實質的內在價值, 當然也不能和其他安全的區塊鏈相提並論說它造成無限通膨. (不過由於還有一堆人無法體認到加密貨幣的價值, 這領域還是一片藍海, 所以一堆缺乏內在價值的糞幣也能亂炒一通, 這在新興領域也是很正常的現象) ====== 結論 ====== 拿目前前兩大的加密貨幣比特幣和以太幣來說, 我可以滿客觀有信心的認為, 以它們的去中心化程度與算力規模, 應該可以挺得過第三次世界大戰的摧殘還能正常運作, 不用依賴特定政府或組織, 這就是他們的價值所在. 和黃金比起來, 加密貨幣還有易於交易/容易藏匿的重要優勢, 真可謂人類史上最重要的貨幣革命了. 話雖如此, 加密貨幣系統安全運作的前提還是在於網路效應所建立的去中心化區塊鏈網路, 在這個資訊網路發達的年代, 整個地球就像是個小村莊一樣, 我們還是不能小看網路媒體的影響力, 就像GME事件一樣, 搞不好哪天莫名奇妙整個比特幣網路就被搞垮了也不一定, 成也網路效應, 敗也網路效應, 所以投資還是要適度分散風險, 不要把雞蛋放在同一個籃子中. 希望未來的區塊鏈網路系統可以跨越星際種族, 成為更加去中心化, 更安全的系統. ※ 引述《x77 (肛之戀精術士)》之銘言： : 請問一下，我朋友每次聽到加密貨幣， : 就會用以下的理論來靠杯，請問如何反駁？ : 1.不抗通膨 : 幣仔整天說政府會亂印法幣 : 但是加密貨幣是有限的，所以可以抗通膨 : 問題是現在加密貨幣種類越來越多 : 而且政府要印法幣，好歹也要顧及貶值問題 : 加密貨幣可是民間就能源源不絕，創造有的沒的幣 : 那只會通膨的更嚴重 : 2.沒共識 : 法幣的背後有政府撐腰， : 基本上一個正常的政府，一定會去捍衛法幣的價值 : 但加密貨幣的價值，完全只建立在「幣仔的信心」 : 政府想搞你就搞你，像最近中國禁止比特幣 : 還有馬斯克這種大神，也是想收割就收割 : 根本就沒有一點可靠性可言 : 或許有人說，加密貨幣的價值就是建立在每個人的心中 : 但加密貨幣這麼多 : 到底哪個大家會有共識，誰知道？ : 3.沒有實體經濟產出 : 以投資來說，長遠看來，投資股票比加密貨幣實際多了 : 因為投資股票，代表著你取得這間公司一定比例的權利 : 公司是有實體經濟效應的，他會產出實際的產品 : 在市場上賺取利潤 : 然後分派股利給股東 : 加密貨幣完全沒有實體經濟產出 : 反而浪費一堆電 : 目前的增值完全只建立在投機炒作 : 4.比黃金更不如 : 綜合以上， : 你想投資一個長遠會抗通膨的貨幣，真的該投資的就是「黃金」 : 因為黃金，它的信心已經有幾千年了 : 它真的有實際的用途，每個政府都認可 : 它的量雖然可能會越來越多，但絕對不會比加密貨幣成長的快速 : 而且戰爭、停電時，它還是有用 : 總之，我的朋友認為，加密貨幣就是在炒作 : 它真正的用處只有洗錢， : 或是想把錢匯到國外，又不想付電匯費時，可以用加密貨幣當中繼 : 現在講的投資加密貨幣，其實就是大家賭博炒作，抓最後一隻老鼠 : 請問他的論點，如何KO？ -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 114.136.173.210 (臺灣) ※ 文章網址: https://www.ptt.cc/bbs/DigiCurrency/M.1621525397.A.07F.html ※ 編輯: guestid (114.136.173.210 臺灣), 05/20/2021 23:47:32
