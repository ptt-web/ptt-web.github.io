Brandon Howlett, 3B 選秀：21th Round, 2018 from George Jenkins HS (FL) (BOS) Age: 19.3 Height: 6' 1" Weight: 205 Bat/Thr: R/R Tool評比 (現在/未來性) Hit Raw Power Game Power Run Fielding Throw FV 20/50 55/60 35/50 40/40 40/50 50/50 40 Howlett在佛州地區已經被關注多年，而且在繳出不錯的春天表現後被認為能打進 前五輪。然而Howlett在NHSI大賽中慘不忍睹，在30支球隊的球探們面前當了工業電風 扇，完全未達到外界認為他會爆發的期待。 紅襪在簽下Howlett之後發現是隱形眼鏡的問題，隱形眼鏡讓Howlett在日場比賽特別困擾 (所有NHSI比賽都是在日場)，所以在Howlett進入職棒之前就解決掉這個大問題了。 Howlett在職棒的43場比賽擊出六轟，三振率有控制住，這讓先前看壞Howlett打擊的 球探們跌破眼鏡。 Howlett常常被拿來跟Northcut做比較，因為兩人都是紅襪在選秀後段挖出來的好球員 (同時都已被長期觀察)，兩人也都是三壘手。Howlett的打擊手感跟game power都略勝 Northcut，而防守則是Northcut較佳。 -- 15. Gilberto Jimenez, CF 簽約：July 2nd Period, 2017 from Dominican Republic (BOS) Age: 18.5 Height: 5' 11" Weight: 160 Bat/Thr: R/R Tool評比 (現在/未來性) Hit Raw Power Game Power Run Fielding Throw FV 20/55 35/40 20/35 80/80 45/55 45/45 40 Jimenez是球探們喜歡的類型，也是這份名單上排名最高的真‧Sleeper。紅襪2017年 國際簽約中有不錯的收穫─Antoni Flores, Gilberto Jimenez, Brayan Bello, Daniel Diaz雖然在GCL都打不到一週，但都已經排進紅襪新秀前20名了。 Jimenez號稱是打棒球的跑鋒(NFL running back)，體型精瘦、運動能力優異，而且打球 很拚，很努力練球。球探們對於他的跑步速度評價稍有歧見，但不是70分就是80分；別 忘了他在壘間和場上的直覺都還沒到位呢。 因為Jimenez沒有什麼power，所以他會是標準的打了就衝的小球戰術型田徑隊球員。這 確實限制了Jimenez的天花板，但是檢視過往突破預期的新秀中，常常是那種運動能力 plus級、makeup plus級的球員。這屆世Jimenez排名這麼高的原因。 -- 16. Mike Shawaryn, RHP 選秀：5th Round, 2016 from Maryland (BOS) Age: 24.3 Height: 6' 3" Weight: 225 Bat/Thr: R/R Tool評比 (現在/未來性) Fastball Slider Changeup Command Sits/Tops FV 50/50 50/55 45/50 45/50 89-92/93 40 Shawaryn評價最高峰的時期是大二，但是大三之後因為傷勢已級stuff退步的關係，在 選秀中大跌到第五輪。Shawaryn以先發身份闖蕩至今，但由於stuff的限制，牛棚角色 應該還是比較適合他。Shawaryn有怪異的low slot投球機制，所以Slider有類似Cutter的 橫向移動，這是他最能製造揮空的球種。Shawaryn的Changeup和Fastball (在AFL中只有 upper-80s)都只有普普等級，單靠怪異姿勢能製造出欺騙性而已。Shawaryn有可能成為 五號先發/長中繼身分；最快能在2019球季就登場。 -- 17. Denyi Reyes, RHP 簽約：July 2nd Period, 2015 from Dominican Republic (BOS) Age: 22.2 Height: 6' 4" Weight: 209 Bat/Thr: R/R Tool評比 (現在/未來性) Fastball Curveball Changeup Command Sits/Tops FV 45/50 45/50 45/50 45/55 88-90/92 40 大聯盟的完投越來越少見，在小聯盟更是如此，因為小聯盟注重謹慎的養成，不會在意 一場比賽的結果。不過超級有效率的好球達人Reyes光是在2018年就丟了兩場完投，而且 都花不到100球；還有一場對Columbia丟滿八局。雖然Reyes球速不快，但是他靠著身體 推進以及超長的手臂製造出大幅地向前延伸距離，讓他的upper-80s Fastball看起來像是 low-90s Fastball。Reyes的Fastball角度並沒有特別刁鑽，但Changeup和Curveball都 蠻有效的(雖然不到制霸武器等級)。Reyes有機會成為高效率後段先發。 -- 18. Brayan Bello, RHP 簽約：July 2nd Period, 2017 from Dominican Republic (BOS) Age: 19.6 Height: 6' 1" Weight: 170 Bat/Thr: R/R Tool評比 (現在/未來性) Fastball Curveball Changeup Command Sits/Tops FV 50/60 50/55 45/55 30/50 91-95/96 40 球速很快的年輕人，夏天在DSL的表現不錯。Bello赴美後只在GCL打一場比賽，然後就去 指導聯盟報到了，主要都丟一些短局數的比賽。Bello在指導聯盟中偶爾秀出三個高於 平均的球種。Bello的體型和手臂動作都有成長空間，有機會能催出更快的球速。若 養成狀況不錯有機會成為四號先發，但目前他距離上大聯盟還有好幾年的時間。 -- 19. Daniel Diaz, 3B 簽約：July 2nd Period, 2017 from Venezuela (BOS) Age: 18.0 Height: 6' 3" Weight: 200 Bat/Thr: R/R Tool評比 (現在/未來性) Hit Raw Power Game Power Run Fielding Throw FV 25/50 55/60 20/50 50/40 40/50 60/60 40 Diaz是2017年國際簽約中拿最高簽約金的球員($1.6M)，比Antoni Flores還多。Diaz去年 在DSL表現不錯，之後就赴美參加指導聯盟。 Diaz體型已經蠻大隻了，有些球探甚至認為他會長到6' 3"、225磅等級，比目前重25磅。 長期下來Diaz留在三壘的機會不高，但是這麼年輕的球員還有機會重塑自己的身型。Diaz 擁有plus級臂力和raw power，但是因為目前打擊野心過高、過度注重power的關係，打擊 能力還有些疑慮在，這些問題都需要被控制住。 Diaz將被限制在角落內野手，而且打擊跟守備位置都尚有疑慮，所以目前得納入觀察 名單。不過Diaz天花板很高，power很強，有機會常為攻擊能力高於平均的三壘手。 -- 20.
