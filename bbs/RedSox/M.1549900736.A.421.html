Triston Casas, 1B 選秀：1st Round, 2018 from American Heritage HS (FL) (BOS) Age: 19.0 Height: 6' 4" Weight: 238 Bat/Thr: L/R Tool評比 (現在/未來性) Hit Raw Power Game Power Run Fielding Throw FV 25/50 70/80 35/65 40/30 45/55 60/60 45 Casas是近年來最受球探矚目的高中球員之一，他在南佛州的American Heritage和一些 美國代表隊中都是扛打線核心位置。他獲得成功的原因之一是因為他在2019年畢業的那屆 中算是年紀最大的球員─因此他加速了學業進程，透過reclassifying選擇在2018年就 投入選秀─提前一年反而讓18.4歲的Casas處在適當位置。 Casas在早期獲得的成功讓對方投手早早改變投球策略，也就是一直投開，不讓Casas打。 許多球探看完整場比賽就只看到Casas揮棒一兩次，因為Casas一直獲得保送。幸好 Casas已經有長久的強打紀錄─在高一暑假參加幾個全壘打大賽都繳出優秀成績，在選秀 前的比賽中exit velocity也很亮眼，所以各隊都知道Casas的打擊潛力很棒。 Casas的手部很棒、具有plus等級的手臂，所以在投手丘上能丟出low-90s球速，不過 防守問題在於跑步速度很慢，橫向移動的敏捷度也不佳。Casas在簽約後守三壘─雖然 因為在六月份守滾地球傷到拇指後得動手術，整季報銷，所以沒多少場比賽能打─但 一般預期Casas的守備位置未來得移到一壘(約2019年或2020年)，而他在一壘的表現 應該會很不錯。 Casas的王牌天份依然是打擊，他有hit 60, game power 70, raw power 80的強打 技能組。雖然高中畢業生 + 一壘手的打擊門檻很高、容錯空間很小，但是我們依然 很看好Casas的打擊能力與未來發展。 -- 2. Darwinzon Hernandez, LHP 簽約：July 2nd Period, 2013 from Venezuela (BOS) Age: 22.1 Height: 6' 2" Weight: 185 Bat/Thr: L/L Tool評比 (現在/未來性) Fastball Curveball Changeup Command Sits/Tops FV 70/70 60/60 45/50 40/45 94-97/99 45 D. Hernandez的未來預測被視為會成為牛棚投手，而且他的stuff很符合一些頂尖左手牛 的特色。依照以先發角色養到現在的成績來看，他遠低於平均的控球能力(control)將使 他無法在大聯盟中投出足夠好球率，擔任大聯盟輪值將有很大的機率無法投超過兩三局。 但轉任牛棚的話D. Hernandez的將會成為很出色的投手，他的Fastball很強悍、角度很 刁鑽，當D. Hernandez把Fastball丟往glove side(右側)時，他的Fastball會有自然的 尾勁，能誘使左打者出棒卻因為尾勁的關係揮不到球。D. Hernandez的變化球種都有 製造揮空的能力，但是他有無法重複投球動作的毛病在─這就是D. Hernandez控球不穩的 一大原因。有時變化球能展現出plus-plus的潛力，但是有非常多球離好球帶太遠，毫無 引誘性。 終歸來說，D. Hernandez的發展預測將取決於球探認為他的控球能修到什麼程度。純論 stuff的話D. Hernandez確實有高張力局數牛棚的手臂潛力，有機會成為大聯盟Top 30 牛棚投手。 -- 3. Michael Chavis, 1B 選秀：1st Round, 2014 from Sprayberry HS (GA) (BOS) Age: 23.4 Height: 5' 10" Weight: 190 Bat/Thr: R/R Tool評比 (現在/未來性) Hit Raw Power Game Power Run Fielding Throw FV 40/40 65/65 50/55 40/40 40/45 55/55 45 Chavis是相當著名的亞特蘭大地區高中球員，在高中時期繳出很棒的數據，具有plus raw power，並能在比賽中展現出力量。比較尷尬的還是位置問題，高中時期和現在 都是。他在高中時期是游擊手，但是他的體能條件並不適合游擊；曾嘗試蹲捕但以失敗 收場。Chavis目前主要站三壘、球探一般認為Chavis長期下來會是三壘手或一壘手，但是 有些職業及球探認為Chavis未來將被限制在一壘。Chavis在2017年秋季聯盟與2018年 季末都有獲得在一壘的出賽機會。 Chavis在2017年打出爆發性的31HR成績，但是接下來就被抓到使用PED，2018年被禁賽 80場。在服完禁賽、升上3A後的半季以來，Chavis再次繳出不錯的表現；但是Chavis的 技能組幾乎只有單靠power拉抬，球探對於這種單一維度的球員類型都偏向持保留態度， 特別還是最近有PED禁賽紀錄的那種。 Chavis防守上的能力限制很有可能讓他成為一壘platoon球員(弱勢側)，或是類似Kevin Millar的普普先發球員。無論是哪種，像紅襪這種要拚冠軍的球隊不太可能會想花時間 養這種天分不夠高的球員到成為每日先發等級為止。我們認為Chavis會成為low-end 每日先發，所以很有可能在Chavis繼續繳出好表現後，被紅襪拿去當交易籌碼。 -- 4. Antoni Flores, SS 簽約：July 2nd Period, 2017 from Venezuela (BOS) Age: 18.2 Height: 6' 1" Weight: 190 Bat/Thr: R/R Tool評比 (現在/未來性) Hit Raw Power Game Power Run Fielding Throw FV 25/55 45/55 20/50 50/45 45/55 60/60 40+ Flores在2017年獲得紅襪第二高的國際簽約金$1.4M (僅次於Danny Diaz的$1.6M)，他在 Southwest Florida指導聯盟繳出亮眼表現，且具有很棒的天分。在去年夏天制霸DSL後 升上GCL，但是打了兩場比賽之後就因為傷勢的關係休戰。Flores讓許多球探聯想到潛在 每日先發游擊手，但是是那種你要看很幾場比賽才會喜歡他的那種。與其他7/2頂級新秀 不太相同的地方是，Flores並沒有特別顯眼的tool，會是非常突出的體能發展潛力； Flores比較偏向穩定發展、低調的球員，對於防守上有不錯的天分、打擊上有高品質的 contact。(防守直覺很常見於委內瑞拉球員上，他們在發展上很注重比賽的掌握) Flores的打擊有above-average潛力、contact能力好，raw power則是average等級。以 Flores的身形來說他的動作算意外的快，跑步速度average，防守直覺足夠彌補腳步速度 的不足，讓他cover更大的range。手臂方面則是plus等級。 Flores的身材有可能會長太大而無法守游擊，但至少能成為高於平均甚至是plus級的 三壘手。除了傳球手臂之外，Flores很有可能成為平均型的球員，各項都拿平均的5分 (2-8 scale)。目前Flores職業經驗尚少，球探們對於看到的少少部分上感到相當興奮， 若打擊發展得不錯，Flores將有機會能成為一線球隊的每日先發球員。 -- 5. Tanner Houck, RHP 選秀：1st Round, 2017 from Missouri (BOS) Age: 22.5 Height: 6' 5" Weight: 220 Bat/Thr: R/R Tool評比 (現在/未來性) Fastball Curveball Changeup Cutter Command Sits/Tops FV 50/50 50/55 45/50 55/60 40/50 90-93 / 95 40+ Houck在大一的球季爆發，並且在該年夏天成為Team USA的Ace；Houck在接下來兩季都 維持高水準表現。當時主球種是91-96 mph的沉重sinker，姿勢是低出手角度的側投， 具有不錯的欺騙性。Houck有不錯的好球率和plus Slider，但是球探們不喜歡Houck的 Changeup、arm slot和揮臂路徑長度，所以認為他會成為牛棚投手。Houck還有個問題是 經過幾局之後，stuff和command會開始變差。 紅襪在2018年頭六場比賽讓Houck改變投球動作和arm slot，試圖投傳統的4-seam，但是 成效不佳。最終紅襪決定讓Houck回到原本習慣的投球方式，而Houck在最後兩個月取得 不錯的表現。 我們認為Houck會成為複數局強力右手牛，能剋右打，製造三振和滾地；有些球探則看到 Justin Masterson型先發。Houck應該能快速通關往上升，特別是如果他每場比賽只需要 對決打線一到兩次的話。 (待續) -- ◣ ◢ ◢███ ◣ █ ███◣ █◣ ◢█ █ █ █ █ █◣ █ █ █ ◥█ █◤ █ █ █ ████ █◥◣█ █ █ ◥ ◤ X John Legend █ █ █ █ █ ◥█ █ █ █ ◥███◤ ◥███ █ ◥ ███◤ █ https://youtu.be/H4a-5BdZ5ck
