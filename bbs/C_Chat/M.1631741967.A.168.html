大概就了解的狀況憑感覺回答 ※ 引述《hundreder (hundreder)》之銘言： : 板上常常會看到美漫標題的討論串 : 不過因為美漫在東方受眾小，在板上的討論串常常都是踐踏的對象 : 反正沒人看，大家踐踏得心安理得，也不管言論到底合不合理、可不可笑 : 且不論某些單純只是出自自己閱讀習慣、甚至是偏見的評論 : 有一件事倒是東西方漫畫讀者共通的共識 : 那就是現在日漫在美國市場比原產的美漫更加吃香 : 關於這個現象，要是在PTT討論的話，畫風啦動作啦之類云云的言論又會跑出來 : 可是站在當地讀者的角度來看，他們會怎麼認為？ : 這段時間看了一些美國YT討論這個問題 : 大致上比較常提到的會是以下幾個原因 : 1. 跨入門檻高低有差 : 在Reddit經常看到有人問想看蜘蛛人、X-MEN的漫畫，要從哪裡開始比較好 : 這樣的問題在Reddit幾乎已經是周經文，大部分美國讀者看到這種問題都會耐心回答 : ，也不介意寫長篇文詳細說明 : 但由此可見美漫難跨入的門檻不分東西，不懂的人就是不懂 : 相較之下要看日漫根本沒有門檻可言，只要從第一集開始看，然後照著卷數一本一本 : 看下去就好了 美漫發展到後來其實不是給小孩看的，美漫的受眾其實是 一批很特別的死忠讀者，所以才會造成早期美漫電影被認 為票房會很糟糕，因為一般認為喜歡美漫的人都是少數的 怪咖，例如影集生活大爆炸裡面的人，美漫市場重生是在 電影熱門之後，但是美漫本身來說對小孩來說還是沒甚麼 吸引力，那些喜歡超英的小孩基本上都是看電影跟衍生 作品為主的 : 2. 內容多元性有差 : 美漫絕對不是只有超英漫畫，很多美劇或是電影其實都是改編自漫畫 : 諸如眾所皆知的陰屍路，魔鬼神探、鹿角男孩、康斯坦丁、雨傘學院、黑袍糾察隊等 : 不過不可否認的是超英漫畫確實幾乎變成美漫的絕對代名詞 : 對美漫不那麼熟悉的人可能還會誤以為美漫除了超英之外沒別的了 : 相較之下，日漫的題材之多，這時候也不需要額外說明了 日漫的角色大多都是從兒童或青少年開始的，帶入感較高，美漫 大多都是成人出發的故事，因為美漫受眾的族群就是那樣， 自然作者就會做出迎合那種喜好的作品 : 3. 美漫內容受到干預而劣化 : 簡而言之就是來自SJW的影響 : SJW的影響範圍遍及美國各種娛樂事業，其中美漫受到的荼毒特別嚴重，漫威與DC皆然 : DC有”我不是星火”與最近討論熱烈的Tim Drake毫無來由突然出櫃事件 : 漫威則是有那個誇張到曾經令人懷疑是不是官方搞反串的SJW版新戰團 : (可惜不是反串，結果因為批評聲浪太大，這批新戰團沒有成書出版) 這個我倒不這麼覺得，在SJW還沒興起的時代美漫就是那樣了， 現在大不了就是石頭奶的角色變成體態均勻一點，然後像是 脫衣舞釀的超人服裝改成正常一點，但是還是沒有脫離美漫 醜到爆的美感，美漫的美感其實就是一種脫離現實， 倒是最近受到日漫影響，畫風、服裝設計都變正常一點， 所以說近期跟日漫相容性還高一些 : 像這類SJW相關的人事物，美國有一種形容詞叫做WOKE : 現在美漫的編劇群就有很多這類WOKE的人，對他們來說傳達政治理念比寫好故事重要 : 很多美國YT搞不懂，為什麼DC與漫威不明白讀者只是想要看好故事 : 在故事優秀的前提下，他們也不在乎接受SJW宣傳的理念 : 可是那些SJW派的編劇不願意花時間潛移默化一步一步來，只想隨便抓一個具有傳統歷 : 史的角色馬上染色改造，成為強迫灌輸自己理念的大喇叭 : 他們甚至把宣傳理念的重要性置於營利之上，讓DC、漫威等出版商與讀者漸行漸遠， : 甚至互相對立 : 這種類似心靈強姦的做法造就出相當不討喜的主角與故事內容，破壞了美漫經年累月 : 累積起來的人物與故事結構，也讓美漫工業更加雪上加霜 : 就我個人認為，漫威受到的荼毒比起DC更嚴重，這或許與漫威頭上是迪士尼不無關係 : 我也真心希望日漫的黑船反入侵能夠讓DC與漫威回歸本心，想像漫畫真正的用途與目 : 的是什麼 其實我不太同意你的說法，美漫會長成這樣就是這些大公司 商業經營的問題，為了商業利益而干涉或甚至主導創作， 多宇宙的設定不就是為了榨乾角色的利益不停重複利用 角色嗎?
