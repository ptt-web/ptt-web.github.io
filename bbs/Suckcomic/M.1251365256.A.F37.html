之前也有想到關於海賊的果實問題, 自己的解釋方式是:"時代不同". 魯夫吃下果實起碼是10年前的事情(當然是漫畫內的時間XD) 那時的海賊世界也許並非以果實為主要戰鬥能力, 當時紅髮海賊團內就沒有任何一個是果實能力者, 同理可以解釋小丑為何只要偷果實去賣, 而非吃掉果實. 黑鬍子卻花了畢生之力去尋找果實. 不過比起這種"解釋方式", 我更相信是設定的bug XD 而尾大自己本身也陷入了果實的牢籠, 從一開始各種不同特色的戰鬥者, 或許是武器, 鍛鍊方式, 種族差異等. 尾大的確可以在不扯到果實的條件下, 架構出許多精彩的戰鬥. 不過現在檯面上卻幾乎清一色都是果實能力者, 的確讓人看得有點乏味啊~ ※ 引述《kisaku ()》之銘言： : 有些王道漫畫 : 比起一堆劇情爆走或者不知所謂的漫畫 : 勝個幾百倍 : 但我仍覺得這些漫畫有瑕疵 : 所以一吐為快 : 首先是最熱門的海賊 : 不諱言他是一部結構完整 : 世界觀宏大的RPG漫畫 : 但隨著故事的不斷發展 : 一個原先的設定產生的效果 : 像蝴蝶效應般越滾越大 : 讓現在的海賊王變得很奇怪 : 那就是為什麼現在出現在這個世界裡的船長和海軍頭目 : "幾乎都是旱鴨子阿阿阿阿阿阿阿阿阿阿阿!!!!!!!!!!!!" : 惡魔果實=不會游泳 : 其實我不能理解作者設定吃惡魔果實就不能游泳的目的是為何 : 當初少了這句話 : 我也不覺得對海賊王的哪一段劇情會產生重大的影響 : 但因為多了這句話 : 就變成現在滿街的船長和海軍頭目通通都是旱鴨子 : (因為大家都得吃惡魔果實才能打仗) : 那當初第一集魯夫吃了果實 : 那些人在擔心個什麼勁? : 照這種設定 : 明明就是吃了惡魔果實才能出海當船長吧 : 難怪他們只能當背景旁白小兵 : 在旁邊窮擔心 : 這奇怪的現象追本溯源 : 就是當一部漫畫 : 作者要合理化新奇能力的產生時 : 往往會給定一個基本的原因 : 進而構築出各式各樣合理的新能力 : 像獵人就是念能力 : JOJO就是替身使者 : 那海賊王把這個原因設定在惡魔果實身上 : 於是就產生上面的奇怪世界觀 : "多數海軍和船長幾乎都不會游泳" : 但是幹麻一定要把新技巧的原因綁定在惡魔果實上 : 不是有各種種族嗎 : 什麼魚人天龍人的 : 那今天一個不是人類的天龍人會特殊能力 : 也是很合情合理的吧 : 那亞馬遜女獵祖傳密技可以迷惑人心不也說的通嗎 : 幹麻一定要吃惡魔果實 : 同樣綁影子的七武海 : 你把他設定成吸血鬼一類的種族不就好了 : 為何又要用惡魔果實來解釋 : 另外 : 像索隆走的劍客道路 : 那邊也不用惡魔果實來鋪陳阿 : 那條路不就發展成格鬥電動最愛的招示名稱對決風 : (雖然好像現在海賊王每個人的對戰幾乎都是如此了 =.=) : 還有像空島篇 : 不也不用惡魔果實 : 就給定了魯夫的對手們特殊能力 : 以上明明有那麼多方法可以給定新能力 : 而且都被採用在漫畫裡 : 海賊王作者為什麼輪到給定船長等級和海軍幹部的新能力 : 就通通都得乖乖吃惡魔果實? : 結果搞到現在 : 每個新能力都用惡魔果實來解釋的結果 恕刪. -- 遵守規則的人才是有獨創性,有個性的人. 憑胡亂的想法是不可能有大發現的. 在遵守規則的研究基礎上,學問才得以進展. 沒有遵守規則的精神,就不可能有學問上的發現 -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 61.62.79.205
