成二種：抽掉"打倒人造人"成分就沒意義的主線劇情，和即使與人造人無關，依然 : 有獨立存在價值的支線劇情。合成獸妮娜事件、艾力克兄弟的過去、在師父那裡的 : 修行都屬於極有意義的支線劇情，然而這些劇情在第8集後急速減少，到12集溫莉與 : 斯卡偶遇後幾乎就沒有了。後面的情節都圍繞在人造人引發的一系列事件上，整個 : 故事變得只剩骨架，而無血肉。用蓋房子來比喻，作者似乎沒有估計到蓋這樣的一 : 棟房子需要多少的磚頭水泥來搭配鋼筋，蓋到後來才發現磚頭用完了，可是鋼筋還 : 有好多根...於是鋼鍊就像個一直沒有進食的難民，開始時渾身肌肉，到後來卻餓得 : 皮包骨。(這裡就不得不提提隔壁棚的獵人，只要覺得肚子餓就坐下來開始吃，不簡 : 單！)另外要提的是，鋼鍊一開始的目的「尋找讓兩兄弟身體復原的辦法」一直沒有 : 被顯出來，尋找過程是：賢者之石=>豆爸的提示=>脫出偽‧真理之門，在這段過程 : 中劇情完全是跟人造人事件勾在一起的，這個..好像顯不出他們有"特別的去找"。 : 簡言之，這是個"打倒人造人"的故事，而荒川弘應該是初次處理這種規模的架構， : 到了中後期她就逐漸疲累了，角色的內心轉變什麼的..變成用一句話帶過。甚至有 : 些劇情她本來應該是打算描寫，卻不得不刪掉。例如豆爸質問父親大人：『你是想 : 要家人吧？』這句話感覺可以多寫一點，可是就這樣帶過了。這種快速帶過的手法 : 會扼殺角色性格，有些伏筆是需要時間去沉澱、醞釀的。例如古力德(麟)和大總統 : 打鬥後，接著就和艾德他們合流，這怎麼看都有種「因為要推王了，趕快找團吧」 : 的味道。大樹的種子灑下後天天澆水，日積月累才長得壯。種下去後馬上長出來的， : 都是矮小的植物。就算有再肥沃的土壤，也要有時間才能成熟啊。作者如果只是用 : 幾句話表現，實在不夠。 鋼鍊的主線是什麼？ 與其說是勇者鬥○○型的正邪之戰 倒不如說是每個人確立自己的「目標」，進而追求「目標」過程中交織起來的長篇故事 我想說這是「目標之戰」 我認為整個鋼鍊從頭到尾都是牛媽的主線。 硬要說是支線的部份，我認為是第一集的三個事件，用一種章回式的冒險故事型開頭 簡單介紹主角的能力還有賣點溫情戲份。 這三個事件就算抽離對整體劇情也不會有影響，我認為這是創作之初試試水溫的作法 從第二集開始，牛媽開始以她的想法去構成這部作品， 剛開始「兄弟倆想恢復身體」，所以找賢者之石 這時候所謂的「賢者之石」的定位就和「龍珠」差不多，只是個傳說中的神物 後來發現賢者之石會牽扯到「人命」，這時候牛媽安排了第一個便當 好爸爸休斯（泣）來做有血有肉的例子，並不是單純的嘴上說說會牽扯到人命而已， 休斯之死我認為是讓他們切身體會到這件事情的安排， 並且牛媽並未單純為了這個目的安排這個便當，她在這個便當上綁上了大佐、 綁上了溫莉、綁上了事件黑幕，但是除了開啟事件黑幕，其他安排都是在後面才完成連結 事件過後，兄弟倆將目標更加明確為 「不傷害任何人而恢復身體」「想看看大家的笑容」 而這時後將鏡頭看看球藻方，從頭到尾的目的都是「湊滿祭品+完成國土鍊成陣」 斯卡一直砍國家鍊金術師，要是砍完了可能的祭品候補就會減少，所以他們要殺斯卡 而好爸爸休斯知道太多事情，要是被更多人發現可能的陰謀可能會礙事，所以要殺他 這邊算是兩邊陣營的第一次交會 之後開始加入「真正明白鍊金術（煉丹術）」的人物，也就是姚麟與張梅等清國人 這邊因為劇情需要，把這角色任務分給兩家 如果姚麟也會煉丹術，那張梅角色就邊緣化了，但又需要雙線進行，於是姚麟不會煉丹術 而是帶了兩個近戰系的護衛 事實上這兩家的信念一致，就是「復興/鞏固家族地位」 只是姚麟同時被拿來和人造人統治者布拉德雷做個對比 之後因為內幕漸漸揭開，所以兄弟倆漸漸進入事件核心 但，「主線」有變過嗎？ 沒有，兄弟倆從頭到尾都還是在尋求「不傷害任何人讓身體復原」 至於為什麼會和球藻團槓上呢？ 廢話，他們要把包括自己親人等等的國民作成賢者之石，除了烙跑之外，就只能阻止了啊 烙跑？ 老婆不准 那就只好打啦 所以我說這是信念之戰。 : 二、感情描寫不夠 : 我對鋼鍊中感情有起伏的橋段都有一個印象：黑體字。是的，我不會注意到他們 : 的動作、表情，而只會記得那句黑體字。表現高亢的情感有很多方法，最著名的就 : 是富士鷹的顏藝+血淚。但是鋼鍊用來表達感情的，就只有那方方正正的黑體字。看 : 到那句黑體字有什麼感覺？有人說平淡，有人說是真實世界的情感表達，這些都沒 : 錯，但我會用兩個字描述：壓抑。想叫卻叫不出來，整部漫畫中充斥這種壓抑感。 : 用壓力來限制感情表達，往往是要在最後一口氣爆發出來，可是鋼鍊就這樣平淡的 : 結束了。這些感情到哪裡去了？真理之門嗎？唯一出現感情爆發的橋段大概只有一 : 個：阿爾跟艾德吵架的那次。兄弟打架，最後躺在地板上回憶過去，言歸於好。這 : 是我認為全書中寫的最好的感情戲，不過也是相當前面的部份了。到後來作者想表 : 達什麼個人信念、角色感情，都是直接用角色的嘴巴說出來。我覺得這也不太現實， : 曾經看過一部描寫二戰的電影，一部份是講述猶太人被送到集中營處刑的過程。其 : 中一名猶太人站到處刑台上，才剛要講話，就被一槍射中心臟。那個人死前最後一 : 句話是：『我以為我能發表一篇名垂千古的演說...』實際上，有太多人沒有說心裡 : 話的機會。鋼鍊裡大部份的人都有主張他們信念的機會，這似乎是作者強行灌輸給 : 讀者的..例如火燄上校與恩維的最後一戰，看了就覺得「這是在說教嗎？作者是在 : 告訴我們『應該怎麼做』嗎？」這種說教的方式並非上策，斧鑿痕跡太重太刻意， : 不是很能讓人接受。最後艾力克兄弟的「打破等價交換法則」出現也很突然，沒有 : 和什麼別的元素連結在一起，應該..有改善的餘地。 : 三、角色塑造不足 : 如前所述，比較有趣的劇情集中在前8集，因此越後面出場的角色越沒有描寫的機 : 會。如阿姆斯壯少將，雖然被叫做「連心臟都是用冰塊做成的」，我根本沒看到她有 : 什麼地方是冰冷的，反倒是個熱血大姐..還是說，這叫冷嬌？下面舉出幾個角色： : 反方 : 色慾──這位小姐最大的問題在沒有獨特性。人造人、情報員，這兩點就算隨便套另 : 一個角色也辦的到，除非是賣肉的部份... 她帶出哈博克是個公私分明的人，她帶出哈博克最後令人振奮一格 她帶出人造人的HP概念 她帶出庫拉多尼對於大佐的恨意這件事情連帶後續愛德+姚麟被吞 而兩人被吞帶出了愛德對於拿回身體的可能性 我同意她的深度沒有其他角色那麼深 不過對於她，牛媽並沒有輕易賜死，同樣讓她的死帶出很多事情 不過因為色慾本身在少年誌上面可能無法描寫得很露骨 牛媽功力不夠，這樣處置我覺得還可以接受 : 貪食──只知道吃和拉斯多的偽‧真理之門。若不把他當人看，算是很有特色？ : 嫉妒──戲份是很多，個性殘暴，二號打手，只是最後那段不太能接受.. : 貪婪──後面洗白的太快了..描寫算多了 : 憤怒──比較有味道的角色，不過總覺得他的感情是(戰鬥的)愉悅，而不是憤怒.. : 驕傲──露出真面目後變得一號表情，看不太出來心裡在想什麼。 : 怠惰──只是個中boss。 : 父親大人──感情都抽走了還能有什麼表現嗎？ : 金普利──瘋狂，甚至稱得上「殉道者」，但是變態度不夠.. : 正方 : 艾德──個性堅強的第一男主角。最在意身高。 : 阿爾──溫柔堅強的第二男主角。身體是鋼鐵。 : 火燄上校──野心勃勃，鬥志頑強的第一男配角。 :
