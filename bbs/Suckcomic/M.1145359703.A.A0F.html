這個作者也是讓我印象深刻 Orz 我接觸他作品是在小學的時候 看了公主軍團 還滿喜歡 因為男主角之一長得很像古早動畫"鎧傳"中的天空羽柴當麻 ( 事後果然證明作者因為喜歡當麻所以把這角色設計成這樣 ) 銀色舞台和東京茱麗葉我也都有看 不過其實銀色舞台的後半已經開始受不了北川的"情敵打地鼠"模式 ( "情敵打地鼠" --> 不斷有情敵冒出來 男女主角向打地鼠一樣一一把情敵擊退 ) 同樣的模式在東京茱麗葉依然沿用 讓人吐血 看完這部後我就不看她的作品了 去年在友人家中剛好看見她的禁忌戀人 好像是朋友剛好借了10~14集吧? ( 反正她的漫畫從中間看也一樣 XD ) 隨手翻翻 真的被嚇傻了 畫風怎麼變這麼多呀???? 人物醜了很多 劇情不用講更是比以往更加可怕 翻完後我真的深深覺得這個漫畫家也墮落了 唉 其實北川以前的畫風還算漂亮 肢體也算ok ( 不過我不喜歡她總是把女孩子的臀部畫得很園 ) 但那不斷出現的情敵模式實在讓人 = = 除此之外 故事的沒內容也讓人黑線到一個極點 就以"東京茱麗葉"來說 特地選了個服裝設計師當做題材 不斷強調女主角多有設計天份 問題是從頭到尾除了那個不怎樣的氣球裝之外 她哪次讓讀者看過女主角的設計功力到底有多強?? 每次遇到比賽或是設計界的名人 總是用素描簿的背面或是只露出局部的衣服讓評審(或知名設計師)大大驚嘆 然後主角就贏了比賽或是受到賞識 這樣怎麼說服得了讀者她真的很強啊!!?? (怒) ( 同樣跟服裝設計有關的"天國之吻"至少禮服很美 ) 然後去國外留學也不見他有受到什麼作品的刺激而成長 ( 男人的刺激倒不少= = ) 最後居然還可以去巴黎舉辦服裝秀 !!!!! 剛好之前看了旅遊頻道播的實境秀 更能體會當設計師的辛苦和不易 這部作品特地選擇服裝設計師這行業卻又搞得如此膚淺 還不如什麼職業都不要選算了 (怒) 銀色舞台也一樣 演藝界的辛苦也不是重點 反正男女主角在演唱會巡迴期間還是可以照樣作愛做不停 不管演連續劇還是做什麼工作都可以湊在一起 每天睡眠也滿充足的樣子都不用練習還有餘力對付一堆情敵 選演藝界大概是比較可以滿足小女生的幻想加上可以穿好看的衣服吧... = = 講到這就一定要推一下仲村佳樹的 Skip Beat -華麗的挑戰 裡面對恭子演技的描述根本是銀色舞台完全比不上的 舉例來說 恭子和奏江去參加廣告甄選那段 作者不但給角色出了題目 也同時出給了讀者 " 要如何表達出吵架的感覺 " " 該如何表達和好的方式? --- 特別是你的點子才剛被盜用過 " 結果作者安排的方式出乎讀者的意料之外 也比讀者預期的要好 在這同時讀者對角色本身擁有的"演技" 也就能自然而然地感受到並且認同了 可是在北川的作品裡我完全找不到這一點..... 不過想想 也許是因為我對這種本來就只想畫俗爛愛情故事的漫畫要求太高了也說不定 演藝界或是服裝設計界在北川的故事中其實並不是真的那麼重要 所以不去傳達這種行業的精神或相關常識對她來講完全無所謂吧.... 再來是不斷冒出新情敵的橋段 裡面每個男角色都很快就喜歡上女主角並且驚為天人 然後總是會先壓倒女主角順便來個強吻 或是在女主角跟男主角吵架後 因為失意或身體狀況不佳時來個偷吻 ( 我已經懶得去算女主角到底被幾個男人吻過了 = = ) 然後在退場前一附好像真的要對女主角怎樣 最後卻突然變大好人說自己贏不過男主角然後說 bye bye 男主角則是會遇到妖艷美麗女人主動投懷送抱 在對方衣服脫得差不多的時候再一附很帥的樣子說"我心裡只有女主角" 然後遇到有別人調戲女主角就是先來一拳或一踢 總之用暴力解決再說 每次都是這種模式 煩不煩哪!!!!
