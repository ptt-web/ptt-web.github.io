※ 引述《o07608 (無良記者)》之銘言： : 等等，《網球優等生》怎麼會被拿來吐呢？ : 別誤會，我非常喜歡這部以網球為主體的漫畫 : 主角丸尾榮一郎（膩稱榮兒）在進高中之前，原本只是個很會讀書的文弱書生 : 結果因緣際會之下，開始打起了網球，不僅打出興趣還打出了成績 : 人生也因此轉往職業網球選手之路 : 這部漫畫無論在平時的訓練或比賽時期，內容都非常充實精采 : 裡面敘述的網球觀念、要領等，都是有所本，而又不會像字典那樣枯燥乏味 : 有時因為劇情需要而插入的心靈輔導，也能讓人心悅誠服，而非相信我之術 : 比賽則是該快速解決的就是幾話內結束掉，該拉成大長篇的就每頁都精采萬分 : 無論是肢體動作還是球的流動，既清楚又充滿張力 : 是那種讀者在看時，會忍不住跟著喊「阿，出界了」「好球！」的感染力 : 如果現在要找一部頂尖的運動漫畫，除了經典的那些以外，我毫不猶豫會推這一部 : 那我到底要吐誰呢？ : 就是本作男主角．丸尾榮一郎先生，該死的人生大贏家 : 他的設定背景是什麼？ : 父親一般上班族、母親一般家庭主婦、從小到高中都是優等生、沒有運動習慣 : 身材瘦削、眼力很好，大概就是這樣，沒有血統問題，也沒有天賦異秉的樣子 : 這樣子的人，原本只打算把網球當做健康運動而已 : 結果三年後居然在打全日本網球選手排名賽，目前進度還到了把第九種子江川逞幹掉了 : 老天，你是打了什麼針還是吃了什麼藥？ 如果以網球來看，榮兒絕對是天賦異秉，因為他有著絕佳的動態視力。就像 他去國外訓練時，那個訓練營裡面就算不是天才也全是一流高手，而榮兒一 進去動態視力就第二名(印象中離開時變成了第一名)，要說他沒有天賦這完 全說不過去。 再來就是榮兒的打法，他那種從對手一發球就能預讀的解析能力也是天賦無 誤，當然這也要他有極佳的動態視力才能配合得到。 用一樣是網球漫畫的人物來舉例的話，榮兒就好比是乾貞治一樣的角色，他 們都一樣是在打"情報網球"。 漫畫裡也有好幾次是榮兒因為體能跟不上對手，一開始就直接照預測打的情 況出現。至於為什麼榮兒會有這麼誇張的解析能力，作者也用了優等生這個 理由來解套。但這也是我覺得不合理的地方，就算這方面再有天賦，可以不 看錄影帶，光靠臨場就解讀成這樣，某方面來說也是網球王子等級的了。 至於體能部份就算了，一方面是有下苦空在鍛練，一方面是這種時期本來就 是男生的發育期。像從小訓練的選手，體能跟基本動作都有一定水準，但不 見得解讀比賽的能力會比較好，簡單來說就是打體能球。而榮兒剛好是這方 面點滿的天才，所以把時間拿去鍛練體能還算勉強能接受，畢竟他表現出來 的也只是剛好及格而已。 結論來說，榮兒會看起來像天才是因為他是打網球的塊寶，只能說作者在這 方面的設定還挺用心的。 -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 1.160.28.101 ※ 文章網址: https://www.ptt.cc/bbs/Suckcomic/M.1433917687.A.D2D.html
