文長，請慎入!!!!!!!!!!! 基本上這個需求看似普通的一個簡單動作實際上是要做上好幾個動作。 給 您(原PO)幾個 GOOGLE 關鍵字 (以藍色為註記) 1.快速建立資料夾 A. md指令(CMD)+ Excel(或是其他免費Office，例如:LibreOffice) 教學範例: http://pcblog.tw/【電腦教學】免下載程式，教你快速地批次建立資/ ( + 複製資料夾內的所有檔案名稱 [非必要，教學範例進階使用] 或是 直接使用 Copy File Name 軟體 ) http://key.chtouch.com/ContentView.aspx?P=1839 開啟Excel: 1.請先於 B 直行複製貼上已複製資料夾內的所有檔案名稱。 2.請再於 A 直行 A 橫列 第一格輸入 md ， 請在此格右下方連點兩次左鍵，A直行補滿 md 至與 B 直行所屬列數相同。 3.選取 A 與 B 整個直行後再貼入BAT(TXT)記事本檔案內容儲存即可。 *md 建立子資料夾命名要點: a.md空格(這個md與資料夾名稱之間的空格空幾格多少沒差)+資料夾名稱 b.資料夾命名名稱: 中間有空格將不顯示空格後方的命名， 例如資料夾命名名稱:測aaa(空一格)111試 => 測aaa (前面有空格，故111試不會出現) B.Text 2 Folders軟體 或 線上快速建立資料夾 http://st.sjps.ntpc.edu.tw/qmkdir/ Folder Frenzy:https://briian.com/7095/folder-frenzy.html(底下的延伸閱讀還有) 或者選擇 想要複製資料夾，但不複製裡頭的檔案（即複製出同名的空資料夾） xcopy 指令加參數可以做，不會可以用 Copy Handler: http://www.techbang.com/posts/7479-drj-how-to-copy-only-the-folder-without-cop ying-the-files-inside 2.使用 n2ncopy 軟體複製到指定的各資料夾：http://portable.easylife.tw/1714 使用簡易說明：左邊綠色為檔案拖曳區，右邊藍色為要複製到的資料夾目的位址。 請用滑鼠把檔案或資料夾選取後左鍵按住拖曳至相關拖曳(綠或藍)區域。 3.將各資料夾內的檔案合併到同一個資料夾 資料夾合併成同一個資料夾(CMD RD或DELTREE做不到的) 接著 a. 與 b. 選一個操作: a. 直接使用 Folder Merger 軟體: http://walker-a.com/archives/2258 (請用內文官網內的下載連結，內文附加下載連結失效) Windows 10 無法使用此軟體，原因不明(可能僅為此測試PC 特例無法使用!) b.使用內建檔案總管搜尋功能(例如:搜尋列輸入.pdf或pdf→因作業系統而異) 再搭配作業系統內建檔案總管重新命名功能:http://blog.joaoko.net/archives/1511 (只適用於 Windows Vista 和最新的 Windows 7，Windows 10測試結果已不支援!) 或是搭配重新命名軟體重新命名(不支援的作業系統適用) 最後，請將所有搜尋結果全選(Ctrl+A)複製或剪下後貼上到 您想要的目的位址(資料夾) 這樣即可完成 複製 N 份 需求。 另外，有個 BAT 直接複製檔案 N 次: http://www.ckizumi.com/2013/09/copy-one-file-to-multiple-files.html Windows 10 安全性問題這邊就不測試了，但 Windows Vista 有測試無法使用 (相容性未測試) 這個 BAT 方法有興趣的煩請測試後將結果推文，謝謝。 (個人覺得是不能使用或是刻意少了幾個步驟) ※ 引述《bookmusic (....)》之銘言： 現有一個檔案： 1-1.pdf 需複製 28 份，檔案名稱可以是 1-1(1).pdf 1-1(2).pdf ... 1-1(27).pdf
