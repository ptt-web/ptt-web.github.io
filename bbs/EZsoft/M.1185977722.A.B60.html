標題[免費] iPig 免費 VPN Server (類似SoftEther)
時間Wed Aug 1 22:15:09 2007
軟體名稱： iPig Client iPig Server Express 官方網站： http://www.iopus.com/iPig/ 下載連結： http://www.iopus.com/iPig/download/ 軟體版本： 2.05 Beta 軟體分類： 網路軟體 執行平台： MS-Windows 語言介面： 英文 軟體特色： 可使用官方 VPN 服務，也可以自己架設伺服器 推薦原因： 使用介面親切，設定比 SoftEther 簡單許多 誰適合用： 需要跳板的苦主、防火牆下的受害者、虛擬 IP 使用者 誰不適合用： 擁有健全網路功能使用者 使用感想： 設定很簡單，當跳板的電腦裝 Server 版，用戶端裝 Client 版 Server 端設定使用者帳戶 http://www.iopus.com/iPig/images/screenshots/ipig-server-gui.png
Client 端設定 Server IP 輸入帳密即可連線 http://www.iopus.com/iPig/images/ipig-user-account.png
不像 SoftEther 或 Hamachi 需新增虛擬網路卡 也不用作橋接分享網路之類的設定 連線後即可以以跳板電腦的 IP 上網了 附註： 1. Client 版預設是連官方伺服器，免費帳號流量只有 10MB 要連自己架的 Server 記得改位址，流量可自訂 2. Server Express 版(免費)只提供五用戶連線 3. 參考教學: 《資訊應用》:Windows上架設與使用免費的VPN http://www.ascc.sinica.edu.tw/nl/95/2208/03.txt 4. 不知道是不是我的問題，還是免費版的限制 我使用時斷線率極高，一分鐘不到就斷一次雖然會自動重連 網頁瀏覽可能影響還小，但如果是 BBS 或線上遊戲想必會想砸電腦 本著作依據 Creative Commons 姓名標示-非商業性-相同方式分享 授權條款為授權 http://creativecommons.org/licenses/by-nc-sa/2.0/tw/ -- Q～ O～ O～ ◢▂ 有種果汁真好喝～ ─ ─ 喝的時候 Q ～ ▼ 喝完 臉- 紅- 紅------ ▇ █ Q～～～～ 好喝的果汁就是 Q ～ ◤◥ -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 140.112.190.59 ※ 編輯: lcwmax 來自: 140.112.25.189 (08/02 02:11)
