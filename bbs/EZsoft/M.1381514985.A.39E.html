※ 引述《thisisshita (..................)》之銘言： : 出來的東西分成兩個 : 一個是區域網路一個是PPP : 區域網路的IP ADDRESS192.168.1.2 : SUBNETＭＡＳＫ　２５５。２５５。２５５。０ : Default gateway 192.168.1.1 ^^^^^^^^^^^ 這是你目前電腦所在區域網路的預設閘道(IP分享器)位址 : PPP的又不一樣 ^^^ 這個常見於ADSL撥接的PPP adapter連線 也就是你的電腦有設定除了區域網路的DHCP之外 又再自行以ADSL的PPPoE撥號連上網路 : IP 跟default一樣 : subnet mask 255.255.255.255 : default gateway : 是哪一個呢 : 我192.168.1.1 .2我都試過了 : 255我也試了.... : 唉 我不懂了 我推文的第一點是關鍵啊 首先把電腦用無線或有線的方式連線到你要查的機器 照你前面所述，如果連過去發現192.168.1.1不是你要查的那台機器 就表示你目前的電腦沒有與你要查的那台機器連線， 或是路由沒有以你要查的那台機器為閘道 你可以 1.把192.168.1.1那台機器先行關掉 2.把你的電腦以直接用有線或無線的方式連線到你要查的機器 (有些機器會禁止以無線的連線方式登入設定頁面,所以最好是以有線連接) 3.執行ipconfig 4.以瀏覽器或telent登入ipconfig裡的預設閘道ip位址 (有些舊型機器預設是以telnet登入) 5.假設仍然失敗,則有可能該機器並未開啟DHCP功能 6.請翻閱該機器說明書(若無請上網搜尋該廠牌型號的手冊) 尋找原廠的ip登入設定，並照手冊說明登入該裝置設定介面。 7.若仍失敗,請翻閱手冊尋找重置該機器的按鈕方法恢復原廠設定 8.重複步驟6 9.仍然失敗，請送修。 補充: 假設你家的網路結構是這樣,以便說明的更清楚 ┌桌機2 192.168.1.3 ├桌機1 192.168.1.2 │ 中華電信的小烏龜┤192.168.1.n (192.168.1.1) └電力線網路+無線分享器 192.168.x.m ╮╮╮IPad 192.168.x.y ╮╮╮筆電 192.168.x.z 以下回答是假設你家的網路結構是這樣子接的話 忘記ip，想要連線至電力線+無線分享器進行設定 你可以有三個確切的方法 1.使用iPad或筆電查詢自己的預設閘道192.168.x.m, 然後使用iPad或筆電的瀏覽器連線192.168.x.m 並輸入該機器設定介面的帳號密碼 2.桌機進入中華電信的小烏龜192.168.1.1查詢IP分配或連線紀錄 即可得知192.168.1.n是甚麼，然後由桌機瀏覽器連線192.168.1.n 並輸入該機器設定介面的帳號密碼 3.將桌機直接以網路線連接該機器(電力線網路+無線分享器)的LAN端網路孔(非WAN端), 然後同步驟 1. 其他就如同前面回文 : : 6.請翻閱該機器說明書(若無請上網搜尋該廠牌型號的手冊) : : 尋找原廠的ip登入設定，並照手冊說明登入該裝置設定介面。 : : 7.若仍失敗,請翻閱手冊尋找重置該機器的方法恢復原廠設定 : : 8.重複步驟六 : : 9.仍然失敗，請送修。 -- 內文無私人ip資訊,故回文至原發文處,提供給其他版友參考、指教。 -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 1.34.163.231 ※ 編輯: newclicker 來自: 1.34.163.231 (10/12 20:53)
