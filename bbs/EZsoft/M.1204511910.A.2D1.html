標題[推薦] 簡單好用的防火牆 PC Tools Firewall 3.0
時間Mon Mar 3 10:38:27 2008
軟體名稱： 免費而且簡單易用的防火牆，PC Tools Firewall Plus 3.0 官方網站： http://www.pctools.com/firewall/ 軟體特色： 覺得Comodo Firewall 太複雜嗎? 不想用花錢的防火牆，但是其它免費的卻都是英文版? 來試試PC Tools Firewall 吧! http://aycu33.webshots.com/image/47072/2000303767490085470_rs.jpg
使用感想： 我個人用過Comodo Firewall跟PC Tools Firewall，比較起來是PC Tools比較好用 因為Comodo會跳出很多詢問視窗，還會吃CPU的佔用率，這都要靠爬文跟設定才能解決 太麻煩了，而且你開BT的軟體，按了允許還是連不到外網，都要自己去設定，更累人 相反的，PC Tools就沒上面這些問題，介面清爽、佔用的資源也少、而且不用設定 在防火牆的評比中，能力大概是中等，但是一般使用者就夠用了，已經比SP2內建強太多 當你開BT軟體後，按了允許就可以直接連到外網，不用設定，如下圖 http://aycu31.webshots.com/image/47230/2004346843254068792_rs.jpg
如果你正在用Ashampoo 或是費爾防火牆的話，也推薦來使用這個 因為這兩個在防火牆的評比中，能力實在太弱，只比SP2內建強一點... 而PC Tools的介面可以讓你立刻上手 附註： 記得安裝的時候，選項不要選Spyware Doctor，他是附加軟體，選了會下載很久 連線巴哈姆特BBS不必等待的方法： 打開防火牆->歷程紀錄->選定那些被攔截的選項(後面有網址bbs.gamer.com.tw那個) ->按右鍵->添加規則。 這個動作做一次就好，然後重開你的PCMan，即可成功連上。 -- CPU ：AMD Sempron 3200+ AM2 | DVD-RW ：BENQ DW2000 20X MB ：Gigabyte MA69GM-S2H | PSU ：康舒 E2-390 350W VGA ：內建 ATI X1250 附DVI | Other ：多合一讀卡機 RAM ：Kingston 1G DDR2-667 | Case ：杰強 幻象2000戰鬥機 HDD ：Seagate 160G S-ATA2 | LCD ：AOC 193FW 19吋寬螢幕 -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 60.244.83.70 ※ 編輯: macro 來自: 60.244.83.70 (03/03 11:00)
