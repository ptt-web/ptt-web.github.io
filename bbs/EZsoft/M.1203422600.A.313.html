: 推 KENyroj:有沒有人知道 2003 及 VISTA 下要用什麼 @@ 我用 XP 不行 應該是因為 Win2003 和 Vista 的輸入法檔案名稱不一樣 也可能是你用的 XP 裏面日文輸入法有問題或版本不同 我沒用過 2003 和 Vista，所以底下步驟只是用推測的，可能與實際狀況不同 1.執行 regedit，到 HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Keyboard Layouts\ 底下應該會有一堆各種輸入法的機碼， 找到 E 開頭、0411 結尾的，那個就是日文輸入法 2.把那個機碼複製一份，名稱格式為 Exxx0404 <- xxx 數字可自訂，別重複就好 複製方法： a) 在 Keyboard Layouts\ 下開一個 Exxx0404 的機碼 b) 在 Exxx0404 新增幾個字串值和內容 Ime File 資料：xxxxxxx.ime <- 填入日文輸入法的檔名 Layout File yyyyyyyy.dll <- 填入日文輸入法的檔名 Layout Text 日文輸入系統 (MS-IME2002) <- 隨你高興 3.到輸入法設定處 (在輸入法工具列按滑鼠右鍵，選「設定值」) 選 新增輸入法 時中文語系下就會出現日文輸入法了 -- 據了解正夯英國研究驚傳宅男大喊吃不消崩潰欲哭無淚柔腸寸斷Ｍ型營養午餐付之一炬 踢爆算一算對當事人來說受害高達八成真是超扯形成壓倒駱駝的最後一根稻草活不下去 爆料低調希望有關單位重視確切的回應官員震怒掉頭離去各說各話羅生門各界傻眼痛批 高層透露指出其中緣由耐人尋味有待釐清靜待真相曝光後自有分曉顯示出公道自在人心 一場風波令人費解憂心忡忡不禁要問景氣差物價上漲股市大跌爭議中政府都在做些什麼 -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 61.229.26.134 ※ 編輯: mstar 來自: 61.229.26.134 (02/19 20:05)
