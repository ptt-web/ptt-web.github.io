標題[免費] MDIE 檔案總管的替代方案 (介紹)
時間Mon Jan 22 14:10:49 2007
MDIE 大概有幾個特色，我在這裡介紹一下 一、支援分頁功能 只要在 [檔案] -> [開啟新視窗] 這邊 就可以開一個新分頁出來，或者是，你也可以在資料夾上 按下滑鼠右鍵，選第一個項目 (新しいタブで開く) 這樣就會開啟另一個分頁 然後，你就可以在數個分頁之間拖了拖去，不用開到好幾個視窗 不過，也因為這樣 所以，MDIE 不允許重複開啟 如果習慣用好幾個視窗來管理的話，這地方可能會不太習慣 二、我的最愛功能 你可以常用的資料夾，加到我的最愛去 這樣就可以迅速跳到你的常用資料夾 只要在 [我的最愛] -> [添加我的最愛] 就可以把你目前分頁所在的資料夾，加到最愛去 至於要怎麼刪除最愛呢？ 只要點工具列的最愛圖示，或是 [檢視] -> [側邊列] -> [我的最愛] 就會出現我的最愛列，在你想刪除的地方，按下滑鼠右鍵，選刪除即可 PS: 也可以任意移動位置哦，只要壓著左鍵，把它拖過去就行了 另外，MDIE 的最愛，還有一個群組功能 在 [我的最愛] -> [我的最愛群組] 這個功能是用在，如果你有習慣一次開啟好幾個固定的資料夾時 比方說，我們在整理下載的東西時 檔案可能放在 桌面、D:\BT、E:\eMule、F:\FTP 那麼我們就把這四個資料夾，設成群組，叫 Download 以後，我們只要開啟 Download 群組，就會自動出現四個分頁 分別是 桌面、D:\BT、E:\eMule、F:\FTP (這個群組是可以管理的，也就是事後編輯、增加、刪除都沒問題) 三、滑鼠手勢 是的，你沒看錯，我翻譯的時候，看到這功能也是嚇一跳 在 [工具] -> [滑鼠手勢] 裡，你可以設定自己的手勢，並對應到選單上的項目 (PS: 快速鍵也是一樣) 預設了幾個手勢 (1) 壓著右鍵，往左畫 上一頁 (2) 壓著右鍵，往右畫 下一頁 (3) 壓著右鍵，往上畫 上一層 (4) 壓著右鍵，往下畫 重新整理 (5) 壓著右鍵，滾輪朝下滾 下一分頁 (6) 壓著右鍵，滾輪朝上滾 上一分頁 四、快捷列 (Launcher) 你可以把常用的工具加進來，比方說計算機、記事本之類的 五、支援 Script MDIE 支援 Windows Host Script 壓縮裡面也附了六個 Script，因為檔名是日文，在繁體中文下會解不開 所以，我也把檔名翻譯了，總共有六個 Backup of registry.js 備份 MDIE 在登錄中的設定 也就是說，MDIE 的設定檔是放在登錄裡的，沒辦法帶著走.... 不過，可以利用這個 script 備份下來，帶到別台去 還原，當然也是點個兩下就可以了 Clipboard Copy.js、Clipboard Copy.vbs 把檔名、資料夾名稱，複製到剪貼簿 New Text Document.js 開新檔 (示範意義大於實用意義吧.....) Open Active Tab With Explorer.js 用檔案總管開啟這個分頁 Shortcut Link.js 打開捷徑 (不過，好像也是示範意義大於實用意義....) 以上這幾個腳本(除 backup of registry.js) 都不能直接使用 MDIE 要使用這些腳本的時候，要先到 [工具] -> [擴充] 那邊 自己手動把腳本加進來 (也許有空的話，我用 inno 幫它寫個安裝檔) 文末，回答一下 chris的部份問題 利用 [工具] -> [MDIE 選項] -> [一般] -> [v] 儲存視窗位置 可以把每個資料夾各別的欄位寬度存下來 如果要套用到全部資料夾的話 就到 [列表] 頁，有個 [適用全部資料夾] 按下去就可以了 (這個頁面的翻譯，實在都翻得不好，有空再慢慢修正[email protected]@) 至於，按下 WinKey + E 的話，可能要去捉熱鍵管理的程式來用了 不然，就是自己寫程式來改..... 最後，希望大家用的開心 如果中文化哪邊有疑問的，歡迎提出，我再修正 或是有什麼心得的，也歡迎分享 也許有空的話，我會試著寫寫 MDIE 的教學文的 本著作依據 Creative Commons 姓名標示-非商業性-相同方式分享 授權條款為授權 http://creativecommons.org/licenses/by-nc-sa/2.0/tw/ -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 60.248.9.254
