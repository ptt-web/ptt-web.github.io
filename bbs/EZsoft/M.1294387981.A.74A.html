※ 引述《tn801534 (踢恩)》之銘言： : ※ [本文轉錄自 ask 看板 #1D9gq6rH ] : 作者: tn801534 (踢恩) 看板: ask : 標題: [請問] 右鍵重新命名 : 時間: Fri Jan 7 14:04:50 2011 : 請問右鍵->重新命名 : 可以設定這個功能的預設名字嗎 : 因為我常常需要備份檔案 : 還是說可以在右鍵建一個叫備份的功能? : 能做到類似的效果嗎 : 我知道右鍵選單在regedit改 : 可是不知道重新命名的命令怎麼下 : 如果有複製兼重新命名的指令更好~~~ : 有高手教學一下嗎~~~感恩... 解說一下做法，有其他需要可至 EzHotKey 板 1. 安裝AutoHotKey 官方網站 http://www.autohotkey.com/ unicode版本下載: http://www.autohotkey.net/~Lexikos/AutoHotkey_L/AutoHotkey_L_Install.exe 安裝過程可參考此網誌 http://jdev.tw/blog/734/autohotkey-introduction-chinese 2. 桌面空白處按右鍵 → 新增 → AutoHotKey Script 會出現一個副檔名為.ahk的檔案 3. 在此檔案上按右鍵 → Edit Script 此時出現記事本 4. 輸入你需要的指令，裡面的文字可以先全刪掉 --以下給你作參考-- ^!a:: send ^c^v sleep 500 send {F2}backup%A_MM%%A_DD%%A_Hour%%A_Min%{Enter} ------------------ 此時出現的新名字就會像這樣: backup01071617 : 以下為說明 ^!a:: : 代表按下 Ctrl+Alt+a 後會輸入下列指令 send ^c^v : 送出 Ctrl+c 及 Ctrl+v 指令 sleep 500 : 等待0.5秒，如果不等待直接送出下一串指令會發生命名到原檔的狀態 send {F2}backup%A_MM%%A_DD%%A_Hour%%A_Min%{Enter} : 依序送出 F2, 預設字串"backup", 目前日期時間的8碼數字, enter : %A_MM%就是月份，%A_DD%就是日期，依此類推 : enter鍵完成命名，如果還需要輸入其他字元就可以刪掉這個按鍵 5. 存檔，然後關閉記事本 6. 雙擊此檔案以啟動常駐，或加入開始功能表的「啟動」之中讓它開機後自動執行 7. 不使用時至右下角常駐工具列找到綠底白字的H圖示按右鍵→Exit -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 218.162.98.29 ※ 編輯: daviblase 來自: 218.162.98.29 (01/07 16:18)
