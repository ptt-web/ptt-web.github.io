2015-08-06 更新內容 2012年在PTT第10篇文章，沒想到還在Google板置底 所以回顧後更新一下，修正失效連結跟整理排版 在近年Google官方說明搜尋技巧， 其實有時間值得一讀，簡單明瞭，比我本篇內容扎實多了 http://www.google.com.tw/intl/zh-TW/insidesearch/tipstricks/all.html 補充電腦常用快捷鍵 F11 全螢幕 (mac除外，當驚慌失措不小心全螢幕試試) Ctrl + F 尋找 (跨作業系統和瀏覽器必學) Ctrl + Z 復原動作 (跨所有瀏覽器跟作業系統) Ctrl + Y 反復原動作 (跨瀏覽器跟作業系統，mac除外) Ctrl + shift + T 開啟剛關閉的分頁 (跨所有瀏覽器) Ctrl + W 關閉此分頁 (跨所有瀏覽器) Ctrl + L 游標跳到網址列並反白 (跨所有瀏覽器，用網址列搜尋必學) Ctrl + P 列印此頁面 (跨所有瀏覽器，也可用列印轉PDF) Ctrl + C 複製 (跨所有瀏覽器跟作業系統) Ctrl + V 貼上 (跨所有瀏覽器跟作業系統) (mac Ctrl 改為 Command) 1. 情報or新聞 連結： 由於科技報橘圖片已經失效，附上本篇新聞英文原文連結 http://mashable.com/2011/11/24/google-search-infographic/ 科技報橘:http://goo.gl/4Ey8F (上) http://goo.gl/D8CGP (下) 科技報橘新聞主要是翻譯六張圖片 2. 內文： 說明圖:http://goo.gl/BwDlr 江湖上有這麼一說：欲練神功，必先谷歌 而你是否每次在 Google 時都只加入空白鍵分隔關鍵字呢？ 最近伊利諾 Wesleyan 大學的研究發現 只有不到 25% 的學生擁有理想的搜尋技能, 而大部份的學生對於搜尋過程上都有相當程度的困難與阻礙。 相對於資訊同樣發達的台灣學生與研究生，可是相當大的警訊， 如何有效率的搜尋可是攸關報告和論文能否準時交差呀！ 透過以下的圖解搜尋小技巧，希望讓大家在尋找資訊時更為順利又迅速 第一招─小試身手 ※想搜尋紐約時報的文章？ Step1： 若只想搜尋 NYTIMES 網站的資料，輸入 site:nytimes.com Step2：關鍵字前面加上波浪符號，輸入 ~college Step3：將不可拆開的關鍵字用雙引號括起來，輸入”test scores” Step4：不想納入的關鍵字前面加上負號，輸入 -SATs Step5：時間區間用兩個句點間隔，輸入 2008..2010 ※想搜尋燕子飛行速度的專業報告？ Step1：若想搜尋 PDF 文件檔案，輸入 filetype:pdf Step2：只想搜尋的標題關鍵字：速度（velocity），輸入 intitle:velocity Step3：搜尋各式各樣的燕子（swallow），在燕子前加上*，輸入 *swallow 第二招─研究生必殺技 ※想搜尋 Dr. Ronald L. Green 和 Dr. Thomas P.Buttz 關於光合作用（photosynthesis）的論文？ Step1：作者關鍵字，輸入 author:green Step2：主題關鍵字，輸入 potosynthesis Step3：想讓搜尋結果更明確，在關鍵字旁加入雙引號， 在此加入作者的全名簡寫，輸入”tp buttz” 第三招─移行換單位 ※字詞定義 想找某個詞的定義，可以在關鍵字前輸入 define: 例如：define:angary ※數學計算 直接輸入數學運算式即可 例如：(2*3)/5+44-1 ※單位轉換 在想轉換的單位前加上 in 例如：54 pounds in kilograms，中文也可：54 磅 in 公斤 第四招─殺手鍵 在這裡 Apple 的使用者使用的 Command 鍵，在一般鍵盤上為 Ctrl 鍵： ※在頁面上做關鍵字搜尋 只要同時按下 Command 加上 F，再輸入關鍵字，關鍵字就會被重點標注 ※頁面上放大縮小 同時按下 Command 加上 + 或 -，即可放大或縮小頁面和 PDF 檔的文件 ※全選頁面網址 同時按下 Command 加上 L，就能夠全選網址列的網址 ※分頁循環 Command 加上 ` 鍵，可以看到所有分頁的預覽（須安裝瀏覽器外掛） Command 加上 tab 鍵 ，可以切換到瀏覽器下一個分頁 --------------------------------------------------------- 感謝emethxyz、pvn，內容有加上自己google到的補充 Mac: Command + ` 循環檢視開啟中的“Finder”視窗 切換應用程式 Mac:Command + tab Win:Alt + tab 瀏覽器切換分頁 Mac,Win : Ctrl + tab 感謝MG2000大大在disp同篇文章補充 Command + ` = 在（同一應用程式的）視窗間切換 Command + tab = 在應用程式間切換 (由於懶維護disp轉錄文章在disp同篇文章刪除了) --------------------------------------------------------- 第五招─萬宗歸一心法 ※善用你的圖書館資料庫 Google也許是十分好用的工具，但也別忘了許多重要的文獻 還是需要從專業的論文資料庫搜尋才找的到喔！ ※別引用維基百科 維基百科給我們相當大的方便做資料搜集，是一個很適合入門的地方，但裡面很多資料已 經是缺乏來源引證的，但好的維基條目還是可以提供我們一些引用資料的線索。 ※找參考文獻 在數位或傳統的研究上，從書、論文或報告上的參考文獻下手是不會錯的，從文獻追溯, 有機會找到更具重要性的資料。 即使 Google 如此方便，大家還是要動動腦、重新消化一番呀！可不要複製貼上大法，到 時老闆也用五招密技找到一樣的資料，被抓包可就得不償失了！ 3.心得: 我印象很深刻我很多同學上大學後還不知道 Ctrl + F 可以搜尋關鍵字 深深覺得很多很重要的使用方法還不普及，所以想藉次分享這些技巧給大家參考 雖然有些可能派不上用場，但學會了工具總比不會用好吧 ^_^ ps. 第一次在 Google 版發文 ，ptt 第10篇文章 所以文章上色方法不慎熟悉，若有錯誤請多見諒 m(_ _)m -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 220.141.161.214
