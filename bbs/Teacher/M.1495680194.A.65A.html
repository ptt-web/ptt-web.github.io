"校內"採用積分制，重點就在校內了，這積分是否通過校務會議，讓全部的人投票通過 這就具法律效力了，校長會要求人事寫在聘書裡，你接聘時(專任教師聘書)，你就要決 定接這聘書與否，不接就失業，接了你要嘛接受這機制(因為是你想要進到這校的)，要 不就是打行政救濟，主訴說明你實際損失的狀況!這也是為什麼我懇求大家多參與會議 多想想投下這神聖一票，決定的後果是什麼(所以教師法只規定低限擔任導師不得拒絕) : 那假如有個職位一直不願意接 : 最後抽籤決定 抽到的那個人能不能不接? : (你會說抽到才反悔不算 那我可以一開始表明我不願意抽 也不願意接嗎) : 用個極端的例子好了 : 假如一個學校"所有"的老師 通通都發表不接行政聲明 : 那學校的主任組長怎麼辦? 出納午秘環教資訊怎麼辦? 既然大家都不想接，怎會有主任組長?這些協行大家也都不接啊? 找不到人，到底是不接的老師有問題，還是想要人接的校長有問題?這看來很清楚明確啊 如同前面板友說的，好的校長領導統御，會拿學校資源作談判籌碼，只要是合理合法的 資源分配，延攬好人才為學校賣命，我個人是可以認同這樣模式的! : 校長能不能自己排? 或者校長能不能決定大家抽籤? : 所以說校長有沒有行政裁量權 : 把願意去這個位置的人 不讓他去? : 或者把不願意去那個位置的人 硬逼他去 : 我想是相當程度是有的 : 但實務能不能只靠這招? : 當然不行 你有張良計 我有過牆梯 : 硬把我拉去那個位置是你的權利 : 但是給出多少產值 是我自己決定的 : 你硬推一個老師到一個位置上 : 陽奉陰違 各種評鑑勉強過關 是很給你面子了 : 如果他各種擺爛不做事 你也拿他沒皮條 : 所以校長選人的時候 領導的藝術就很重要 : 不管是動之以情 威之以勢 誘之以利 : 至少你要讓他不排斥才會把它擺到這個位置 : 不然只是拿石頭砸自己的腳 : 至於硬幹行不行 : 個人覺得可以 我個人覺得硬幹(要老師接行政)是行不通的，至少我看到的案例很清楚了， 教育局公務員拿不出法條究責教師(近幾年有縣市教育局自創法條限制教師不在此討論) 只能要求校長，而這位校長自恃站得住腳，錯判局勢誤以為長官會挺，所以下場就是 回任教師，而臉皮薄回不去就自請退休了... 結論:教育局公務員只管學校新學期有沒有承辦行政"人名"，讓他核銷經費，誰管你在 這位置的人是野心勃勃想學更多東西，還是夜夜哭泣的不願意，接了聘書，就表 你自己願意在這位置了，否則你幹嘛接?(有人用槍指著你嗎?)什麼?你說校長同事 說了很多讓你擔心受怕的話或行動讓你不得不接?證據呢?我還被公務員嗆過:"覺 得不合理，你可以去提訴願啊"，和公務員交手略過不提... : 縣府對校長的控制力比校長對老師的控制力高多了......... : 經過考量後 大部分理性的人不會選擇硬幹 所以可能我比較得天寵啦，我遇到過很多很多不理性的人^ ^，也被該主任說 我這年輕人可憐啊，但建議我轉個念，在當老師初期就遇到別人可能直到退休都不會 遇到的狀況，相信這條路，我會越走越順的...至於和校長擋假、校長人事合擋我介聘 、和教育局公務員交手、自提行政救濟、調查站喝咖啡，又是其他支線故事了(遠目...) -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 120.116.123.12 ※ 文章網址: https://www.ptt.cc/bbs/Teacher/M.1495680194.A.65A.html ※ 編輯: wasiau (120.116.123.12), 05/25/2017 10:49:44
