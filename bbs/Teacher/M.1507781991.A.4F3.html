※ 引述《Qunnie (Qunnie)》之銘言： : 各位老師大家好， : 我是第一次發言， : 若發言格式有問題再麻煩告知！ : 最近剛接手四年級課後班（原本老師受不了班級離職）， : 我本身並沒有修過教程也沒有教師證， : 是諮商輔導的背景， : 基本上是第一次帶班， : 班上約有近20名四年級學生， : 秩序非常亂， : 有些學生講話沒大沒小直接回嗆（男女都會）， : 甚至不尊重我， : 認為我只是課後班老師， : 本來希望自己不要發脾氣， : 應該要從輔導角度出發， : 只要同理學生、告訴學生彼此的感受應該就可以了， : 但完全沒有用， : 他們不僅變本加厲， : 甚至有的學生認為我不夠兇， : 所以我還是會大聲制止或是吼學生， : 但每次一吼就覺得自己好錯亂， : 也覺得違背了我原本的初衷，， ㄧ問你的初衷是什麼呢? 二問你認為課後班老師該有怎樣的樣貌? 三問你自己1和2答案是否有衝突? 釐清自己的觀念，才能夠採相對應的策略手段 輔導、管教都只是策略手段 你想要ㄧ個怎樣的班級，這才是採取這些策略手段的核心價值 沒大沒小回嗆已是學生間相處的常態，差別在於當有外人在的時候是否仍維持這樣模式 沒錯，老師就只是個外人，因為一天幾乎8個小時無時無刻相處的，才是學生自己圈內人 圈內人就會互相尊重嗎?想也知道不可能，但那不是你現今緊迫要處理的。 較輕微:你的學生把你當圈內人，所以沒大沒小回嗆你只在於他認為你沒有威脅性， 你跟同學ㄧ樣，有什麼糾紛我們就圈內人自己解決，沒大沒小回嗆不尊重只是"剛好" 較嚴重:你的學生把你當圈外人，但你拿出的對應態度讓學生衡量過後，還是覺得對你 沒大沒小回嗆不尊重你"沒關係" 不論哪一種，都不是你要現在處理的，先提出來是希望你往後孤單寂寞有點冷時回想 可以怎麼作就好。不論上述哪一種，你要作的是建立起屬於自己的場域。 我接新班，最基本最基本就是臭臉ㄧ個月，讓學生摸不著我的個性，這ㄧ個月可以好好 常規建立。臉臭到學生就像債務人還完錢了，還會懷疑自己是不是沒還清利息。 這ㄧ個月裡學生作對事，臭臉到讓學生覺得這本來就該是理所當然要做好，作錯事的話 不用多說，飆下去就對了，利用這ㄧ個月讓學生衡量，寧願要家長來幫他辦轉學， 也絕對不要惹到我。(我都還沒發脾氣咧) ㄧ個月過去，你想要學生搓圓捏扁，學生都會接受，因為他已經知道你 是ㄧ個怎樣的"老師"，也找到跟你相處的態度了。 : 而且這也不是有效的帶班方式， : 因此想詢問板上有經驗的老師， : 有沒有一些帶班的方法可以傳授， ㄧ點拙見，你可以開始借錢給學生了，如果你看到欠錢不還的人還可以笑臉迎人 我只能說，別人的帶班方式永遠都不會是你的。 輔導須長期、管教很立即，但影響學生長長久久帶得走，卻是態度。 管教、輔導本來就是相輔相成，ㄧ昧地壓迫只會養成奴性或是暗地偷雞摸狗的態度 ㄧ昧地安撫只會養成欺瞞或是明著反對你的態度，有的孩子在家看到的家長都不尊重 人了，你要怎麼寄望還沒教育前的學生懂得尊重、同理、感恩? : 不然真的好崩潰！！！ : 以上，謝謝老師們！ 想要學生成長，也得要讓他們先懂得吸收啊，對吧? -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 120.116.123.12 ※ 文章網址: https://www.ptt.cc/bbs/Teacher/M.1507781991.A.4F3.html
