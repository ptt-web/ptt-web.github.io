※ 引述《tabehoudai (吃到飽)》之銘言： : 我們班學生今天疑似被阿飄附身了……… : 1.叫他不理 : 2.差點要從樓上跳下去(這真的超可怕) : 3.拿剪刀背割手 : 4.在課本上寫“我要殺你朋友”（那字體不像他平時寫的） : 5.超兇狠的瞪我= =(我只是狂念鬼把"殺"寫成"剎"寫錯字而已= =) : 6.出現哀號跟抽搐的狀況…… : 他醒來後就開始跟我描述意識清楚前他看到了些什麼 : 例如看到 : 有鬼被鬼推下樓(兩位鬼是在表演嗎?)、 : 鬼要進教室抓學生、 : 女鬼好幾隻(這性別比例好失衡) : 說的時候還說他現在看到那些鬼正在對面看我們笑= = : 後來學生還說 : 其中有個鬼很像在器材室裡有張陳年黑白照片中的女人 : 因為他們很害怕我又想查清楚還跑去翻…… : 結果一翻照片超幽默der竟然是首任校長= = : 剛怕學生出事已經跑去廟裡幫他們拜拜 : 我其實有點懷疑學生是否誤觸毒品或是有精神疾病 : 但這很難開口去跟家長講....... : 不知道有沒有老師遇過類似狀況 : 可以分享一下的？ : 謝謝~ 我遇過學生在校一直三太子附身的~ 因為症狀的明顯程度和壓力情況有顯著的關聯 所以一開始我以為是故意的 經過幾次觀察 發現即使沒有真的附身 大概也會是心因性 附身的時候 也是口吐白沫 全身抽搐 附身後寫字 說話方式 表情和個性都會明顯不同 老實說我根本不信 但以往的輔導經驗告訴我 不要有預設立場 所謂的沒有預設立場指的並不像是那種口中說說 我會相信他感受到的 但事實上表現出來的反應根本不是那麼一回事 (嘴巴說相信 行為上直接當病人) 而是真的相信他所見 所感 去跟他討論他到底感受到了什麼 我在跟他談話的過程中 常常被三太子中斷 所以我後來選擇在他沒有附身的時候 請教他要如何和三太子溝通 他教我怎麼踩七星步 怎麼拜請 用什麼態度什麼語句 後來幾次談話我很認真的跟三太子溝通 如果在學校時上身 會給學生帶來怎樣的困擾和問題等 當然一開始免不了被神明斥責 但慢慢地有辦法對談 說也奇怪 學生在校附身的情況就漸漸減少了 或許有人會問 我有宗教信仰嗎? 其實沒有 我自己本身是無神論者 即使到現在 我都還是懷疑學生是不是自導自演 但這不是重點 而是 他想說的話 想表達的事情 有沒有被聽進去 我們念教育的時候 都學過防衛機轉 學生很可能透過(不論真假 不論是否有自我意識到) 這些方式在表達某些感受 我也遇過真的思覺失調的患者(也就是已被汙名化的精神分裂症) 即使他們的邏輯.語句都充滿了混亂 但還是能從他們的話中聽出他們在意的 不願面對的事情 看到你的文章 恕我直白一點 字裡行間 其實給我一種你不太在意學生的所感所見的感覺 如: 兩位鬼是在表演嗎? 這性別比例好失衡) 那些鬼正在對面看我們笑= = 一翻照片超幽默der竟然是首任校長= = 等等 當然 或許那可能是因為你想增加文章的趣味性 我指的在乎 並不是相信他說的那些景象 而是 你有沒有在乎學生真正想表達的 換個方式 你可以讓學生感受到你的關心 如: 他被附身的感受是什麼?? (一般來說 若真的看到像你學生說的景象 應該是蠻可怕的 那他的感受是?) 鬼被鬼抓下樓是發生什麼事了嗎? 你怎麼會覺得鬼是在笑? 當他寫下我要殺你朋友的感受是? 當被附身差點跳樓時 自己是否有感覺 我自己也遇過多起 其實他就是透過這種方式在表達自己內心的想法 例如 長期被霸凌 怒氣累積而不敢宣洩(敢怒不敢言) 缺乏成就感 找不到生活目標與動機(無意義無希望感) 有時候光是認真地和他聊一聊他看到了什麼 感覺到了什麼 症狀就會減輕了 最後我想說的是 如果直接送去鑑定是否有精神疾病 不管鑑定出來是有還是沒有 他的問題終究還是得處理 就算真的是精神疾病(非生理造成 如腦病變.吸毒等) 那一定有壓力源 就算吃了藥 但他的壓力源仍在(如霸凌.家暴等) 正向症狀(如 幻覺.被害妄想等)或混亂症狀(沒有邏輯的表達或行為)獲得緩解 那也只是徒然增加負向症狀(沮喪.失去自我價值.悲傷等)而已 更甚者 原來只是替代性的防衛機制在作用 結果因為眾人的處理方式 就讓他的附身自驗預言了 -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 163.22.79.30 ※ 文章網址: https://www.ptt.cc/bbs/Teacher/M.1430356432.A.954.html ※ 編輯: smallfinger (163.22.79.30), 04/30/2015 09:17:14
