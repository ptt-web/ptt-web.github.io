※ 引述《winered ()》之銘言： : 一個月前 我對醫生說：我覺得應該不用吃藥了 : 帶著笑容和醫生的討論 醫生的祝福 對自己的信心 : 現在回想起來格外諷刺..... : 剛從二年級接起時 我不知天高地厚地 投入了過份的熱情 ^^^^^^^^^^^^^^^^ 年輕時容易理想化容易因現實趨向和期待不符合,因對自我的要求,產生了抑鬱... : 把所有希望他們的好全都向同學傾倒下去 看起來剛接時可能有一番改革,推測是由此而此風波, 其實新帶後母班,如果原來的學生狀況不是太差的話, 其實先讓學生照原來的作息及規矩走, 自己先和學生培養感情,並且照原來的規定稍作修改,也許會更理想喔.... 學生也是人,人性會有"習慣性",即便新的規定不見得不好, 但也許會因為習慣性而打從心裡排斥.. 前陣子剛和一位本校十分會帶班的資深老師偶然閒聊到,她之前也是第1次帶後母班, 所以雖然明知道班級經營和前位老師不同,但是還是大部分維持了原樣, 直到九年級才全面改由照他的規矩,理由是"九年級了, 目標要轉向學測,要有新的作息及規矩" 班上就沒什麼反抗聲浪了, 那班也是和原po的班上有點類似的,聰明,但會有人帶頭的那種.... : 我付出的時間與心力 雖然不敢說很完美 但都很真心 : 大小事積極管理 聯絡簿賞罰兼具 : 想學習校內嚴格的名師們 在常規 秩序 學生愛戴方面一手包辦 : 雖然一定會經過一番反感 但當時天真地想 : 總有一天 你會感謝我....... 欲速則不達,其實我也覺得校內嚴格的名師們,帶班真的蠻有一套的, 只是真的從一年級開始比較理想,學生從一年級和從二年級帶,有差喔... : 然而我真是好傻好天真 : 原來一年級時人人誇獎的班級 其實有1/3的以上的向下拉力 : 這個班很聰明 不是隨便拐得倒的 但他們的聰明卻是於不良的影響 : 犯錯被處罰後 但根本不想改 而是在老師面前敷衍 ^^^^^^^^^^^^^^^^^^^^^^ 不管是多乖的班,很多聰明的學生都有這種個性說... 反向思考是,至少有敷衍比連敷衍都不想好多囉.. : 事後抓住老師處理時枝微末節的小辮子 相互洗腦全是老師的錯 通常代表其中有學生對老師起了排斥心理,在心理層面上,基於人性, 當你打從心裡排斥一個人時,不管他做了什麼,都有理由編派他的不是.... 所以一旦排斥的人多了,就會變成"相互洗腦".. 所以建議先收心,抓住一些學生的心向著妳,這些學生如果是女學生們會更好.. 女學生在班上,有著潛在的影響力.... : (ex 有人對主任態度不好 被主任輕戳了肩膀幾下 一群人鬧著主任推人) : 於是這股勢力開始串聯 人多壯膽 互相包庇 : 漸漸從整潔秩序的崩盤上浮現到與各科老師的相處氣氛 通常,整潔可以用卯起來盯解決,秩序亦同.... 我任教某個科任班,因為班上常讓老師覺得吵,所以導師上課前都會先出現整頓秩序... 其他的慢慢來.... 我今年還有個很深的體認是,班上的讀書風氣影響其廣,如果從這邊下去和家長溝通, 也許會更有力,這也是資深老師教的... : 才教第三年的我 經驗 威嚴 氣勢 勇氣 全都不足 : 管了沒效 罰了不怕 只好靠寫聯絡簿期待家長一同約束 管和沒管一定有差別,原po不要小看自己的力量,只是要學習面對及轉化自己的負面情緒, 不要讓自己受學生影響,進而影響了你看學生的想法及自己的心情.. 罰了不怕通常也只是學生嘴硬或硬撐.. 我今年科任有一班學生常有教學情境題出現, 例如:3-4位男生很喜歡主動說要罰站,趁機作亂... 擺明是沒事可做,打算和老師鬧著玩,所以我想說,要站就站吧,但是手貼住大腿, 沒貼住就登記準備交罰抄,大概半節課,就一個接一個被登記後,自己主動坐下了.. (心裡OS:吼,無聊啊~我巴不得坐下,你們還想站,透逗啦,就陪我站吧!) 這班狀況還有很多,都是些無聊的事...如果是以前剛教書的我,一定是氣炸大吼, 然後對槓,最後對立.... 現在則是有狀況,一項項處理,然後還是上我的課,保持好心情.. 例如:有時他們又很會起鬨,一個問問題,變成全班一團亂,停不下來,這時候1,2,3很好用, 喊完就只好宣布"戒嚴" 戒嚴時期:不準問問題,不準說話,不準大笑,不可比手畫腳,全班只有老師可以說話... 學生說:老師我們有言論自由 ... 師:誰叫你們太吵了,戒嚴時哪來的言論自由,上課時老師最大... : 結果當然換得那群學生對我的怨恨 以及家長的麻痺以致真正重要事也嚇阻不了 : 即使他們大膽到敢公然對台上的老師挑戰嗆聲 : 即使抽菸 慣竊 言語暴力 集體作弊這些都找過學務處處理過了 : (處理也無法讓他們心服 背後說我是跟主任有一腿 一起整他們) 和家長通話不能太常,也需要一點技巧,讓學生感受到威脅性,讓家長覺得有施力點, 偶爾煩學生1,2次有效,太常就會失效.... 偶打給家長最大的用意是讓學生知道導師的底限在哪... 如果不是太常打電話,學生會因為不想被父母念而有收斂, 從學生會怨恨就知道,學生其實不想老師打電話的... 另外有時會覺得,學務處已經和以往不同了,功效有限, 大部分的學生問題,建議老師自己處理下來會更好, 除非該校有強而有力又能讓學生信服的學務處, 抽菸及慣竊可以找學務處,言語暴力及集體作弊應該可以想辦法處理... : 也無法拆散他們那股自以為人多勢眾的現象 : 而我 已經是個不折不扣的紙老虎了 : 並不是第一年教書 面對班上只有少數這種人的現象自然不擔心 : 只是沒想到一個班這種分子居然如此多 : 若全班皆如此我倒還狠得下心放棄 : 偏偏乖巧的分子又好得讓人心疼(私下寫信為我打氣 但不敢面對那群人) : 讓我加倍自責：為什麼不能保護他們的權益? 乖巧的小孩,也會自己找到生存之道的,一個很好的班固然好, 在傾向混亂的班裡,也等於是逆境,怎樣在逆境中學習,老師和乖小孩都可以當成一門學問, 所以我覺得也不用太自責,畢竟妳是有努力的,只是暫時情況不如預期 : 對其他老師的愧疚 對好學生的保護責任 對來自家長的期待 對自己的質疑..... : 種種壓力讓我從四月開始逐漸崩潰 : 人後大哭 失眠 惡夢 人前連發飆的力氣都沒有 : 而我的焦慮他們一定也感受到了 更是肆無忌憚地嗆聲 他們知道他們贏了 : (有天請病假 接到十幾通暱名騷擾電話 全都是下課時間打的....) : 之前最嚴重時是五月 那陣子每天開車都在注意兩台並行的大卡車 : 幻想著開到中間緊急剎車.....(真是笑話 連面對學生的勇氣都沒有 何況是死亡?) : 利用暑假期間當成療癒期 一邊看醫生吃藥 一邊計畫如何改善之前的帶法 : 許多症狀都改善了 讓我在開學之前儲備了不少能量 : 學期初 可能是之前的衝突經過暑假冷靜了下來 還算順利 : 只是沒想到能量這麼快就消耗光 經過幾次試探與衝突 : 即使我再怎麼跟全班說明我的堅持與不退讓 堅持保護全班的學習權益 : 忍受他們在班上公開的質疑 一件件用強硬的語氣說明回去 獲得冷潮熱諷的回應.... ^^^^^^^^^^^^^^^^^^^^ 老師堅持不退讓也是對的,如果理由是讓班上一起變好會更好, 其實班上已經有一團學生連成一氣時,用語上就要比較謹慎, 避免讓學生挑語病 認為"老師又暗示我們是老鼠屎,影響她眼中那群乖學生了"的這種心理發酵 另外一個是要避免"冷嘲熱諷這件事",一定要想方法全面禁止這種不尊重導師的行為, 即使在背後,在家裡罵都沒關係,但是在老師前就是不能,不然不管老師宣布什麼都會無效. : 改不了教不動的事實還是不斷刺著我 事實最壞就已經如此,頂多就是降子而已, 而一個班,有導師和沒導師真的是有差的,不管這導師管不管得動 有導師在就是有個力量在的,要相信自己,即使只有一點點小小的影響力, 但至少可以讓自己心想努力的學生,記下導師曾有的努力.... 而不是記下老師曾經承受不住壓力而崩潰這件事.... 即使妳很努力了還是老樣子,還是混亂到畢業,但中間你所承受的一切, 這些學生還是會記得的,我想那就夠了...所以要以這樣的信念對自己打氣 : 我好想保護值得我心疼的孩子 可是到最後或許他們會對我失望 : 因為當初說得信誓旦旦的我 實際上什麼都不能給 : 即使還有八個月就畢業 我想我撐不下去 : 暑假時情緒的改善 跟本不是醫藥的效果 也不是我自己的進步 : 只是當時不用面對夢魘而已.......... : 開始找人傾訴的我 大部分得到的答案都是 這一關過了 以後就風雨無阻 : 但我反而會想成：這一關都過不了 未來三十年都完蛋了 : (憂鬱症就是這樣 好與壞 該與不該 自己都知道但跳不出來) : 可是若每天踏入這個充滿敵意的環境 我沒有一顆堅強的心去成長 遇到很難帶的班,的確很難熬,有渡日如年的感覺,後來我想,轉換個心境, 即使在當下會不斷有問題發生,也只是在當下,時間總會降過去的, 不管發生什麼事,就想辦法處理,發生一件事就處理一件,發生二件事就處理二件, 一天處理很多件時,就笑著對自己說:"快變超人了" 但是不要打從心裡為著將來可能更嚴重而害怕,憂慮, 甚至形成了更大的恐慌及壓力.... 時間不會倒退,是會過去的,頂多就上班上課時要處理狀況,但總會處理完的, : 我好想衝去校長室求著換導師 : 我想保證我不想打混只是想換環境休息、調整、學習、成長 : 讓我學出心得後 在新的環境一定會不會辜負大家 : 但真的這樣的話只會讓我在學校黑掉吧.... 這樣的話,不是黑掉的問題,是一關沒過,沒調適好,下一關如果又不行呢,你會徹底崩盤的 可是實際上隨著年資,一年年當導師,妳真的會變強的...所以就讓時間一天天過 你把每天都當做在練功打怪吧...反正怎麼打也不會掛,頂多就勞心勞力, 但是不用把小孩的話當真,不成熟的人說的話,聽聽就算,不要被學生影響心情啊~ : 有人安慰我 不要覺得他們變壞是因為我 也不要覺得他們變好是我的功勞 ^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^ 沒錯,大部分學生變壞,變好 還是和家庭教育及本性有關 你要學習的是怎樣在一年年中,熟悉各種學生會出的"教學情境題",然後你會發現 其實教到5年,10年後,發現每年的學生問題,還真是換個形式而已,本質都差不多... 然後就會發現,這我以前處理過,只要如何,就有用,或是,以前如何不行,換這樣好了, 或是:這我聽某老師提過,如果如何處理好像不錯.... 很像在帶導師時,收集各種情境變成自己的心得,將來的情況是,手上有各種方法在箱子裡, 遇到了就會恰好的拿對東西處理掉... : 放輕鬆 神經大條一點 : 有人鼓勵我 導師的角色很重要 這個班再這樣下去會崩壞 : 我沒有選擇餘地 只能繼續戰鬥 : 有人責備我 跟本是我自己的問題 是我自己要鑽牛角尖把自己弄成這副德性 : 值得嗎? 乾脆不要管了! : 哪邊都有道理 但我連做出選擇的智慧都消失了 : 當老師當成這樣 我也是大草莓教一群小草莓吧(笑) : 一點睡著 四點醒來 等一下是慣性的每天早上拉肚子 : 這篇毫無建設性的文章PO在這 也就不怕被笑或丟臉了 : 或許對大家沒有幫助 也可能沒辦法從大家身上得到幫助 : 但 就當作我真的很想抒發吧....... 有的問題,如果自己暫時沒能力,就多請教別的老師,再修改成適合自己的方法去嘗試 但不要先信心滿滿想說,降一定會成功,只是想著,要試看看有沒用,沒用再換別的 如果自己已有方法又處理得不錯的,就繼續維持原來的模式.... 至於導師一職,已經是該班導師的妳,即使班上再吵再亂,要相信 有導師的班,比沒導師的班好多了....不要忽略自己的價值,多加油....
