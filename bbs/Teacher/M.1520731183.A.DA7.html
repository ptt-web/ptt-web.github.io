不好意思，難聽的話寫在前面。 如果不懂制度如何運作，請不要貼 文害人。系統改成多角調先跑後， 每年都有人誤信只要找到對象，彼 此只填對方一個志願，就可以100% 成功。結果公布結果時，找到的對 象變成同事。原因就是只要有同科 同事要調，不管他要調那個縣市， 都很有可能讓自己的多角調破局。 以原po的三角調為例。他有同科 同事也要調，所以稱他是A1，他 同事是A2。A1找到的三角調是 A1-->B--->C-->A1。三人都只填一個 志願。A2沒找對象，多填志願亂槍 打鳥。結果在他某一個志願裡出現 A2-->D-->E-->F-->G-->C-->A2 的六角調組合。因為六角調先執行 ，所以C在六角調階段，就與A2先 調成功。等到系統執行三角調時， C就不能再參與A1的三角調。結果 A1，B都調不成。A1與C在新的學 年當同事。那為什麼會這樣? 因為填志願時，是填校名，不是 人名。你沒辦法指定某人參與你的 調動。系統只會針對你填的學校， 搜尋該校所有參與調動的該科老師 ，從這些老師的志願中，再找下間 學校，直到形成迴路。而越多角的 迴路越先被執行。所以自己找的三 角調打不過同事亂槍打鳥的六角調 。除非同科同事也找到自己的多角 調，也只填一個志願。這樣就可以 形成不互相干擾的迴路。大家都能 調的成。 ※ 引述《carpfish (帳號已死)》之銘言： : 如果同校沒有老師要跟你填同一間目標學校, : 三個人又都是唯一志願 (只填一個, 不要填其他的) : 基本上, : 100% 成功 : ※ 引述《rt324 (())》之銘言： : : 這問題我已爬文過 但是我又聽到不同的說法 所以還是想上來請教一下前輩 : : 倘若 我是A縣市 已找好了 B縣市某校、C縣市某校的 三角調 : : 在彼此誠信的原則之下 假設我們三人都願意填唯一且第一志願 : : 但得知我的學校內 也有人想填縣外介聘(可能有兩位老師) : : 但是 他們想去的地區不是B也不是C : : 請問我們ABC三個老師 仍可確保成為一封閉系統嗎? : : 我爬文的結果是 : : 要百分百成功 須滿足三條件: : : 1.互填唯一志願 : : 2.同科無人介聘→目前就是卡死在這裡QQ : : 3.學校不開單調缺 : : 因為非常希望今年可以回雙北 : : 若依舊能確保成功 我很願意只填唯一志願 : : 但是又擔心有風險 畢竟如果這個志願失敗 就完全沒機會了...... : : 我去年剛結婚 但年資尚淺 積分不足100 所以勢必只能由高分的老師來推動多角調 : : 不知道我的情況 是仍舊填唯一志願賭賭看(當然也還有另外兩位老師的變動可能@@) : : 或者是將說好的學校填第一志願 而後繼續多填其他的志願呢? : : 當中若有我未考慮到的因素 也請前輩多指點 : : 再煩請大家不吝指教 也預祝大家都能介聘順利 謝謝大家 -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 1.34.241.211 ※ 文章網址: https://www.ptt.cc/bbs/Teacher/M.1520731183.A.DA7.html
