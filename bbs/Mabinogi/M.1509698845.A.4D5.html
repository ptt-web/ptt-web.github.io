3.回音石瞄準速度，不影響瞄準速度 4.攻速套有影響瞄準速度 根據貓窩說攻速套增加瞄準速度15%~25%，那麼 4.95秒 / (200% + 15%) = 2.3秒 4.95秒 / (200% + 25%) = 2.2秒 4.95秒 / [200% (1 + 15%)] = 2.15秒 4.95秒 / [200% (1 + 25%)] = 1.98秒 我這邊測試出來是2秒，所以我覺得攻速套，應該是對瞄準速度做乘法的加乘 難怪人類那麼愛用攻速套，旋風箭300%瞄準速度，加上攻速套就375%瞄準速度 這就差不多和開了視線的穿心箭差不多快了 5.無聊測個弩，結果嚇到吃手手，讓我聯想到貓窩寫的 『弩(g16s2的加成，同射程下弩比短弓快，但射程比不過長弓)』 原來更改的是瞄準最大距離的時間阿~ 6.根據貓窩，弓箭在200%瞄準速度下零距離瞄準需要0.225秒 巴弓(射程2000)，距離1800： 理論瞄準時間 = 0.225秒 + (2.475秒 - 0.225秒) × 1800 / 2000 = 2.25秒 巴弓(射程3400)，距離1800： 理論瞄準時間 = 0.225秒 + (2.475秒 - 0.225秒) × 1800 / 3400 = 1.41秒 測試值皆符合理論值 7.我測出來，弩在200%瞄準速度下最大距離瞄準需要1.95秒， 猜測，弩在200%瞄準速度下零距離瞄準需要 1.95秒 / 11 = 0.177秒， 弩(射程3200)，距離1800： 理論瞄準時間 = 0.177秒 + (1.95秒 - 0.177秒) × 1800 / 3200 = 1.17秒 測試值與理論值接近 也有可能，弩在200%瞄準速度下零距離瞄準需要的時間和弓一樣0.225秒， 弩(射程3200)，距離1800： 理論瞄準時間 = 0.225秒 + (1.95秒 - 0.225秒) × 1800 / 3200 =1.19秒 測試值與理論值接近 8.相同細緻射程和距離下，皇家弩的精遠瞄準速度比巴弓快 9.細緻精遠瞄準20等的頭，情報說+60%瞄準速度，實際上測出來，時間是1.9秒 4.95秒 / (200% + 60%) = 1.90秒 4.95秒 / [200% (1+ 60%)] = 1.54秒 所以細緻精遠瞄準加速，60%是直接加上去的 10.攻速套 + 細緻精遠瞄準20等的頭，實際上測出來時間是1.6秒 4.95秒 / [ 200% ( 1 + 25%) + 60% ] = 1.59秒 4.95秒 / [ (200% + 60%) ×( 1 + 25%)] = 1.52秒 所以疊加上，先計算攻速套，再計算細緻 也就是說， 精遠的瞄準速度 = 技能瞄準速度( 1 + 攻速套25%) + 細緻精遠瞄準等級 × 3% 以上僅供參考，錄的影片FPS只有20，誤差難免 然後穿心呢? 能加速嗎? 弩瞄準比較快嗎? 把上述的東西用穿心又測了一次，答案是... 弩和弓穿心瞄準速度相同， 攻速套、活潑版、回音石瞄準、細緻精遠瞄準，都不影響穿心 -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 220.134.1.81 ※ 文章網址:
