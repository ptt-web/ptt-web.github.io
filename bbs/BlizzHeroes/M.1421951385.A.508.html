標題[心得] 縫合怪超詳盡心得之噁心霸主
時間Fri Jan 23 02:29:41 2015
縫合怪一直是我很喜歡的角色 因為他的鉤子真的是非常的討人厭 如果可以把鉤子玩的夠好的話 可以在拿地圖資源的時候佔盡優勢 更可以在順風的時候把雪球越滾越大 這也就是為什麼這隻角色的勝率可以維持在高檔 提供參考 人權MMR : http://imgur.com/f7oef3A 從A測開始玩 打對戰也有遇過幾個知名實況主 像是StillSleepy 以下正文開始 1. 定位.技能詳解 2. 1級會戰 3. 天賦 4. 中後期團戰 1. 定位.技能詳解 胖子 ( 重要!!!!! ) 這隻角色是一個開戰型+團戰型的坦克角 但是他的開戰卻非常安全 因為他不像 桑雅或是穆拉丁一樣直接進場 直接進場很容易被往後拉打+控場 造成被風箏和耗血的困擾 而用鉤子開戰 如果是勾到軟皮 很容易就讓交掉位移技 在這樣的會戰就能取得大量優勢 ------------------------------------------- 肉鉤(Q)Hook 指向技能 消耗法力 75 冷卻 14 秒 將第一個擊中的敵人拉到自己身邊，造成48傷害。 ------------------------------------------- 這個技能絕對是縫合怪的精華之一 使用的技巧非常多 另外他帶有一點點小暈眩的效果 ( 可以中斷某些技能 ) 我通常在Gank人的時候 會把這招扣住 ( 我會特別關照對面沒有位移的軟皮去Gank ) 當我和線上把對面血量壓下來了以後 他想往砲塔走的時候 我們在這時候出鉤 就非常容易命中 因為我們的範圍就只有兩座砲塔中間的門 另外對於有位移技的角色 我傾向等他用掉位移在鉤他 我寧願讓隊友用掉他的控制技能去逼掉對面位移 因為我們縫合怪的鉤子是非常安全又強大的 他也可以勾對面正在吃的傭兵 讓對面沒辦法很順利的吃下這坨傭兵 可以達到干擾對面 或是拖延時間等隊友支援的戰術 ---------------------------------- 猛敲(W)Slam 扇形 消耗法力 55 冷卻 8 秒 對扇形範圍內的敵人造成61點傷害。 ---------------------------------- 這招如果單看技能的話其實非常的廢 耗魔略高 但傷害很低 但是他有99.99%的機率一定會搭配16級的天賦"粉碎(傷害提高50%、暈眩0.5秒)" 以下我的說明都包含此天賦 這也是讓縫合怪這麼好用的精華之二 他的範圍以及距離還不錯 冷卻8秒 在暴雪英霸眾多技能裡面算是滿短的 也就是說 在會戰我們可以一直有一個小小的控制 但是他可以帶來不少的效益 像是暈住敵人增加鉤子準度、追擊敵人、會戰結束撤退、保後排 但最最最重要的一項功能 就是他可以斷法 泰科斯的機槍、維拉的大絕轉轉轉和巫醫的引導法術等等 ( 你可以想像你的角色的重要技能被肥肥秒斷 在你還沒回過神來前，你又被他勾走了那種傷痛... ) 所以如果可以利用這個技能中斷對面一些關鍵的技能 我們可以很容易贏的每一個大大小小的會戰 --------------------------------------- 吞噬(E)Devour 消耗法力 75 冷卻 30 秒 對小兵造成320傷害，或對英雄造成60傷害。 恢復縫合怪相當於最大生命值20%生命。 --------------------------------------- 這個技能沒什麼特別的 他不能對建築物施放 天賦也不會特別去強化他 唯一比較重要的是 他的冷卻時間偏長 因此在每次準備要拿地圖資源或打會戰的時候 確保這個技能沒有在冷卻中就可以了 小技巧：快要死的時候身邊沒有東西可以啃 可以勾小兵或是傭兵過來啃一口 ＊＊＊＊＊　Ｒ 技能我們在天賦的部分再解釋 ＊＊＊＊＊ ---------------------------------------- 污穢臭氣Vile Gas (被動) 受到傷害時，會散佈一陣持續4秒的汙穢之氣， 任何接觸到的敵人會受到64傷害持續3秒。 ----------------------------------------- 怎麼說呢 這個技能不算很強 但也不算太差 他就是一個很煩的技能 在對線時期 很廢 他要挨打才能觸發 觸發了還必須碰到這坨毒氣才有效果 但是到了會戰 他的效果就出來了 因為毒氣如果毒到軟皮的話還算是滿痛的 而且會戰也很容易毒到3個人以上 (後面做解釋) 有時候對面追擊你的時候還會一直被毒 被毒下馬 毒到沒血 毒到不想追你 你說這招技能煩不煩? 2. 1級會戰 1級會戰真的可以說是鉤肥的show time 很多人看到對面有鉤肥就直接放棄打1級會戰 1級會戰講究的並不是能獲得多大量的經驗值和優勢 我認為打的是一個氣勢 鉤肥打1等會戰 絕對不要一開始就出鉤 我習慣先讓自己上去賣 等到自己血量壓低 同時對面血量也壓低的時候 利用E技能把自己血線稍微往上抬 對面血線低 自然會開始逃竄 就在這時候 大鉤一出 我們的目標是殘血的人 取得了 First Blood 了 !!! 我們要的就是氣勢啊 !!!!!!!!! 3. 天賦 --------------------------------------------------------- ( LV.1 ) 格擋 Block 定期降低敵方英雄基本攻擊對你造成的傷害50%。觸發次數可累積至2次。 細嚼慢嚥 (E) Chew Your Food 使用吞噬同時將在3秒內治療你相當於生命值上限10%的生命。 打扁你 (W) Heavy Slam 猛敲的傷害提昇50%。 恢復專精 Regeneration Master 每收集3顆恢復之球，每秒額外恢復4點生命值，永久有效。 ---------------------------------------------------------- 1級我會在「格擋」和「細嚼慢嚥」做選擇，而大多數的時候我會選擇格擋 畢竟鉤肥是一個前排且沒有位移的英雄，很容易被遠程普攻騷擾 因此這個技能可以讓我在對線的時候壓力比較小一點 後期也有不錯的減傷，至於有些人會點「打扁你」 我認為W技能的價值不是他的傷害 而是他的控制效果 如果需要傷害就交給後排就可以了 我們也不會一直去用W技能去騷擾 不然很快就會沒魔 「恢復專精」雖然效果不錯，但這個技能需要農恢復之球 反而會讓鉤肥失去他跑線騷擾的優勢 因此「打扁你」和「恢復專精」都不考慮 ---------------------------------------- ( LV.4 ) 增強治療 Amplified Healing 所有受到的治療效果和生命恢復提高30% 英雄威能 Superiority 敵方非英雄單位造成的傷害降低50%。 污穢菜刀 Vile Cleaver 基礎攻擊會在目標腳下產生污穢臭氣。 腐臭殘渣 (W)Putrid Ground 被猛敲擊中的敵人會受到污穢臭氣感染。 ---------------------------------------- 這裡的話我通常只選擇「增強治療」 我認為這招非常的OP 他給的效果是所有受到的治療效果和生命回復 也就是說當我們點了他之後 我們 「天生的回血、E技能補血、隊友的治療、恢復之球、治療之泉 甚至是你按爐石回家他都不是補500滴血而是補650滴血啊!!!! 」 對於一個前排來說 維持血線的高度是很重要的 在爭奪地圖資源、打傭兵 有血線的支撐可以讓自己的隊伍能一直處在優勢方 「英雄威能」我覺得這招沒什麼用 不考慮 另外有些人會去點「汙穢菜刀」和「腐臭殘渣」 這兩招都是讓被動更有存在感 也是變相的增加自己的輸出 我認為都沒有「增強治療」來的好用 --------------------------------------------- ( LV.7 ) 斷筋碎骨 Tenderizer 基礎攻擊將造成敵人緩速25%，持續1.5秒。 猛毒臭氣 Toxic Gas 增加污穢臭氣的作用範圍25%，以及延長持續時間2秒。 最後一口 (E) Last Bite 如果吞噬殺掉敵人，那冷卻時間將減少15秒。 回味無窮 (E) Savor the Flavor 對敵方英雄使用吞噬，將永久+2每秒生命恢復。 ----------------------------------------------- 這邊我選擇「猛毒臭氣」 他給的範圍和延長時間都是非常有用的 他讓很煩的被動變得更煩 範圍變大讓敵人更容易踩到 一踩到就是5秒 而「斷筋碎骨」是一個有點雞肋的天賦 看似效果不錯 但其實對於一個沒有硬CC 沒有跑速 攻速又慢的肥 其實實戰效益真的有限 而「最後一口」和「回味無窮」都很差 不考慮 ------------------------------------------------------- ( LV.10 ) 腐爛膽汁 (R) Putrid Bile 8秒內不斷從屁股噴出膽汁到地板上，對範圍內的敵人每秒造成23傷害， 並對其緩速35％。施放期間移動速度增加20%。 一口吞掉 (R) Gorge 將目標英雄吞進肚裡去3.5秒，造成125傷害。 結束之後，被吞掉的英雄將受到額外125傷害。被吞掉的英雄在期間內無法行動， 但也不會受到其他傷害。 『※吞食期間縫合怪的移動速度將降低15%』 『※無法對大型英雄使用』 ------------------------------------------------------- 不能不說 縫合怪的精華之三 就是他的大絕 兩招都是好用到不行的技能 這邊我選擇了「腐爛膽汁」 這招絕對是極度OP的技能 尤其是搭配20級的強化後更令人瞠目結舌 「8秒給你在地板緩敵人35% 讓你增加跑速 持續傷害 這是什麼意思?」 就是讓你去噁心對面軟皮 開了大絕之後衝到軟皮角臉上開敲 他有位移? 不怕 位移了你照樣追得到 敵人會不會扁這個討人厭的縫合怪? 會！所以對面軟皮要吃你大絕的傷害，被動的毒，還有你的普攻 對面用一堆技能把我留住了 我追不到軟皮了怎麼辦? 不怕！用W暈他 大鉤一甩 對面表示 這遊戲有夠難玩的 ..... 對面必須花很多時間阻止這個縫合怪 而因此會吃到許多的大絕傷害+被動的毒 扛了很多傷害的同時 我們有剛剛的天賦「增強治療」和E技能補血和被動「格擋」 如果還強化了大絕 我的大絕還能幫我補血 補血是可以疊加的 非常驚人的補血量 這麼多的優點 以至於「一口吞掉」就相形失色了 雖然能控制4秒 但是只能進戰施放 我覺得在後期會戰影響力還不如「腐爛膽汁」 -------------------------------------------------- ( LV.13 ) 奮戰不懈 Relentless 英雄受到沉默、昏迷、緩速、定身與變形效果的持續時間降低50%。 敲很大 (W) Mega Smash 猛敲的角度和距離增加25%。 消化不良 (E) Indigestion 使用吞噬會生出一隻小縫合怪。 樂於鉤人 (Q) Helping Hand 肉鉤現在可以拉隊友（但不會對他們造成傷害）。 ---------------------------------------------------- 這邊的話幾乎必選「敲很大」 他給的距離+25%非常有感 往往可以在不可思議的距離暈到對手 尤其是當對手在很遠的距離放一個很可怕的招式 ( EX. 巫醫 ) 範圍增加在會戰也更容易暈到更多人 至於「奮戰不懈」、「消化不良」和「樂於鉤人」都是滿差的天賦 尤其是「樂於鉤人」 你應該不想噁心自己又噁心隊友吧........ -------------------------------------- ( LV.16 ) 粉碎 (W) Pulverize 猛敲的傷害增加50%，並擊暈目標0.5秒。 釣鉤 (Q) Fishing Hook 肉鉤的距離增加50%。 串燒 (Q) Shish Kabob 肉鉤可一次拉兩個目標。 英氣逼人 Imposing Presence 敵人攻擊你時，攻擊速度會降低50%。 --------------------------------------- 絕對選「粉碎」不解釋 報鱈櫻爸最OP的天賦沒有之一 ( OP到我都打錯字了 ) 你就當作縫合怪16級只有一個天賦吧...... ------------------------------------------------ ( LV.20 ) 風暴閃現 Bolt of the Storm 冷卻 40 秒 啟動後傳送至附近地點。 癒傷膽汁 (R) Regenerative Bile 腐爛膽汁的持續時間延長2秒，並且獲得相當於所造成傷害50%的治療。 肚子餓餓 (R) Hungry Hungry Stitches 一口吞掉在吐出目標後可重複施放。吞掉的時間延長1秒。 ------------------------------------------------- 基本上只選「癒傷膽汁」 傷害的50%治療對於一個會吃大量傷害的坦克來說非常有用 「風暴閃現」除非有什麼特別考量才會選擇 「肚子餓餓」即使我大絕點了「一口吞掉」我應該也不會考慮他 效果不是很好 4. 中後期團戰 中後期團戰通常是5打5或是4+1(花魔or龍騎士) 這邊我通常會盡量用鉤子去開戰 鉤的目標優先順序為 「軟皮沒位移的 > 軟皮有位移的 > 前排沒位移 > 前排有位移」 除了考慮到鉤子命中的問題 也考慮如何贏下會戰 因此能勾到軟的 只要隊友跟得上 就可以開大絕去纏住他 對面輸出角非死即傷 而且開大絕的同時 我也輸出了大量範圍傷害和緩速 如果有需要斷法 就把W扣住去斷對面關鍵技能 反之則多用W去做群體CC 我認為縫合怪在中後期會戰真的是非常強的角色 因此我不會去拿花魔和龍騎士 但是我們跟花魔和龍騎士有一個共同點 就是我們的「定位」 那就是 我們是胖子 !!!!!!!! 胖子在會戰中還有一個關鍵的作用就是「卡」 卡住那些想跑的人 很重要 尤其是那些殘血的人 如果善用胖子的體型 可以讓他完全被卡到死 超~好~用~的~ 心得打到這邊 如果有任何問題 歡迎一起討論 也希望有縫合怪愛好者繼續開發這隻有趣的角色 底下是在巴哈PO的攻略有圖片比較方便閱讀
