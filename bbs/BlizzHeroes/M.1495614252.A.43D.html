※ 引述《habbit24 (BlindJustice)》之銘言： : 大家好 : 我是剛玩約一個禮拜的新手 : 快速對戰場次也一百五十場了 : 有些感想想說說 : 由於有LOL約六年的經驗 : 對於MOBA類型遊戲並不陌生 : 但在玩HOTS的過程中 : 有些遊戲體驗是令人感覺很糟糕的 安安，我LOL從S1只有美服就在玩了 算算年資應該跟你差不多，雖然近幾年沒碰了，但大型賽事還是都有在看 稍微看了一下你的原文和下面的討論 我覺得最大的問題你把英霸當LOL在打 : 1.韓港台玩家放在同一個伺服器 : 選單可以選擇排韓服或台服 : 但我選台服依然會遇到韓國人 : 伺服器似乎沒有完全分割 : 這麼吃重溝通及團隊合作的遊戲 : 我只能看著韓國人打出一串韓文 : 然後各打各的 : 尤其快速可以多排 : 通常韓國人多排就是自己玩自己的 : 其他人遇到他們下場通常是被放生 : 然後再被冷嘲熱諷一番 : 想call還會被回Chinese pig/Taiwan pig : 真的玩的好而且有團隊意識的玩家並不多 這的確是個問題，不過比起溝通問題更要解決的是排不到人的問題 跟韓國一起排也是個沒有辦法中的辦法 不過說真的，英霸其實並不太需要太多的"文字"溝通 我從B測的時候就開始玩，雖然玩的場次並不能算極多 但是那麼多場下來我印象有打過字的好像寥寥無幾 一部分是因為英霸節奏相當緊湊，其實沒太多空餘時間能打字 另一方面是多數狀況想要表達戰略或狀態都可以靠遊戲內的ping來表示 前面剛好有一篇文有所介紹，建議你可以參考看看 : 2.地圖機制太過於強大 : 可能是角色問題吧 我不確定 : 但是遊戲取勝方式幾乎要倚賴地圖機制 : 以LOL來說 : 當團隊劣勢的時候我通常會選擇分推 : 一來 我擅長的角色幾乎都是分推角 : 二來 分推是團隊處於逆風時很有效的戰術 : 優勢方抱團 劣勢方分推幾乎已成LOL的戰術定局 : 但在HOTS 分推的空間很小(可能是角色問題 我不確定) : 很容易被抓 隊友也拿不到什麼東西交換 因此選擇也變得很少 : 劣勢方要抱團跟優勢方拚團 只能靠操作硬打 沒有其他優勢(兵線、裝備等等) : 萬一輸掉就是掉地圖機制->掉節奏->無限循環到結束 : 贏了這波也只是扳回平局 祈禱隊友(或自己)可以回神 很難有一波大風向 先說LOL的部分，"分推"並不能說是lol逆風最有效的戰術吧 事實上對於順風且上路選強勢角那方而言，分推反而更適合順風方 S2 TPA也就是靠41分推這當時近乎無解的戰術拿下該年世界賽冠軍的 而最近的MSI比賽也很多場是優勢方在執行此戰術 當然比賽和一般配對有所不同 劣勢方與其說是用分推，更準確的說法應該是拖延比賽節奏 因為lol有裝備，我方兩件大裝對方三件，此時對方可以說是比我們強了50% 而當我方三件對方四件的時候，對方就只剩比我方強33% (以上說法只是概略示意) 回到英霸，在有坐騎和各能種飛天遁地英雄的情況下 其實英霸比lol好分推太多了 可問題是，你為什麼要分推？ LOL比賽分推是為了讓自己優勢上路和對方的一打一 對方守不住就必須派人來幫，派人幫就可能被4打3硬上，不幫可能就穩定掉塔 可英霸沒這種問題，我隨便一個能洗兵的角色就能爽守一條路 甚至還有時間跑去別路幫幫忙 而除了10/20等這兩個階段，其他時候你在英霸靠分推拖延時間並不會像LOL縮小裝備差距 事實上，除非等級差距到三四等以上，說真的兩等以內的等差在很多時候對團戰影響並沒 有你想像中的大，哪怕差一個小天賦 (除了10/20等這兩個大招階段以外) 而且英霸對於落後方是有額外的追趕機制的，目前最容易逆轉的MOBA可不是說假的 所以在英霸分推更多是戰略需求 就拿我最近打過的一場來說，地圖是永恆聖所，雙方基本上有來有往實力相當 最後一波機制爭搶，我們的普羅比斯就默默一個人跑去下路單推 而我們剩下的其他人就在天使惡魔這邊跟對方耗著，不求打贏只求拖住對方腳步 就因為這樣對方產生了一時混亂，最後我們就靠這一波直接獲勝 玩英霸更多時候你要用戰略方式去思考整個地圖的走勢 最基本的做法例子就是：在出機制前先吃野怪 這樣雙方在機制點僵持的時候我們的兵線就能推爽爽 但是同樣的如果你能猜到對方正在這樣做，你就能集結一波抓掉對方正在吃野怪的人員 所以一直賴在一條線上是沒意義的，洗完小兵確認對方沒辦法推你的塔後 適當的遊走反而能幫助隊伍取得風向 所以當機制出的時候，你大可先去參與機制爭奪，結束後再繼續做你原本的事 雖然你們弱勢，正常五打五或許還能換到對方一些人頭 但是你放隊友四打五，這完全就是一個爽送機制並且還送對面隊友人頭的節奏啊 除非你能保證隊友只是騷擾拖時間，不會被硬上死人且你能快速拿下塔的經驗 不然這樣的分推只是在擴大自己的劣勢而已 在輸機制的情況下，最該做的事保持冷靜，擋下機制後繼續做你們該做的事 穩穩的賺兵，盡力爭取野怪的經驗，看能不能抓對方吃boss的時候反打一波 我個人就打過不少場機制全輸最後反打一波直接獲勝的場 遇到整場崩到底的大多是整隊沒清兵角被對面爽爽洗兵 在這邊也告誡新手，洗兵角真的很重要，尤其是多排一定要有人能穩定洗兵啊 : 3.太吃重陣容選擇 : 在下新手只能玩玩快速 : 這一個禮拜內只要遇到瓦麗拉跟澤拉圖都很頭痛 : 因為快速常常沒坦沒SUP 沒人能偵隱 : 一堆軟皮遇到雙隱就是被殺成智障 毫無反制手段 : 但HOTS的角色技能組真的很強 : 強到需要特定角色特定技能才能反制 : 尤其可以透過天賦選擇產生技能的質變 : 這是LOL做不到的 : 以這點來說我覺得是雙面刃 我看到瓦麗拉澤拉圖諾娃伊利丹源氏也真的都很頭痛，在我們隊伍裡面的時候 一堆玩家選到這幾隻以後好像除了殺人以外就什麼事都不會做一樣 遇到隱形角就是要多注意，看不懂水波紋就多往隊友或小兵裡面擠 在不確定對方在哪的情況下，就別離砲塔太遠，反正英霸不需要尾兵有經驗就好 會戰的話就要看坦的意識，當對方影角只想肛人就表示對方正面只剩四個人 抓準時機直接上，就算自己後排被抓掉一個，對面基本上損傷會比你們大更多 至於在平衡上不管怎樣英霸怎樣都比LOL好 畢竟英霸因為地圖的關係讓更多角色有出場的機會 我是沒覺得英霸有什麼技能真的很OP，或許你可以提出來大家討論看看
