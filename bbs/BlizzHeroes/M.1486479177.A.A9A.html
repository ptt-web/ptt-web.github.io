來討論討論熱熱場子 目前古爾丹18級，快速對戰勝率55% 說說我最常點的天賦，以及戰場上的觀念 ※ 引述《zacks ()》之銘言： : 第一層天賦 : 腐蝕迴響(E) : 任務型天賦，用E擊中英雄40次後，第三圈腐蝕波會在朝反方向轟擊3次 一般來說13-15等左右時會完成 如果有大劍師維京人會更快約10-12等 任務完成後在後期對薄皮的削血非常管用 會戰時對方一群人擠在一起，總共6波亂轟下去的傷害很可觀 對方如果沒有補師，通常就要跑了 : 第二層天賦 : 生命通道(W) : 若敵人死於W下，則W的CD立刻重置 我本來都點強化分流(回魔25%->35%) 後來都走生命通道 第二層就擁有此天賦後，整場可以只回家2-3次 吸小兵不斷的保持血量跟魔量 初期要先Q一次小兵才吸的死，中期後可以直接吸滿血的小兵 : 第三層天賦 這邊比較活，我比較常點嗜求力量 : 弱肉強食 : W可對50%生命以下的敵人造成額外的傷害 不常點，對面沒控場可以斷古爾丹的W時，可以吸死對方 : 疲勞詛咒 : E的持續傷害結束後，會造成緩速50%，持續2.5秒 對方腿短+我方有追擊角(丹哥、凱利根、維拉等) 因為是傷害結束後才有效，對方發現血不夠時才會發現跑不動了 : 嗜求力量 : 技能傷害提高15%，但從盟友來的治療效果減少25% 因為可以一直W自己回血 不是很在意盟友的治療(人啊一定要靠自己) 技能更痛對面才會怕 : 第四層天賦 : 暗影恐慌(R1) : 短暫延遲後，對目標區域的敵方英雄造成 124 點傷害並且使其陷入恐懼，持續2秒。 : 受到恐懼影響的單位會往被命中位置的反方向持續移動(即呈現放射狀散開) 古爾丹沒有保命技，這是自保的唯一手段 使用時機很活，可以先丟也可以後丟 隊友爆發強可先丟，丟中心抓掉落單的人，有補師就丟他一起抓掉 後丟要等對方交出技能，最好是對方的控場技出完準備進場抓人時把對方趕跑 可攻可守超級萬用 : 第五層天賦 : 收割生命(W) : 對英雄使用時，W的回血效果額外增加50% 會戰時回血更多，對面有殘血的先吸 因為吸乾他可以重置W，繼續吸下一個殘血的 如果再配合第三層的弱肉強食一對一完全不會輸 : 治療石(主動) : 啟動後可恢復25%生命，CD60秒 應急保命用 : 第六層天賦 : 苦痛災厄 : E每波額外增傷，以該層天賦16級的傷害為每波增傷146點， : 若敵人吃到完整三波傷害，第三波的額外增傷變為436點。 E的傷害大增，軟皮吃到三波基本上就快掛了 先靠近用Q打人，等對方落跑時放準一點3波就會全中，沒死算他撿到 : 第七層天賦 : 惡魔法陣(主動) : 招喚一個法陣，使作弊單可傳送到法陣位置。傳送引導時間為1秒，CD為60秒。 終極保命技 點了這個沒粗心大意通常就不會再死了 對方有放毒的(小鹿、巫醫等)延遲傷害的(火球、澤拉圖丟球等) 結界就放在溫泉裡，不然還是可能因為血被磨光而死 不然都放在有治療活泉附近，回傳後馬上可喝水應急 或出去吸小兵血，順便吃傭兵 回傳後記得馬上重設新的結界 在CD60秒結束前不要太深入敵境，以免跑不掉 : 簡易心得 我QWE都用，不算專精什麼流派 古爾丹的工作 吃線跟削血 古爾丹常常攻城傷害跟英雄傷害第一，但不一定都MVP Q用來吃線，清蟲胎 前期對線首選大劍師 次選加茲魯維跟阿茲莫丹跟蘇爾 一來阻止他們推爽爽，二來那些招喚物都可以吸血 W什麼都吸，隨時保持血量80%以上與W常駐 E前期一直扣著，只用來打人不打兵 後期E造成的削血很煩，被轟到通常就要離開戰場傳回家 古爾丹很怕衝臉角 屠夫、伊利丹、閃光、瓦里安 如果對面有這幾隻，前期自己顧線不要跑太超過中線 不然被貼上就等著復活 最好跟隊友一起走 古爾丹也不喜歡對面有補師 傷害會比較沒力，E造成的扣血可以被補師救回來 所以R暗影恐慌就用來抓補師 有隱角就隨時用Q掃一下可疑的地方 惡魔法陣也很活 會戰時可以出去當誘餌給對方交出技能後馬上回家 跑去偷拆時等人來抓再馬上回家 開花魔跟龍，爆掉後被圍毆也能把Q跟E交完馬上回家 非常煩 大概就這樣 祝大家打GAME愉快 -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 111.254.172.46 ※ 文章網址: https://www.ptt.cc/bbs/BlizzHeroes/M.1486479177.A.A9A.html
