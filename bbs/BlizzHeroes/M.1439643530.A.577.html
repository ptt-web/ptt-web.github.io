※ 引述《ctx705f (鍵盤小妹)》之銘言： : 趁現在在等回體的時候，來回一下這篇騙P幣，順便講一下最近玩泰科斯的心得。 : 首先，為什麼人氣會這麼低落？推文都講了，沒有爆發輸出是一個主因。 : 沒有爆發輸出、沒有夠力的強控、存活能力也不好(雷諾表示：) : 難操作讓泰科斯變成一隻實在是不親民的角色，大家寧願無腦四大後排+澤拉圖。 : 但操作得宜、能搞清楚自己要幹嘛的話，泰科斯玩起來非常有樂趣，對面壓力也不小。 : 來講講我摸索過後目前最常使用的點法，總歸一句：(戰績43/74，大多都是快速) : "提升奧丁的輸出時間和存活率" 忘記今天PTT停機,睡醒後沒辦法發文QQ 來分享一下我目前玩泰科斯的心得和天賦點法~ 首先泰科斯確實是一名不太適合新手的角色 由於輸出方式跟一般刺客不同,打團戰時必須要勇敢上前找機會輸出,因此很吃操控+判斷 泰科斯的爆發主要還是在變身成奧丁後,在這之前的團戰主要都必須靠Q、W、普攻來打 存活能力其實並沒有想像中那麼差,E就是要扣著當逃命技用的,W的榴彈也能炸退敵人 除非確定可以追擊,否則E千萬不要亂用 而W技除了拿來洗兵+丟塔外,另一個非常重要的用途是"拿來斷招" 在會戰時看到牛頭人的大、老陳的喝酒...等,拿W丟過去就對了 玩得好的泰科斯無論是在攻城傷害或英雄傷害上都會有很亮眼的表現~ (扣除現階段版本的火法不談的話啦...) 以下則開始說明我的天賦點法~ : Lv.1 恢復專精 : 好，我承認點這個只是因為我是個球奴。但吃球到了中後期之後會戰間的喘息對輸出來說 : 是很重要的，專心吃的話整場回個四五十以上不是問題。而且點吃球這天賦還有一個最大 : 的好處，就是你的人生有了目標：吃球就對了。 Lv.1 穿甲彈藥(Q) 儘管爆裂榴彈(W)的範圍增加可以增加爆破範圍+提高容錯率 但我個人認為1等點Q技才是最適合泰科斯的天賦 泰科斯雖然沒有極大的瞬間爆發力,但相反地卻有Q技這招威力強大的持續性輸出技能 Q技無論是拿來洗兵+單吃傭兵+追落單角,都是很有用的技能 但由於Q技會被斷的關係,開下去後要注意對手的走位和斷招技能 在對線期或會戰的收尾階段,如果你看到對方把一輪技能都繳完或準備後撤時 就勇敢的Q上去吧~ 猛衝(E)則是增加容錯率的天賦,新手還在練習泰科斯時可以考慮點出 至於恢復專精?我完全不會想去點... : Lv.4 高熱榴彈 : 雖然傷害已經被nerf，但在丟小兵以及建築的時候仍然相當夠力，配合Q的話帶線壓塔的 : 能力還是很不錯的。通常1等和4等這兩個天賦點出來後，就抱著玩專家的心去玩他吧。吃 : 球帶線就是你最重要的工作，除了地圖機制外的零星會戰幾乎都不去，效益太低。 Lv.4 吸血之擊 點了吸血之擊後的泰科斯可以大大增加自己的魯小程度 由於機槍射速快的關係,吸血之擊搭配普攻輸出時的回血速度是非常不錯的 有了吸血之擊後,泰科斯能明顯減低回補的次數 在自己剩下極為尷尬的1/3血量時,只要找機會用普攻吃一波小兵,血量立即回升 另外吸血之擊也能幫助泰科斯很順利的單吃傭兵(尤其是攻城) 小會戰時,只要搭配走位和地形的障礙物來繞打(Ex:巨龍郡的祭壇) 吸血之擊就會發揮很大的效用~ 甚至有時在雙方都殘血+一時又逃不掉的情況下,可以站著用普攻跟對方對打! 把對方KO掉之後,就可以體會到什麼叫極限鎖血了 : Lv.7 應急治療 : 這天賦對於這打法來說太重要了。雖然即使不點奧丁也會點這個居多就是。 Lv.7 應急治療or高熱射擊 7等時這兩種天賦點法都是可以被接受的,但我目前主要都點應急治療 應急治療一樣可以增加自己的魯小程度+容錯率,也能欺騙對方達到反殺的效果 在前期對付突然現身的澤拉圖非常有用! 另外應急治療的使用也不要太吝嗇 如果看到自己的血量太少,又急需準備會戰時,開下去補就對了 單吃傭兵而血量不足時?一樣開下去就對了 至於高熱射擊則是我以前的點法,搭配變身成奧丁後的普攻,輸出非常可觀 但經過許多場次的試驗後,最終還是決定主點應急治療 想追求恐怖輸出+對自己容錯率有絕對自信的人可以點出 不過我真的很喜歡那句話:死人是沒有輸出的。 : Lv.10 奧丁降臨 : 既然都是這個打法的主軸，那就不必多作解釋。 Lv.10 奧丁降臨 達肯雷射鑽洞機在一些特殊地圖可以考慮點出(Ex:永恆之戰) 不過要在什麼時候放?放在哪裡?都是很大的一門學問... 在雷射鑽洞機放不好的情況下,我還是會主點奧丁 會戰時主要躲在大後方用普攻+Q+W來輸出,E技一樣是拉開距離或追擊用 由於在變身成奧丁時需要前置動畫,因此開啟奧丁的時機和地點都非常重要 假如開啟時離會戰點太近,對方一定優先集火 會戰時太早開或太晚開也會影響輸出效果 (太早開,對方看到奧丁立刻後撤,無法達到有效輸出) (太晚開,隊友都殘血落跑或死光了,奧丁只能跟著跑或變成活靶) 因此開啟奧丁的最佳時機跟地點,應該是等雙方確定打起來了之後 從己方後面或戰場的側翼進場(側翼進場的主要目的是達到包圍的效果,但風險很高) 除了會戰時一定要開啟奧丁之外,在我方拿到地圖機制推塔時,奧丁也可以開下去跟推 20等點出核彈後,推塔的能力和速度更是可怕 甚至能直接rush,硬拆對方主堡結束比賽 : Lv.13 巨人殺手 or 彈如雨下 : 為什麼用or？用巨人殺手的話，奧丁不知道是否會加成，但就算有加成也是0.5%，有夠少 : 。而且泰科斯用普攻平和地A對方英雄這種情況是相當少的，大多都是在用Q和W作戰。 : 點彈如雨下的話就可以有緩速的效果，但是最高才20%，能不能黏到都是問題，珍娜隨便 : 一個技能A到就帶25%緩速了。(還附加其他技能爆擊，幹) : 所以這兩個天賦都滿雞肋的。食之無味，棄之可惜完美體現在13等的天賦選擇上。 : E衝刺也有考慮過，但無法套用在奧丁身上很可惜。奮戰的話，對於目前路人團仍喜歡高 : 爆發套路的話，基本上沒甚麼差，因為被丟到幾乎就是死。 Lv.13 巨人殺手 這邊我想先推翻原PO的說法,泰科斯用普攻A對方的機會其實不少 在會戰把Q和W都繳完了之後,剩下的輸出方式就是靠普攻了 善用地形跟敵人周旋時,普攻的影響力更是大大增加(Ex:巨龍郡的祭壇) 因此普攻對泰科斯而言也是非常重要的 雖然乍看之下0.5%的加成不高,但別忘記泰科斯的射速 在機槍高射速堆疊下的普攻輸出量是很巨大的 彈如雨下在搭配團隊進行追擊時的效果不錯,可以考慮點出 但我個人仍偏好使用巨人殺手 : Lv.16 石膚 : 與主軸相呼應，提高存活率第一選擇不做他想。 Lv.16 石膚 除了搭配開啟後仇恨值超高的奧丁外,有了應急治療+石膚的泰科斯也能擋一些隱角的傷害 當諾娃或澤拉圖現身時,先開石膚擋住他們的一波爆發,再視情況使用應急治療 可以避免泰科斯在雙方還沒正式會戰之前就被搞到半殘的窘境... : Lv.20 限制解除 : 用W的時候自帶一個核彈，不過相較於W本身會更延遲一些才有輸出。另外額外50%的存在 : 時間也是非常有感，多這大約11秒就可以把QW再各丟一次。 Lv.20 限制解除 選了奧丁後一定要點的天賦,讓奧丁的影響力更上一層樓~ 有了核彈的W技,傷害更加可怕,後期會戰有時就需要核彈的傷害才能扭轉劣勢 奧丁的拆塔能力也藉此天賦完全解放 以上大致是我玩泰科斯時的天賦點法和心得,如果大家有疑問的話歡迎在推文詢問~ 順便推廣一下自己的實況台XD:http://www.twitch.tv/jmarve 最近習慣開深夜時段,之後應該會再調整開台時間~ 感謝大家(鞠躬) -- 標題 [新聞] 小熊老大驚天一轟 梅登：以為飛過計分板
