※ 引述《ctx705f (鍵盤小妹)》之銘言： : 一、前言 : 雷諾是我第一隻買的英雄，為什麼？當然除了星海爭霸系列所帶來的親切感之外，還有 另 : 一個很蠢的原因。當初新手教學時是用雷諾，教學完之後要投入戰鬥(無論是打電腦或 快 : 速)時，很直接地就想....『是不是一定要買雷諾啊？剛好都給兩千』 : 所以我就買了這樣。也因為如此，雷諾是我在英霸的入門角，場次也是最多的，當然心 得 : 也最多。其實泰蘭妲或者泰瑞爾的心得也不少，有空再分享泰蘭妲的貓頭鷹狙擊流或者 泰 : 瑞爾的純坦流。 : 普攻流當然有相對應的操作，更後面再說。 : 二、天賦點法 : Lv.1 老練射手 : 因為是普攻流，所以普攻能夠造成的傷害很重要。點了老練射手之後，顧好兵線和血量 的 : 控制就是前期的功課。吃更多的小兵，到後期就能夠有恐怖的傷害。通常到20分鐘左右 大 : 概都會疊到30層以上(詳細沒記)，真的隨便尻都會痛。 : 反正在10等前你打會戰也沒什麼用，1等會戰你沒位移技自保、Q也吃手感，跟著別人A 根 : 本就是經驗值，不如好好吃兵。 : 至於其他天賦，補血量增加50%好像沒什麼用，因為是漸進地回血，所以如果對方抓夠 快 : ，根本就撐不到血回滿就死了，更何況那時候還可能會中控場。 : 傭兵就不說了，好好顧兵線是前期最該做的。 : 無人偵察機，我是沒用過啦...不過以皮薄又沒位移的雷諾來說，開個視野好像也行。 不 : 過要記得定期補眼就是了。 : Lv.4 精準攻擊 : 不太解釋了，先講為何其他的天賦不予採用。 : 吸血才吸15%的傷害，以會戰那種瞬間爆發的傷害量來說的話，吸血其實沒什麼用。雖 然 : 在前期可以增加在線上的續航力，不過在後期會戰中就會少一些額外的輸出，尤其是雷 諾 : 這種高攻擊次數的英雄。 : Q減CD...因為這是普攻流，減CD並不是我們要追求的。 : E主動開啟並減CD...照理來說若能在會戰前期就開下去藉此拉開戰場的話，有助於增加 在 : 戰場上輸出的時間。不過若陣型拉扯不好，可能後來剩你一個殘血跑掉，也沒什麼傷害 。 : Lv.7 革命搖滾 : 加攻速的同時還可以加跑速，理論上最高可以提升30%，無論逃命或者是追擊或甩槍走 位 : 都很好用。之前也想過點"生存本能"，不過理由和前面也是一樣...漸進式的回血就是 會 : 在時間上稍微輸人，不如追求最高的操作空間。 : Lv.10 雷諾突擊隊 : 不解釋。 : Lv.13 巨人殺手 : 點了這個之後雷諾的傷害開始痛了，每一下都有額外的%數傷害，搭配7等的跑速提升讓 雷 : 諾的操作空間又大了一些。雖然說強化W也可以考慮，但是從另一個角度想，沒有這1.5 % : 的額外傷害，在後期還是會有差。 : 有時間再來試試看點W的效果如何。 : Lv.16 狂暴 : 攻速+40%，跑速+10%，根本就是雷諾的夢幻天賦。之前也考慮過處刑者，不過如果是單 挑 : 或者是隊友並無控場類的技能的話，還是自己控制的狂暴會好一些。 : Lv.20 萬象之怒 : 攻速+20%，射程+20%，再次提升雷諾的操作空間和傷害。考慮到強化R若在會戰中仍會 被 : 亂攻的指向技AOE給打爛，還是萬象會好一點。當然閃現也可以考慮，可以提升自己的 存 : 活率以及追擊能力，不過我個人比較喜歡點萬象。 : 三、操作 : 雷諾受惠於英雄特性，射程和視野會比其他的英雄要多一些些，前期利用這些優勢就很 重 : 要。不要隨便給人黏，泰科斯和伊利丹或弗斯塔德黏上來就Q讓他退，然後龜自己塔下 ， : 乖乖吃兵。 : 通常吃兵都是用W強化自己的小兵一起吃，偶爾用Q打一排。但是若對方有隱形角，我平 常 : 就會把Q給扣著，如果發現隱形角的話就把他打出來並且退到塔下等安全的地方。因為 雷 : 諾的體質和自保能力差，抓隱形是一定要的基本功。 : 另外由於攻速快+手長，所以甩槍也是一定要會的。時刻把自己維持在和對方英雄或者 小 : 兵的最遠距離，除非要抓對面。並且儘量不要讓雷諾站在原地射，保持移動。 : 會戰中陣型的拉扯就不必多談了，但是如果對方英雄如阿努巴或泰瑞爾、伊利丹這種英 雄 : 硬要衝後排，記得一邊甩一邊拉開距離，增加對方的操作難度和破壞對方的陣型。 : 如果普攻流的雷諾成型，後期的會戰常常就可以看到對方近戰衝進人群中，但沒過幾秒 後 : 發現自己已經死了，然後一邊往前甩又開了W和狂暴的雷諾朝你衝過來。 : 曾經有一次對方女王沉默箭中了之後很開心地往前衝來結果還是被我秒掉了，他大概發 生 : 了甚麼事都不知道...... : 四、結論 : 以天賦點法和所需要的技能來說，其實這套打法不太適合新手玩。因為你必須會甩槍、 會 : 拉開距離，還要會抓隱形，而且是用那範圍並不會大的Q，即使是深得我心的瞬發招。 : 但是如果玩起來之後，通常最後統計時，英雄傷害並不會輸給凱娘或者珍娜，有時還會 更 : 高。而且攻城傷害也不會難看，因為前期都紮實地吃兵，所以數字很好看。 :
