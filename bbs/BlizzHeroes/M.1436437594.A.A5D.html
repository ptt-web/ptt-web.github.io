稍微看了一下你的RP，我們先討論幾件事喔 第一，很多時候會戰的輸贏不是看當下會戰的處理，而是從前面累積而來的。譬如說， 前期強但中後期弱的陣容有沒有打出等差? 清兵強的角色有沒有最佳效率推線，或在很 漂亮的時間點吃傭兵讓對方要處理不是，不處理也不是。 我的做法是，檢討會戰為什麼輸，不如檢討第一個走位失誤。 第二，觀念=做對的事=每個動作都有意義 先不管單排隊友怎麼樣，每個人都可能遇到，小弟常常R1三排四排，也會給一個R40的， 然後4v5輸掉，重點是自己打的怎麼樣，有沒有盡可能做到觀念=做對的事=做最有意義 的動作。 附帶一提，你的恰恰在我看來，一點都不恰，R1五排vsR1五排那個恰恰才可怕，一個手 指的位置錯就是被抓出來秒殺。不過打習慣了是還滿好玩的啦，打一般路人會覺得很沒 勁，哈哈 在你的RP裡面，我看到幾件事，只要這幾件事改變了，一定會成長到R1 1. 沒有在判斷這時機對線的清兵是安全或危險，你隨時都好像覺得對面會跳出來。 但是HOTS不是這樣喔，很多資訊可以判斷的出來對面在幹麻，人大至上在哪裡。 像是你第一次死亡就是沒判斷下面已經一堆人，他們下一步一定是往上走或推下路， 走位就要走安全的位置。 對面往上走，所以你被抓到了。 2. HOTS現在就是要跑線，非常積極的跑線。 所以快速清兵，然後快速跑線或者跑線在最佳位置stand by。 女王清兵超強的優勢沒有利用的很好，而且你很多很多的走位沒有意義。 3. 黑心灣這張圖，誰先敲第一次砲，誰有最大優勢。你也沒有積極做這件事，雖然， 可以說隊友也沒有想要趕快吃錢開砲的樣子，但隊友你影響不了，想要勝率高，從自己 做起吧！ 4. 雖然R1很多水貨，小弟也是一枚水貨R1，也覺得單排很可怕，但卡在R30一定有原因 的，也許你可以push等級先上10打會戰，多ping進攻或徹退燈，世界就不一樣了。 以上，純分享。 ※ 引述《kodatakuya (TakuyaLin)》之銘言： : 因為一開始爬輸得比較慘所以神秘的MMR好像跌得不輕 : 上上週在R34~R36間瘋狂跳恰恰跳到我都累了 : 不過上周開始奇蹟似的連勝, 感謝同場的大大carry : 但是出來混總是要還的(?) : 本周開始在R26~R28徘徊, 而且一直在連輸...(目前11敗4勝) : 看了Hotslogs, 目前MMR在1550左右 : 是因為離起始分數太近嗎? : 還是因為暑假到了 : 本周瘋狂遇到新手玩家 : 當然不是說新手玩家不好 : 畢竟誰不是新手起家的 : 只是真的真的很不熟練的也別來害人阿 : (BOSS綑綁完全不躲活活被綁死我也是服了他,還是他太專心打boss忘了躲呢... : 雖然我也沒很強, 不過自認為觀念還算可以, 難道只能多排了嗎? : 一度都覺得就這樣輸下去吧,掉就掉吧 (也太負面) : 不過剛剛打了一場黑心灣 : 雖然我們一直是逆風 : 後來我們這隊4個去硬推上路 : 留一個(雷諾)守家清對面叫的傭兵 : 然後我們推到主堡時被反殺 : 剩下的雷諾還在清小兵 : 一開始覺得沒什麼問題畢竟剩一個, 兵線也壓到家門口 : 不過對面的諾娃一現身 : 雷諾就無視小兵越追越出去...(我知道諾娃很美也別這樣...) : 狂按撤退燈也都被無視 : 單挑之後就被收掉了... : 遊戲結束後看了一下發現雷諾是R1 整個傻了 : 也不是要抓戰犯 : 就.... : 追出去是對的嗎? : 是我覺得清兵才是錯的嗎? : RP : https://goo.gl/5a6FEU : 是因為不了解其中的奧妙所以我才會卡在R20幾嗎? : 歡迎大家來找我多排Q_Q : BATTLETAG:Stoneheart#3170 https://www.hotslogs.com/Player/Profile?PlayerID=1425609 分享一下自己的戰績，從R50~R1的RP都有上傳的記錄。 沒有很強都能上R1了，其實加油一下都有機會。 我跟本覺得R1不值錢，有三個團我怎麼打都打不贏阿～～～(抱頭) -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 114.38.243.233 ※ 文章網址: https://www.ptt.cc/bbs/BlizzHeroes/M.1436437594.A.A5D.html
