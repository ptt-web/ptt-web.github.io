一座城堡，即使是第一次玩的新手也不至於跟不上遊戲的節奏 : 3.英雄很具特色：我只玩過LOL啦~ 別的遊戲不知道 至少跟LOL比有趣多了 : 像是魚人寶寶:只要蛋不滅就是打不死的小強 又或著像蟲后：默默的就帶領遊戲勝利 : 4.天賦系統：省去了考慮出裝的困擾，只要選擇你喜歡的天賦就可以暢玩了 : 而且就算亂點天賦，大概也不會被隊友發現被罵吧ＸＤ 玩到一定程度後就會常按tab看雙方天賦 天賦無腦亂點一定被隊友罵 譬如之前遇過一場對面先上10 E.T.C大絕點範圍熱舞暈 巫醫點魂靈 都是引導技 結果我方唯一能出暈技的維拉點掃射 整場被隊友念翻 : 5.配音系統：就像原PO说的一樣，包括PING野怪的時候也會聽到隊友的英雄說: : "我來幫忙了" 之類的 頗溫馨XD : 6.等級共享: 再戳也不會讓你的隊伍面臨無法逆轉的情形 : -------------------------------------------------------------------- : 缺點：　 : 1.地圖大同小異: 五張地圖都擺脫不了"蒐集要素->招換大生物or某些優勢->開始進攻" : 然後通常就是一座主堡+1-2個等級的領先。Loop...五張地圖除了主題不一樣以外 : 本質是一樣的 感覺有些可惜 本質上完全不一樣 先說我覺得設計最失敗的地圖: 海盜 這張圖的重點就是打金幣的時機和gank對面交金幣 也是唯一一張對面不用進攻我方基地就能獲勝的地圖 因此優勢方幾乎能一直保持著優勢 其他地圖譬如龍騎 花園 被推掉軍營還可以在家守等對方來反打 海盜這張對方只要吃光地圖傭兵和寶藏 守方根本沒什麼機會逆轉 且因整體地形設計的關係大概是最好猜對方位置的地圖 礦坑某種程度上來說是最容易一面倒的一張圖 如果隊伍沒有搶頭骨又不趁對方搶頭骨拿騎士營地的話大概就沒什麼救了 不過兩邊都有概念的話 打起來就是很刺激的一張圖 此圖重點:礦坑開啟倒數前趕快補滿體魔 頭骨拿到一定數量後搶營地 頭骨差距太大全隊防守 烏鴉詛咒則是最吃組合的地圖 這張圖重點是每次的祭品會戰和把握下個祭品出現前打傭兵的時間 也因次前期續戰力高和有長射程ae的英雄很有優勢 另外此圖也是唯一一張有兩個boss營地的地圖 後期的重點會在boss的攻防 (傭兵都不受詛咒影響) 龍騎和花園相較上面幾張來說單純許多 就算沒搶到也不會損失太多 也很好防 對方搶到龍騎推線時看其他人有沒有配合進攻 如果沒有通常三人防守就擋得下 注意被瞄準踢的人卡視角或是控場龍騎 而植物進攻時就需要較多人防守 在他丟出盆栽時一定要第一時間集火打掉 閃好範圍控場 植物可怕的是在後期的普攻傷害 種了控場常會直接被拍到死 另外控制植物和龍騎的人也是要配合隊友進攻才有用 常看到那種隊友死光還自己去開植物 或是拿到龍騎以為自己能開無雙亂衝的 每張地圖包括海盜我都打過逆轉勝的 有些英雄就是要17等的關鍵天賦點出來才算成形 譬如莫奇lv16的泡泡時間內回滿血 甚至像泰科斯lv20的雷射大絕 點出來根本就是不同次元 所以這遊戲絕對別隨便放棄 : 2.英雄能力過強: 這可能也是被LOL影響了吧 永久隱形+一套秒人,直接定身5秒 還有 : 各種OP的控場招式, NG場遇到一些組合真得很難打 : 3.沒有反制機制: 以LOL為例 中路遇到AD刺客 會出沙漏反制 : 遇到強POKE 會出女妖 : 遇到強控 會出水銀 : 隱形 會出紅眼 : 可是這個在暴雪英霸幾乎無法反制 : 遇到潛行英雄只能考驗眼力,遇到澤拉圖超久的定身只能干登眼 : 選個沒有位移的輔助英雄 遇到各種刺客衝臉根本難打 不到20級沒閃現被抓一次 : 死一次 :
