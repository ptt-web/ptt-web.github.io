我是前陣子開始玩快速， 雖說最近台服快速排隊時間有改善， 但還是常常要等5,6分鐘以上。(晚上10:00左右) 所以在中國伺服器又開了一個帳號。 在中國伺服器排隊時間真的減少很多， 單排通常在兩分鐘內就可以開打了。 基本上玩家觀念台服相當，很少遇到奇葩。 我玩了幾十場只遇到一個惡意掛機， 但偶爾也是會有掉線的。 最大的問題是有時ping值會飆高。 像前兩天ping值飆到幾千一萬，根本連不上。 或者稍大ping在200,300左右也不用想玩。 ping 200,300玩亞拉瑞客就是Q不到人， 想W拉人，人早就跑遠了。 然後大絕R來不及放就被打趴。 不過還好一般都在100以下，還算順暢。 通常我會先去玩AI確認一下ping值ok再排快速。 避免雷到人。 喜歡玩快速又不想排隊排太久的可以考慮去中服玩玩。 至於只玩AI在台服其實不用怎麼等就沒差了。 祝大家場場666 ※ 引述《atony355083 (謝謝你9527)》之銘言： : 小弟弟我最近在台服排快速，以單排來說，平均等待時間約450秒以上（晚上也一樣) : 五排的話，不用說！一週七天，任何時段都是600秒，而且沒有好的遊戲體驗。 : 再來是聯賽，台服單排一律都是400-500秒（宗師分段)，雙排不清楚。 : 團隊聯賽，無論韓服台服都是600秒以上，偶爾韓服有強團在排，可以秒撞，其餘情況 一? : 這情況相信各位都會等到獨攬，因此我轉陸服試試看。 : 在五個月前，有連過陸服，當時ping約500，但是實際玩起來肯定破千，所以當時快速 我? : Ok，屁話說完了，我來說一下這兩天的陸服心得。 : 玩陸服有啥好處， : 1，人多，小弟我等級很低，但是排快速都在百秒以內（有幾次破200秒)，但是還沒有 破3 : 2，人多，所以匹配會較為精准，不像台服，人少到聯賽也要賽垃圾給你。 : 3，語言上可以溝通，不像韓服，無法溝通（我書讀的少，英文不會，只會一點髒話) : 4，他們沒有分線路，像台服有kr有tw，所以每個玩陸服的都在同一條伺服器上，不會 像? : 5，連線順暢，ping值比韓服還低，穩定50-70，非常順暢。 : 6，和諧化，減少個人暴力傾向。 : 目前玩兩天覺得唯一缺點是，我不知道怎麼儲值啊…TAT : 個人遊玩兩天，感覺就是爽！ : 明天補上陸服tag，想一起玩可以加好友。 : 補上tag: opDog#5708 : 版上蠻多人在問怎麼連陸服 : 很簡單，在登入介面，伺服器選擇改成中國，注意，要申請帳號，不過沒關係，因為申 請 : 登入後開始遊戲他會說語言不相容，會叫你轉簡體字，按下確定後，下載約800mb資料 後? : ----- : Sent from JPTT on my Samsung SM-G900I. -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 39.8.106.248 ※ 文章網址: https://www.ptt.cc/bbs/BlizzHeroes/M.1491379919.A.01E.html
