※ 引述《whoiam (胡愛晏)》之銘言： : 天宇，曾經可以算上霹靂之外，最廣為人知的布袋戲。除去那政府下令禁止 : 播放的史碑級代表外。不管當今一派獨大的局面多麼顯明，背後仍是黃氏血脈傳 : 承，無人可否認。不論是多遠或多變調或多大改變，黃族一脈至少有三代奇蹟。 : 其中的雙生，以獨特自我風格帶起一波波戲迷的天宇，在新東家推出新天宇二十集 : 後斷層，這中間的效應，我稱之「後天宇時代」的來臨。 這段為「名詞解釋」，何謂「後天宇時代」 : 誰堪與當代掌中戲企業化之龍頭霹靂一較長短？是其父？是其弟？是另有他人？ : 想要開創新時代的「新世紀國際多媒體」在目前看來是最具公司模式的一支。天地 : 霹靂、新時代(新天宇-短暫)仍否變成三雄鼎立？或是黑馬突出？未來有無限的可能 : ，但是愈新、愈華麗、愈精品化、愈高科技，就算稱不上第二大，但在發片初期勢必 : 會引起那些失望、疲憊、厭煩的收視、租看之戲迷些許的探頭張望，試試水溫、嘗嘗 : 味道、看個幾次，一定有的。 說新世紀算是最有機會開創新時代的公司。 : 有不一樣的，總是比較吸引人。不管後來變得如何。 : 從無天宇到有天宇，從天宇到危機天宇，爾後新天宇，新天宇到無天宇，現在… : 是「後天宇」時代的標誌性，想打下一片天，大有人在。天宇也許暫時消失，但它帶 : 起的效應還有曾經的開創都是值得在偶戲史，記上一筆。後天宇，明天過後，宇宙… : 也許又不同了。 天宇有其歷史意義 : 一黨獨大，從來就不是好事。就算有，也是暫時。 這句說明了，作者不希望只有霹靂，而且應該是「從來都不希望」， 推文中說他希望霹靂去併天宇的，顯然有誤。 : 文創事業，本該百花齊放、萬紫千紅。沒人規定只能一言堂。真的有爭豔現象也未 : 嘗不是好事，相互刺激、進步。不是你死我活或爭食大餅，而是造福觀眾與活化這多媒 : 合的多元複合文化體。 : 只有一朵花開在世上，不是值得歡喜鼓舞的事，尤其是面對僵化而不自知、虛壯而 : 不覺、盲化而不自察時。當一朵朵的文創之花盛開之際，養份不會被吃完，反而是魚幫 : 水、水幫魚，生生不息、源源不絕。 作者指出只有霹靂不是值得高興的事，「僵化……盲化而不自察」應是批評霹靂 : 掌中戲是一朵自由行走的花，花在手上，花…都開好了。 ------------------------------------------------------------------ 作者大概覺得掌中戲史上，至少還有天宇和霹靂同時站起， 但現在天宇又斷了，只剩霹靂一家獨大， 不過總是會有人想出來佔一席之地的， 目前就是等著看誰能站出來的「後天宇時期」。 只是，我不知道怎麼解釋標題的「霹靂爆炸」...... -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 219.71.14.25
