時間Fri Jan 22 09:41:03 2021
各位板友大家好： 關於有板友在問到底什麼是轉讓？什麼是贈與？判定的依據為何？ 以下針對 交易文 各項過去 有板友詢問過的規定 給予 認定通案 說明： 贈與認定：將自己的權利或物品 無償 轉送於他人 稱為 贈與 轉讓認定：將自己的權利或物品 有償 轉讓於他人 稱為 轉讓 或 交易 割愛認定：將自己的權利或物品 有償 轉讓於他人 稱為 割愛 或 交易 合購認定：於網路上揪購物 但 未同時於一地點集合購買 稱為 合購 如有約定地點一起購物 則 歸類為玩樂文 代買/刷 ：於網路上請求他人協助購物，而無約定一地點一起購物 稱為 代買/刷 租賃 ：於網路上登載租賃物件 稱為 租賃 以上說明僅為 通案判定 概念舉例，實際判決 仍依 個案判決 為主。 以下相關爭議案件舉例： 轉讓健身房：轉讓者 說明願意自付 轉讓金 => 贈與 轉讓健身房：轉讓者未說明自付轉讓金 => 交易、轉讓 轉讓健身房：轉讓者願意自付會員轉讓金，但要求受轉讓方支付課程轉讓金或剩餘 課程金額 => 會員部分屬贈與、課程部分屬 交易、轉讓。 轉讓健身房：轉讓者要求對方支付轉讓金 => 交易、轉讓 轉讓健身房：徵求他人轉讓會員，並說明由對方支付轉讓金 => 爭議內容文章 刪除 轉讓健身房：徵求他人轉讓會員，無說明轉讓金之處理 => 交易、轉讓。 好市多合購：揪人一同前往好市多購物 => 玩樂文 課程合購 ：揪人一起學習技能，並於一地點一起學習 => 玩樂文 網路合購 ：XXX商品大特價，揪團網購 => 合購交易。 好市多代買/刷：揪人一同前往好市多購物，說明自備零錢等 => 玩樂 網路代買/刷：網路請人協助前往一定點購物 並支付金錢 => 代買/刷 以上為相關舉例說明，本說明屬 包含 而 非所有部分。如有疑義於此公告底下提問 會一併說明。 有鑒於可能因為過去認定說明不夠清楚導致板友有違規行為。 故於本公告前已發之違規文章，處理如下： 於 2021/01/23 0:00 前 自覺錯誤者 並 自行刪除者 不判決。 於 2021/01/23 0:00 後 所有違規者 依交易文適用規定 處 水桶 7 日。 高雄板板主 simonjen -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 180.177.212.242 (臺灣) ※ 文章網址: https://www.ptt.cc/bbs/Kaohsiung/M.1611279665.A.494.html
