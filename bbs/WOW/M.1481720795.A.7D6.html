原文恕刪 借題分享一下神聖牧師在 M+ 幫打的經驗 希望能拋磚引玉，有錯誤請版上的大大指正 一、幫打的原則 只要確定不會死人就可以幫打 (廢話) 有危險的時候還是以補血優先 (廢話) 二、配裝 2詞綴以下：加速 20% 剩下堆致命，帶輸出飾品，打起來很舒服 3詞綴以上：加速 15% 剩下堆致命，帶治療飾品，有空加減打，別死人比較重要 不過我每週也只是打打低保而已，10+ 的經驗不算太多，希望大大補充 三、輸出手法 神牧幫打非常簡單，AE 的場合直接上去炸新星就好 新星的半徑足足有 12 碼，其實非常大 介於庇(10碼) 和群驅(15碼) 之間，可以參考這兩法術的施法範圍 點殺的時候如果不用考慮 暈/ 癱瘓 的問題： 譴(找七巧板) > 神火> 懲擊 如果有分配到暈或斷法的任務，請一定要算好譴的 CD 務必要在輪到你暈之前把譴的CD刷掉 如果時間掌握得不好，或是控場遞減算得不好的話 譴就捏在手上別亂丟吧 如果七巧板已經出來了，譴也可以先捏在手上，等七巧板消失了再丟 四、輔助巨集 #showtooltips 懲擊 /cast [harm][@targettarget,harm,exists]懲擊 這個巨集的作用是： 點怪物的時候對怪物施法，點隊友的時候對隊友的目標施法 幫輸出的時候可以不用換目標，點隊友後跟打就好 懲擊可以自行替換成其他攻擊法術 如果用這個巨集幫打的話，建議不要點著那些常換目標的職業跟打 對，說的就是我們牧道會裡的那群暗牧，跟他們打是有風險的…… （那群信仰暗影的瘋子根本不知道自己在幹嘛） 想知道原因的話，可以版上搜尋一下關鍵字「TAB」 五、輔助插件 Focuser SHIFT+滑鼠左鍵直接設定專注目標，幫打的時候快速監控危險目標 比如說玄鴉堡的斥候 讓你可以在他讀完刃舞的第一時間譴/鎖他，方便繼續偷打 沒控到的話就得老老實實的補血了，真的很沒意思 當然也可以用巨集做到一樣的事情，或是直接看怪物名條上的施法條 不過我還是習慣用這個插件了 RangeDisplay 搭配 Focuser 和輸出巨集使用 在不換目標的前提下確認目標是否在譴的範圍內 (30碼) MSBT 複數目標的法術傷害/治療會整理成：[X擊中,Y爆擊] 可以用來確認新星炸到的數量 ======以下閒聊分隔線====== 打低層的時候，全程十幾二十萬秒傷不是問題 隊友犯錯越少，火力越猛，輸出自然就會越高 英勇+持久全程覆蓋的話，單體王也可以打到二十幾萬 (我有護腕) 10+ 的話，因為我和隊友都還不算太熟練，常有犯錯的情況 打每周保底的話大概是全程十萬左右，難度越高就越低 我真心覺得，神牧其實不弱，只是需要多花點心思罷了 牧道會的兄弟姊妹們一起努力，願聖光常伴我們左右 -- __ __ _____ _ _ _ | \_/ |_ __| ___(_)___| |__ (_)_ __ __ _ | |\[40m /| | '__| |_ | / __| '_ \| | '_ \ / _` | | | | | | _| _| | \__ \ | | | | | | | (_| | |_| |_|_|(_)_| |_|___/_| |_|_|_| |_|\__, | |___/ -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 114.36.167.202 ※ 文章網址: https://www.ptt.cc/bbs/WOW/M.1481720795.A.7D6.html
