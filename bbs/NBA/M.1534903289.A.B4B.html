還沒奪冠前, 勇士只有一個all star K湯跟嘴綠, 是奪冠後才變成all star的 當然這樣的做法比較淚也很艱辛 直接蒐集all star組隊來拼總冠軍是比較容易的道路 但不代表只有一個all star就沒辦法奪冠 勇士之前也有一個球隊做到過, 那就是2003的底特律活塞 https://imgur.com/77ptX7y.jpg
該年活塞只有大班蛙入選all star game 然後活塞4-1奪冠終止了湖人的霸業 只不過活塞陣中仍具重要性的大溪蛙, 2001有進all star game就是了 ※ 引述《kenny1300175 (蘇湖)》之銘言： : 虎撲8月22日訊 : 隨著最近幾年的NBA總冠軍都包含了2個或更多的全明星球員，有實力的球員轉投強隊的 : 現象開始出現，包括Charles Barkley、Michael Jordan等名宿也對現在這一情況發表了 : 自己的意見。近日，雷霆球員Paul George也對“超級球隊”發表了自己的看法。 : “沒有一支球隊在只有一名球星在隊中的情況下獲得了總冠軍，現在不是那個（單核帶隊 : 奪冠的）時代，”George說道，“我不是很清楚那些名宿、傳奇對這個部分怎麼理解。” : “現在的比賽不一樣了，對那些只會插嘴說我們球隊建立的方式和你們不一樣的傢伙... : 我永遠不會理解，如果我們都獨自作戰，然後嘗試對抗勇士，又有誰會說我們是傻子呢？ : ”George接著說道。 : “聯盟裡最好的傢伙（LeBron James）也沒有做到這個，他被橫掃了（2018年總決賽）。 : 所以現在這個階段，你只要展現出你能為了勝利帶去什麼，因為你也需要充滿天賦或者技 : 巧過人的隊友來爭奪冠軍。” : George還認為現在的球星必須理解他們有做出自己的選擇的權利，並且在參與球隊建設時 : ，要更加積極主動。 : “在過去的日子裡，球員們被困在球隊中，”George說，“球隊總是圍繞一名球員建隊， : 然後讓那名球員為他們帶來冠軍。現在一切都不同了，人們能夠理解，我們也知道要帶去 : 什麼。” : https://voice.hupu.com/nba/2336404.html -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 60.250.135.23 ※ 文章網址: https://www.ptt.cc/bbs/NBA/M.1534903289.A.B4B.html
