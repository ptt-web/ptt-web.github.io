※ 引述《vbangus (電機小胖翔)》之銘言： : 不好意思有個問題想請教大家一下： : (現在是很"真面目に"的在問) 我也以自身經驗まじめに回答一下。 : 一般如果說以念的科系非日文系，EX:理工背景的， : 若欲從事日語教學工作，或當日文家教老師， : 大家會建議"至少"要通過新制的多少級數呢？ 這個問題很難回答。 你可以思考一下，自己要有幾級不會被學生問倒 或是有一級的人就不會被學生問倒？ 自己的程度相對影響的是你能夠教哪些人。 : P.S. 我動詞分類是學五段、上一段、下一段、カ行、サ行 : 變化是學未然形、連用形、終止形、連體形、假定形、命令形。 : 品詞分類部份是學"形容動詞"派的，非"な形容詞"派的。 : 原來日語教學的文法上還有各自表述，不同解讀的情形。 其實這無法讓我們知道你有多少能力 事實上，坊間最多的課本是採用「大家的日本語」那一套 用I類(五段)、II類(上下一段)、III類(タサ変) 教原形、否定形、マス形、テ形、タ形、仮定形、命令形 形容詞有い形容詞、な形容詞這種。 所以如果只會你上述的國語學那一種，在教學上跟日語補習班不合 家教的話你還不能用一般坊間的教科書。 (國語學一般用未然形、連用形、終止形、連體形、已然形、命令形 假定形是已然形+助動詞ば) 另外日語教學的差異不算是各自表述 只是對象不同用的方法不同， 就跟華語教學面對國人跟外國人方法是不可能相同的。 : 背景： 四月底開始正式學日文，12月份考了N4, N3(結果尚未公布) : 台大部份也有在這學期選修日文一上和日文二上(同時修)當複習。 : 因為"0級到2級只要8個月的課程"真的上起來超趕的，吸收力並沒那麼好。 : 曾經被學校推薦在大三和大四出國留學兩年(美國、雙聯學位) : ，因此目前有從事英語會話和發音(KK)的教學(家教)。 : 經過日文老師的鼓舞，她希望以後被她教過的學生能夠把日文學成自己的專業， : 然後也跟我談說是否以後也可以順便考慮教日文。(最少可以當作賺外快) : 於是萌生了這個想法.... : 謝謝各位！ 我面對過的實際情況。 我大學的時候念中文系 在大四的時候過日語一級 有想過接家教，會遇到的問題如下。 明明不是日文系，憑什麼教日語？ 面對一群對日語沒有概念的人，你就算有一級對方有可能認為你比日語系三級還弱。 為什麼時薪這麼高？ 吃米不知道米價，有些人會用自己那微薄的薪水來衡量 認為你不該要這麼多，何況你不是日文系的。 而家教最大的問題點，在求人的時候就看的到了。 教小孩：限女 教女人：限女 教男人：女佳 為什麼？只有求人的人知道原因了... 要走補教業，可能需要一些コネ。 以前公司的一個後輩，因為跟某村櫃姐很熟 清濁不會分，依然可以教50音。 我離開公司後，後輩叫我去問問看 畢竟保聖那給我住日10年以上程度的認定 能力也電爆那些台灣人老師了 教個基礎入門應該不會有太大的問題 何況連他都可以教了我應該ok吧(他自己說的) 但是我碰壁了XDDDDDDD 那個後輩後來打電話去問 櫃姐連我的履歷都沒有送出去 理由是非日語系，沒有留學經驗。 後來問了幾家，也是強調留學經驗。 或許有些人會覺得 非日文系的又不一定比日文系的差 (日文系的去一下某村的高級班應該會抖) 沒留學過的又不一定比留學過的差 (很多留學回來的去某村的高級班受過震撼教育) 但是學歷社會就是這樣 無論你能力高低，非本科系、無留學經驗 有的地方連測試你的機會都不給你。 畢竟供過於求，有一堆留學生可以挑的情形下，何必著眼於非本科系的？ (某村裡面的台灣人老師根本10年沒換，而且沒被砍就不錯了，怎麼可能進新血？) 所以與其走日語教學，不如以日語為輔，讓你的專業更受重視 至於方法你可以多參考板上理工前輩的文 薪水也絕對比日語教師多很多。 -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 114.32.192.145
