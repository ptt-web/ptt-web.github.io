我的比你簡單好分辨（個人覺得啦:P），例外也不多，參考看看。 五段一樣，跳過。 上下段我是這樣教的: 看る前面的那個字，只要是i或是e段音的，就是上下段。 不是的通通丟回五段，解決！ 至於例外有哪些， 切る、帰る、いる（這三組是五段、上下段皆有，也不太算是例外） 入る、知る、走る、握る 就只有這七個算是例外。 不過對初學者來說，握る其實也是中高級的字， 所以負擔會比較小。 還有幾個啦，不過因為初學者用不到，所以就無視了 wwwwwwwwwwww 有錯請指正 :P ※ 引述《sssn1 (找一個字代替~)》之銘言： : 結尾無る--- 五段 ex.買う立つ呼ぶ志す死ぬ泳ぐ : / : / : \ る前為多音節漢字---五段 ex.盛る陥る眠る蘇る香る喋る握る : \ / : 結尾有る アウオ段---五段 ex.在る売る折る : \ / : る前為單音節漢字或假名---イ段---上一段 ex.用いる過ぎる : \ : エ段---下一段 ex.駆ける寄せる : 例外：要る切る減る蹴る散る照る練る知る etc. : 當然サ変和カ変本質上已是例外，即使符合上面的規則也不是五段。 : 結論：結尾非る與多音節漢字均五段，因此只需考慮單音節漢字且符合上下一段 : 特徵(i/e+る)卻是五段的動詞即可 : -------------------------------------------------------------------------- : 剛剛回答別人問題時順便做的表 希望對版上有幫助 -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 111.248.210.111 ※ 編輯: akumahirosi 來自: 111.248.210.111 (11/14 14:16)
