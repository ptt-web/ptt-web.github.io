標題[心情] 爭與讓
時間Thu Oct 7 13:51:09 2010
看到這個標題,頗有感觸~~ 過去這幾個月與先生吵得很激烈 有幾件主軸大事,加上牽連其他里里摳摳的小事 演變成兩人只要一講話,就很容易一觸即發,吵到後來累兩人精疲力竭, 舊事吵,新事件也不斷加入,後來兩人乾脆避免多說話, 家中冷戰.緊繃的氣氛搞得很難受..... 後來我們去尋求諮商的協助 這也是我三催四請,最後先生才同意去聽聽他口中所謂"外人"的意見與協助 幾次諮商過程當中,我們仍舊有幾次的大吵 但是開始會有意識地注意到我們溝通上的問題 譬如: 1.吵一件事情,就針對那件事情,不要牽扯其他事情 有時候因為會提及其他事情,對方會辯駁或是另舉新例,變成又衍伸為另一個議題的辯論, 主要議題仍未獲解決,... 2.容忍對方表達負面的情緒 有時候夫妻對某件事沒有共識,即使最後接受了不滿意的決定,仍應容許對方表達負面情緒 比如:先生不喜歡太太每個月回娘家一次,太太還是堅持要回娘家,最後可能協調出 先生不回去,太太自己回去,這雖然不是最完美的安排,但兩人都接受下,即使偶爾雙方 仍會為此碎碎念,要容許對方有表達情緒的自由.... 3.每一方都想想"我該怎麼做,可以讓狀況改善?" 吵架時,常會想:為什麼對方不怎麼做,要是對方怎麼樣,事情就會改善~~ 諮商師要我們學習去想: "自己"可以怎麼做,讓狀況改善? 而不是"對方"該怎麼做,事情可以改善 4.區分哪些爭吵的事情是"可以解決"(solvable), 哪些是比較重大，積習的長久性議題(perpetual) 譬如:夫妻可能為了牙膏怎麼擠而吵,如果單純只是爭論牙膏該怎麼擠, 這種議題其實不難解決,也比較好溝通 如果擠牙膏的背後，是更大的議題,比如說夫妻間的溝通方式,夫妻間欠缺信賴,信任,與愛 那才是更需要正視的議題 對於這點,不久前我才經歷的一次"實例"~~ 先生因剛回國,除了時差還未調整過來,好像連家中東西在哪裡都有點錯置 總之，他做了一件讓我非常無法忍受的事情:連續三天沒刷牙!! 我也沒發現，直到那天要去一個朋友家作客時,在車上一陣惡臭,我才問:你沒刷牙? 他才承認:找不到牙刷~~ 我說:找不到牙刷,不會問我嗎?有這麼難嗎? 他:我想說你在忙,所以沒問你;可是我都有用牙線(在我看來是"辯解") 我:家裡離 Walgreen這麼近,你找不到,也不問,也可以自己去買啊!? 他開始無言以對... 老娘本來想繼續數落,一陣提醒聲(這是"可以解決"的事情...) 心裡開始想,我已經表達了,適可而止,此時也沒地方臨時去買牙刷給他,我停止咆哮了~ (雖然心中還是很氣....) 5.諮商師介紹我們一本書,裡面談到讓婚姻work的七個原則 作者:Dr. John M. Gottman "The Seven Principles for making marriage work" 我覺得受益很多 如果版友借的到,推薦可以一讀...不知道有無中文版?! 後來在報紙上看到一篇婚姻諮商的文章，在知道原來這個作者算是婚姻諮商的大師! 6.除了去諮商外,最近也我常上這個版 本版的好處,除了"比上不足 比下有餘",看看別人婚姻的問題,有時想想自己與先生的爭吵 根本是小問題,會比較感恩,去看自己婚姻中好那一面; 另外我開始會轉寄一些文章給先生,佐證我的想法 有時候好像同樣的話從老婆口中說出,他比較不能接受 從別人口中說出或寫出,他就相信"確有此事"~~ 效果奇佳!!! 最近Ponima那篇閃光文,特別得到我那正經八百的老公的注意 再我解釋一番Ponima過去po文的簡歷後,我們兩人都慢慢對我們的婚姻重拾一點信心~~ "果然,他山之石可以攻錯!" 還有一些心得,無法一一贅述,覺得婚姻真的是一個很難很難的功課! ※ 引述《drama (OoooOooOooOOOoooO)》之銘言： : 這標題對一些六年級的人應該很眼熟 : 某次聯考的作文題目嘛！ : （這算是梗還是幽默？） : 然而，不論是看婚姻板至今的討論 : 或者是實際婚姻生活上的溝通 : 愈想愈覺得這其中的標準拿捏真是微妙 : 我猜很多人在寫這個題目時 : 大多都會引用到這句話 : 「忍一時風平浪靜，退一步海闊天空」 : 好了，問題來了 : 許許多多在婚姻生活中的摩擦與爭吵 : 不外乎圍繞在「計較」二字上 : 「為什麼我要忍？」 : 「為什麼我要退？」 : 「為什麼對方不能體諒我？」（要對方忍） : 「為什麼對方不能包容我？」（要對方退） : 既然說了忍「一時」，當然不是要你一直忍 : 而是要掌握到溝通的時機 : 在摩擦爆出火花的當下，忍不住而硬要去爭 : 豈不是火上加油嗎？ : 就像我從來不會在我先生火大開始大吼摔東西的時候 : 硬要去和他講理，這點他倒是很好認XD : 如果是一些脾氣較溫和的，潛藏在水面下的冰山很多人都沒察覺 : 然後就笨笨地開船撞上去了 : 而退「一步」，也就是要你以退為進 : 退了一步，兩人之間就有了溝通的空間 : 可以冷靜下來，清楚地看到對方，理解對方的心情與想法 : 這才是退一步最重要的溝通關鍵 : 每次我在板上分享我與我先生的溝通經驗 : 常常不少人以為我總是委屈自己或是無限退讓 : 其實我只是根據我先生的脾氣個性 : 去拿捏這「一時」與「一步」的時機與彈性 : 畢竟像我先生個性這麼強烈的人 : 除非不得已，大多數人都是能避免接觸就避免接觸吧 : 所以自然無法理解到底要怎麼樣和這種人溝通 : 因此會覺得我付出與忍讓的總是比較多 : （你說像現在我在電腦前打字，他做好我和兒子的早餐 : 還自己做好便當準備出門上班去，不是過得很爽嗎？XD） : 曾經我在板上分享過，夫妻之間，沒有對錯 : #16g7Oukl，這是我們吵了十幾年的心得 : 當然我們仍然有很多觀念和想法是大相逕庭的 : 但我們已經不會去爭這種事了 : 當學會尊重與包容的同時 : 也學會了理解對方的個人特質 : 所以我們不會想去改變這些地方 : 既然當初我們在一起就是因為某些個人特質而互相吸引 : 那麼，沒有理由非要對方抹除其他你看不順眼的個人特質 : 因為那是之所以對方會是對方，成為你生命中這麼特別一個人的理由 : 我先生當然有很多地方我受不了 : 但只要他並不是表現在我無法接受的某些生活部分就沒關係 : 例如不要干涉我的做事方式，不要用上司的態度評估我的能力 : 有的時候，我也學會了拐一個彎去溝通 : 例如他整晚都坐在電腦前看影集，完全不顧小孩哭喊吵鬧 : 這時候我就會按捺住心裡的火 : 不過走到他背後給他用力地「按摩」下去 : 我先生會唉唉叫，可是同時我也找到了溝通的時機 : 他會邊抱怨工作多忙多累，但也會問一下剛剛小孩怎麼了 : 我就可以順勢地跟他抱怨，然後按摩完他就會起身來幫我XD : 我也常常買一個蛋糕回家，然後問他要不要吃蛋糕喝咖啡休息一下 : 這十分鐘的空檔其實可以很容易地溝通很多事:P : 在溝通的時候，我從沒想過要改變他的想法 : 我只是讓他挪出空間來容納其他的想法 : 如何整合安排那就是他的問題了 : 溝通，要的就是這樣一個彈性空間 : 可進可退，可攻可守 : 那麼，什麼問題都好談了:) -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 70.225.180.3 ※ 編輯: regent2 來自: 70.225.180.3 (10/07 13:55)
