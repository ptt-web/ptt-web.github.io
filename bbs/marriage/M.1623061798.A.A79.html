※ 引述《clare247429 (啾啾)》之銘言： : 最近家中長輩在催生… : 最常講的幹話就是「生的出來再說，搞不好生不出來」（從這句話就知道上一輩生小孩 都 : 沒什麼規劃和責任感） : 因為不想當沒責任感的人，我和先生想先存一筆錢再生（之前的錢都買房子去了，因為 不 : 想再繳房租了）… : 還有同事說… 沒懷孕幹嘛結婚？說我外表看起來很會玩的樣子，居然會沒懷孕就結婚 了 : … : 被我白眼後，她解釋說 因為結婚等於被綁住 : 如果真的因為生不出來要離婚 離婚有紀錄以後難找好對象， 現代人都先懷孕再結婚居 多 : .. : 其實我和先生是買房子後再結婚的，然後再重新存錢生小孩，但這社會風氣 好像都導 向 : 當然我也很怕我同事說 萬一生不出來… : 「結婚後沒懷孕就是不正常」 : 尤其是長輩一直念，我和先生覺得壓力大 : 唉.. 先講結論：問看看你自己跟你老公，要不要生？ 永遠不要把你們夫妻的想法，塞給別人來決定。今天人生是自己的，如果生了小孩之後， 他or她也會有自己的人生，不要幫自己加戲、也不要硬性的幫孩子加戲。 不要說一些養兒防老、不生小孩不像結婚....你生了小孩，就該為了他or她未成年的日子 負責。 不愛小孩就生、沒考慮清楚就生、想要他幫忙完成你的夢想才生...這些種種都不是在為 你的小孩負責任的態度。 天底下，一定有不愛自己小孩、甚至怨恨自己小孩的父母。奇了怪了，小孩是欠你的嗎？ 對那些父母來說，小孩只是簡諧運動後、多巴胺分泌下高潮結束的副產品。爽是你們父母 在爽，真的生下來後又要被靠北一堆 + 塞入一些孝順父母、聽從安排的倫理大戲。 拜託，只是因為你們忘了買保險套。至於這樣為難一個孩子嗎？ 當然，如果你問過自己的內心，確信自己的確是想要這個孩子、想要豐富夫妻、孩子的人 生。願意付出愛跟美好。那就去做。但永遠不要把夫妻之外的人拉來救援，什麼同事、什 麼隔壁老王、菜市場大嬸、甚至是自己的父母。 任何會帶著未來的怨懟、事後的後悔、對未知的恐懼。都不要生下孩子。孩子不會是你人 生的解藥，不要以為你把你未完成的夢想套用在他身上，你就圓滿了。 生孩子，不是老公出了精子、老婆出了卵子。一陣哆嗦後就結束的場景，事後它會伴隨著 生活的不便、經濟的困頓、母親的不適、父親的憂慮、孩子的調皮、成長的喜悅與各式不 同的難題。 不要讓 “被催生” 這件事情變成你的枷鎖，你先問問自己，你想生嗎？想負擔嗎？想要 愛一個孩子並願意溫柔的承受他未來的一切嗎？ 記住，孩子，一條沒辦法回頭的江湖路。 ----- 你可以有後悔的念頭，但是孩子生下來就不能回頭了 ----- 婚姻點： 我跟老婆沒生孩子，因為我自己沒耐心、我老婆跟我自己父母家的陳舊觀念也沒辦法改變 。 當我確信自己沒辦法溫柔的去接受怎麼照顧孩子、傾聽孩子，我就知道自己不是一塊當好 父親的料。 不是每部電影都應該有續集，同樣的，也不要以為生下孩子讓他完成我未竟的夢想就是一 個好的續集。 這應該不算婚姻點，但是跟自己老婆講好就好 -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 27.247.135.243 (臺灣) ※ 文章網址: https://www.ptt.cc/bbs/marriage/M.1623061798.A.A79.html
