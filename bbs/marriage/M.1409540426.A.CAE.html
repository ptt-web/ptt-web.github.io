※ 引述《bearshoney (HI)》之銘言： : 原本就沒有和婆婆一起住 : 因為有一些溝通上的問題，加上婆婆對我講話刻薄，雙面人，所以內心對她有些恐懼 : 一直以來，都是希望保持距離以策安全 : 婆家，娘家就在附近 : 每次回來，我就在婆家待個幾小時或一個白天，然後回娘家住 : 我不住婆家，也不要求先生陪我回娘家住 : 最近婆婆整個大爆發，指責我的行為，說哪有媳婦不住婆家的？ : 還說每次親戚 朋友剛好來家裡，都問說妳媳婦呢？ : 她覺得很沒面子，開始對我先生施壓，說我這樣不成體統 : 我覺得要住哪裡是我的自由 : 況且我也有回婆家，只是很少在那裡過夜，為甚麼三姑六婆一兩句話，我就要去聽從？ : 接著，先生可能也被逼瘋了 : 開始說什麼他的朋友娶的老婆，每個都跟公婆住，沒有人像我過這麼爽的 : 然後開始說我不想融入他家 : 我說你媽講話刻薄，我要怎麼融入她？他之前聽了就衝去把我婆婆罵一頓，我心想他這樣 : 做只會把我們關係用更糟 : 長期以來，一直為這個問題而吵，之前就討論過先生愛拉著我陪他一起回家， : 現在又說除了回家，還要過夜才像是他們家的人 : 我承認婚後我看清婆婆的真面目，能閃多遠就閃多遠 : 也不是很喜歡先生的家人 : 但他就覺得我嫁了，就應該遵守一些規則 : 從未想過現在連回家陪父母住，都是一種奢求 : 先生還對我說，他當夾心餅乾很痛苦，還說我這樣做，也會害他跟他父母的關係變糟 : 還說天底下沒有一個媳婦像我這樣，一年在婆家過夜只有過年及重要節日 怎麼會沒有。我啊！我大嫂啊！我們只有一年幾個節日在婆家過夜而已， 並不是「只有你」這樣。 我不知道這種狀態要給什麼建議，說說我的想法而已。 1.你婆婆可能以前也是被她婆婆規定不能回娘家 所以，她現在「升職」成婆婆了，希望把前代的規定傳承下去， 自己並沒有意識到她正試圖把她當時的痛苦轉嫁到你身上。 破解的方法：如果有大小姑，那就好辦。請問一下大小姑是否常回娘家？ 有的話那就直接跟你老公面前戳破。「她們也常回娘家啊」 2.你跟他結婚，並不是「你嫁到他家」，先破除這個基本概念，才有可能往下講。 況且，他當夾心餅乾很痛苦，難道你被婆婆酸言冷語不痛苦？ 試著告訴他，他是婆婆的兒子都可以逼成這樣，你一個外來者會好過到哪裡去。 3.「他的朋友的老婆每個都跟公婆住，哪有你那麼爽的」 請提醒他，在結婚那一天他是如何承諾你爸要給你過「爽日子」的， 既然他都知道他朋友們的老婆跟公婆住沒有很爽，請問為什麼要逼你過不爽的日子。 為什麼要打破他自己跟你爸許的承諾。 以上。 （大嫂歹勢我出賣你了XD ） : 千錯萬錯都是我的錯 : 這樣吵真的好累，可是我也不想讓步 : 先生就罵我，都只想到自己 : 都沒替他著想 : 我這樣做有很過分嗎？ -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 111.250.44.84 ※ 文章網址: http://www.ptt.cc/bbs/marriage/M.1409540426.A.CAE.html
