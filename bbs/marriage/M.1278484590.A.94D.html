我覺得pooh大寫得很中肯。 我也覺得關於再婚最重要的， 第一，應該是對於新人生各方面的規劃。 第二，應該是對過去人生或婚姻關係中，所留下的各項事情人事與責任的安排。尤其是有 孩子的人。 第三，才是時間點。我也佔成時間要慢一點，緩衝期要長一點，但那無關旁人眼光，而是 要有時間餘裕，來規劃第一二點的事務，以及有充裕的時間來尋找並認識下一個共度人生 的對象及其生活背景。 其實我想這些大家都了解，我也不想批評版友處理婚姻及孩子的態度， 只是我想說的是，希望有些父母明白， 對孩子來說， 父母婚姻維持，但無法照顧而被祖父母隔代教養， 跟 父母離婚，且無法照顧而被祖父母隔代教養， 孩子心裡的感受差異是很大的。 如果沒有帶走孩子，父親一邊再戀愛，母親一邊也去再婚了。小孩心理的痛一定無可言喻 。也許年紀很小時不會有狀況，但之後就會影響。 如果無法妥善安排孩子，帶在父或母身邊，是否再婚的計畫要延緩。 我並不是認為，父母要為孩子犧牲一切，父母當然可保留追求幸福的權利。 但是重新追求幸福時，我認為要將自己的責任-孩子，做最好的安排， 畢竟那是自己選擇的婚姻所帶來的責任。 看了版上的另一個例子，女人選擇離婚，放下一切包括孩子，追求自己的人生。 我想到的是，那她是否對孩子做了交代？是否讓孩子了解母子之間不能斷的責任與牽絆， 是否仍將自己對孩子的義務盡了？ 人生，本來就是不斷的選擇，我同意每個人都有追求幸福的權利。 但是相對的，前提是你必須以尊重他人的方式(例如不能用外遇拋家棄子的方式) ，並且不能拋棄自己的義務，捨棄自己的責任。 其實活到現在，我想，人生沒有絕不後悔的選擇，選擇的重點在於值不值得(客觀來看)。 以及對自己的選擇負責。什麼是負責，不是指選了以後不能改變傻傻衝。 而是例如因為新的選擇捨棄舊的責任，那也是不負責任的表現！ 我認識一個父母離婚，住在祖父母家的朋友， 她父母也認為祖父母把她照顧的很好，但是父母親一邊把他託給祖父母， 一邊兩人各自交男女朋友婚嫁。 老實說，我朋友創傷很深，她認為父母有時間能力再交友再婚，卻沒時間帶著她。 當然她長大慢慢恢復，體諒父母。 其實我不是針對個案在說，只是說一下自己的想法~~~結果有點離題了...... ※ 引述《poohw (pooh)》之銘言： : ※ 引述《it855407 (盜小君)》之銘言： : : 想請問各位對於離婚後的男女，關於再婚的"時間點"，有什麼看法和建議?! : 以我鄉民的眼光來看 >1年 : : 就我自己的部分，我是會傾向拉長再婚的時間點。 : : 因為第二次的選擇，總是希望觀察彼此更久一點，讓時間多考驗一下彼此的適應性。 : : 還有就是擔心外在的觀感。 : : 我所謂的"外在觀感"是指旁人可能會覺得太快投入新的婚姻，會擔心是不是只是想找個 : : 脫困的浮木，未必是真的考慮了清楚....等等因為擔心而產生的不看好。 : : 不想讓旁人為自己擔心，所以對於"再婚"這件事我有著很大的猶豫。(以上是個人想法) : : 不過如果另一半基於某些考量，希望彼此能不要拖太久，在這狀況下，雙方的想法是 : : 有所衝突的。 : : 這時雙方如何在想法上取得平衡點，就是需要透過不斷的溝通去取得共識。 : : 至於怎樣的講法比較適當，不至於讓對方以為自己根本不想與其再婚呢?! : : 這部分還懇請板友提供建議。 : : 另， : 不過坦白說 我覺得小君的考慮很怪 比起"時間點" : 更重要的是 你對於生活 過去婚姻留下的種種 (財物 孩子 親屬關係 朋友圈) : 整理安排好了沒 對於未來新的婚姻關係 規劃好了沒 : 全部都妥當 要隔天馬上登記去也很ok : 該處理的事沒弄好 等個50年也不夠 : 舉個不倫不類的例子 有點像責任制的工作 問說什麼時候打卡才好 : 我覺得要考慮的 首先是你人生規劃吧 : 現在看似你有很多選擇的機會 (小孩監護權 再婚與否) : 但是這些機會抓不好會全落空 : 請先決定好 你如果得放掉一個 你想留哪個?
