我認識一對美國中年夫婦 頂客族 也是雙薪家庭 養了兩隻貓 住鄉下兩層透天 也是雙周請人來家裡打掃 先生熱愛工藝,多把休息時間花在自己用原木做家具,修繕家裡東西 太太熱愛園藝及縫紉,把後花園弄得美輪美奐,並兼種蔬小菜園,常常可以吃到新鮮蔬果 剛開始知道他們沒有小孩 還請打掃阿姨 也是很驚訝:"你婆婆不會不高興嗎?" 太太回答: "不會啊，婆婆知道我們都有工作,很忙,沒空做家事...." 當然，他們不是真沒空做家事,只是時間都花在自己的嗜好上, 那些家具,或蔬果,也不是市場上買不到, 只是在自己喜歡的事情與DIY的感覺,跟被迫不得不"去打掃"的家務 他們選擇了前者! 的確 如有些網友說,夫妻協商,你們高興就好~~~ 既然經濟能力許可,為何不把時間花在你認為有意義,你喜歡的事情上 讓自己快樂一點過日子~~ 否則即使家裡窗明几淨,自己卻因打掃而疲憊不堪, 也失去在乾淨的環境下本應有的享受,不是嗎? 題外話, 真的無法理解,為什麼很多人無法接受找人來家裡打掃, 卻要把自己或家人搞得很累的這種邏輯(特別是在經濟許可下)? 我夫家也是四層透天 已經多年的過年大掃除都是由所有媳婦包辦(媳婦平日是職業婦女) 某年當我們提出請人來打掃的建議,立即被封殺~~ 理由，當然不外乎是"費用"之考量 (即使子孫都願意負擔這個費用...) 最後不了了之,那一年的年前大掃除,又是媳婦包辦了.... 我想,已經有免費的媳婦,誰還會認為有需要找人來打掃呢? ※ 引述《girling (晴空萬里)》之銘言： : 我和老公目前兩個人自己住，是三樓透天，每層大約三十坪 : 因為我從小家裡就有請阿桑在打掃，而且媽媽很愛乾淨，所以家裡總是一塵不染 : 我只需負責我自己房間的物品擺放整齊就好 : 其實我家也不是多有錢，只是爸媽都在上班，家裡小孩多， : 媽媽想說請個阿桑來固定打掃，減輕負擔，一請也就上癮，直到現在 : 我對於自己的打掃功力實在沒有把握，也不喜歡做， : 所以婚前就和老公討論希望我們家也請阿桑，價位都打聽好了， : 兩週一次，一次約1500 : 我們兩個都是公教人員，月薪合起來快十萬，房貸一個月一萬五， : 兩邊都不用給孝親費，我覺得負擔阿桑的費用很輕鬆 : 但老公覺得我們沒有小孩，兩個又年輕，請個阿桑好像太誇張 : 堅持自己掃就好，如果我不想掃，他掃也行 : 從去年結婚後，我和老公大約兩、三個星期打掃一次家裡， : 但是我只要用吸塵器吸完一到三樓，幾乎就累到沒力氣再拖地，別說擦桌子、洗廁所 : 老公也會幫忙，其實他做得比較多，因為他也愛乾淨，幾乎做的頻率是他二我一 : 我只是出於不好意思都丟給他做，我才做 : 但家裡有養兩隻貓，雖然只有兩個人住，每次好不容易打掃完， : 過個四、五天，家裡又髒了，白色石英磚上又有數不完的頭髮、貓毛和污垢 : 久而久之我也懶了，反正很快又髒，我根本也懶得整理了 : 就演變成，誰先看不下去，誰就去打掃，但是老公最近在寫論文， : 他幾乎都沒時間也不想整理，看著亂糟糟的家裡， : 我真的很想去和阿桑約時間了...... : 想請問大家，真的都是很認真的常常打掃嗎？ : 以我們的狀況，請阿桑真的會很誇張嗎？謝謝大家... -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 70.225.179.178
