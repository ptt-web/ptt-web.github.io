他一共演過4部戲說台灣，按播出順序如下︰ 1、水并姻綠樹（正派） 2、土地公還俗（反派） 3、王勳千歲展神威（正派） 4、青春不老翁（反派） 唯一主演也戲份較多的是「王勳千歲展神威」 如果whyy大覺得他演技好，那是因為他算科班出身 據查資料顯示，他是先參加客家「加里山劇團」演出 爾後因個人在劇團演出興趣，繼而棄電機從戲劇 （意即他大學原是就讀電機系，放棄原科系轉考文大戲劇） 不過他的作品很少，幾乎看不太到他有什麼演出 這點是最可惜的 ※ 引述《whyy (melody)》之銘言： : 上一檔[王勳千歲展神威] : 對黃振尉演文昌醫生還滿有感覺 : 所以來談談 : 會對黃振尉有印象 : 是他在戲說[土地公還俗]演個反派 : 覺得他演得不錯 : 起先以為他是第一次演戲說 : 結果一搜查才知幾檔前[水井姻緣樹]他演個智障角色 : 而這個角色我明明還有印象 : 卻無法把這兩個角色聯想成一個人演 : 當時就覺得這個人演技真不賴 : 對他印象就滿深刻 : 經過大半年他再演[王勳千歲展神威] : 就不會記不得他了 : 雖然說反派角色讓人印象深刻 : 但真正會讓觀眾有好感度會想追戲還是需要一個[非反派]主要角色 : 而文昌醫生不只非反派還是個純正派角色 : 造型穿長袍不錯,若是青衫會更佳 : 不是無能正派,他會醫人又努力學武 : 聲音表情也不錯 : 雖然黃振尉外型不算是個帥哥 但總總條件加起來則是斯文得體 : 若有缺點就是拿大刀時顯不出武力值 沒有威武感覺 : 就像[鐵馬尋橋]鄭嘉穎演不出顧汝章草莽武夫感覺 : 不過他演文昌醫生還是會帶給我一些聯想 -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 220.130.11.223 ※ 文章網址: http://www.ptt.cc/bbs/DramaTalk/M.1413124187.A.58F.html
