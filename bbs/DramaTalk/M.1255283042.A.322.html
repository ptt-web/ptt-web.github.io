標題Re: [討論] 請大家多支持碩果僅存的神明戲劇

其實戲說台灣的收視率是僅次於 娘家 天下父母心之後的連續劇 媽祖聖誕進香陣仗過小 這是我提出的問題點 其實很簡單 近年政府為恢復傳統文化藝術也投入不少心力 彰化甚至還有所謂的政治遶境 先照會相關廟方去取景 之後再以剪接的方式讓聖母因為馬超的垃圾步倒退回宮內 這樣就OK啦 因為再怎麼樣 出巡的陣仗都不太可能就像本集裡面這麼小 除非是一般家庭所供奉神明要回祖廟回香進香 既然是首屈一指的宗教類本土劇 當然要教育觀眾正確的概念 ※ 引述《BAOY (抖抖抖)》之銘言： : 很多戲說台灣的劇迷都會感嘆 : 「為什麼最近常常會重播？」 : 「為什麼不找大牌來演？」 : 「為什麼神明出巡的陣仗這麼小，跟現實不符？」等等的一些疑問 : 其實這樣「愛之深，責之切」心情，製作單位都能體會 : 可是這一切都只能歸咎於「阿共仔的陰謀」（誤） : 不，是「製作費的限制」 : 大家應該都很清楚，電視台的收益來自於廣告業務的收入 : 而企業主願不願意買廣告，我想是取決於二點： : 1.本身產品的規模能否負擔起廣告支出 : 2.產品針對的消費族群 : 所以，大家可以看見鄉土型的節目就很多賣藥的，家庭用品之類的廣告 : 而都會型節目就會比較多流行商品的廣告 : 但是以上這些因素卻都不敵一項更重大的因素「經濟大衰退」 : 大家都知道去年金融海嘯以後，各行各業都陷入大低潮 : 戲說台灣當然也不例外，廣告業務收入直線下跌 : 電視台入不敷出的情況下，只好祭出大絕招「重播」 : 儘管戲說台灣在今年年初還創下高峰收視率３．８％ : 可是廣告收入卻是直線下滑，電視台在無計可施的情況下就只好重播 : 雖然電視台高層也知道這樣做會流失很多觀眾，但是不這樣做的話會造成虧損更擴大 : 「存活下來就有機會」我想這是大家都知道的道理 : 再者，為何戲說台灣的製作規模不能大一點？ : 這一點只能說「巧婦難為無米之炊」 : 大家也知道，戲說台灣可以說是台灣目前碩果僅存 : 「以台灣各地神明神話傳說為題材」的戲劇節目 : 一集十多萬的製作費，能作到這樣的品質，我可以說是全球無人能比的 : 這位我所認識的製作人，她對於節目品質上的要求非常嚴謹 : 「小品戲劇就要有更高更細緻的的質感」這句話是製作人的目標 : 既然在場面規模上不能驚天駭人，就只能在劇本上讓劇情更扣住人心 : 大家要是有機會看到這位製作人為了劇本更完美，可以不眠不休的修改討論劇本 : 窮盡各種手段要讓畫面有更細緻的呈現 : （偷偷說）甚至用腦過度到年紀輕輕頭髮花白 : 唯有喜愛戲說台的觀眾越多，企業主才會注意到這個節目 : 企業主認為有足夠的廣告效益，自然會願意買更多的廣告 : 電視台廣告業務收入增加，自然就會增加戲說台灣的製作經費 : 有更多的製作經費，就自然會有更好更大的製作團隊與場面 : 最終的關鍵因素還是在各位手上的遙控器 : 所以，請大家樓頂揪樓腳，阿母揪阿爸，厝邊揪隔壁 : 多多支持戲說台灣　～　謝謝　岡溫 -- Muy jijia gao long e hiao gong Taiyi.
