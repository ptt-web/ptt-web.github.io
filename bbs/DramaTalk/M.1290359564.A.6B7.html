標題[心得] 美人梳妝穴
時間Mon Nov 22 01:12:39 2010
※ 引述《whyy (melody)》之銘言： : 【假日完整版】11月感慨美人無美命，情字這條路為何如此難走！ : 消息來源：戲說台灣部落格 : http://blog.iset.com.tw/dramatalk/?p=2035&cpage=1#comment-4171 : 11/21【美人梳妝穴】 : 阿善師是唐山過台灣的茅山道士，走遍台灣山頭想要為父母找到一個福地安葬 : 以蔭子孫。 : 這日他在北投紗帽山下發現美人梳妝穴，只是這個穴蔭女不蔭男， : 只要有先人葬此，當出: 富貴皇后命。此時，正好小紫霞欲將小鳥埋葬此穴， : 阿善師覺得可惜，於是花兩文錢將紫霞的穴買下來，回程路上， : 他遇上火旺因為生活無著，帶著女兒小文雀欲自殺， : 阿善師於: 是將穴指點給火旺，並約定二十年後火旺要將家產分給他一半…… : 演員：邵姿菱、何依霈、丁華寵、官家宇、王麒、劉明仁、葉青田 這部戲架構跟我滿欣賞「一世姊妹情」有點像 同樣也是講因緣際會從小兩個要好的女孩因為命運不同走向不同人生 比較坎坷的女孩則對好運女孩心中充滿了嫉妒怨懟之情 巧的是演不幸的女孩幼年都是由童星梁馨如飾演 演壞女孩長大後的女演員演壞人也都演得很誇張 （沒看到片頭,所以不知道誰演誰） 當然劇情內容就不像了 戲說常演有關於風水寶穴的故事 這部藉寶穴引出的故事則比較像是港劇無頭東宮故事 我通常不喜歡看壞人囂張太久劇集 所以港劇無頭東宮也是斷斷續續看 (該劇女主角陳妙瑛巧得也演過「布袋和尚」、三立八點檔新女主角六月也是） 這齣戲說壞女孩剛成就了「皇后夢」不久就被掃地出門倒滿俐落 不過結尾太草率以致於劇情不能自圓其說是敗筆 如「美人梳妝穴」為何不能庇蔭壞女孩紫霞長久，劇情沒解釋 如阿善師先前說紫霞沒父母親人可安葬寶穴，才要紫霞騙挖好友雙眼代替 最後怎麼又找到紫霞母親的遺骸作法？ 或許兩個小時的戲無法像四個小時的戲一樣鋪排 : 11/28【澎湖七美人】 : 村子有一口井，常在半夜的時候無緣無故出現一隻女人的手…原來二十年前， : 玉女和其姐: 妹原本在村莊裡無憂無慮的生活著， : 玉女和一位賣雜貨的憨厚男人墜入愛河，原本以為能夠從此長相廝守， : 卻沒想到某天村子倭寇來襲，玉女和其姐妹為了守住清白之身，遂投井自殺。 : 死後的玉女仍然無法忘懷她的情人，於是便發誓如果有人對著香花樹求婚， : 她便下: 嫁於他。而木納的世豪剛好無意間來到香花樹下，練習說情話， : 無意間向香花樹求婚後連帶將玉手環套到樹枝上，意外的讓玉女復活， : 不計一切要將世豪帶回陰間成親，但世豪早已經了有了未婚妻， : 他該如何處理這難解的三角習題呢？ : 演員：莊奕勳、羅巧倫、陳曉菁、鍾天、王麒 這齣跟改編自俄羅斯民間傳奇故事成為英國背景美國動畫電影「地獄新娘」 不知會有哪裡不同？是我最期待的地方 提姆波頓電影我覺得棒的地方不是在於劇情 而在角色的塑造 他劇中的女主角往往有冒險的心及不讓鬚眉的行動力 男主角則多是顯得陰柔卻又特立獨行與眾不同 「剪刀手愛德華」「斷頭谷」及「地獄新娘」都是這類型 除了角色塑造外 「地獄新娘」也有超浪漫感人的劇情及畫面 不知道改編成台灣背景的「地獄新娘」是否還會浪漫？主角是否另有特色？ -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 118.232.185.102
