今天播出大結局還是很好看 很希望戲說偶而拍攝戲說單元 也可以選這類與神明、鬼怪皆無涉故事 雖是台灣以前常見傳統老套戲劇故事 但自從台語八點檔變成長壽劇之後 一部戲演不到一個月就劇情邏輯不通、角色嚴重變調讓人看不下去 對於這種傳統故事真的懷念不已 若傳統故事舊酒能套上童話、卡通、西洋故事新瓶 就更讓人驚豔了 「一世姐妹情」的主要場景---「張家」房子非常漂亮古典 客廳大塊石材裝飾牆壁也顯得非常大器 但這一兩年戲說好像都不在此拍攝滿可惜 甚至常把中庭當客廳來使用 感覺滿奇怪 演阿蕊的周星辰 說台詞腔調尤其是心中OS很像是歌仔戲班出身 不然她就是郭美珠的親戚了 演技不像當時是新人 但一些裝腔作勢卻演得太舞台化 不過這一兩年看過她的演出已經修正滿多 華承佑時期陳建隆演正邪難分的張建中 實在帥得一塌糊塗 但這一兩年看他演出 不知是演技稍有退步 還是那時既有年輕活力已經不見 或是近期都沒有演出很酷角色 很難讓人眼睛一亮 後面劇情還是有令人看起來卡卡地方 問題也是出在張建中這個角色身上(是我太喜歡張建中嗎?) 明明他是最早懷疑大娘收容碧玉居心 在劇情其他方面如懷疑阿蕊等也顯示他是個聰明的人 但當大娘因為反對碧玉跟他或弟弟耀宗交往不小心說出碧玉真實身份 卻沒見到他懷疑耀宗的真實身份 感覺很奇怪 是因為發現他跟碧玉是兄妹打擊太大？ 但我覺得若他不想跟碧玉是兄妹關係 應該對大娘一些說不清楚地方追根究底 不然也可能他大娘只是因為討厭他來騙他 就這樣罷手 連奪走家產也還給大娘 只因為大娘應也是個善良的人原因未免太虛弱 那以前謀奪家產是在奪好玩？ 用碧玉來當藉口會不會比較好些? 或許更能顯示建中長情及不得不離開張家及故鄉原因 ※ 引述《whyy (melody)》之銘言： : 以下消息來源轉錄自 : http://mypaper.pchome.com.tw/bobo945/post/1267297882 : 清朝時期，在艋舺地區有一間由政府所辦的孤兒院，叫做『育嬰堂』，專門收留被遺棄、 : 沒有父母的孩子。在這個育嬰堂裡，有一對感情如同親姊妹的女孩，因為命運的捉弄，碧 : 玉成為千金小姐，阿蕊卻變成小偷，為了扭轉命運，阿蕊不顧姊妹情，害死碧玉，李代桃 : 僵變成千金小姐，就在她歡喜享受榮華富貴時，卻出現一個和碧玉長得一模一樣的人，阿 : 蕊精心策劃的謊言是否會被拆穿呢？ : 　　而在這個家庭裡，一個驕縱的紈?子弟，原本以為父親死後，他就能接受一切的財產 : ，卻因為細姨所生的兒子設計脫產，變成一個一文不名的人，就在他落魄潦倒時，又發現 : 一個和他身世有關的漫天謊言… : ═人物介紹═　　 : 周姿君飾演碧玉　 : 梁馨如飾演碧玉(幼)　 : 周星辰飾演阿蕊 : 小芳芳飾演阿蕊(幼) : 陳霏飾演張建中 : 許定南飾演張建中(幼) : 莊濠菖飾演張耀中　 : 李文成飾演張耀中(幼)　 : 張龍飾演張金旺 : ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~ : 陳霏 應是 陳建隆 之前的藝名 : 恬娃 演大老婆的女僕 : 吳帆 演個壞人 : 恬娃和吳帆都是此劇重要關鍵人物 -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 118.232.185.102
