相關專案連結：https://github.com/bbsdocker/imagebbs 簡單來說就是做個各版本 BBS 站臺的 Docker Image 以供測試 這邊目前皆以 MapleBBS 系列為主 只要自己用的作業系統能安裝最新的官方版 Docker 即可拿來使用 == 如何使用 1. 安裝 Docker 2. 執行服務 PttBBS: (sudo) docker run -d -p 8888:8888 -p 8889:8889 dockerbbs/imageptt Maple3-itoc: (sudo) docker run -d -p 8888:8888 dockerbbs/imageitoc Maple3-WindTop-DreamBBS: (sudo) docker run -d -p 8888:8888 dockerbbs/imagedreambbs Maple2-SOB-WindDustBBS: (sudo) docker run -d -p 8888:8888 dockerbbs/imagewdbbs 3. 連線至站臺 請先將終端機環境設定成 Big5 編碼， 或直接用 PCMan / PCManX 連線到 telnet://127.0.0.1:8888 即可連線。 == 僅適用 PttBBS: 在 Linux / Mac OS X 終端機下直接 telnet 127.0.0.1 8889 即可連線。 (UTF-8 Encoding) == 註： 若有用來開放外連服務等需求，還是建議先自行研究相關細節(如網路設定、資料儲存) 不然目前仍建議還是優先按各 BBS 版本文件指示自行架站 -- 你知道PTT有小天使嗎? 祂能幫助你唷！ 小主人我來了^Q^ － ◥ ●β 、ＨＥＬＰ 想呼叫屬於你的小天使嗎？ /◥＝ ` 〈○〉 ▃▃▃ 只要按ｈｈ，就可以了唷！！ Δ 新 ◥ >_<～ // ▅▅▅ 快按下"hh"呼叫你的小天使吧！\(≧▽ν≦)ωgarnier -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 140.116.112.110 ※ 文章網址: https://www.ptt.cc/bbs/AppsForBBS/M.1534458114.A.718.html ※ holishing:轉錄至看板 BBSview 08/17 06:25 ※ holishing:轉錄至看板 BBSview 08/17 06:25
