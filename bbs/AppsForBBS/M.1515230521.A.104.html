標題[分享] 目前各版本 BBS server 安裝資訊
時間Sat Jan 6 17:21:58 2018
本來是要貼到 SetupBBS 板的 不過該板已經唯讀，且這邊板旨也明定可以討論相關的技術 所以先簡單分享在這邊 太久遠的版本就無法保證可以裝起來了.. 僅介紹一些主要 近期還可以在較新作業系統安裝的 BBS 程式版本 建議一開始先在 32 bit 環境下測試 作業系統大致上則是建議用 GNU/Linux 或 FreeBSD 來作為安裝環境 1. Current Ptt 目前 ptt.cc / ptt2.cc 正在跑的程式運行版本 可在：大部分 Linux 發行版、FreeBSD 11.x 等作業系統安裝 新手可先用 Ubuntu Linux 來作為開始嘗試的作業系統環境 熟悉相關設定的話則是更建議在 Debian GNU/Linux 下編譯安裝 相關專板： PttCurrent 板 https://github.com/ptt/pttbbs 安裝教學 Wiki: https://github.com/ptt/pttbbs/wiki 2. Maple3.10-itoc 目前現存大部分 BBS 站台在跑的程式版本就是以這個為基礎了 ( 現存 BBS 中文站臺：https://bbslist.github.io ) 建議可先用 Ubuntu Linux 或 CentOS Linux 來作為開始嘗試的作業系統環境 以相關教學文件多，且有相關討論的專版(其他相關站台裡的 itoc 版) 及目前還有在運作的專門 新聞伺服器(News Server): news.tfcis.org POST(119) 新聞群組(Newsgroup): TFCIS.personal.itoc 為其中特色 先前主要官網都在這邊: http://cpu.tfcis.org/~itoc 相關程式碼放在: https://github.com/xeonchen/maplebbs-itoc 主要官網復活前可以參考資料的據點: https://www.ptt.cc/bbs/Maple/M.1504354938.A.AED.html 其他還有早期以 Windows 也可以輕易安裝為著名的 ColaBBS 以及中國現存 telnet BBS 常用的 Firebird BBS 等 甚至是非常早以前 PttBBS 除了 Current Ptt 以外， 其他站長還有開發過在當時相當容易安裝的 OpenPtt ( https://github.com/clamtestbbs/openptt ) (詳見 ptt2.cc 的 PttSrc 版) 甚至還有以 Ptt 草創時期程式版本為基礎 改良使用者介面以及加上大量外掛為著名的 Wind-Dust BBS (又作 WD-BBS) ( https://github.com/clamtestbbs/wdbbs ) 主要著名的 BBS server 安裝版本大概是這些， 當然之後會視情況分享更詳細的相關紀錄. 而著名的 PttBBS 以及 MapleBBS-itoc 等都是以 清華大學楓橋驛站早期開發的 Maple BBS 為基礎的各個分支 以下再分享由 Dopin 大在 亞特蘭提斯站 ( telnet://125.227.52.214 ) 提供的 Maple BBS 「族譜」： >
