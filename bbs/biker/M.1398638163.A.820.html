標題Re: [問題] 初步接觸檔車的新人，CBR150是好選擇嗎？
時間Mon Apr 28 06:36:00 2014
※ 引述《jademesses (古今皆醉夢)》之銘言： : 大家好，我是即將踏入檔車領域的新人。 : 我今年大一，這將會是我人生第一台檔車，也因為是自己的第一部車，所以非常重視。從三月以來，每天觀察網路文章至今，到現在對於檔車終於有初步的認識，心中也有個大概的底。 : 在做檔車OR速克達這個決定前，我看過無數車友說明檔車是如何的不便，但說實在這些都打斷不了我檔車的興趣，也所以最後還是選了檔車。 : 另一方面，也是因為爺爺家裡有一台閒置的速克達的緣故，讓我更下定決定要買檔車。 先確認那台速可達狀況穩定 可以在你不想騎檔車 不能騎檔車 檔車有毛病時 他是可以安穩送你上課的~~~~~ : 回歸正題，自從我下定決心要買檔車後，我從原本嚮往的T1和酷龍，到目前已經徹底放棄國產車的想法，目前我心中的首選是CBR150，備選是250的車，但除了CBR250，NINJA250外，目前沒有太多想法。 : NINJA250我有個疑慮就是覺得250的車太大，畢竟我身高只有180，我比較嚮往向CBR150這樣比較小一點的檔車。 這身材250不會過大 我才171 cbr150老實說還稍微小一點點 : 也因為是學生，所以預算有限，所以8~12萬上下大概是我的極限，目前初步的最後選擇是落在這個價位的二手CBR150。 : 如果有其他可以的選擇，也希望能有大大分享。國產或進口都可以，我的需求就只是，仿賽的檔車... : 我可能比較注重加速性能，所以有慎重考慮NSR過，但NSR畢竟是老車，在爬完很多文看完很多大大的心得後，最後放棄。 原廠加速不算很快 改管改化油器就還不錯 : 但也想在這裡順便問一下，CBR150的加速是到底算好，或不好，因為: : http://ppt.cc/sG8G : 在這討論串裡的momo2代大大分享了他的心得，他的說法是"CBR150R我覺得加速只比小速克達快一滴滴"... : 這讓我有點困惑，畢竟想買檔車，也是因為要跑的贏速克達。 加速很普通 比fzr150快 但比fzr220慢 而fzr150單論直線加速 如果都油門一敗的話 不會贏大多數的新型速可達 更不要講滿街跑的改裝速可達 還有加速快 跟跑得快 是兩回事 要看路況而定 給你影片參考 https://www.youtube.com/watch?v=8JBzx2G9bwo
跑前面的是你所謂的不快的速可達 後面的是加速最快的紅牌車... : 想在這裡詢問各位大大，有關CBR150的騎乘心得，或其他建議、注意事項，或者可能有更好的選擇等等。 : 第一次在BIKER板PO文，標題可能下的不大好，還請見諒，也在此先謝謝大家。 要達成你的目的 250比較簡單 另外如果載女生的話 你會希望你當初買的是速可達 -- Bon jovi-Aerosmith-Led Zeppelin-The Who-Nirvana-Eagles-Guns N' Roses Def Leppard-AC/DC-The Ventures-Kiss-Europe -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 114.41.83.123 ※ 文章網址: http://www.ptt.cc/bbs/biker/M.1398638163.A.820.html
