Ｑ３-1：請問我可以拿別人機器附的光碟，來給我的機器裝新版的作業系統嗎？ 不行，且不說隨機光碟會鎖機型，在你電腦根本不給你安裝。 隨機附贈的軟體，也只有一個使用權，不能同時使用在兩台電腦上。 從法律上來說，你的朋友借你安裝之後(假設你裝的上去)，那麼兩台電腦都是不合法的。 [補充] 從 10.7 Lion 之後，因為在 Mac App Store 販售，所以授權方式也跟著走。 http://www.apple.com/hk/osx/apps/app-store.html App 可以安裝到所有授權供你個人使用的 Mac 上，甚至可再次下載。 這方便你為全新購買的 Mac 裝上你原先已有的 app。 ------------------------------------------------------------------- Ｑ４：那我在Windows上用的軟體,Mac有嗎? 要看該公司有沒有推出 Mac 版本的軟體，就像XBOX的遊戲，要看公司有沒有做PS3版。 不過一般軟體都是以應用功能為導向，你不出，別人會出啊。 所以功能取向差不多的軟體，都會有的。 Google 一下那個軟體名稱，加打 Mac、likely 等關鍵字，可以查到很多東西。 另外，你也可以尋找 Mac 上的替代軟體 http://alternativeto.net/software/?platform=mac 比如我打入「Visual Studio」，就會出現一堆程式開發工具作為替代。 替代軟體跟原本軟體不一定一樣，但是可能值得一試。 ------------------------------------------------------------------- Ｑ５：那有Office嗎? 辦公室軟體目前比較常見的這幾套： *微軟Office 2008(2011) 付費軟體, 微軟自家的辦公室軟體，英文 http://www.microsoft.com/mac/default.mspx *iWork 09 付費軟體(有試用版)，蘋果自己出的，非常有蘋果風格的軟體，有中文 http://www.apple.com/tw/iwork/ *OpenOffice.org 免費軟體，有名的開放源始碼軟體。 http://porting.openoffice.org/mac/ X11版是 Linux 直接移植的版本，需要安裝 X11 來用，而且不是 Mac 原生介面 建議使用重新寫過的 Aqua 版，介面跟操作上會比較符合 Mac 使用習慣。 *NeoOffice (其實是最早的 OpenOffice 重寫移植版)免費、好用、有中文，推薦使用。 http://www.neooffice.org/ *LibreOffice 算是 OpenOffice.org 後續版本。(這是另一段八卦了... WINE:他說要個視窗啦。 OS X:喔… (畫個視窗) 大概的概念就是這樣… 所以相容性不是很好，不過如果能用的話，速度一般都不錯。 2.CrossOver 標準版39.95 專業版69.95 http://www.codeweavers.com/products/cxmac/ CrossOver 是 WINE 的商業版本，做了很多的包裝， 以及針對一些常用程式，做了執行上的修補，理論上相容性會好很多。 另外還有一套叫 CrossOver Game，是特別為遊戲(尤其是DirectX)做過強化的。 http://www.codeweavers.com/products/cxgames/ 這兩套在他們官網，都有使用者回報的相容性列表，可以參考看看。 其實，在 Mac 上使用 Windows 應該是當成一個萬不得已的決定， 如果你還是比較常用到 Windows 的功能，平常也不會使用 Mac 的軟體， 那麼市面上很多 Windows 電腦比較適合。 還有, 提醒一下 Windows 作業系統為版權軟體，買蘋果電腦也不會隨機附贈。 請留意一下取得管道是否為正版軟體，也請勿公開徵求。 ------------------------------------------------------------------- Ｑ１３：那 BootCamp 和 Paralles Desktop/vmWare Fusion/Darwin 那些的有何不同啊? BootCamp是由官方支援的雙開機方式，其他的都是用模擬的方式虛擬出 Windows 環境。 而 PD/VmWare/VirtualBox 這種虛擬軟體，模擬的是完整的 X86 機器， Darwine 則是純粹模擬 Windows 的指令集。 大概來說，比較如下： BootCamp 虛擬機器 模擬指令集 方便性 差<------------------------------->佳 相容性 佳<------------------------------->差 執行效能 佳<---------------------->差(兩個差不多) ------------------------------------------------------------------- Ｑ１４：好吧,那那一台Mac適合我啊? 主要看需求囉… 首先看有沒有攜帶需求，決定筆電或桌機
