標題[閒聊] 測試 PCMan Websocket 連線
時間Mon Jun 12 13:08:05 2017
目前嘗試將 PCMan 加了 Websocket 連線支援, 不過想必會有些 bug, 先釋出這個測試版, 希望大家能幫忙測試. < 系統需求 > Windows XP SP3 以上 (再戰十年) < 如何安裝 > 下載: https://goo.gl/DJy6EY 解壓縮後選擇 PCMan 或 PCMan Combo 資料夾裡的 PCMan.exe 直接執行. 兩個版本 PCMan 和 PCMan Combo 均為 Portable 版， 有獨立的設定值，不會影響系統內已安裝的 PCMan。 如果打開顯示缺少 vcruntime140.dll 表示未安裝 「Visual Studio 2017 的 Microsoft Visual C++ 可轉散發套件」 請至下列連結下載： 32位元系統(x86) https://go.microsoft.com/fwlink/?LinkId=746571 64位元系統(x64) https://go.microsoft.com/fwlink/?LinkId=746572 < 如何連線 > 使用我的最愛裡寫有安全連線的選項連線, 抑或在位址列輸入以下: wss://ws.ptt.cc/bbs 即可連線至 PTT. < 常見問答 > Q: 為什麼圖示這麼大? A: 這是由原始碼直接編譯的版本, 不包含一般版 PCMan 的圖示. 未來如果正式釋出或許會換成大家熟悉的圖示. Q: Windows 說我沒有 vcruntime140.dll A: 請見上面安裝說明. Q: 我發現 bug 了!!1 A: 請記錄詳細的操作流程, 最好也能附上錯誤時的螢幕截圖, 將這些資訊寄站內信給 robertabcd, 或是 email 至 robertabcd [at] gmail.com Q: Code 在哪? 怎麼編譯? A: https://github.com/pcman-bbs/pcman-windows/pull/5 見 Dependencies.txt 檔案. Q: 這個版本的 PCMan 是官方版本嗎? A: 不是. Q: 還有 PTT 之外的 BBS 站支援 Websocket 連線嗎? A: 目前沒有. Q: 為什麼做這個? 不是已經有 SSH 了? A: 目的是希望大家都有方便加密連線能夠使用. 一般手機 App 都已支援 SSH 加密連線, 唯獨 Windows 下無 BBS 瀏覽專用的 SSH 客戶端可用. (PuTTY/PieTTY 很好但是並沒有 PCMan 這麼多功能) 不選擇 SSH 的原因是不太好做 key rotation, 也沒有 CA 可以使用. 選擇過 https 的 Websocket 可以直接使用現有的 CA 基礎建設. -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 140.112.30.38 ※ 文章網址: https://www.ptt.cc/bbs/PCman/M.1497244087.A.23E.html ※ 編輯: robertabcd (140.112.30.38), 06/12/2017 13:35:25
