判決主文： 未達處罰標準，不予開罰。 判決說明： 本板所要處罰之不當名片檔，主要處罰對 特定個人、特定 ID 之謾罵或其他相類 似侵害他人法益之行為。(參見本板對類案 #1GQIF45J (Violation) 之說明。) 惟本案被檢舉人之名片檔既未直接指稱特定對象，也無普羅大眾可旁敲得知之對象，且觀 作家魯迅「墳．論『他媽的』」及張愛玲「面子理論視角下詈詞『他媽的』詞匯(彙)化研 究」，皆認為「他媽的」是純粹的口頭禪，縱有輕蔑之意，但沒有貶抑他人的語意(惟若對 具體對象使用，仍有可能侵害他人法益而構成本板所稱的不當名片檔) 綜上所述，本案被檢舉人仍屬言論自由之合理行使，故不予開罰。 申訴救濟： 如不服本判決，得表明理由，至 Pttlaw 板發文申訴。 ※ 引述《QQKKQKQK (QK啦!)》之銘言： : (檢舉水球或信件違規，應將證據由個人信箱直接轉錄至本看板，並不得為任何編輯。 : 並請變更其標題與檢舉文一致。違反規定者得駁回之。詳情請見置底文檢舉規範。 : 另，檢舉點歌違規請附上日期。) : ┌╮* * ＊ * ~ * * ╮ ☆ ＊* ˙οξ∵ 。﹒ : ╰┘。 ╭═╮╮╭╮╭═╮╭═╮╭═╮╭═╮╮ ╮╭╬╯╰╦╮╭═╮╮═╮ ∴ ★ : ˙· ║ ║╠╯ ║ ║╰═╮╠═╝║ ║ ║ ║ ║ ║ ║║ ║ ∴ : 。∴ ║═╯╰ ╰═╯╰═╯╰═╯╰═╯╰═╰ ╰╯╰╩═╰═╯╰ ╰ ．┌╮ : ο ☆ ╯* * ＊ * ~ * *～ ψCAMILLECHOU ξ*＊~ ＊ * ~ * ˙ ∴ξ╰┘ : 　 ╭═════════════════════════════════╮ : ψ 檢 舉 人︰QQKKQKQK : ψ 被檢舉人︰l11111111 : ψ 檢舉事由︰不當名片檔 : ψ 檢舉說明︰ : （五）不當編輯或使用暱稱、簽名檔、名片檔或其他個人資訊設定者。 : ψ 所附證據︰若為信件水球證據,請記得直接自信箱轉錄至本板, : 轉錄後得事後刪除涉及隱私及個資之文字. : 《ＩＤ暱稱》l11111111 (一一) 《經濟狀況》普通 : 《登入次數》3845 次 (同天內只計一次) 《有效文章》98 篇 (退:0) : 《目前動態》不在站上 《私人信箱》最近無新信件 : 《上次上站》05/23/2022 16:55:56 Mon 《上次故鄉》42.72.89.28 : 《 五子棋 》 0 勝 0 敗 0 和 《象棋戰績》 0 勝 0 敗 0 和 : "幹你媽的Q三洨！？" ▁▁▁ : ◢◢▇▇ ▁▂▃▄▅▄▄▃▂▁ : ◢██…||| ⊙目）█████…████◢▇▇▇▇▇▇▇ : ▂▁…████⊙██████…███…███…◥▁▁▁ : ◥◥不 ▇▅▅▄▃▂▁▂▃▅ : ψQSWEET ◥▁▁▂▃▄▅ : ╰═════════════════════════════════╯ : ＊上述資訊請填載明確，秉實詳述，充分真實，否則不予受理。＊ -- Ptt法務部法務 -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 61.14.210.235 (韓國) ※ 文章網址: https://www.ptt.cc/bbs/Violation/M.1653313712.A.A9A.html
