時間Thu Apr 3 03:41:05 2014
不管是經濟, 社會, 政治議題 要讓每個人完全了解意涵是不可能的事情 社會上由各式各樣的人組成 有各種不同的專業 非專業的看得霧煞煞 專業的在不同領域上可能也無法完全了解 對複雜議題沒興趣的更不可能花時間去看 服貿議題這種經濟中又牽涉到政治, 法律 就算把所有文件內容都看完 把所有相關新聞全部看完 就真的可以知道要如何處理嗎 就真的知道要用什麼樣的心態去面對嗎 別再說什麼媒體誤導, 民智未開 有誰可以保證他完全理解這個議題 然後可以說服每個人自己觀點是正確的? 說要退回這個版本的服貿 不可以出賣台灣 那麼對台灣好的版本在哪裡呢~ 反對的人有沒有辦法提出更好的意見呢? 只提出反對意見, 卻無法提出正面建言 說服力真的不夠 社會不是只由參與這次學運的人所組成 更多的是無動於衷, 看熱鬧的平凡百姓 你可以批評他們怎麼這麼不關心時事 怎麼這麼不愛自己的國家 很抱歉, 你所處的社會就是這樣 不管怎麼樣都會有一群冷眼旁觀等著看戲的人 你很難說服他們要跟你一樣熱血為國家 在他們眼中 我自己的生活好比較重要 而在民主社會中 這樣的冷漠也是得被尊重的 如果社會只有一種觀點, 那會是一種鄉愿 這次反服貿議題的觀點有很多 但通常都只圍繞在一個層面上 程序正義, 或是統獨問題, 或是弱勢勞工 因為A不行 所以行不通 因為B不可以 所以無法認同 所有見解幾乎都只限於 "這樣不行 所以要反" 而沒有 "這樣不行 所以得這樣做修正才能過" 的積極做法 一味提出問題, 提出反對的理由 卻未提出具體的做法, 未提出更全面的解決方案 這樣自然是無法去說服那些冷漠的人 "我已經知道你反對的理由, 接下來你有什麼更好的做法呢?" "你反對的很有道理, 但我更想知道要如何才能變得更好?" 我相信沒有一個專家可以完整回答, 怎樣才能更好 學法律的, 可能對經濟了解不如經濟學家 學經濟的, 可能對政治問題不夠敏感 學政治的, 可能太重視操作手法而忽略問題的本質 這是一個龐大且複雜的議題 賣台的政府會犧牲國民利益 但換成你我來做也不見得能做的完美 因為不管怎樣總是有人得利有人受害 總是有人贊成有人反對 一個決定, 對51個人有利, 對49個人有害 這51個人說民主是少數服從多數 另49個人說這是多數的民主迫害 社會是由這麼多的族群組成 要讓所有人思想與利害關係一致 是威權, 而非民主了 ※ 引述《balahaha (小妹哥)》之銘言： : 媒體自己搞不清楚+存心讓大家誤解的媒體跟政治打手 : +怕簽完ecfa之後很難選所以把整個學運抹成不簽服貿也沒差的民進黨 : +自己口號混亂外也容易讓人誤解+臺灣一堆白痴與莫名的意識型態 : 原po你怎麼會覺得身邊的人都搞錯很意外呢？ : ※ 引述《ChianNiau (小千鳥我愛你)》之銘言： : : 最近這幾天跟家人、親友、路上看起來可以聊的路人、靜坐時旁邊的阿姨...等聊天 : : 發現還是很多人認為所謂退回服貿=死都不可以簽=學運在阻礙經濟 : : 當然我會再三解釋其實是這版本不能簽，因為內容....以下省略 : : 然後所謂退回服貿其實是退回此版本，待訂定遊戲規則後再重新簽訂一份 : : 真正對台灣好的服貿 : : 有的人聽了會靜靜的思考 : : 有的人聽了會猶豫，繼續嘴砲一下 : : 有的人聽了你會知道他是鐵桿9.2 : : 但我發現，還是很多人不相信，尤其是長輩。 : : 可能是某幾台媒體的渲染、可能是極少接觸外界資訊，僅靠街頭巷弄口耳相傳（樁腳） : : 就像那些我們怎麼講他都堅持朋友/電台才是對的的親人 : : 真的要用證據狠狠打臉才會清醒的去思考 : : 有點無力 : : 有時候我在想，到底還有什麼方法可以當"證據"可以當七大武器之首 : : 如果下次在登報紙，只出現幾個大字 : : 退回服貿！我們不想出賣台灣！ : : 我們只想簽一份真正對台灣好的貿易協定！ : : 當然這只是我臨時想的，肯定有更好的說法。 : : 這樣的話是不是能更吸引到那些誤會的人，更能讓不清楚但有點興趣的人去查探詳細呢？ -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 220.129.132.171 ※ 文章網址: http://www.ptt.cc/bbs/FuMouDiscuss/M.1396467667.A.09C.html
