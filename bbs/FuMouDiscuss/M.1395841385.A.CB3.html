標題Re: [閒聊] 「退回服貿」口號是不是容易落人口實？
時間Wed Mar 26 21:43:03 2014
真的，社會上很多人都不懂服貿，口號與標題， 很容易深植人心。 退回服貿 = 不要服貿 = 逢中必反 = 鎖國孤立 = 重創經濟 ..... 無限迴圈的想像 .... 讓得利的工商團體，或是不知事情大條的民眾，有反擊的空間。 不過，我個人覺得『 重審服貿 』，這個詞比較好。 重審是指，我支持服貿，但是我不要黑箱作業的服貿。 重審是指，我要維護民主國家，正當的立法審議程序。 重審是指，我要更好的服貿，不是財團政客私相授受的服貿。 一但，讓服貿真的實質審查 (在委員會審查，不是院會喔)， 那將會真的非常精采。 各相關部會官員，都要列席報告，立委可以提出質詢， 甚至要求官員修改內容。 黑箱將會打開，太陽花會死盯著它，讓它見光死。 誰去談的？ ( 幹！原來是你們這些財團、政客 ) 為啥談這些項目？ (馬的，原來你們都私相授受，大搞利益輸送 ) 誰爽翻了？ ( 喔 ~~ 原來財團政客爽翻天了！ ) 誰虧大了？ ( 知道虧大的人，會群起激憤 ) 數據報告在哪？ ( 經濟部那種濫報告，曝光會被鄉民電死 ) 補償損失預算的錢要從哪來？ ( 操！國家財政困難，根本沒那麼多錢 ) ......等等 ，絕對很精采。 最後，民眾發現，這根本是，寧靜和平統一台灣。 老共隨經濟發展，策略思維也進步了！ 銀彈(金錢)買下台灣，把台灣香港化，完成祖國統一大業。 一但所有東西攤在陽光下，KMT 還想用人數優勢，強渡關山 ，那以後選舉不用選了！ 真相大白，政治人物沒那個膽，敢做出忤逆的事情，自絕政治生涯。 況且，馬卡茸頂多剩下兩年，白痴才會跟他陪葬。 這時，若再配合網友募捐買電視廣告，不斷放送， 中國共產黨培養的首席評論員阮次山，在節目中的那席談話： 『 因為過去台灣與中國已簽訂了 18 項協議，沒有一條需 要逐條審查，全都是直接打包通過，立法院若逐條審查 《兩岸服務貿易協議》，等於馬英九背叛「祖國大陸」 』 學生也不用辛苦，死守四行倉庫，光是排山倒海而來的民意， 沒有一個獨裁的強人，可以受的了的。 最後，就是 ---- 牆倒眾人推..... 。 ※ 引述《joanrey (發呆中)》之銘言： : 這幾天紛紛看到各大團體呼籲，「支持服貿！」 : 不支持就會如何如何 : 感覺就好像說「退回服貿」= 不支持服貿 : 然後學生們不支持開放的大帽子就隱然被扣上了 : 接著就可以套上爛草莓，沒實力所以怕競爭等等的論述 : 我覺得口號若是「重簽服貿」是不是更有說服力？ : 因為一堆 沒時間 不願了解 沒能力了解的群眾 : 跟他們說黑不黑箱根本就無法理解!!! : 對他們來說「XXXX」這樣的口號就是最最最最最最簡單的懶人包 : 對他們來講「退回服貿」聽起來就像是不支持對外貿易協定 : 也就是不支持開放，搞鎖國，怕競爭 : 所以是不是喊「重簽服貿」會更有說服力和感染力！ : 而且重簽勢必就要退回阿！ : 且重簽聽起來就更具有建設性 : 聽起來也比較積極~~ : 民調不也顯示超過半數的群眾支持重簽嗎 : 重簽還簽不簽得成是一回事 : 一旦贊成重簽，等於就是贊成退回服貿啦！！（擊掌） : 這樣是不是可以省去很多將「退回」誤解成「再也不簽」的壓力!? : 就算整體活動的口號為了前後一致不適合改變 : 但起碼鄉民們做的廣告是不是可將「重簽服貿」與「退回服貿」勾勾纏 : 讓民眾只花3秒鐘！ 就將『退回=重簽』做一聯結 : 以此達到太陽餅與太陽花在問題本質上是一樣的論述呢！！！ : 不要再說 我們不支持開放與簽定對外貿易協議了哦>.* -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 114.40.206.142 ※ 文章網址: http://www.ptt.cc/bbs/FuMouDiscuss/M.1395841385.A.CB3.html
