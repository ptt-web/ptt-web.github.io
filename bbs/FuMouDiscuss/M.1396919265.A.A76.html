標題Re: [心得] 將能量轉換成罷免立委
時間Tue Apr 8 09:07:43 2014
在討論這個話題前，先確定一個前提 國民黨是全世界最有錢的政黨 我國代議制度為什麼失靈? 因為立法權被行政權透過黨意間接挾制，國會無法自主 黨意為什麼可以凌駕民意? 因為選舉要有樁腳和金錢，國民黨的黨產和一堆1945年劫收來 的地方黨部，與地方派系掛勾等等要素提供了候選人重要的競選資源，換而言之，國民黨 提供了一條當選的捷徑，西瓜都能當市長。所以這些人往往本身沒什麼實力，不靠黨就垮 了。所以做什麼當然要以黨意為優先，別跟自己的位子過不去。 在這個情形下，不要期望看到國民黨員良心發現。除非自己很行的，行到不靠黨也能選的 相對來講，民進黨沒有黨產，沒有辦法提供足夠的選舉奶水。所以黨對候選人的約束就沒 那麼強，很多有實力的候選人脫黨還是能選得上。相對來說，因為沒有黨的資源，所以這 些候選人往往都是一時之選，當選後也必須兢兢業業，否則翻盤的機會就高了。這種情勢 之下，例如民進黨南部縣市首長兢兢業業經營，因為他們的腳都被釘上鐵釘。 民進黨能提供的是舞台，只要沒舞台就會有黨員出走，這和國民黨不一樣的。 大家會說可是藍綠一樣爛，財團兩邊送錢。民進黨沒有黨產，當然為了選舉不得不接受政 治獻金。沒有錢，如何和全世界最有錢的政黨打選戰呢? 一旦接受了政治獻金，某些程度 上還是必須和金主低頭。然而這和國民黨長期與財團利益交換的情形並不相同。民進黨不 接受政治獻金，無法和國民黨競爭，這個國家一黨獨大的情形就無法改變。 所以我們要大家建立一個共識，宣傳一個理念，就是如何選出無法被黨紀綁住的政治人物 要解決這個問題該怎樣做? 關鍵就在你我身上，睜大眼睛選賢與能，讓民意超過財團的 影響力，超過黨的影響力。支持真正優質的候選人，別再挺丁丁了。當然，要達到百分之 百優秀的候選人不多，但是至少選一個比較聽我們話的吧! 所以，想想問題不只是聽從馬意的立委，而是整個結構的問題。若要改革代議制度失靈， 先從罷免黨意立委開始。 另外 希望太陽花決策團隊能理解到，台灣社會普遍接受的透過間接民主(即選舉)來解決問題。 公民憲政會議不只是產生的機制有許多問題要克服，最重要的在於你還要花很多力氣時間 說服百姓接受新觀念，這不是一個聰明的做法。 ※ 引述《worldleader (那飽滿的稻穗)》之銘言： : 剛剛好好聽完了黃國昌老師的意見 : 的確很有道理 但聽完也很生氣 : 本來個人是反對學生如此癱瘓國家運作的行為 : 但少數立委（也許是少數）長期瀆職也是事實 : 服務貿易協議的審查只是其中一個引爆點 : 既然立法委員沒有善盡監督行政之責 : 明知審查於法無據 : 卻恐怕任期屆滿不被黨團提名或者種種個人因素而噤聲 : 因而放任影響民生重大法案（服貿只是其一）草率通過 : 而不是退而研究 建立一套能取信於民的審查機制（此次命名為兩岸監督條例） : 還得勞煩學生被社會公幹 翹課 來喚醒社會 : 理應好好揪出幾隻雞來殺 讓整個佔領立法院的行動有一個回到法制常軌的結局 : 否則曠日時久 原本合理的訴求會被因長期佔領國會的不正當行為的批評聲浪所掩蓋 : 如果能將原本的“反”服貿轉化成為罷免廢材立委的訴求 然後化為實際行動 : 那麼比之二十多年前的野百合學運最終促成民選總統 今朝的太陽花學運會因為 : 落實罷免機制讓台灣民主真正的深化 也讓眾多幹ㄍㄧㄠ立委的台灣民眾好好出 : 一口鳥氣 : 所以 誠摯地希望 讓反服貿成為開端 讓罷免孬種立委成為實際行動 不然 : 不管再怎麼耗 還是沒有正當性 終究要由民選的立委來解這個鈴啊 -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 219.85.135.203 ※ 文章網址: http://www.ptt.cc/bbs/FuMouDiscuss/M.1396919265.A.A76.html
