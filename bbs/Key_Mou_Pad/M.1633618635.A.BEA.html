標題[鍵盤] ortholinear鍵盤及其他一些心得
時間Thu Oct 7 22:57:13 2021
從去年封城開始WFH，陸陸續續買了一些冷門鍵盤 其中一半是ortholinear人體工學鍵盤 我的需求基本上是空集合，要真對稱人體工學、數字鍵在左邊、機械鍵盤熱插拔、藍芽 所以每一款都有些滿意及不滿意的部分 順便給中文圈參考一下很少有的使用心得 一、Typematrix 2030 https://i.imgur.com/lazSNcs.jpg
這把其實是三年前買的，用了挺久 之前工作一直用筆電，完全忘了機械鍵盤存在 所以選這種直列鍵盤時只考慮到配置和功能鍵 對，這把是薄膜鍵盤 但很奇怪的是觸發壓力是55g 導致我這個指力很重的人打了兩年才忽然想起還有機械鍵盤 優點： 1. 鍵位功能非常完善，原生支援Dvorak (雖然我沒在用) 雖然不是100%鍵盤但透過numlock或fn組合可以使用數字鍵區 事實上這把是尺寸和鍵位功能上我最滿意的 2. 以薄膜來說打感不差 3. 偏輕(709g)，攜帶還算方便 4. 價錢不貴，加上運費和保護套是$136.5USD 缺點： 1. 別裝保護套，裝了手感超爛，但不裝很容易髒 2. 不是機械鍵盤 3. 僅有有線模式且非鍵線分離 結論：目前放在公司使用，打起來不會吵到同事，功能數字鍵區做分析報告很方便 二、X-bows Knight Plus，佳達隆青軸(光軸) https://i.imgur.com/sbsRfl1.jpg
這把是想從Typematrix出來換點不同的配置 加上數字鍵可以自由選擇放在左邊或右邊 所以趁去年Black Friday特價入手 買了半年後出了新版可以支援QMK 害我超幹，猶豫要不要再買一把 優點： 1. 配列是扇形放射，比直列更符合人體工學 2. 100%鍵盤，鍵位完整 3. 數字鍵盤是磁吸式可拆卸，可以自由組合成左手或右手鍵盤，也可以單獨拆下來用 4. 機械鍵盤熱插拔 5. 可自訂鍵，新版支援QMK 6. 也不算貴，$207USD含運，新版好像貴一點 7. 有RGB 8. 鍵線分離 確定： 1. 扇形配列對小手會偶爾無法單手按組合鍵，距離太遠 2. 熱插拔僅限光軸，無法更換特殊軸 我這款是青軸但手感比一般青軸還是略差一點 可能光軸比較適合線性軸 3. 竟然會少home & end鍵，但重複的鍵夠多可以自己改 只是我用的舊版韌體不能改fn層，改的不算很理想 4. 字母、數字、F鍵以外的鍵帽是特殊規格，無法更換 所以改完鍵後看著上面印不同的字有點錯亂 5. 沒有藍芽 6. 比一般100%鍵盤大一些，比較佔空間 結論：放在家裡給繪圖機使用，因為右邊放了繪圖板，數字鍵盤就一定要在左邊 三、微軟designer compact keyboard + number pad https://i.imgur.com/YHy4H0T.jpg
新公司每週有一天WFH，家裡桌子上被x-bows填滿 需要小鍵盤才能在桌子上放兩組鍵盤 但同時又需要數字鍵區，所以買這套可以把數字鍵盤放在左邊 但沒想到觸發過輕導致手指超痠 優點： 1. 藍芽，可配對三組 2. 非常輕，主鍵盤288g，數字鍵盤78g，且非常薄攜帶很方便 3. 續航力不錯 4. F鍵區的快捷功能設計不錯，也可以透過微軟的軟體部份自訂 缺點： 1. 打起來比筆電鍵盤還輕薄，對我這個重手來說反而力道壓在自己手指上，打了兩天就痠 到不行 2. 睡眠喚醒時間有點久 3. 竟然是用鈕扣電池 結論：配給一台鍵盤快報銷的舊筆電，偶爾要出門時用 四、keychron k4v2，自改凱華Box Navy軸 https://i.imgur.com/P9D8GOE.jpg
封城全面WFH後，微軟那組打到手受不了了 想找一把藍芽機械鍵盤，不要太寬但仍然有數字區 另外支援熱插拔可以試觀望已久的box navy 所以別問我keychron自帶的軸打起來如何 我買它就是換軸用的，自帶的一次都沒打過 優點： 1. 100鍵配列不太佔空間，但功能鍵仍然很足夠 2. 機械鍵盤熱插拔，box navy打起來非常過癮，可以放心敲不怕打爆 3. 支援mac(雖然我沒在用) 4. 雙模，藍芽可配對三組 5. 關燈續航力還算可以 缺點： 1. 數字鍵盤還是在右邊，右手使用滑鼠時配置還是歪的 (我以前因為數字鍵在右邊會左手用滑鼠，但現在滑鼠側鍵都是給右手用的…) 2. 打習慣typematrix和x-bows後，還是很不喜歡左手腕會拐到的傳統配置 3. 燈光無法軟體控制 4. 有點厚，一定要用手托，否則手腕會懸空 5. 和對岸一堆相同配置的鍵盤比算貴？ 我是$149AUD含運入手，外加box navy軸一百顆$63AUD 6. 喚醒需要一秒 結論：暫時WFH搭配公司電腦用，但這大小無法和私人電腦的x-bows一起放在桌子上，只好 又去買了鍵盤架分開上下放了 五、cool master sk622，青軸 https://i.imgur.com/MiBLUpm.jpg
懶得拍燈光了，這把才是真正打算拿來攜帶用的鍵盤 原本是和微軟那套之間猶豫 結果用完微軟後還是回來買了這款，幸好趁特價$101AUD入 優點： 1. 尺寸小(60%)、輕(446g)，方便攜帶 2. 即使方便攜帶，仍然是機械鍵盤 3. fn功能鍵非常完整 4. 雙模，藍芽可配對三組 5. RGB 缺點： 1. 這其實是矮軸，手感比標準軸還是差了一點 即使是青軸，感覺也有點接近薄膜鍵盤 2. 承上，不是熱插拔所以無法自行換軸 結論：去年底組了新電腦，之前的minipc換下來當影音機以及長期旅遊攜帶用 這把鍵盤就是特地買來配minipc的新用途 6. ID75，配凱華Box Navy https://i.imgur.com/HIqRMwq.jpg
在每一把當WFH鍵盤都不滿意後 和ergodox之間猶豫了好幾個月 最後因為ID75鍵數比較多，以及ergodox比較佔空間，選擇了ID75 優點： 1. 和typematrix配置相似 2. 機械鍵盤熱插拔，搭確定很愛的box navy 3. 支援QMK，加上有75鍵，功能鍵非常夠但不用組合太複雜 我做了一層左右都有數字鍵盤，非常好用 4. 不佔空間，只比sk622大一點 5. 但也不會太小，把字母鍵區分在最左右兩邊，手的位置還是順的 (planck就是太小，雖然是直式但仍然會折手腕) 6. RGB 7. 鍵線分離 8. 價錢也不算貴，含運$193USD 缺點： 1. 沒有藍芽 2. 全1u鍵帽有點難買，要買帶增補鍵兩套才能湊齊 和套件一起買的鍵帽不太好看，而且竟然沒有定位點 結論：換下keychron k4當WFH的主力鍵盤使用，只是那條線有點擋礙 插曲：奇怪的觸發壓力變化 ID75的box navy軸是從原本給keychron k4那邊再換下來 而keychron k4另外換上了box jade 但是ID75 x box navy打起來卻沒有keychron k4 x box navy重，後者明顯比較硬 不知道是不是和鍵帽有關？因為目前ID75的鍵帽是XDA 另外ID75 x box navy的聲音竟然也比keychron k4 x box jade清脆 按理來說box jade的聲音應該是比box navy清脆一點的 另外，keychron k4 x box navy，超級吵，吵到不行 比ID75 x box navy和keychron k4 x box jade都吵 雖然我個人是覺得不錯啦，但僅限在家用不然會被圍毆 -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 49.199.215.128 (澳大利亞) ※ 文章網址: https://www.ptt.cc/bbs/Key_Mou_Pad/M.1633618635.A.BEA.html
