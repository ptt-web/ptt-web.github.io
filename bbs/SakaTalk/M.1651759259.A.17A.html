時間Thu May 5 22:00:57 2022
※ 引述《FW190 ()》之銘言： : 剛爬文看到月初一篇關於高本的文章 : 底下推文有人提到5ch乃團版許多藍黑及紫藍的恩怨情仇 : 想說就來po一下我從16年中平假一期入團後到現在的觀察 : 主要來自於b站,虛實吧及ptt等華人論壇 : 1.16年5月到18年:西線無戰事 : 這個時期我幾乎沒看過兩邊粉絲的衝突 : 乃團飯當時主要是跟AKB和漢字欅的粉絲在戰 : 平假當時非常弱(尤其是16-17年),粉絲還很少,多數是"被"漢字欅粉罵 : 一直到18年才有比較大的反擊,但還是差很多 : 我的感覺是當時乃粉應該沒有注意到平假 這個時候漢字跟平假就戰爛了，乃團這個時候應該算是黃金時代，櫸坂出道後單曲大賣， 加上前所未有的世界觀，導致櫸粉覺得自己家偶像是阿踢斯特，有那個本事囂張到處嗆， 連好好的乃櫸合作活動都可以噴生駒，起衝突的基本上是漢字櫸跟乃團，平假嚴格來說在 2018年前不成氣候。 : 2.19年:戰爭爆發 : 情況到了19年有了改變,平假改名日向並且在3月單曲出道 : 乃團從13年開始一年都發3單,其中首單通常在3或4月發表 : 結果那年23單5月底才發,而且整年只發兩單 : 就有乃粉認為是因為日向出道的關係排擠了乃團正常的發單時程 : 開始對日向產生不滿,不過這個意見在當時不算主流,吵得不算兇 : 真正炸開的是年底兩件大事 : 首先是12月初日向聖誕宣宣布隔年日向聖誕在東蛋舉辦 : 乃粉整個大爆炸,認為出道第二年上東蛋根本就是兒戲(乃團是第五年) : "憑什麼?""坐不滿"之類的評論非常多 : 接著就是紅白三坂共演乃團20單,不少乃飯認為這嚴重排擠到自家成員的演出 : 不過這不一樣的地方在於紫粉是綠白兩團一起罵,不是只罵日向 基本上飯圈就是這樣，眼紅是很正常的，櫸坂出道3年就上東蛋，比乃團早了兩年多，乃 粉照樣噴，基本上就是怕別人超越自己團，粉絲面子掛不住，每一個粉絲都希望自己推的 團能夠永遠上行，這樣感覺自己走路有風。 日向坂粉絲也希望日向能夠超越乃木坂，但基本上以目前狀況不可能，現在就是天花板， 不過全選拔能做到日向坂做到這樣已經很不容易。 噴三坂共演的就是粉絲人多，其中必有白癡，也不想想兩團動員襯托你一個白石，夠有面 子，畫面也很美好，結果這些瘋起來人連內村光良都可以噴，人家跟Down Town同輩的藝 人大前輩幫忙站台還要被你們噴，面子都給這些白癡丟光了。 : 3.20年:烽煙遍地 : 這年兩團粉絲開始徹底槓上,什麼都可以吵,真正的血流成河 : 除了前一年的事件外,另外兩點我認為更是關鍵 : a.原本乃飯眼中的大敵-AKB和欅坂到這年都已經衰落,日向剛好取而代之 : 不夠強的話,紫粉還不會鳥你咧 : b.開始有紫飯變成紫藍兼飯(彩虹人),甚至由紫轉藍 : 這讓一些乃專粉更看不下去,相對於AKB是被乃團吸走許多粉絲 : 欅粉不少是阿提斯特粉根本就瞧不起傳統偶像(飯),他們要變成乃飯可能性不大 : 更年輕,走元氣/綜藝風的日向被許多乃專認為是最危險的敵人,一定要往死裡打 : 這年的高潮就是年底CDTV兩團合作的環節,日向唱乃團16單(壽司C) : 什麼樣的髒話/酸言酸語都出來啦.其中這兩句個人覺得很經典- : "你們日向是沒自己的歌可以唱嗎?" : "你們日向憑什麼唱再見的意義?根本不配!" 這個時候有趣的現象有兩個，不爽自己被日向超越的櫸粉跟乃粉一起噴日向坂是吸血坂， 櫸坂吸完吸乃團。 另一個現象是AKB粉隔岸觀火，想說乃木坂如今也有臉說別人吸血了，果然是翅膀長硬了 呢~ 基本上也是飯圈意識到日向坂是威脅，壽司跳再見的意義被噴，可以看成是當初乃櫸合作 的翻版，只是這次是乃粉噴日向，由此可知粉絲都是一樣的，不希望自己團被超越甚至是 不樂見他團慢慢爬上來，不過她們還是把日向想得太強大，也太小看自己了。 我自己提出另一個可能的想法，這或許也是老粉對於乃團後輩的沒有信心，尤其是他們看 到遠藤跟小坂的無情對比之後，覺得乃團靠四期生就完了，必須趁現在斷絕日向的上行之 路，才能永保江山。 : 4.21年:戰火稍歇 : 這年兩團飯還是繼續吵,但我覺得不管是規模和激烈程度都有下降 : 猜測大概是幾個原因,首先是兩團都飯的粉絲越來越多,自然越不容易吵起來 : 然後日向和16-18年的AKB及欅坂相比,不管是火熱程度和粉絲人數/規模都還差很遠 : 最後就是乃團自己也是逐漸在下滑,和16-18年比起來有差,至於原因就要另文討論了 : 5.未來:繼續觀察 : 目前坂道三團,大概就紫藍還會戰的比較兇,紫白基本上我沒看過 : 藍白現在有但不算多,有也是小吵 : 之後變數還很多,繼續看下去 這段時間兩團都處在轉型期，乃團不少強將陸續離開，世代交替進行中，日向粉絲也有意 識到東蛋之後也會有所改變，兩團粉絲都在為了轉變中的團體而緊張與期待著，戰？沒空 啦！ 紫白粉戰不起來是因為其實沒有怨仇，而且白對於紫沒有威脅，粉絲根本不當你是個咖， 沒事戰你幹嘛？白粉也知道自己現在團需要的，就是慢慢發展，也沒有那個本錢去戰來戰 去的，默默推就對了。 -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 39.13.99.75 (臺灣) ※ 文章網址: https://www.ptt.cc/bbs/SakaTalk/M.1651759259.A.17A.html
