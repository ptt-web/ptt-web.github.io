※ 引述《drkk (一言難盡￼ ￾N￾ ￾N￾N)》之銘言： : 我貼上網頁的原始碼在內文裡面 : 自己寄給自己發現都是原始碼 : 我有收到那種信件一收到就是htm的 : 不適用夾帶檔唷 : 是在預覽的時候跟點開那信件的時候都是顯現網頁 : 請問該怎麼用呀 : 編碼我是選支援http的 EDM ( 電子廣告,or...俗稱垃圾信件? ) 特色. 郵件檔案小,圖片放在網頁空間上,收件者開信的時候會連到絕對路徑抓取圖片 如果收件者開信的時候沒網路會看不到圖片 方法. 1.先把你要寄的東西(EDM)做成網頁A 2.把網頁A中有用到的圖片上傳到你的網頁空間 3.把網頁A中原來圖片的路徑改成絕對路徑, 這個圖片使用絕對路徑的稱網頁B ( ex. <img src="pic.gif"> ---------------------------網頁A <img src="http://你的網址/pic.gif"> ------------網頁B ) 4. Outlook Express -> 工具 -> 選項 -> 傳送 -> HTML設定 -> 傳送圖片與郵件 不打勾 5. Outlook Express -> 建立郵件 -> 檢視 -> 來源編輯 打勾 6. 在剛剛的新郵件按原始檔, 把網頁B的原始碼貼上 7. 預覽沒問題 -> 寄出, 檢查信件大小 如果 10KB 以內應該就成功了. -- ※ 編輯: accordingto 來自: 61.218.51.174 (11/18 14:29)
