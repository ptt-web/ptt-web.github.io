※ 引述《sidawei (不去意識)》之銘言： : width:660px; : padding: 5px 0px 5px 40px; : 這樣的方式在FX中，總寬度會是700px : 而在IE之中則是660px : 是否有其他比較好的方式來解決呢？ 假設你要設定 .box 的總寬度為 660px, 左內距 40px, 上下內距各 5px, 你可以這樣做: .box { width: 660px; padding: 5px 0px 5px 40px; } html>body .box { width: 620px; padding: 5px 0px 5px 40px; } 第一個是給 IE 用的, 而因為 IE 不認得 ">" 的用法, 所以第二個會被 IE 忽略, 而 Firefox 認得 ">", 所以第二個會被 Firefox 採用 這種技巧叫 CSS hack, 不同瀏覽器有不同 hack 的方法, 想要進一步了解的話, 你可以上 google 找找看 -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 218.35.56.37 ※ 編輯: eliang 來自: 218.35.56.37 (02/05 03:20)
