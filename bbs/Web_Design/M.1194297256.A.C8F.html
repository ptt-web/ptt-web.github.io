我覺得最快的方法就是直接去寫一個網站～ 試著寫寫看，遇到了甚麼問題就去弄懂他並且做延伸。 這樣會學得很快。 記得一開始我也和你一樣，只會用一點HTML， 而且我那時候更糟糕，都只會用FrontPage拉... 可以說HTML都不太懂。 之後也是因為接觸類似的工作，才慢慢開始學的。 一開始就試著寫一個網站，有不懂的地方就去查書查google， 很多資料範例可以讓你找到，跟著練習看看就會了！ 看你的需求在哪裡就學多一點那部份的東西， 基礎一定要會的就是HTML， 更好一點就學學CSS，XHTML等 想要功能強大就深入學PHP／MYSQL(ASP.NET／MSSQL)，JAVASCRIPT等 更深入就學學AJAX囉！ 想要華麗的畫面就學學FLASH，PHOTOSHOP，ILLUSTRATOR 想要富有設計感簡單線條的網頁...就靠你獨特的設計感囉 供你參考囉～天下無難事，只怕有心人 :) 這是我最近的成果，欣賞一下巴 :) http://www.liveinlow.idv.tw/ ※ 引述《subsummer (噹噹)》之銘言： : 其實我是想請問各位大大過去學習的經驗 : 因為最近開始從最尾端的地方開始土法煉鋼亂做 有需要想做就去翻書 : 但是很有興趣 一直想要做出很了不起的效果 所以就要一直找資料一直借書 : 可是覺得基礎實在很薄弱 想問問看怎麼樣學習會比較順利 : 大略上來說我大致上了解html了 (怎麼那麼粗淺的資歷...) : CSS概念了解 想得到的格式大概弄得出來沒問題 (不過我不是很有創意) : javascript 恩 書借了但是還是處於onClick window.location 跟 print.window : 之類的html小應用 (書是O'Relly的大全...真大本) : flash也開始摸了 可以做出一些教學影片等等的但是很想真的學一下Action Script : php跟資料庫之類的 有聽說了 因為一直在版上潛水常常看到 也想學一下 : 我的學習模式從後面學起 就是先從dreamweaver flash這種套裝軟體開始 : 然後一開始想學CSS就又去找O'Relly的動物書 從頭念起 : 通常是想做某某效果就來版上爬 然後就發現有新的東西要學 : 然後試一試用樣板硬作之後發現不懂 所以就找書要從頭開始看起 : 現在我覺得還算爭氣可以直接寫html跟css這樣 不過也就到此為止了 : 我的問題應該是從後面這樣學回去是不是不太對勁 : 因為我是工作需要要一直做網頁 作的時候我就一直想要加新的效果 : 所以一邊做一邊學 常常書念半本就硬上了 (通常是第一頁開始前半本) : 偏偏念前半本的時候會看到很多不會的東西 會提到程式語言等等 可是我完全不懂程設 : 我現在做得很有興趣 想要認真的學一些東西 : (目前學得最熱衷的是flash 因為flash一開始只要會插影格寫寫非常簡略的 : Action script就會動了 js還學不會) : 不知道板上的大大會不會建議我先看什麼樣子基礎的書 上什麼課 還是念個什麼學位 哈 : 還是照我現在這樣猛衝回去 二十年後終究會把東西都學完的這樣... : 想學的東西越來越多有點慌張 謝謝大家的指教囉。 -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 59.115.218.10
