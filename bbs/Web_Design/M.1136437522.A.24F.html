HTMLGate Free http://www.mpsoftware.dk/ 有不少可視元件可以用，而且完全免費。還有 HTML/CSS/JavaScript 之類的 Code Library 可以讓你選擇各種語法或函式。對初學者而言是很不錯的東西，對進階者而言 也還不賴，至少不用再找 Dreamweaver 或 FrontPage 的破解了。 NVU http://moztw.org/nvu/ 以 Mozilla 為核心的「所見即所得」網頁編輯器，開放源碼，跨平台.
