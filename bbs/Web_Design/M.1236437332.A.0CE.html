※ 引述《harryjoker (我很幸福)》之銘言： : 我想架設一個留言板 : 使用的軟體是VS2008 : 下載了tinymce3.2.2的版本用來作為文章編寫的介面 : 我嘗試將範例html的原始碼複製到aspx網頁來執行 : 這部分沒問題 : 但是當我想在網頁中放入asp的控制項(按鈕)時 : 會出現一個訊息 : "插入asp.net控制項會將網頁中的html表單轉換為asp.net表單， : 並移除他目前的內容。是否要插入asp.net控制項(y/n)" : 選擇"是"後一樣會讓我放置asp控制項 : 但在執行的網頁裡按下我設置的asp按鈕後會出現錯誤訊息 : http://0rz.tw/Y3IBW : 請問我該怎麼作才能正確的將tinymce放入asp網頁裡與資料庫作存取呢? : 謝謝 我的做法是先在設計模式下， 直接在頁面上拉好一個 TextBox， 這個 TextBox 的 ID="TextBox1"， 然後再切到原始碼模式，在 JavaScript 這樣設定： <script type="text/javascript"> tinyMCE.init( { mode : "exact", elements: "<%=TextBox1.ClientID%>" } ); </script> 存取資料庫的內容的時候， 就直接把從資料庫撈出來的東西指給 TextBox1.Text。 請注意 <%=TextBox1.ClientID%>，第一個百分號之後一定要加上等號， 因為這句是要取得 TextBox1.ClientID 轉換後的結果的。 如果執行後出現 "tinyMCE 未定義" 的錯誤， 則可能是引入主檔案時路徑寫錯了，請檢查路徑。 -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 61.62.185.27 ※ 編輯: fannys23 來自: 60.249.135.214 (03/09 13:14)
