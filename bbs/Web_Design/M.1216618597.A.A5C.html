※ 引述《colosturm (小封)》之銘言： : 可以請問一下 : 我做的網頁是用div做排版的 : 整個版面分為上下左右四塊，上下兩塊沒什麼問題 : 左邊主要是選單，右邊為內容 : 但是右邊不同頁面其內容高度皆不相同 : 我有什麼辦法偵測右半邊div的總高度(DIV沒設死高度是讓內容往下延伸) : 因為我要在左邊的DIV要以CSS語法使背景Y軸延伸 : 但是由於該DIV無內容，所以我想看有沒有辦法可以將另一塊DIV高度抓過去 : 方便延伸背景圖 : 不管何種語法皆可 只要能插入網頁即可 : ---------------------------------------------------------------------- : 以上內容，倘若不順口請多多包含，感謝各位前被瀏覽本頁 照你的說法來看，你的左右部份應該是用float的方法來實現的吧，像這樣？ ======================== header ======================== nav || content ======================== footer 我總覺得這是一個CSS上的常見難題 orz 原因是因為用純CSS沒有辦法像你所說的一樣「去抓別的box的height」 所以用float的兩欄式排版並不好做到兩欄同高 一個workaround是用一個box（<div id="container">）包住nav和content 在這個box中放y軸repeat的背景，然後把背景的寬度設成和導覽列一樣 這樣就勉強可以做到你想要的校果了 -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 140.112.242.241
