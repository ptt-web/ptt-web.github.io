問題不出 cross-browser， 例如 IE 只有在適合的播放器有安裝時，才會支援 embed。 因為 embed tag 不是標準，而解決這個的辦法就是不要用 embed。 你可以選擇使用 HTML5 或是 Flash Player，只是前者也未必每個瀏覽器都支援， 這時建議你可以找一些有 Flash fallback 功能的 HTML5 播放器，餵狗一下，不難找。 一般播放器有許多事件可綁，你裝上去之後， 寫一點 javascript 去控制，我想必可以解決你第一個問題 -- 更新 想到更簡單的 HTML5 解法： 把你的 video 轉成以下格式： mp4, webm, ogg audio 檔案轉成以 下格式： mp3, wav, ogg 寫法像這樣： <video controls="controls"> <source src="movie.mp4" type="video/mp4" /> <source src="movie.ogg" type="video/ogg" /> ... </video> 如此市面上五大瀏覽器都可播放 ※ 引述《anniekinki (我是安絲琪!)》之銘言： : 大家好~ : 終於找到這裡了~ : 想請問各位高手 : 我使用dreamweaver製作了一個網站如下 : http://www.music-rm.com/rco/index.html : (這是用網路抓的版型去套的,所以我其實並不是真的會做網頁) : 在網頁的左下角我置入了影片及音樂 : 想問各位高手 : 1.有辦法可以讓影片播放時音樂自動停止,影片播放結束後又自動撥放音樂嗎 : 2.有人告訴我他的電腦無法看影片,連播放鍵都沒有就是一個黑黑的方型在那邊 : 這有可能是什麼問題呢?因為我的電腦看都是沒問題的~ : 以上~~ : 有請大家幫幫忙了 : 估狗好久都沒答案 : 我快被老闆煩死了(哭哭) : 謝謝~~~ -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 61.227.147.207 ※ 編輯: tonytonyjan 來自: 61.227.147.207 (08/11 20:25)
