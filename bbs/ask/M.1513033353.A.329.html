最近才發現 3月左右，RSS功能(atom feeds)就已經移置到 ptt 網頁版( https://www.ptt.cc/atom ) 繼續提供了.. https://github.com/ptt/pttweb/commit/cdd4e78 跟原版 RSS 的差異在於 以下提到的三個bug/限制都有所改進了XD ※ 引述《holishing ()》之銘言： : ● 1. 大小寫要正確： : 像原po提到就可板(joke)訂不到rss，但我試的結果是可以的： : http://rss.ptt.cc/joke.xml ←點點看，裡面有xml的文字標籤格式，代表這是 : 可以用RSS閱讀器訂閱的。 : 大小寫不正確，就無法找到你真正想要訂閱的內容。 : 像 Joke, JOKE, joKe ...等名字都是錯誤的。可以試試看，頁面會顯示： : Not found. : 建議去各板裡的右上角或看板選單，找出正確的「大小寫板名」。 大小寫要正確的限制已經取消囉！ https://i.imgur.com/i4aOuWD.png
另外原本網址的格式是 http://rss.ptt.cc/joke.xml 現在會自動更新轉址成: https://www.ptt.cc/atom/joke.xml : ● 2. 十八禁看板無法使用RSS訂閱功能 : 資料來源： #1D61joOw (SYSOP) : 因此，Gossiping板、sex板、......等非普遍級看板無法 : 使用RSS訂閱功能。 (這應該是很少鄉民之後都不用這個功能的原因之一吧!) : 頁面同樣顯示：Not found. 現在也可以訂閱了 https://www.ptt.cc/atom/Gossiping.xml https://i.imgur.com/t9wR6W9.png
因此有需要訂閱八卦板的鄉民... (純粹想看最新文章那種) 或許可以嘗試看看...XD : ● 3. 英文板名有「.」的看板也無法使用RSS訂閱功能 ( F.Cuz , S.H.E ...等) : 資料來源： #1Gv5JXAs (PttBug) : 經過測試，頁面會跳至 http://www.ptt.cc 網頁板首頁， : 這我就不知為何了，可能是系統上的限制? 現在也可以訂閱了 https://www.ptt.cc/atom/S.H.E.xml https://i.imgur.com/Yhz6hwt.png
