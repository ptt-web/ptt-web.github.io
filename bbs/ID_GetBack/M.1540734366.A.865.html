標題[公告] 常見申請錯誤
時間Sun Oct 28 21:46:02 2018
申請領回密碼及保留帳號之使用者常見錯誤如下: 申請領回密碼者 1. 申請人 ID - 空白 - 與發文者 ID 拼寫不同 2. 電子信箱: - 填寫兩組以上之電子信箱但未換行填寫 - 電子信箱前或後方包含其它符號或文字 - 填寫無法收件之電子信箱(如拼寫錯誤或信箱失效等) 3. 確認是否詳閱公告 - 未確認 4. 格式 - 刪除既定格式虛線上方內容 - 未使用既定格式填寫申請單 5. 其它 - 除修正前次申請錯誤外, 一再提出申請 - 催促案件處理. 申請[保留]/[盜帳]者 1. 申請人 ID - 空白 - 與發文者 ID 拼寫不同 2. 保留 ID - 帳號不存在 - 帳號仍可登入 - 無特殊原因申請保留符合〈密碼取回規範〉第一條第 5. 項要件之帳號 3. 確認是否詳閱公告 - 未確認 4. 格式 - 刪除既定格式虛線上方內容 - 未使用既定格式填寫申請單 5. 其它 - 除修正前次申請錯誤外, 一再提出申請 - 要求寄送密碼 - 催促案件處理 由於過多使用者僅注意自己申請領回帳號之權益, 不願事先詳閱各規範與說明, 造成處理密碼領回事務時極大困擾, 自本公告起, 違反上列事項者, 一律逕行退文, 且不接受申訴. -- 帳號站長 disFabulous -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 123.241.153.66 ※ 文章網址: https://www.ptt.cc/bbs/ID_GetBack/M.1540734366.A.865.html 2018/11/25 新增項目: - 新增未確認是否詳閱公告者 2018/11/28 新增及修正項目: - 移除與證明人制度相關項目 - 新增其它項目及其子項目 2020/02/08 新增項目: - 新增申請保留仍可登入之帳號者 2020/03/09 新增項目: - 催單或於受理後與發放密碼前一再申請 2020/05/06 修正列表方式並新增: - 無特殊原因申請保留符合〈密碼取回規範〉第一條第 5. 項要件 之帳號 2022/04/25 修正標題 申請[保留]/[盜帳]者 ※ 編輯: disFabulous (203.72.182.30 臺灣), 05/06/2020 09:12:31 ※ 編輯: Makotoyen (210.61.91.224 臺灣), 04/25/2022 10:51:01
