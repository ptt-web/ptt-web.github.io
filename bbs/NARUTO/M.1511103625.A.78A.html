標題Re: [討論] 為什麼螺旋丸不需要血輪眼?
時間Sun Nov 19 23:00:18 2017
※ 引述《canandmap (地圖上的流浪者)》之銘言： : 引言恕刪 : 問為什麼螺旋丸不需用到寫輪眼這個問題，何不如問有什麼忍術是不需要用上寫輪眼的? : 看來看去，也就非血繼限界一類的忍術而已 : 從這角度來切入，就可明白其原因 : 撇去前面回文提到的威力、攻擊方向、應用等不談 : 螺旋丸這個忍術已經快變得人人都可以學的程度 : 目前已知用過的人有: : 波風湊(四代，創始者)、自來也、卡卡西、鳴人、木葉丸、博人 : 光一個術就有六個人有用過，再者這個術只需要懂形態變化並練至極致便行且還不需結印 : 況且以當時鳴人這個笨蛋的程度都可以學會，其他人會螺旋丸只是遲早的問題 : 再來與之相對的千鳥，坦白講我覺得它和寫輪眼綁定的設定有點自打臉 : 因為攻擊速度太快而看不清對方動作，那雷影(艾)及被穢土出來的三代雷影要做何解釋? : 他們的雷遁也是有可能因攻擊速度太快而看不清對方動作 : 難不成他們的眼睛比寫輪眼還厲害? 因為他們仗恃著的是雷遁盔甲和超堅韌的肉體強度 : 而本傳中，佐助與鳴人恰巧與上述兩人對戰過 : 佐助用千鳥攻擊艾的重流暴，就重演了一次搭配寫輪眼的設定 : 鳴人沒有寫輪眼，他開仙人模式來感知三代雷影的攻擊方向，再用螺旋丸進行反擊 : 至於有人說到千鳥很吵，我在別的板有提過 : 卡卡西曾經用雷切(後來的千鳥)捅過角都，而且角都還說「這麼快的雷遁之術...」 : 表示他完全沒想到自己會被捅 : 如果千鳥聲音很吵，角都應該就會發覺有人要攻擊他而採取對策了吧 當初角都說的是「你竟然可以看穿我結印的速度，再加上這麼強的雷遁」 於是就推測出了捅爆自己的就是寫輪眼卡卡西 雷切有個特點其實後期比較少著墨的就是「超高速衝刺」 也就是卡卡西使出這招的時候是可以快到連角都都反應不過來的 : 總之，我個人認為千鳥綁定寫輪眼的設定，是屬早期的設定 : 四代在當時還補充了「這是個不成熟的術」，表示不可輕易動用 : 但是岸本是屬於會吃書的作者，後期設定會推翻前期設定早在意料之內 這倒是沒有。 因為當卡卡西獲得了帶人的雲端雙萬花筒寫輪眼之後， 還說「因為寫輪眼回來了，所以雷切這招也復活了」 當然事實上雷切這招本身無須寫輪眼也可以發動， 但就只是說沒有寫輪眼，自身的視力反而會跟不上自己本身的高速移動而已 當然這種理由跟說明的合理性實在不高啦但至少是個解釋囉 -- 段譽一見之下，身上冷了半截，眼圈一紅，險些便要流下淚來，心道：「人道習主席是人 中龍鳳，果然名不虛傳。馬先生對他如此傾慕，也真難怪。唉，我一生一世，命中是注定 要受苦受難了。」他心下自怨自艾，自歎自傷，不願抬頭去看馬先生的神色，但終於忍不 住又偷偷瞧了他一眼。只見他容光煥發，似乎全身都要笑了出來，自當選以來，從未見過 他如此歡喜。兩人已走近身來，但馬先生對段譽視而不見，竟沒向他招呼。段譽又道：「 馬先生心中從來沒有我這個人在，從前就算騙我投票給他，心中也只有他主子。」 -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 219.85.86.66 ※ 文章網址: https://www.ptt.cc/bbs/NARUTO/M.1511103625.A.78A.html ※ 編輯: m19871006 (219.85.86.66), 11/19/2017 23:00:58
