標題Re: [討論] 為什麼螺旋丸不需要血輪眼?
時間Mon Nov 20 12:50:16 2017
※ 引述《canandmap (地圖上的流浪者)》之銘言： : 引言恕刪 : 問為什麼螺旋丸不需用到寫輪眼這個問題，何不如問有什麼忍術是不需要用上寫輪眼的 ? : 看來看去，也就非血繼限界一類的忍術而已 : 從這角度來切入，就可明白其原因 : 撇去前面回文提到的威力、攻擊方向、應用等不談 : 螺旋丸這個忍術已經快變得人人都可以學的程度 : 目前已知用過的人有: : 波風湊(四代，創始者)、自來也、卡卡西、鳴人、木葉丸、博人 : 光一個術就有六個人有用過，再者這個術只需要懂形態變化並練至極致便行且還不需結 印 : 況且以當時鳴人這個笨蛋的程度都可以學會，其他人會螺旋丸只是遲早的問題 其實非血繼限界的忍術本來就是人人可學的 但是還關係到天份、訓練方式、查克拉量、作戰方式、熟練度...等等 所以自來也和卡卡西幾乎沒有在戰鬥中用過 型態變化也不是那麼容易就練成的 連四代也沒完成 鳴人也是用非常人的方法才研究出來 以鳴人訓練的時間乘上影分身人數 應該也是要滿長一段日子 博人要嘛是天才要嘛是bug= = : 再來與之相對的千鳥，坦白講我覺得它和寫輪眼綁定的設定有點自打臉 : 因為攻擊速度太快而看不清對方動作，那雷影(艾)及被穢土出來的三代雷影要做何解釋 ? : 他們的雷遁也是有可能因攻擊速度太快而看不清對方動作 : 難不成他們的眼睛比寫輪眼還厲害? 如果單就動態視力而言 我相信雷影們的眼睛確實不亞於寫輪眼 簡單來比喻就是 千鳥是F1賽車 沒開寫輪眼的佐助卡卡西是高速公路警察 而雷影們是F1賽車手 雷影們經過長期累積下來的訓練和戰鬥經驗 還有雷遁的加持 神經傳導和動態視力絕對是比一般人高上很多 佐助卡卡西的體術雖然也在中上 但和雷影們還是有差 講得誇張一點 雷影們在雷遁的狀態下 隨手一戳就是雷切千鳥了 而且要用幾次都可以 : 而本傳中，佐助與鳴人恰巧與上述兩人對戰過 : 佐助用千鳥攻擊艾的重流暴，就重演了一次搭配寫輪眼的設定 : 鳴人沒有寫輪眼，他開仙人模式來感知三代雷影的攻擊方向，再用螺旋丸進行反擊 寫輪眼和仙人模式本來就是強大的buff 眼睛跟不上雷影們只好開buff : 至於有人說到千鳥很吵，我在別的板有提過 : 卡卡西曾經用雷切(後來的千鳥)捅過角都，而且角都還說「這麼快的雷遁之術...」 : 表示他完全沒想到自己會被捅 : 如果千鳥聲音很吵，角都應該就會發覺有人要攻擊他而採取對策了吧 我不記得戰鬥的細節 不過之所以叫千鳥就是因為真的很吵 卡卡西戳得到角都有可能是因為 1.卡卡西搭配土遁壓低千鳥的噪音 2.卡卡西先移動到角都身後再近距離發動千鳥 3.角都自己就很吵了所以也聽不太到 : 總之，我個人認為千鳥綁定寫輪眼的設定，是屬早期的設定 : 四代在當時還補充了「這是個不成熟的術」，表示不可輕易動用 我認為「不成熟」可以解釋為 千鳥的高速衝刺 需要與之搭配的身體能力和動態視覺 而這不一定是要寫輪眼 若擁有阿凱或雷影般的肉體條件 當能運用自如 增加成功率的同時降低危險 : 但是岸本是屬於會吃書的作者，後期設定會推翻前期設定早在意料之內 -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 223.137.146.43 ※ 文章網址: https://www.ptt.cc/bbs/NARUTO/M.1511153418.A.37D.html
