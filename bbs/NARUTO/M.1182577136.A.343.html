標題Re: 關於鳴人的修練
時間Sat Jun 23 13:38:51 2007
※ 引述《Extrasensory (歸路無途)》之銘言： : 主要是關於螺旋手裡劍的問題。 : 當卡卡西等人知道鳴人是風屬性之後， : 為什麼要他去把螺旋丸加強，而不是去學習其他的風屬性的忍術呢? : 螺旋丸本身就已經很強了， : 即使兜這種有治癒能力的人，中招之後雖然沒死卻也無力再戰， : 如果是其他的人中螺旋丸，不死也只剩半條命， : 一招已經會讓對手支離破碎的忍術， : 有必要再練成讓對手灰飛煙滅的程度嗎? 螺旋丸命中兜使他無力再戰是因為兜太弱了。 就算兜再強，定位頂多就跟卡卡西或是靜音等上忍差不多。 要拿來和曉對戰，恐怕只有吃鱉的份。 如果我們把曉定位在與三忍同等級的忍者的話， 對方根本不會給你「第二次」出手的機會。 以對角都為例，假設鳴人跟卡卡西一樣強，可以打中角都。 螺旋丸很成功地擊中，秒掉角都的一個心臟，角都也可以用別的術秒掉鳴人。 鳴人的速度不夠快，攻擊動作又太大，都是很明顯的缺點， 如果命中沒有直接打死敵人，大概就等著被打死了。 至於「其它風屬性」的忍術，也得要把性質變化附加在某些東西上， 否則也只是單純的切割而已。與其練其它不熟的忍術， 不如就搭配鳴人最熟悉的忍術來練習。 另一方面，以團隊作戰的角度來看，低破壞力的招式已經有很多了。 卡卡西可以用四次雷切，雷切等級的敵人應該綽綽有餘， 但是更缺乏更高級別的致勝武器，因此使練螺旋手裡劍是必要的。 其它破壞力低的忍術用來牽制敵人，以求取鳴人一擊致勝的機會。 在現在的戰力配置上，我認為這是很明智的。 : 再者，螺旋丸本身單純只是型態變化，所以不會有屬性相剋的問題， : 可是一旦加入風的型態變化，反而會受到火屬性忍術的克制， : 這樣去修練真的對鳴人有幫助嗎? 這樣的修練對於鳴人去 pk 其它超強的忍者顯然沒有什麼幫助。 甚至講得更直接一點，恐怕不管怎樣的修練都不會有幫助。 我們還不知道鼬有什麼屬性，如果不幸他也是風屬性的， 也許看一眼就學會螺旋手裡劍了。但是就團隊而言， 鳴人學會螺旋手裡劍是很有用的，至少在接近敵人後， 螺旋手裡劍可以給予敵人十分有效的致命一擊，其它忍術則不行。 : 當初鳴人花了一個晚上的時間就把S級的(?)多重影分身學會了， 多重影分身被當作是禁術原因並不是因為它很困難， 而是因為它對於施術者本身很危險， 因此學會多重影分身並不能當作鳴人悟性的證明， 事實上我們要知道，大部份複雜的術鳴人可能都練不太起來。 就算練起來也不見得可以好好應用。 : 而現在他又會用影分身來縮短修練的時間， : 如果把時間拿來學習其他風屬性的忍術，想必應該很快就可以學會， : 多學些忍術在實戰上也比較有幫助， 我認為這樣的假設是不存在的，鳴人悟性低是角色設定。 打破這樣的設定會讓角色的特質消失，雖然這樣的設定會一直被罵， 永遠不會長進，永遠只有那一兩招。但是如果招招都會， 那乾脆把漫畫名稱改為卡卡西、佐助或是鼬算了。 鳴人如果練很多術，也許會像段譽使六脈神劍一樣，亂無章法。 反而單使一脈的時候，可以更加專注，以致更可以牽制敵人而獲勝。 : 怎麼會把所有的時間拿來修練一個未必能成功的忍術呢? : 大家討論一下吧~ -- 活著的目的是為主活 然後為主死 死亡的目的是為主死 然後為主活 -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 140.112.31.132
