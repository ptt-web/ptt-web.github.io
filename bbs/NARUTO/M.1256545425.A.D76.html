忍者小隊"執行任務"是以四人為主 三人應該是底限 一方面負責誘敵(需要戰鬥), 另一方面執行任務(例如護送、情報採集等) 可採 2+1 或 1+2的戰術 (1+2比較危險, 除非那一個人能夠完全牽制對手) 而實際戰術上, 人數比底限要多, 主要是讓小隊有更多變化性, 也可在損失隊友後繼續進行任務 不採用五人以上的小隊, 則是指揮系統的考量, 以及為了避免容易被發現 但是, 若任務需要、許可, 仍然可以增加人數(例如純粹的廝殺、或困難的任務) 當小隊人數提升為四人, 人員分配轉為 3+1為主(如鷹小隊), 或 2+2(需要分開行動時) 做為戰鬥員, 至少要有一人有強大攻擊輸出, 其餘人員負責助攻 例如第十班, 阿斯瑪、丁次是強攻, 鹿丸負責助攻, 井野則除了負責醫療, 也可以輔助 當然, 當遇上戰力不足而後勤有能力戰鬥時, 也會應要求出戰(例如綱手) 而依照任務的需要, 不同專長的組成隊伍 例如阿凱班就屬於主戰型的隊伍 紅班(牙、雛田、志乃)屬於探查型隊伍 但是漫畫始終是一種人為創作, 多對多的交叉戰鬥不容易規劃、安排 一對一(或者一對多)的方式, 比較容易規劃 另一方面, 敵隊通常也是四人小隊 (剛好每一人有適合自己的對手) 所以戰鬥大多都以一對一展開 而且常常的, 屬於非戰力輸出人員也會參與作戰 (海賊王就是這樣) 曉則是自認戰力超群, 且任務單一, 即打敗人柱力 所以只有負責戰鬥的人員 2名 ※ 引述《AbcPig (一款豬肉飼百款人)》之銘言： : 這些內容我不知道有沒有雷 因為沒在看單行本 所以還是空一頁保險點 @[email protected] : 最近看了許多火影的動畫與漫畫 (我在補之前沒看到的進度啦 XD) : 有關三忍 & 醫療忍者有幾個小問題想跟各位板友討論一下 : 1.有關醫療忍者: : 就是 忍者出任務時 通常都是3~4人一個小隊對吧 而且好像以三人為主 : 三人的話 通常是兩個主攻+醫療忍者 或是一主攻一助攻+醫療忍者呢? : 還是説三個包括醫忍都能當主攻呢? 目前我有印象的醫忍:千代婆婆 綱手 小櫻 琳 : 千代婆婆的傀儡術不用說 : 綱手我記得的就只有召喚無殼蝸牛跟一拳三百磅 三忍對決時有沒有用其它術我忘了 orz : 小櫻好像就只剩下拳頭了... : 而在卡卡西特別篇中琳好像也沒有特別的表現 連爬出洞穴都要卡卡西拉她上去... : 講到這裡我就要小抱怨一下 醫療忍術不是要有很好的查克拉控制技巧嗎? : 連腳底的查克拉都控制不了 居然可以移植眼球&視神經等等...難道她天生神力!? orz : 前面對上兩隻岩隱忍者時明明就是走在水上啊... : 我該說岸本不意外嗎? = = 還是說移植時把查克拉耗盡了? 唔 這也是有可能啦...囧... : 2.有關三忍: : 在自來野外傳中 以及一些回憶過去的畫面時 都可以看到一些三忍齊心協力戰鬥的畫面 : 我想提出一些他們在小隊中個人定位的看法 : 綱手不用說 當然是以醫療為主 (僧侶或聖騎的角色? 能補又能打 XD) : (但在對上山椒魚半藏那幕時誤觸陷阱 反而成為拖後腳的 orz) : 自來也在少年時期同時學忍術和仙術 有仙術加持威得很 體術應該也不錯 可以做主攻 : 至於大舌頭 光看外型較為陰柔(?)就覺得不太適合在前線作戰 : 身體軟趴趴(?)的打人也不痛吧 XD 頂多放潛影蛇手把人給勒死 : 而且他的畢生的志向就是學會所有的忍術 (包括蛇形刁手? 金蛇纏絲手?
