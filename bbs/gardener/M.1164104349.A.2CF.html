我以前在私立幼稚園當過美術老師。學校的走向就是才藝為主。 老闆的重點就是快樂學習。 不過我覺得"快樂學習"有時被濫用了。這裡就不多說。 前面幾篇有版友PO文說學校大方針怎樣，老師只能也這樣。 我覺得，老師本身的自覺性要高。今天是站在教育的立場，為了小朋友好， 就應該以小朋友為出發點。真的覺得學校方針不對，也要多多替小朋友爭取。 家長選學校也是，不管是公立私立，要先去瞭解那一間學校的老師。 我覺得先要從園長及主任先瞭解起。要知道他們的走向及態度。 如果園長及主任的觀念很圈叉，當然就不用考慮，因為長官的勢力本來就比較強。 除非底下的老師真的很用心。 再來瞭解帶班老師及各科老師，找機會跟老師聊聊， 可以模擬一些狀況問老師的處理態度寂及從中看他們的態度。 這樣才能知道該間學校適不適合你的小朋友。 小朋友幾歲開始上幼稚園。來我們學校上課的小朋友從兩歲半開始就有。 不過我覺得如果家長非得很早就讓小朋友去學校，應該也要三歲半左右吧。 我上美術課幼幼班堅持最多8個小朋友，所以從兩歲半開始上還可以。 但上起課來真的是超累.........괊 我要表演要說故事要逗他們要聽他們每人急於跟我說的小秘密又要注意每個小朋友的狀況 不過說真的，這些小朋友跟了我三年，他們的成長及學習態度讓我很感動！！ 另外，家長在啟蒙階段扮演著比老師更重要的角色。我常跟家長說，我最多一週 只能跟小朋友玩美術兩堂課，但家長可以天天晚上或假日都可以跟小朋友玩美術或是 他們有興趣的東西。所以家長的態度肯定影響著學生的學習及成長。 我會主動跟家長分享一些教學技巧，美術知識，甚至做一些學習單，是要給家長看的， 讓家長可以回家陪小朋友玩美術。 因為我覺得給家長一些技巧及觀念分享，對我的教學，對小朋友的成長只有加分作用。 總而言之，我覺得幾歲上學要看小朋友的狀況。小朋友最主要是處理他們的情緒問題。 學習是過程遠比結果重要。所以帶班老師很重要。也要看整個大環境是怎樣。 不過我本身是不贊成太早送小朋友去上學，因為小朋友的世界也是很複雜， 有的心機也很重.....除非家長跟老師給小朋友的心理建設很好，老師又時時能注意到 他們的狀況（不過這不太可能，他們暗地裡小動作太多了）。有些太小的小朋友 就會被影響。我以前上班有個小女孩，才四歲（兩歲開始上全天班）， 就宛如一隻老鳥，心機也很重（真的是這樣....會搞小團體，會說謊）， 當然他也有很可愛的一面。但因為太早整天接觸這種群體生活，他不得不開始察言觀色， 知道什麼時候該說甚麼話，讓我覺得很難過。 所以囉。家長選學校要睜大眼睛。用心觀察。不能只有公立或私立二分法。 ※ 引述《pinkbomn (本姬妹妹)》之銘言： : 我是文章的原po.... : 其實我的小孩還沒出生辣..我現在懷孕8個多月快9個月 : 沒想到引來這麼多的討論.... : 我也很仔細看過大家給的意見... : 其實...我自己的想法是... : 小孩快樂比較重要... : 我會兩者比較是因為 : 第一..光費用方面就差滿多的.. : 現在大家生的少..想給小孩最好的教育..什麼東西都用好的 : 每個小孩都很寶貝...但真的需要花那麼多錢在幼稚園身上嗎 : 我自己的爸媽家庭教育是...公課不是很重要.. : 重要的是品行..生活能獨立自主比較重要 : 第二..想了解一下到底公立跟私立他們上課都在教一些什麼東西... : 我看到有大大說..公立的小孩生活比較獨立..相較之下私立的 : 就比較差一點...雖然公立的小孩在課業上比私立的落後.... : 但這就是很多家長的迷思吧..想要小孩贏在起跑點上.... : 不過很謝謝大家的意見.. : 不過我有一個問題... : 幾歲上幼稚園會比較好呢 -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 24.18.186.146
