標題Re: 關於嘉義縣教保員考試簡章疑問
時間Wed Jun 28 23:39:02 2017
我認識其他縣市教保員考試承辦人員， 可以稍微回答一下。 引述《lilianchan (水面鏡子)》之銘言： : 今年嘉義縣教保員的考試簡章及報名表呈現十分 : 鬼異的狀態，令我感到好奇的是討論度竟然如此 : 低！首先先從簡章公告開始端詳。 : 1.承辦人載明這次考試是各鄉鎮市委託縣主管機關 : 公告主辦，此一說法明顯與過去慣例不同， 考試的確是由教育局處會發文調查各園所要自辦還是委託教育局處辦理（跟教甄一樣）， 所以的確是各鄉鎮委託局處統一辦理無誤。 教育部的確是最上級單位，但要弄清楚的是教育是地方自治，在一切沒有違反母法之下， 各縣市教育局可以決定怎麼辦理。 且鄉鎮 : 市幼的主管單位乃是縣教育局，而教育部則是教育 : 局的上級單位，教保員怎可能是由鄉鎮市委由縣 : 舉辦，而非由教育部委由縣市政府主辦再分發至 : 各鄉鎮市國小或鄉鎮市附幼呢？ : 2.教保員甄試乃屬於全國同ㄧ性的考試，計分標 : 準應不分鄉鎮市幼或國小附幼的不同而有不同， : 而依照慣例，各縣市進行的教保員考試，不管是 : 報名程序或考試日期，資格要求，考試分發皆有 : 一致性，然而今年出現花蓮縣提早考，而嘉義縣鬼 : 異的積分計分方式，讓尚未考試的考生，在積分的 : 計算上處於未考就失去58分的劣勢！此ㄧ考試讓 : 所有不是公所出身的考生在本此考試就注定初試將 : 無法通過！有興趣的人可以參閱嘉義縣教保員各組 : 報名表！ 教保員並不算全國同一性考試（這沒有規定哦），從第一屆教保員考試開始，因為第一年 各縣市都是第一次，所以那年全國一樣試題，也因為一樣試題所以必須同一天考試。 但也是可以不同天的哦，只是不同天的話考題就不可能同一份。 考題怎麼來？是由承辦的局處去跟中央買來，如果同一天考試的（就教甄而言，例如：中 區聯盟，他們就是由參加的縣市一起share這份考題的費用） 可是自己招考的縣市，如上面所說的花蓮提早考試，花蓮縣是自己承擔考題的費用。 就我承辦的同學說：通常要跟別的縣市聯盟啊，不然考題的費用很貴。如果自己辦，通常 會虧錢。 教保員跟教師甄試一樣，都是地方自治，沒有規定要同一天考試，只要它最後的任用辦法 沒有違背母法，一切都是合理的。 * 而一個園所要自辦，那個壓力非常之大，我的實習學校曾經自辦獨招教師一名，校長也坦 承要承受很多黑白道關說壓力。所以真的很少願意獨招。 -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 101.14.165.116 ※ 文章網址: https://www.ptt.cc/bbs/gardener/M.1498664344.A.D8F.html
