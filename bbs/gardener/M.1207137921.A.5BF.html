我以前做過一個活動是 我先用把學校附近商家的招牌拍下來，列印出來、加上軟磁鐵 自己畫一張學校附近的簡單地圖 再準備一些各種商品、食物、文具的圖片，也是加上軟磁鐵 活動開始時，先把這些招牌秀給小孩看 問問他們這家店是什麼店、在地圖上的什麼地方 如果小孩有能力的話，可以請小孩把商店招牌貼到黑板上去 商店位置都放好了之後，把商品裝在小盒子裡 請小孩輪流抽一張圖卡，說出要去哪一家店買這樣東西，請他把圖卡貼到那家店旁邊 也可以搭配錄音檔 前一天自己先去錄附近商家會有的聲音，像是老闆叫賣聲、便利商店叮咚聲 把錄音檔放給小孩聽，請他說出是哪一家店，從學校出發要怎麼走去 ※ 引述《pinkswing (有壓力才能專心。)》之銘言： : 有另兩本繪本：來我家玩，照著地圖走喔。和"我的地圖書"也很棒！ : 我之前帶過這個主題，你可以做很多延伸，例如藏寶圖，家裡到學校的地圖等等。 : 我們最後是把方向轉到"標誌"上，因為孩子有興趣! : (標誌可用繪本：逛街and兔子先生去散步) : 其實你也可以結合這兩個概念去做，相信會很好玩唷! :D : ※ 引述《ireta (時間的聲音)》之銘言： : : 有一本繪本「我家附近的野狗」裡面有地圖 : : 可以透過圖畫書的引導 : : 來讓小朋友先認識自己家附近有些什麼 : : 然後試著把它畫下來 : : 再慢慢延申 : : 其實光是地圖就有很多東西要先認識 : : 像是座標的概念 : : 還有方向 : : 所以可以慢慢來唷！ -- 我的播客：http://media.webs-tv.net/kashin 喜歡唱 ACG 歌曲和老歌 XD 歡迎給我建議 ^^ -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 218.167.123.242
