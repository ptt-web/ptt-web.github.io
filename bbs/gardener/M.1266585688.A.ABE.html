我不能算是在幼教很有經驗 因為其實我比較常帶小學生 (我算是營隊的人) 但是因為也有玩過幾個幼教的班級 我一開始頭真的很大 因為我習慣跟小學生講道理 可是...幼稚園有時候真的聽不懂人話 不過之後就自己用對待小學生版本的修正版 -------------------------------------- 1.給選擇題~不要給是非~不要太把指令用問句型態來給予 比如說之前因為有個類似成果發表的東西 所以要小朋友有一些作品~寫一些123連連看... 寫那些東西勢必得花到上課時間~所以他們就不能上一些喜歡的課程 我有看過比較沒有經驗的老師是這樣問 "因為我們要成品~所以我們來寫作業好不好?" 小朋友十個有八個會說不要~我會用一種方式 "現在你要寫作業還是要在旁邊面對牆壁靜坐?" 2.口氣一定要夠堅定~可以溫柔~但是態度一定要堅持 去服務的地方規定~小朋友點心用多少就要吃多少~所以不能倒掉 某個四歲的寶貝~看到是吃番薯片用了一大碗 中間老師也再三地問他~真的要這麼多嗎?吃不完等一下就不能上課.. 最後吃不完了 因為下堂剛好是我主帶的畫畫課~共同創作 所以也就跟那個寶貝說~吃完才能上課...哭也是得吃完 另一個老師一面幫我一起上課~一面去盯他吃東西 最後~他是一把眼淚一把鼻涕地把東西吃完...期間當然小孩有鬧~但是就只是重複一句話 "吃完就可以上課...不吃完就不行上課~哭也得吃完" 我講重要事情開頭常常是 "我很認真地跟你們說~~" "這很嚴肅唷.." 配合把我那超小眼睛睜大的技術~ 3.一定要說到做到 被我帶過的小朋友都會說 "大鳥老師會說到做到~所以我們要聽話乖一點" 也常常有這樣的對話 我:大鳥老師是說到然後哩? 小朋友:做~~~到~~~~ (接著他們就知道該仔細聽該小心了) 因為小朋友很愛跟我閒聊~所以對他們而言很大的獎賞是全班閒聊十分鐘 或者是因為我平常不是很喜歡抱小孩..所以抱也變成超級大獎賞 甚至某天在尋求上級的同意之後~就帶去廁所玩水 (當然是有設計活動的讓他們玩耍) 可是大鳥老師的處罰也都是言出必行~沒有討價還價的餘地 然而因為處罰也都是我已經告知~你怎麼樣~我就會怎麼罰你 所以執行上偶爾會有阻礙~但是我會很堅持要執行完整個處罰 4.練音量...不是說大聲~但是一定要亮 之前去某個機構幫忙他們成立幼兒班(下午一點到晚上九點的班級) 因為請的老師~不是正規老師~基本上就像是社區媽媽...無經驗 不過她很認真也很拼...理論上我只是協助應該不算主帶~但是因為我比較有經驗 所以反而我是主帶~她是助手 她的音量真的不大聲也不夠亮~也是一個沒有什麼氣勢的人 可是她會每天在騎車來上班的途中~一直練音量 "安~~靜~~" "集~~合~~" "小~幼~班~~" 我想再騎車時~旁邊的人應該常常被嚇到 但是也真的音量跟氣勢在她的練習之下超級有成長~ 最後~我覺得有氣勢跟兇不一樣 而且如果是有道理的兇~我不覺得會被多討厭! 至少我常常都是那個"最害怕但也最喜歡的老師" 因為當你是有道理的兇時~小孩也不笨....他們知道~ 另外一個原因也是~現在被小孩討厭無所謂~只要可以正規地成長 ※ 引述《tomatoma (千載難逢)》之銘言： : 我是班上的助教... : 我一直以來都有不太能掌握班上秩序的問題冏~ : 因為我平時不兇...比較溫和 : 小朋友也就因此都不太怕我 : 之前有同事跟我說過 : 建議我不能像老太婆一樣嘮嘮叨叨 : 這樣小朋友會覺得說我也只是念一念而已不會怎樣 : 可是那該怎麼做會比較好呢？ : 有時候我說的小朋友會跟我討價還價 : 但換主教老師說的話...他們就不敢反抗 : 他們在我面前也比較調皮 : 有一次大家在練跳舞的時候 : 一個小朋友不想練 : 一直來找我要抱我 : 說不想跳、很難、很累的 : 那時候我們主教老師不在 : 旁邊的老師看到就說 : 他不想跳才這樣 : 你要負責教他啊 : 我一直鼓勵他說我們一起跳好不好 : 或者是跟他說現在大家很累可是都在練習 : 可是他還是不跳 : 我說那大家都要練習他不跳的話就只好請他出去了 : 都沒有用 : 後來是另一個老師把他帶去訓話 : 他就哭了...不敢再耍賴 : 這下真的很冏 : 我跟他們相處了快一年 : 卻還是沒辦法搞定他們 >"< : 尤其是午休的時候 : 我們班有培養大家都睡午覺的習慣 : 如果主教老師在的時候 : 大家都會乖乖睡覺 : 可是如果剩我一個人的時候 : 他們就常常會一直講話不睡覺 : 我罵他們的話我的聲音也會變得很吵 : 之前我會一個一個去哄他們睡覺 : 可是最近變得不太管用 : 在這邊哄這一個，另外一邊的就會吵起來 : 大家還是睡不著 : 之前帶我的老師說該兇的時候就兇、他們哭也關係 : 他們乖的時候就要對他們超好的 : 可是我不太明白那個兇的界限在那裡 : 之前有一次午休大家都不睡覺我很生氣 : 講道理講了...兇也兇了都沒用 : 我後來不跟他們講話 : 可是班上的小朋友似乎都不覺得怎樣 : 如果是主教老師生氣的時候只要板起面孔來出現嚴肅的表情 : 他們就知道情況不妙會馬上安靜 : 這種境界該怎麼達到啊... : 請大家給我一些建議...謝謝 -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 84.3.27.92
