私幼老師適用勞基法嗎? 這是個好問題，不過今天問題癥結不全在這邊，先略過。 基本上，像板友回文的說法一樣，法規規定，工作"一年"以上者， "十天"以前必須要"函"知雇主。為什麼要"函"因為要有"憑據"。 可是由企業倫理來說，和SIBI說的一樣，一個月可能是比較能讓人接受的時間。 這時候妳會很擔心自己想走又走不了，或者園所不願意發給你最後一個月薪水一類的。 這樣的情況雖然比較不希望發生，不過遇過很多這樣的案例，所以還是提一下。 因為私幼老師不全受勞基法的保障，我建議你找上自己所屬縣市的勞工局， 先點網頁進去看看吧，勞工局會有義工律師、相關的人員提供你諮詢、備案服務， 這樣的動作不代表馬上進入法律途徑，可是私幼老師沒有教師法可以保障， 勞基法又不完全適用的情況下，這是保護自己最好的辦法。 站在雇主的角度上他當然不希望你離職，再找一個人要花費的人事成本太麻煩了。 不過，如果你已經受不了這樣的環境，(看了原PO的暱稱與文章還滿心疼的)， 感覺就是被折騰的很難受，才會做這樣的決定。我相信園所會用盡各種理由留人， 因為要我是園長我也會這樣做! 不過大家都是"離職之前"就先提早早工作! 沒辦法，生活的壓力是現實的。 至於你所謂的是否轉換跑道! 我想，這真的是太多人想過的問題了。 可是，別人沒辦法給妳太多建議。 我國幼教環境真的不是很好，重點在你喜歡當一個幼教老師嗎? 一個人必須對自己有認同感、堅持，才可能面對工作環境的一切考驗與挫折。 不妨，你可以思考下面幾個問題，再決定自己的未來: 1. 請想像你未來的五年在做什麼?從事此工作需具備哪些條件與能力? 2. 再逐年列出這五年內，每年你想做的事情，及達成上述目標的實踐方針。 加油啦~ 祝你順利! ※ 引述《excellence (相信未來..我能嗎???)》之銘言： : 目前的園所讓我很厭煩 : 因為一個月有三週以上的周末要到學校---原本談的是隔週休的 : 強迫老師要招到小孩念書…否則寫悔過書 : 把老師當工人….搬著30公斤以上的重物上下五層樓 : 所以想換工作環境 : 且福利跟當初我進去時談的不一樣 : 所以想辭職找別間 : 可是又不知道怎麼提 : 因為園長是那種很會盧的人 : 且我們班陸陸續續換了不少帶班老師 : 我也覺得小孩很可憐 : 但是再待下去真的會瘋掉 : 園長沒有給老師基本的尊重 : 辛苦的過程她不看的…成果不滿意就發飆 : 連一句謝謝也沒有 : 且要老師留下加班就加班…時間無上限….還沒加班費 : 一開始還覺得自己剛出社會 : 應該懂得吃點苦 : 但是連最基本的尊重都得不到 : 我覺得應該要離開了 : 我9月初進去的 : 還不到半年 : 如果要辭職勞基法有規定要多久前提辭職嗎?? : 因為我怕他會用一個月前提辭職壓我 : 所以我又想快點找到下一份工作 : (因為爸媽不給錢囉…住外面得自己想辦法) : 不知道板上大大有沒有人有經驗 : 如何提辭職….和快點找下一份工作接上 : 或是我可以哪一個版參考?? : 感謝大家了 : 私幼都這樣嗎?? : 好累阿…..做教具跟佈置就算了…做粗活讓我很不能接受 : 在猶豫要繼續待這行業或轉行>”< : 感謝大家看完 這樂樂長的文章 -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 140.116.143.95
