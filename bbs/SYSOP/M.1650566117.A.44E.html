一、您的帳號是否仍為登入狀態 　【是】-> 您有下列兩種方式取回帳號 　　1. 設定聯絡信箱自行重設密碼 　　　 路徑：主選單 -> (U)ser -> (I)nfo -> (M)聯絡信箱 　　　 設定完成後於進站畫面輸入 /recover 自行重設密碼 　　　 （建議開一個新畫面操作，先不要登出帳號） 　　2. 向帳號部申請領回帳號 　　　 請至 ID_GetBack 板閱讀置底「密碼取回規範」公告 　　　 依照公告內流程發文申請領回帳號 　【否】-> 請依下方步驟繼續操作 　　若您的帳號當初是使用信箱註冊或曾設定聯絡信箱 　　可以直接嘗試於進站畫面輸入 /recover 自行重設密碼 　2022/04/25 更新：由於被盜帳號者，站方皆已主動停權，若您成功取回帳號， 　　　　　　 　　　請至 ID_Problem 板申請復權，詳情請看置底公告。 二、確認您的帳號是否遭到盜用 　請至主選單 -> (T)alk -> (Q)uery 輸入 ID 查詢您的帳號資料 　您將會看到如右側的畫面（範例）：https://i.imgur.com/wFexDZK.png
　請確認《上次上站》及《上次故鄉》的時間及 IP 位置是否為您本人使用 　（註：IP 位置查詢網路上有很多工具，請多加善用 　　　　站友 BigCat 有整理目前已通報有問題之 IP 位置 　　　　可閱讀本板 #1YNxYgPn (SYSOP) 文章來比對是否遭到盜用） 　如您可確認帳號遭到盜用，請依下方步驟繼續進行 三、前往鄰近派出所報案 　請攜帶您的身份證及相關證明至派出所報案 　（註：相關證明例如您的帳號資訊、註冊時的 Email 通知 　　　　建議您可將步驟二所查詢到的帳號資訊截圖印出） 　案由可使用《刑法》妨害電腦使用罪 　　盜用帳號 -> 第 358 條 　　更改密碼 -> 第 359 條 　（註：係依據司法院各法院過往判決書所引用法條給予建議） 　請務必確認取得「受(處)理案件證明單」才算完成報案程序 四、至 ID_GetBack 板申請 　請將「受(處)理案件證明單」以站內信寄給本站帳號站長 Makotoyen 　並在 ID_GetBack 板依 [盜帳] 公告按帳號部流程發文處理 　由於尚未開放被盜用的帳號領回，後續請靜候站方公告 五、問與答 　Q1：為什麼我要自己報案？ 　A1：帳號為您個人所有，只有您自己才可以報案，站方沒有權力代您處理。 　Q2：我一定要報案嗎？如果我已取回帳號，是否仍要報案？ 　A2：若您要追究對方的刑責，那一定要進行報案流程； 　　　且您無法確定帳號遭盜用期間，是否已經被用以非法用途， 　　　報案也是幫助您自保的方式。 　Q3：我人在國外/目前不方便前往派出所報案要怎麼辦？ 　A3：各縣市警察局有提供網路報案服務，您可以先利用網路報案的方式來處理； 　　　惟刑事案件仍需製作筆錄才能完成報案程序，後續您仍需至警察單位製作筆錄。 　Q4：我的帳號目前仍可登入，該怎麼確認是否有異常登入？ 　A4：您可依下列路徑確認您的帳號相關記錄 　　　主選單 -> (U)ser -> (L)My Logs 　Q5：我要如何確保帳號不被盜用？ 　A5：設定高難度的密碼（英數混雜、大小寫）、定期更換密碼、與其它網站使用不同的 　　　密碼等 　 強烈建議您務必至個人設定區設定聯絡信箱，忘記密碼或遭盜用後仍有機會領回帳號 　　　　　　【本篇內容由 s5048218 依個人經驗提供，不代表批踢踢站方】 -- 　　　　　　　　　　　 走過的，叫足跡；走不到，叫憧憬。 　　　　　　　　　　　 -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 219.91.2.63 (臺灣) ※ 文章網址: https://www.ptt.cc/bbs/SYSOP/M.1650566117.A.44E.html
