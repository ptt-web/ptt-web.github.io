※ 引述《hsuans (大豆大豆國貨好)》之銘言： : ※ [本文轉錄自 baseball 看板] : 作者: hsuans (大豆大豆國貨好) 看板: baseball : 標題: Re: [聯晚]巨蛋4公頃大 4萬觀眾席 : 時間: Thu Nov 13 17:07:56 2003 : 事隔數日，我想我花一點時間再來談巨蛋的問題。 : 1.經費從何來? : 按照台北市政府現在的搞法，就是把松煙這塊地拿出來。 : 看誰要bot，要取得這塊地所有權的人就蓋顆蛋。然後其他的地你要拿來作商場也可以。 : 所以理論上台北市政府「不必」出一毛錢。 松菸的地是國有地, 中央政府希望台北市買下來 不過台北市希望用換的 目前台北市要拿信義區A21的地來換松菸的地 但是A21現值只有一百零五億, 松菸十八公頃則有兩百五十億 原本台北市是希望只換巨蛋所需的八公頃用地就好 但是中央不同意, 而希望台北市先用A21換松菸十八公頃 不足的一百五十億則分五年分期付款 所以, 如果台北市要取得完整的十八公頃松菸用地 必須付出一百五十億元的代價 : 2.為什麼廠商要蓋? : 對廠商而言，市府所提出來的條件是很吸引人的。 : 蓋一顆蛋經費估計要150到180億左右，但是實際上給民間作的話花不了那麼多錢。 : 而這十公頃的地，去掉之後還有約六公頃的地可以運用，而光是這六公頃， : 市價大概要200億以上。 換句話說，對廠商而言，蓋顆蛋最少賺20億。 : 而廠商的這些投資，可以從哪裡收回來? : 想想看為什麼信義計畫區百貨公司一座一座蓋，就可以知道了。 松菸在松山區, 地段比信義區差, 所以地價比較沒這麼高 不過六公頃的地也有八十多億的價值 但是這六公頃的地也不是可以全部拿來開發商場 還必須符合建蔽率跟容積率 : 3.租金的問題 : 巨蛋蓋了會不會成為一座空蛋? 我個人的想法是「不可能」 : 為什麼? 可以從政府、廠商及巨蛋三方面來談。 : 1.市政府 : 蓋一顆蛋的確是一大政績，但是這個政績也要建立在「蓋了能夠常用」的基礎上。 : 沒有辦法(因為租金太高)使用的蛋，對市政府的政績只會造成傷害，無論是現任的馬 : 市長，或是繼任者而言都一樣。 : 在這個考量之下，前述市政府的招標，必須能夠確保巨蛋能夠被使用。因此要求 : 廠商訂租金時，不僅是「合理的」甚至是「可接受的」的租金都是很重要的。 其實政府在考慮興建大型公共設施時 不見得真的考慮過使用率的問題 就拿天母運動公園這個耗資三百億的重大建設來說好了 2001年使用天數還不到五十天 : 2.廠商 : 第一點，廠商在BOT爭取時，就必須提出條件，包括低廉(甚至是不符成本的)租金 : 來滿足市政府及主要的使用者(球團)，其次，我們要來觀察，如果巨蛋被使用，對於 : 廠商有什麼樣的好處? : 顯然地，巨蛋被使用時，所帶來的人潮是不算少的。這是顯而易見的部分，但是， : 對於廠商而言的另一優勢是，它可以以折扣或是送票的方式(買5000送一張票)來增加 : 營業額，這個促銷方案對於廠商而言成本幾乎是零。(因為巨蛋有比賽時，一萬人和兩萬 : 人觀賞的成本幾乎相同)，同時也可以作為減免球團租金的條件之一。 另一方面這也能 : 使的許多球迷會利用此點來此消費，而不是去隔壁的京華城。 不管怎麼促銷, 每年兩億元左右的維護經費 每天五十萬左右的使用費都是必須面對的開支 除非巨蛋能創造比這些費用還高的經濟效益 否則不足的部分終究要轉嫁到場租上 : 3.球團 : 球團如果能夠付出低廉的租金就可以使用巨蛋，在台北市場地缺乏的情況下，除非球團 : 完全放棄台北這個市場，不然使用巨蛋是必定的。 : 4.票房的問題 : 很多人把巨蛋將來的票房界定為「和現在差不多甚至多一點點」，「金剛和太陽比的話 : 大概更少」，我想提出一點不同的想法。 : 為什麼信義計畫區明明地價那麼高，但是廠商們寧可付出高租金或地價，也不願意將 : 百貨公司蓋在郊區? : 因為「位置」 : 信義商區這幾年漸漸形成人潮聚集之處，交通便利是一個主要的因素，每日在信義商 : 區流動、消費的人口，和天母、新莊是「不同等級的」 : 我請大家想一想，今天很多人不去球場的原因，是不是因為「天母、新莊」好遠? : 或是「天母、新莊」交通不方便? 或是「天母、新莊」附近沒有什麼玩的地方? : 再考慮到巨蛋球場的硬體設備與兩地不同，更可以吸引球迷進場，如此一來， : 進巨蛋的球迷必定會較兩地多的多。因為這個地方是上班族上完班，只要十分鐘，二十 : 分鐘就可以到達的地方。同時，如果有設置專用的包廂，對於下班後的商務應酬也是 : 十分適合的場所。 台北市立棒球場就在南京東路、敦化北路交叉口 以地段來說比松菸還好 但是在1997年, 也就是職棒榮景仍在時 台北市立棒球場的票房, 只比平均票房多不到五百人 : 當然，巨蛋不可能每場滿座，不過，他的確是必要的。 : 以上是一點點不成熟的淺見啦。 -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 220.130.2.97
