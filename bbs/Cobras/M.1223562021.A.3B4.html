撰文：林易萱 : 　　曾經在網路上發表文章，稱自己「聽說」中華職棒仍有打假球現象的米迪亞暴龍 : 隊老闆施建新，在還沒有查出職棒球隊簽賭事證前，施建新本人的出生年齡，以及學 : 經歷，卻先被檢舉有造假之嫌。 : 　　七月下旬，中華職棒球隊米迪亞暴龍隊的老闆施建新在批踢踢網站上發表（球隊 : 難管的真相）一文，指出中華職棒至今仍有打假球的現象存在。 : 　　這篇文章一出，各界譁然，施建新為此召開記者會，甚至被調查局請去說明。沒 : 想到事情發生半個多月，還沒有找出球員簽賭、打假球的具體事證，施建新本人的背 : 景，卻先被檢舉有造假之嫌。 : 　　今年初，誠泰將所屬職棒隊，出售給米迪亞系統科技的關係企業賽亞數位科技， : 球隊也改名為米迪亞暴龍隊，米迪亞執行長施建新因此受到外界矚目。 : 　　在此之前施建新已擁有一支超級籃球聯賽隊伍「米迪亞精靈隊」，後來又買了職 : 棒球隊。當時，許多媒體都以施建新年僅三十五歲，「年紀輕輕」就同時擁有籃球隊 : 、棒 : 球隊為題，報導這位台灣運動界的新面孔。 : 　　但是，本刊發現，施建新對外宣稱的個人資料，其中有不少疑點。 : 　　在施建新買下籃球隊和職棒球隊後，知名度也隨之提高，他的過去也逐漸被認識 : 。根據報章媒體報導，施建新曾經就讀過台灣第一學府台大，並曾在美國名校史丹佛 : 大學深造。學歷漂亮，加上執行長的身分，施建新一直給予外界「年輕有為」的形象 : 。 : 　　然而根據本刊記者去電台大註冊組、台大電機系、所辦公室，甚至是台大夜間部 : 求證，卻統統得到「查無此人」的回覆。 : 　　施建新過去對外的說法，皆表示自己僅是在台大「念過書」，並沒有畢業；但台 : 大表示，即使念過書，也會留有註冊的學籍資料，施建新連學籍資料都沒有，顯示他 : 並未註冊，不可能曾經是台大的學生。 : 　　　　　　學歷不符 並未在台大就學 : 　　事實上，本刊查證結果顯示，施建新的確是電機系的學生，不過，並不是台大， : 而是位於新竹的私立中華大學。 : 　　至於在史丹佛大學深造，以及申請博士學程一事真實性亦值得商榷。因為施建新 : 未從中華大學畢業，在沒有具備同等學力的情況下，如何申請進入史丹佛大學？ : 　　不僅施建新並非台大學生，本刊調查發現，施建新實際是台北市私立東山高中的 : 畢業生，他的年齡如果按照他畢業的時間來推算，與施建新目前所宣稱的明顯不符． : 根據查證，施建新是台北市私立東山高中第２６屆畢業生．創校於１９７２年的東山 : 高中，第一屆學生於１９７３年畢業，也就是說，施建新應是１９９９年高中畢業． : 　　若以台灣高中生正常的畢業年齡計算，１９９９年高中畢業的施建新應該出生於 : １９８１年，與施建新自稱的１９７３年出生，整整差了八歲．也就是說原本被認為 : 是六年級前段班的施建新，合理看來應是七年級生．而中華大學也證實施建新註冊時 : 所登記的出生年是１９８１年． : 　　施建新從未坦承自己真正考取的是中華大學電機系，卻宣稱曾在台大唸過書；關於 : 史丹佛大學的求學經歷，也同樣採取有誤導之嫌的說辭，施建新的同事．同業許多人都 : 誤以為他是台大．史丹佛高材生．可議的是，施建新如此自述，在許多媒體報導刊登後 : 他也不曾明確澄清． : 　　　　　　財力、經歷無從查證 : 　　去年１０月，施建新接受本刊專訪時表示，米迪亞系統科技，是自己用過去 : 創業所賺得的資金 成立的；今年初，施建新買下職棒球隊，更宣稱用來養球隊的資金 : 多半是自己的"私房錢" : 　　而攤開米迪亞的主要股東名單卻發現，米迪亞的資金，主要有兩大來源： : 一是施建新的家族；二是其他投資者，也就是上市公司松翰科技和上櫃公司環天科技的 : 董事長陳賢哲．從股東名單中，看不到有他個人的資金． : 　　施建新在接受本刊採訪與其他媒體訪問時所自述創業經歷，包括與友人在美國一起 : 研發的技術，被Newton買下，自己也成為Newton的員工．到１９９７年，Newton被蘋果 : 電腦購併，施建新因此獲得用來購併Newton而換股的蘋果股票；然後隨著蘋果股價上漲 : ，自己的身價也跟著增加．他並表示，為了籌措買球隊的資金，變賣不少蘋果股票，期 : 間並且因高買低賣，賠錢套現． : 　　如今Newton這家公司已不存在，無從查證．但是蘋果電腦購併Newton時，１９８１ : 年出生的施建新年僅１６歲；換句話說，施建新豈不是在14.5歲時，就已經和友人在美 : 國創業，才可能躬逢購併案；然而，他如何能夠在台灣的國．高中就學，又能同時在美 : 國工作，可說是匪夷所思． : 　　施建新的出身良好，父親是上櫃公司基泰營造的前總經理施義烈，施義烈於今年六 : 月卸任，轉任基泰營造子公司右泰建設的總經理。米迪亞成立之初，施義烈也曾經擔任 : 團相比，米迪亞是否有足夠的現金週轉能力，是最讓人霧裡看花的地方了。 : 　　 到底拿的是哪國身分？ : 　　 : 　　面對記者當面求證，施建新仍堅稱自己的確是1973年出生。他表示1981年，也就是他 : 九歲那一年，因為設籍日本，在台灣沒有公民身分，為了回到台灣居留，同時避免兵役問 : 題，因此用另一個身分，少報年齡在嘉義登記，此後，這個新登記的身分就成了他台灣的 : 身份證明。 :
