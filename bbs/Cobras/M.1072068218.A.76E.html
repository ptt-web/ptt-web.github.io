標題[新人]關於SO/BB, OBP. SLG,OPS 的簡單介紹
時間Mon Dec 22 13:24:34 2003
這篇單純只是想讓新手比較好找而已。因為整個PTT好像沒有人介紹。 全部都是行家對談，新人根本無從下手。 反正列在標題上，要找的時候按/就找得到了。 下面的資料主要來自 http://www.gida.idv.tw/page/2w/a030410.htm 感謝大大的整理，我只把重要的拿出來放。借花獻佛 不過要轉的話請先徵求允許。 BB Bases On Balls 四壞球 SO Strikeouts 三振 SO/BB 選球眼 三振/四壞比。 用來判斷打者是否會選球的標準。 接下來是OBP(上壘率)、SLG(長打率)、OPS這三個數據。 謝 佳 賢 (55 ) 打擊率：0.267 全壘打：18 打點：59 出場數:100 打席:426 打數:356 安打數:95 勝利打點:4 得分:50 一壘打:53 二壘打:23 三壘打:1 壘打數:174 上壘率:0.381 長打率:0.489 三振:91 雙殺打:11 三殺打:0 盜壘成功:2 盜壘刺:2 盜壘率:0.5 犧牲短打:1 犧牲飛球:2 四壞球:60 故意四壞:6 死球:1 四死球:67 H Hits 安打 BB 四壞球 HBP Hit By Pitch 死球，打到打者的球，打者會因此上一壘。 AB At Bats 打數 SF Sacrifice Flies 犧牲飛球 OBP On-Base Percentage 上壘率 (安打+四死球)/(打數+四死球+犧飛) 所以他的上壘率是 95+67/356+67+2=162/425=.381 OBP TB Total Bases 壘打數 (安打+二壘打+三壘打x2＋全壘打x3) SLG Slugging Percentage 長打率 (壘打數/打數) 所以他的長打率是174/356 =.489 OPS On-Base Percentage＋Slugging Percentage OBP+SLG 所以他的OPS值是.381+.489=.870 好吧，我承認我查了好久都沒找到OPS的中文翻譯，但是大大們在說OPS 的時候，是指一個打者的重要性，有的球員OBP高，有的球員SLG高。 如果要評論誰比較好，那麼一般的標準是看OPS，是這樣子的嗎? 其實評斷一個打者的綜合評量不只OPS，還有 TA(Total Average)、RC(Runs created)、 XR(Extrapolated Runs)等。 可是為什麼很多人愛用OPS，因為好算。 大部分的記錄都會將上壘率和長打率直接列出來，而不用像我們上面 一樣呆呆的計算，而OPS只是簡單的加法。 比較起來，像TA的計算 [壘打數+盜壘+四壞球+觸身球-盜壘刺] / [打數-安打+盜壘刺+雙殺打] 或者是RC ((安打+四死球-盜壘刺-雙殺打)x(壘打+0.26x(四死球-故意四壞球) +0.52x(?打飛+盜壘)))/(打數+四死球+?打飛) 甚至是XR (.50 x 一壘安打) + (.72 x 二壘安打) + (1.04 x 三壘安打) + (1.44 x 全壘打) + (.34 x (觸身球+四壞球-故意四壞))+ (.25 x 故意四壞)+ (.18 x 盜壘) + (-.32 x 盜壘刺) + (-.090 x (打數 - 安打 - 三振)) + (-.098 x 三振 )+ (-.37 x 雙殺打) + (.37 x 犧牲飛球) + (.04 x 犧牲觸擊) 的計算，哪一個比較好我不知道，但是在聯盟的統計上如果沒有直接列出來。 算下去是很累的.... -- 「安西先生..バスケが..バスケがしたいです.......」 -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 218.167.36.213
