現在是怎樣？台灣上ＢＢＳ的知識份子判斷力太低了嗎？還是邏輯思考太差勁？我 剛剛看了許多板的留言反應，果然跟我想的一樣：台灣人果然是世界上最好被愚弄 的人民。也就難怪今天統派媒體來一篇邏輯不通的社論，明天獨派媒體也來一篇是 非不分的評論，儘管錯誤百出，但是也都會有成千上萬的支持者寧願被愚弄，被操 控。 拜託，要識破蔡明里這種把戲很難嗎？比起玩政治的人，這還是雕蟲小技。 ※ 引述《shujoe (聖氣凌人)》之銘言： : 發言者：蔡明里 : 發言題目：關心聯盟裁判執法水準者,必看!(歡迎完整轉載) : 討論內容： : 近日,聯盟裁判組再次通知本台,自即日起不再配帶迷你麥克風(該設備主要是收本 : 壘附近的現場音效,讓觀眾較有臨場感)! OK，這是這一篇的導火線，起因於〔中華聯盟通知緯來體育台，即日起裁判不再配 帶迷你麥克風〕，其實這本來是一件不大不小的事情，我們來看蔡明里怎麼把它搞 大，怎樣把它跟〔誤判〕扯在一起。 : 如果,讓觀眾在家收看轉播的感覺變得比較不理想,這都要怪我及我們多角度的慢動 : 作重播! 從上一句到這一句的邏輯而在？ＯＫ，後面有寫，我們來看邏輯通不通。 : 這是今年聯盟裁判第二次拒絕本台,時機都發生在一連串的重大誤判之後!部份裁判 : 覺得本台"不夠意思",鏡頭,重播及主播都不太"友善",主播的部份幾乎完全就是針對 : 我,他們認為我講得太狠了!尤其在明顯的誤判之後!而且,在球季初,當裁判以不幫忙 : 配帶麥克風來給本台施壓之後,蔡明里居然不買帳!後續有意請和事佬來約個"鴻門宴" : ,沒想到蔡明里還是不買帳!轉播小組也是照常拍到什麼就播什麼! : 所以,今天的發展我一點都不意外! ＯＫ，這一段就很厲害了，巧妙地把聯盟不願意裁判配帶迷你麥克風跟誤判的事情 緊密連結在一起，用一句〔時機都發生在一連串的重大誤判之後〕和後面的所謂種 種不友善（其實多半都是蔡明里自己的猜測）把聯盟打成是〔因為誤判而不爽蔡明 里〕，而把蔡明里和緯來體育台塑造變成的受聯盟施壓的英雄。 現在眾多的球迷因為裁判過多的誤判而覺得非常不爽，包括六支球隊的所有球迷大 概沒有一個人會對裁判滿意，也因此這篇文章在這個時間點就很成功的挑起了球迷 的情緒，間接成為緯來的打手。 老實說，這對聯盟的一連串的指控請問證據何在？時間？地點？聯盟的說法又是如 何？ 狠一點來說，聯盟如果要搞蔡明里，有成千上萬種方式，幹嘛用這種〔不再配帶迷 你麥克風〕很驢的方式？這種方式對蔡明里而言有懲罰到什麼嗎？他還不是可以球 照撥，薪水照領？ : 聯盟裁判確實沒有義務為本台配置那小小的裝備(合約內未載明),但是如果能讓球 : 迷有更好的臨場感,這舉手之勞為何不為?而且拒絕的時機皆是最敏感的時候!是藉此 : 來懲罰本台的"不友善"? ＯＫ，一連串的 question mark，表示蔡明里之前的推論根本就是臆測之詞，單 方面的說法。 : 我必須強調該設備的用途並非要收錄裁判或球員在場上的對話,相信大家多年來也 : 未曾聽到一些私秘對話,因為這是我們與裁判的協議,所以,裁判總是以此作為藉口,總 : 讓我們感到啼笑皆非! : 我們的成音組會尋求替代的方法!絕不會去向部份裁判低頭或妥協!我們也會一本以 : 往公正轉播的立場,該重播就重播,是誤判就說誤判,判得精彩就說精彩!也不會刻意去 : 找裁判的麻煩! : 我相信,球迷絕對不會希望本台與裁判有任何私下交易! : 你們一定很難相信聯盟裁判是用這種心態來進行改革!或許,還有人把緯來當做聯盟 : 的"官方電視台"吧! 厲害厲害，從〔不再配帶迷你麥克風〕可以推斷到聯盟裁判的改革心態，這頂帽子 扣的也太大了。 的確，聯盟的裁判素質實在低落，有待改進。但是請眾多球迷自己想想，〔配帶迷 你麥克風〕跟減少誤判和提昇裁判判球品質有什麼相關性？難不成〔不再配帶迷你 麥克風〕就代表裁判不思改進？不想改革？ 換句話說，裁判的改革難不成一定要佩帶麥克風？ 這種因為私利（緯來希望能配帶迷你麥克風），然後藉由巧妙的言語技巧，把一件 大家都關心的事情（裁判品質）和這件事情扯在一起，挑動群眾的情緒，用群眾進 而對另一方施壓（聯盟）。這種事情已經屢見不鮮了，之前是兄弟象和中華職棒在 用，用來對付那魯灣聯盟，扯著什麼虛偽正義的大旗鬥爭那魯灣。現在比較好笑的 ，鬥人者人恆鬥之，這種手法被蔡明里學到了（雖然未必是有心的，但是在中職待 久了，手法自然而然就學會了，一出手就有），用來反批中華職棒聯盟。 拜託，台灣的球迷，你們還要被耍，被愚弄多久？一個ＥＳＰＮ耍的還不夠，現在 還要被緯來蔡明里耍？ -- ※ 發信站: 批踢踢實業坊(ptt.csie.ntu.edu.tw) ◆ From: 210.202.193.29
