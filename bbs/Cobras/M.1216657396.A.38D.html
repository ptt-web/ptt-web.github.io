其實對於戰績我都看得淡了，這個板的板眾應該都習慣了這隻球隊永遠的虎頭蛇尾。 然而我們還是會支持，因為這隻球隊還有我們支持的球員。 有不輸統一獅的打線，有台灣左打最大炮，有聯盟最頂級的中繼，有一票傑尼斯外野， 有三個年輕的新進左投，再交易來三個可以期待的年輕投手，有受傷的王牌竹子 這隻球隊就算一年打不起來，養得好的話三年內一定是隻奪冠勁旅。 最讓暴民傷心的，莫過於前程似錦的幾個年輕投手，先發救援兩頭燒， 結果就是賴俊男，這位有望接班球隊未來王牌投手燒到本季報銷， 陳建輔，變成今年新人投球局數最長的投手。 球隊的希望被燒掉了，今年戰績低迷，明年又看不到希望，而主因是什麼? 是投手不夠? 是"先發"投手不夠? 那請洋將啊，請來的又是什麼? 拿破崙不是球隊最迫切需要的，看在他打擊還算小有貢獻， 食之無味棄之可惜，先不鞭他。 那雷霸龍是怎麼一回事? 擺明是不能用，先發鐵暴，後援也像隨時會暴， 到頭來就是本土投手燒光光，剩下快贏的比賽又被雷霸龍給砸掉，球迷要怎樣看得下去? 再來，去年誠泰Cobras打線大爆發，一些看起來N年才一隻的打者都全壘打滿天飛 鄧老師、黃大頭、九尾名人、嗡嗡嗡，打教孫小毛功不可沒，連黃大頭都說孫小毛是恩師 結果米迪亞一接手，把去年中職最好的本土打教、誠泰不玩時主持全局帶隊訓練的孫小毛 換掉，先叫他去當二軍教練，後來叫他去當球探，最後讓他離隊，這又算什麼? 接替的人選有比較好嗎? 為什麼球員還是說現在打擊練習就照以前的菜單在做? 一隻有人氣的球隊要有讓球迷可以力挺，可以拿出來說嘴的東西，我們暴民有什麼? 新生代左投，兩隻受傷，一隻正在過度使用， 造就暴力打線的打教也沒了， 還有墊底的戰績，也許一部份球迷不離不棄，但能吸引多少全台灣的棒球迷? 減少虧損需要人氣，但是我看到球團許多做法卻都只是造成球迷流失， 希望球迷繼續支持，請球團三思自己花的錢做的事，是否能夠讓球迷去支持。 ※ 引述《dadny (小新)》之銘言： : 各位暴民好: : 在這裡 先對今年上半季的戰績不佳道歉 : 以及到現在為止的戰績道歉 : 也在這裡為我這半年來不常在台灣道歉 : 希望我們再來的戰績能跟著洋基一起反敗為勝.. : 言歸正傳 : 我星期四冒著颱風回到台灣 目的只有一個 就是為了看明星賽 : 當看到六局下 : 由我們(紅隊)連得八分 : 真的是感動到連眼淚都悄悄落下 : 更高興的是 分數是我們暴龍隊拿下的 : 晚上在台中的金色三麥慶功吃飯的時候 : 更巧的是..王傳家遠遠的認出我來 : 還特別走過來跟我打招呼 : 真是讓我感動啊!也讓我深深感受到揹負一個球隊的重任啊! : 今天有很多話想說.. : 因為今天是我們被通知接手球隊正好滿六個月的一天 : 這半年來 真是一種煎熬啊... : 我們真的完全沒有經營球隊的經驗 : 一切都是在錯誤 學習 成長的過程 : 我們也希望能讓球隊越來越好 : 就像今天我們下了決定 讓球隊的成員有了很大的變動 : 有離開的 有新加入的 : 都是為了讓球隊更好 : 但是 : 經營球隊 這是一個燒錢的過程啊 : 洪領隊說今年會賠四千萬 : 我們何止啊!!! : 我們非但不是網友說的不願意花錢 : 我們有可能是花錢花最兇的一隊 : 到六月底 我們已經花了一億四千七百萬 : 老實說 到年底 兩億元真的是跑不掉的 : 縱使有四千萬的轉播費 : 我們也要賠上一億五千萬 : 我們真的是萬丈高樓平地起 : 從完全沒有設備到可以參賽 這一路的心路歷程 真不足與外人道也 : 本來 : 要接球隊時 我們接到的數字 應該一年只要賠上六七千萬 : 但是 : 後來才發現 實際金額可能一倍還不止 : 事實上 這些錢 花了只要球迷高興 這一切都是值得的 : 但是 那天洪領隊跟我講了一番話 讓我心有戚戚焉 : 洪領隊:花錢沒有關係 但是不要花了錢又被罵.. : 事實上 現在被罵得最慘的球隊老闆 大概就是我們兩個人了吧 : 今天又看到 洪領隊講了有關球隊轉播金的談話 : 又看到了棒球板上大家對洪領隊的批評 : 我真是為他抱屈啊 : 那天談轉播權利金的時候 : 除了六隊老闆 還有聯盟和緯來八個人 : 每個人的發言都在比窮 比誰賠的多 : 而且大家都有拿數字出來佐證 真的都有賠這麼多 : 結果 沒想到 我們是賠最多的.....(哭哭) : 我想 : 今年度對米迪亞來說 真的是建立起一個新的球隊 多花錢是應該的 : 但是 : 就像我在公司常講的:失職是沒有理由的! : 所以在這裡 再為我們的戰績 向各位道歉~~ : 職棒不是有錢的公司就可以經營的 : 我真的覺得在棒球板上亂罵的鄉民 真的... : 有人說棒球不是兄弟這種小公司可以經營的 : 也有人點名叫我們米迪亞不要玩了... : 叫我們自立自強 : 我之前講過 : 如果棒球淪為大公司的廣告工具 : 那今天 棒球界就完了 : 中信 不是大公司嗎?
