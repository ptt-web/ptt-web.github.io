今年在墨西哥的成績。 Emil Kamar(Monclova) Game 23 Game Start 22 Innings Pitched 116.0 Hits Allowed 139 Homeruns Allowed 3 Strikeouts 50 Base on Balls 60 6-9 ERA 4.66 Emil Kamar，從記錄上看起來，應該是很典型的墨西哥投手。大概有點像以前的 安力魁或是羅魁那一種。他可能沒什麼球速，但是快速球的尾勁可能很賤。他沒 有三振能力，保送多了一些，控球不一定差但是可能比較閃。他能讓球留在場內 ，不丟三振也不被轟出場外。這些留在場內的球，有很多變成安打，加上那些保 送使他WHIP有夠高，可是，卻有還不錯的防禦率。可見他一定有些特別的本事， 可以讓壘上的跑者回不來。 以上都是推測，不過如果是真的，這個人可能前途看壞。實力是有的，不過，他 的弱點都恰好是在台灣會被嚴重放大的。首先不管他是控球不好還是投的太閃， 遇到台灣財判可能都會死。然後，他被打進場內的球越多，太陽的守備發生XD還 有財判婊人的機會當然也越多.... Jose Hernandez(Veracruz) Game 55 Game Start 0 Innings Pitched 36.1 Hits Allowed 41 Homeruns Allowed 3 Strikeouts 19 Base on Balls 9 4-1 1S ERA 2.48 我們終於等到一個左手中繼了！Jose Hernandez，今年出賽55場，排名聯盟第二， 可見他是球隊戰力中的重要角色，一個非常被信任的中繼投手。不過，投球局數非 常的少，可見他可能是那種專門出來面對左打者的短中繼。但是，在台灣，他別想 得到這種輕鬆的工作。只要他投的好，被當作肌肉男潘尼亞來用，也不是不可能。 他的球路應該有不錯的水準，配合他老到的經驗，以及左投優勢，應該會很好用。 重點只在於他有沒有辦法適應超過3局的「中繼」任務。他可以讓對手打不好，不過 也是沒有三振能力，也是要小心守備XD或財判婊人。 這兩個一起來，算是很好的組合。一個是正職的主力先發投手，一個是專任後援投 手，只要照著用，就剛好可以把投手群的洞補起來。萬一兩個都是先發或都是後援 那會很難排。通常台灣找洋將，很少能找到正職先發，通常都是找到後援投手或板 凳先發，然後到台灣再改為先發，運氣不好就會發生體能調整或是經驗不足的問題 。能找到這種在墨西哥先發20場的，很不容易。不過我在想，能夠這樣完美的找到 恰好一個先發一個後援，到底是特別要求的，還是瞎貓碰上死耗子？ -- ※ 發信站: 批踢踢實業坊(ptt.csie.ntu.edu.tw) ◆ From: 210.85.6.138
