標題李維拉………………劇場？
時間Sat Nov 5 21:45:05 2005
這篇本來是八三一前後要po的，但李杯杯因故沒回來就做罷 現在大家開始展望明年，所以又把它挖出來修一修。 ------------------------------------------------------------ 之前我po過李杯今年球季在墨西哥聯盟的成績 不知大家是否注意到，李杯前後待過兩隊 先是Laguna Cowboys（以下簡稱L隊） 然後是Aguascalientes Railroadmen （以下簡稱A隊） L隊上半季在他們那個聯盟（應該是北方聯盟吧，另外有一個南方的） 八支球隊裡面排倒數第二（咦？我們季初也曾在這個位子上坐過好一陣子……） 李杯在隊裡可以說是Ace級投手 （成績就請參考我八月那篇，不過那篇只記錄到7/25） 奇怪的是到了六月底L隊卻把李杯交易出去，原因不明 只看到一則短短的英文評論裡面記者在罵 說他搞不清楚L隊在想啥，幹嘛把陣中王牌賣給A隊， 難道是嫌輸球輸得不夠多？（西班牙文我看嘸，沒辦法找到更進一步的資料……） 沒了李杯的L隊，下半季是以倒數第一名做收 （難道墨西哥聯盟也有墊底搶狀元這回事？不是吧？？） 至於李杯到了聯盟中排名大約三、四的A隊後，表現如下： IP H R ER BB SO HR ERA 備註 6/29 1.0 0 0 0 0 1 0 3.72 10:3 領先時上場 6/30 1.0 0 0 0 0 0 0 3.65 6:2 領先時上場 7/02 1.0 1 1 1 0 0 1 3.75 7:2 領先時上場 7/25 1.0 1 1 1 0 0 1 3.86 4:8 落後時上場 7/30 1.0 1 0 0 0 0 0 0.00 5:0 領先時上場 8/04 0.2 4 4 4 2 1 0 20.25 6:5 領先時上場 (結果6:9 輸球) 很明顯，A隊幾乎只在差四分以上才派李杯出來 （他們另有一個no. 1 closer，巧的是也姓Rivera，另外還有幾個中繼後援） 但就算是這樣李杯的表現也抖抖的， 以墨聯打者的「泡瓦」和地理位置來看，hr多一點不意外 但把李杯L隊A隊成績綜合來看，會不會太多了啊？ 何況他是一個投球局數不多的的CP…… 另外，李杯不知道為什麼七月曾至少三星期沒出賽 當時他們例行賽還在進行，也不乏中繼救援機會 不知為啥就是沒他──受傷還是怎樣了？ 他老人家好不容易回來後，馬上被賞了一支HR問候 下一場雖沒失分，但再下一場就完全Orz…… 接下來沒再出場，不久後他們的球季也結束了， A隊似乎是以第四名做收。（時間有點久了懶得回去查，有錯請指正） 總之，八三一那時我想說的是，李杯若回來，劇場上演的可能性還是不小…… 現在則是，如果球團還要找他，最好把情況搞清楚： 他為什麼被L隊交易出去？那三個禮拜是怎樣？ 受傷了嗎？有沒有復原？前後成績落差怎麼說？ 我個人是沒有很期待李杯 但如果誠泰明年還是堅持要找些比克大魔王之類的娛樂我們 那麼──李杯你還是回來好了～ 反正我今年被誠泰鍛鍊了一整個球季，心臟夠強了 看你的劇場，我應該撐得住………………………………Orz -- ※ 發信站: 批踢踢實業坊(ptt.cc)
