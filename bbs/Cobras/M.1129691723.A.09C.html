※ 引述《musicscoop (天天要唸書~)》之銘言： : 在職棒場上，總是會有球員萌生退意的情形發生！ : 然而，這一位七年級生也不例外！ : 這一位曾經兩度想要放棄棒球的人，正是今天的主角【許竹見】。 : 在高中聯賽，以六成一一的恐怖打擊率，一舉拿下打擊王的竹子！ : 就在回學校等著國手通知時，卻發現代表隊的名單上沒有自己！ : 此時的竹子，在人生的道路上迷了路。 : 在沒有辦法給自己一個交代的情形下，一度想要放棄棒球！ : 竹子的母親告訴他：『當初是你自己要決定要打棒球的，現在如果你要選擇退出，不 : 會有人反對，但千萬不要做會讓自己後悔的事情。』 : 這一句話讓竹子知道，打棒球不是為了什麼，只是為了一個理想！ : 就因為他不想讓自己把遺憾留在球場上。 : 高中畢業後的他，曾投入那魯灣大聯盟的比賽中！ : 這時候，兩聯盟的整合、面臨兵役的問題，讓竹子當兵兩年都沒有機會碰球！ : 兩年過去，竹子收到了誠泰球團的通知！ : 也在郭總的親自召集下，竹子再一次投入職棒！ : 與竹子同梯，也是ＩＢＡ國手的林英傑在第一年繳出了亮眼的成績！ : 就在這時候，竹子想退出了！ : 第一年進入中職的他，不但是一勝難求，自責分率更是高的嚇人！ : 竹子又開始找不到答案，一個讓他繼續留在球場的答案！ : 左思右想的他，遇到了一位貴人，【孫昭立】教練。 : 毛哥依舊不改他操體能的臉色，只告訴竹子：『不要讓自己後悔！』 : 今年剛開季，竹子還是面臨跨季連敗的厄運！ : 就在４月１２號這一天，竹子投出了不到１００球的完投敗！ : 帳面上看起來是一個敗投，但內容上已經有國手的架勢！ : 隨後對統一的比賽，竹子找到了自己！ : 不單單是拿下一場勝投，也找回他在投手丘的霸氣！ : 例行賽在今天結束了，竹子也漂亮的拿下雙位數的勝投！ : 如果說，有一個獎項是『最佳進步獎』 : 我想沒有人比竹子更適合這個獎！ : 這個獎會永遠告訴球迷：竹子打球不為了什麼，只為了一個理想！ : 一個從他高中就開始堅持的理想：不把任何遺憾留在球場上！ 那我也來說說我心中的竹子吧.. - 以下全憑印象 有錯請見諒 高一那年我第一次看到竹子投球 那是金龍旗的一場比賽 對上三民高中 一開始我根本不認識許竹見這個球員 只是單純的地緣關係 我特別喜歡善化高中 熬夜看了看了重播的比賽 發現這個先發投手有讓我想在半夜尖叫為他加油的能力 他在投手丘上的霸氣 那種捨我其誰的氣勢 (ooh..真的是很帥 XDD) - 回頭翻了一下資料 這場比賽 竹子完投丟兩分 九次三振 善化9:2贏三民進八強 球速最快141km 從那場比賽開始 許竹見就變成我心中最愛的投手第二名了 接下來的IBA青棒選拔賽 隊上擁有陽森 許竹見 黃小偉 林智盛 許人介 郭曙偉 等的善化高中 一路闖進了冠亞軍 可惜對手是綠色怪物 有一左一右兩大王牌投手曹錦輝 林英傑的高苑工商 就這樣 善化只能拿下第二名 不過這也是竹子第一次當選國手 當時 大家的焦點或許都在曹錦輝 沈鈺傑 尤志峰身上 但是我知道中華隊還有一個值得期待的投手 縱使他真的沒什麼上場機會 = = - 1999年的IBA世紀青棒賽 在高雄澄清湖球場舉行 印象中每天都下雨 最後中華隊在冠亞軍決賽中輸給美國隊 只拿到亞軍 先發的是曹錦輝 高中畢業之後 竹子沒有繼續升學 加入合庫 我想這是為了加入職棒的暫時選擇 中間有一個插曲 有一篇報導寫說 善化高中的投手吳耀庭 在某場比賽中投出不錯的成績 而他的手套上繡著許竹見的名字 他說"這是學長送給我的 他要我好好加油" 看到這篇報導 驚了一下 發現竹子還有這麼細膩的一面 XDDD 好啦 然後一樣99年 竹子入選洲際盃成棒代表隊當國手 這就是大家對他印象最深刻的對上古巴完投敗的那場比賽 因為這場的好表現 讓很多人認識了許竹見 更讓他得到更多發展機會 還去到了美國陪曹錦輝一起練球 也傳出教士隊有意要簽下他 可惜最後忘了什麼原因(還是根本就不知道原因??) 沒有成行 接著竹子和同班同學楊森 一起加入了台灣大聯盟 挑戰役男打職棒的規定 其實我有點想不通 是不是當初的竹子比較傻阿 役男的問題吵得沸沸揚揚 當時的中華棒協也有封殺台灣大聯盟的潛意識 但他們還是加入了 我也只好乖乖支持竹子 看比賽就是了 竹子加入台中金剛隊 為了看他 我還跑去台中參加球迷會 結果主辦人就特地給我竹子跟楊森中間的座位 >//< 但是依稀記得當時的竹子蠻台的 XDDD 這一段不是重點..我承認 金剛隊那個時候有許銘傑和謝佳賢兩個大球星 成績也是很棒 竹子上場機會也不是很多 表現也是普普通通 開始有很多人會質疑開放役男打職棒是對還是錯 後來查到陪我們一起看王建民的袁博士有寫一篇關於這個議題的評論 主要是說其實舊傷也有關係 適應也需要一段時間 我的看法是 打職棒的年齡並不是很重要 重要的是心態問題 美國人沒有兵役問題 他們高中畢業就投入職業運動的很多 成功的也不在少數 而那個時候的竹子 我覺得呢 就是年紀小 心態也不夠成熟 所以表現平平 既然在金剛成績不是很棒 那就快速略過 就在兩個聯盟快要整合之際 竹子接到兵單 跟英傑拔還有陽森一樣要去當兵了 役男挑戰職棒的結果 就是被國訓隊封殺 意謂著一年多碰不到球 以每天都跟棒球為伍的棒球員來說 球感會消失 尤其是投手 傷害大嗎? 我那個時候非常擔心 以後的竹子還能不能打棒球??跟以前一樣好?? 接下來很長一段時間 我不看職棒了 原因是在兩大聯盟的整合 中華職棒聯盟有點像是在欺負台灣大聯盟的球員 說什麼以前的成績不列入計算 甚至懷疑這些球員的實力 我連聽緯來轉播都覺得好刺耳 >< 索性就不看了 去年 我聽到英傑拔回到球場的消息 還是以非常快的速度恢復水準 投出比當兵以前還要好的成績 報紙節目都有訪問他 也有提到他在當兵期間的自我訓練 是他能在當完兵之後短時間就能回到職棒場上的主要原因 聽到這個 我開始生竹子的氣 為什麼英傑拔能做到 竹子卻做不到 去年投出那麼高的防禦率 一勝難求 整個自信都不見了 這不是我記憶中的許竹見 直到今年 報紙開始出現竹子勝投的消息 一場一場的 感覺到竹子重新在投手丘上找回他的自信還有霸氣 讓我又開始有了看中華職棒比賽的衝動 開始注意竹子還有誠泰的每一場比賽 甚至在球場還會遇到小朋友跟我說"我最喜歡許竹見了 他很強" 也才驚覺到 竹子真正成為了一個優秀的投手 每一次聽著他的MVP感言 真的能感受到他內心的成長 對於教練的感謝 隊友的感謝一直掛在嘴邊 也很珍惜他的球迷 努力的學習電腦還有網路跟球迷互動 在那件事爆發後對球迷的信心喊話我還印象深刻 那個時候眼淚好像掉了不少 ^^|| 對於媒體的訪問應對 謙虛又得體 也會有純真的一面 對於球隊贏球的使命感更是沒有斷過 還記得他說 "每次三連戰都是我投第一場 我都覺得 只要我能搶下第一勝 他們兩個就會跟進" 他還有說過"只有球隊贏球我才會開心" (這句我是覺得有點over啦 XDD) (這樣的竹子誰會不愛咧 XDD 哎呀 我超愛的) 最後 球季賽的最後幾場 竹子的成績不是很好 澄清湖的那一場 我在現場看到心都碎了 傻眼 我也很擔心 季後賽竹子能不能投出之前的好成績 更擔心他的身體狀況 不過呢 咳咳 我也要來喊話一下 這季大家一定要一起走到最後 和球員一起努力 大聲的加油到最後 畢竟 像我這種球季中才開始認真看比賽的人 都已經從他們身上得到太多美好的回憶了 希望這個球季能成為屬於誠泰所有球員還有球迷的球季 (這樣應該沒有期待到吧 XD) 竹子加油 誠泰加油 大家要手牽手幫誠泰加油阿!! -- 謝謝有看到最後的人阿 寫了一堆 都是因為某人寫竹子給我的刺激啦 -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 211.74.244.79
