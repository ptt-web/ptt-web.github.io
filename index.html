What is Ptt?

批踢踢 (Ptt) 是以學術性質為目的，提供各專業學生實習的平台，而以電子佈告欄系統 (BBS, Bulletin Board System) 為主的一系列服務。期許在網際網路上建立起一個快速、即時、平等、免費，開放且自由的言論空間。批踢踢實業坊同時承諾永久學術中立，絕不商業化、絕不營利。(詳細的權利與義務以及隱私權保障請參閱 使用者條款)
目前在 Ptt/Ptt2 註冊的人數超過一百萬人，尖峰時段兩站容納超過十五萬名使用者同時上線，擁有超過兩萬個不同主題的看板，每日有上萬篇的新文章被發表以及閱讀，並且擁有前輩朋友留下數量可觀的資料文件。
本站並時常與 教育界 、 公益界 、 各大媒體 、 政府機關 、以及 Open Source Software 等等各界的朋友合作，在 link 中可以查詢與本站合作朋友的列表；亦同時歡迎各界的朋友與我們合作，詳情請見 與我們聯絡 。(恕不接受任何政治、商業的合作申請！)

批踢踢實業坊支持網站分級

批踢踢實業坊支持網站分級政策，已經在 批踢踢 、 批踢踢兔 的 telnet-based 介面，各板板主可依各板討論內容，自行決定該板是否開放十八歲以下使用者閱讀。一旦設定，將強制拒絕十八歲以下使用者進入。同時在 web bbs 及 網頁版精華區 對限制級內容加入機器分級標示。若您尚在使用微軟的 Internet Explorer ，還未更新到更快更好更安全的 FireFox ，請參閱 安裝說明 ，或拜訪 財團法人網站分級推廣基金會。

批踢踢提供的服務

Ptt BBS

電子佈告欄系統 (BBS, Bulletin Board System) 最初僅是用來提供一個資訊公告交流的空間，功能就如同其他各式的線上論壇一般；批踢踢提供更甚於論壇的功能，包括線上交談、電子郵件、小遊戲....
目前有 批踢踢實業坊 ， 批踢踢兔 ，以及 批踢踢參 。 (詳細說明)

Ptt Source

批踢踢將所有的程式碼透過 Git 管理，放置於 GitHub 上的 pttbbs project 內；批踢踢並將所有的程式碼以 GPL 的方式授權散布，任何人都可以自由的下載使用，以及查看相關修正，並歡迎各界提供修正改進與指教。 (詳細說明)

Ptt Video

您是否苦於在社團或系所舉辦大型活動所拍攝下來的數位影音資料，找不到地方放置供大家下載？或是雖然找到空間，卻來下載的全是自己人，沒有辦法讓其他的人一同來分享您的成果呢？ (詳細說明)
