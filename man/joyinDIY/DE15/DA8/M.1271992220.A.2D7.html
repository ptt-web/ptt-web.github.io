關於乳化劑的問題　其實沒那麼複雜啦 我自己DIY的經驗　所用到的乳化劑也不過就三種 1.透明乳化劑--->製作卸妝油專用 2.簡易乳化劑--->冷作(不需加熱)乳霜或乳液之用　 　　　　　　　 　　　　　　　　這個真的超級簡單的　油相跟乳化劑攪拌均勻後 　　　　　　　　再和入水相　攪一攪就完成了 　 　　　　　　　　如果是做乳液　就更容易了　 　　　　　　　　我都把東西加入瓶子裡(水先加一半) 　　　　　　　　用力搖一搖　均勻後再把剩餘的水加入　繼續搖　很快就好了 　　　　　　　　缺點：某些乳化劑容易油水分離 3.植物乳化蠟--->熱製乳化劑　需要加熱至70度左右　再加入油相攪拌 　　　　　　　　這個比較難　一開始做容易失敗　算是它的缺點 　　　　　　　　所以要靠經驗累積　熟了以後一樣可以亂搞...XD 　　　　　　　　(這個部分我不詳述　因為初學者最好找標準比例按部就班比較好) 　　 如果原po是油性肌膚的話　第三項幾乎可以不用考慮 我算是混合性肌膚　只有冬天這幾個月會用到熱製乳液 平常都只用簡易乳化劑製作的乳霜而已 但熱製乳化劑成品的好處是比簡易乳化劑的成品放得久 (我都不加抗菌劑的) 至於油水比例問題　之前推文有提到 簡易乳化劑　水＞油 植物乳化蠟　油＞水 還有　各家廠牌的乳化劑　乳化蠟多少都有點差異 有好用　有難用的　都要去試過才知道 以上... ※ 引述《netab (紫花霍香薊)》之銘言： : 這個問題我真的困擾了很久， : 也很努力的找了書跟各個網頁， : 不知道是我搜尋的方向不對還是怎樣， : 一直都沒有辦法找到多一點的資訊， : 考慮了很久， : 終於決定要問問大家^^ : 關於乳化劑 : 市面上有很多的種類 : 乳化臘、簡易乳化劑、清爽型乳化劑、冷作型、熱作型...... : 作乳液用、作卸妝油用、食品用... : 甚至連精油都有專用的乳化劑 : 有混合的(每種混合的又有不同的用途)有單一成分的 : 有的時候每本DIY的書裡名稱還都不一樣~"~ : 去各個化工行也有各種不同的名稱 : 不同種類的乳化劑適合的配方也會不同， : 油跟水的添加量也有差別， : 感覺上根本就是迷失在乳化劑海中了 : 想請問各位 : 大家都怎麼去挑選適合的乳化劑？？ : 還有配方(既然是DIY那油跟水的量應該可以自行調整吧) : 還有要做o/w 跟 w/o 的劑型時所需要的乳化劑應該也會不同吧 : 希望各位前輩可以提供相關的書籍或是介紹 : 感謝大家^^ : 如果認為我沒好好做功課可以噓我沒關係 : 可要記得提示我方向喔 -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 210.192.246.216
