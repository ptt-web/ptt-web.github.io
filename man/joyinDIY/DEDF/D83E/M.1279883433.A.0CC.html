標題[心得] 澳胡類馬賽皂
時間Thu Oct 23 23:41:19 2008
澳胡類馬賽皂，顧名思義，就是以澳洲胡桃油取代橄欖油在馬賽皂的位置。 製造日期：2008/10/14 熟成日期：2008/12/14 總油重：1300g 材料：未精製澳洲胡桃油 936g ( 72% ) 未精製乳油木果脂 182g ( 14% ) 棕櫚核仁油 182g ( 14% ) 氫氧化鈉 164g ( 鹼化率90% ) 純水 500g 色染：澳洲天然礦石泥 ( 淡藍 ) 1小匙；備長炭粉1/2小匙 精油：花梨木 Lavandin Lavender 蘇門答臘安息香 快樂鼠尾草 佛手柑 苦橙葉 作法：1.先以500g純水溶164g氫氧化鈉，待降溫。 2.棕櫚核仁油與乳油木果脂切小塊隔水加熱，待融化後再加入澳胡油。 3.油溫37℃、鹼液36℃左右，將鹼液倒入油中並充分攪拌均勻。 4.材料油呈濃稠狀時，加入精油攪拌後取少許皂液作色染。 5.所有皂液倒入模型並依個人喜好將染色皂液置底or置中。 6.放入保麗龍箱保溫24小時以上。 7.脫模後切塊並置於通風陰涼處等待熟成，熟成時間約2個月。 後記：未精製澳洲胡桃油呈淡黃褐色，在加入鹼液時會變成豆沙色， 隨著皂化的進展顏色會越來越淡，最後變成芋頭色。 澳胡油是前田京子相當推崇的植物油，因其含有多量的棕櫚油酸 ( 16-23% )， 而棕櫚油酸是延緩皮膚與細胞老化不可或缺的成分， 是故澳胡油常被用在熟齡肌膚保養or其他抗老化產品中。 以澳胡油為主做出來的手工皂洗感非常好，泡沫柔細好沖洗， 洗後清爽又不失滋潤，是做皂的優質材料油之一。 參考圖文連結：http://blog.xuite.net/jessica8945/blog/20245742 -- 　 ～～自由自在　～　來去如風～～ 　▁ ▂ ▃ ▄ ▅ 頑皮俠 ▅ ▄ ▃ ▂ ▁　 http://blog.xuite.net/jessica8945/blog -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 123.204.134.136
