標題[分享]甜杏酪梨油漩染皂
時間Sun Nov 16 17:24:30 2008
上一篇才說過幫皂皂上色必須使用耐鹼又不畏光的素材，怎麼這會兒就自打嘴巴..？ 怎麼說咧…當我一打開內有大馬士革玫瑰花瓣粉末的包裝袋， 那玫瑰特有的甜而細緻的芳香就撲鼻而來，再加上花瓣粉末是美麗的淡紅色， 真的讓我很心動！唉，就當我意志力不夠堅定好了， 明知道玫瑰花遇到鹼的下場是很悲慘的，我還是心一橫，給他撩下去了。 製造日期：2008/11/2 熟成日期：2009/1/2 總油重：1300g 材料：甜杏仁油 520g ( 40% ) 未精製可可脂 130g ( 10% ) 未精製酪梨油 325g ( 25% ) 椰子油 195g ( 15% ) 棕櫚油 130g ( 10% ) 氫氧化鈉 168g ( 鹼化率90% ) 純水 505g 色染：超細大馬士革玫瑰花瓣粉 1 小匙 精油：花梨木 玫瑰天竺葵 甜橙 苦橙葉 粉紅葡萄柚 維吉尼亞雪松 快樂鼠尾草 作法：1.先將168g氫氧化鈉以純水溶成鹼液，等待降溫。 2.可可脂切小塊與椰子油、棕櫚油隔水加熱，融化後再加入其他材料油。 3.油溫與鹼液同步36℃左右，將鹼液倒入油中並充分攪拌均勻。 4.材料油呈濃稠狀時，加入精油攪拌後取少許皂液以玫瑰花瓣粉調色。 5.將調好色的皂液倒入原來裝皂液的容器，邊倒邊徐徐攪動，此處勿過度攪拌。 6.最後把全部的皂液倒進鋪好耐油紙的皂模裡。 7.皂液連同模型置入保麗龍箱保溫至少24小時以上。 8.取出後脫模、切塊並置於通風處等待熟成，熟成時間約2個月。 參考圖文連結：http://blog.xuite.net/jessica8945/blog/20783874 後記：原本色彩美麗的玫瑰花瓣粉末，加入濃稠的皂液裡頭立即香消玉殞， 留下慘不忍睹的大●色….真的不知該說什麼才好….~_~ -- 　 ～～自由自在　～　來去如風～～ 　▁ ▂ ▃ ▄ ▅ 頑皮俠 ▅ ▄ ▃ ▂ ▁　 http://blog.xuite.net/jessica8945/blog -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 123.204.126.153
