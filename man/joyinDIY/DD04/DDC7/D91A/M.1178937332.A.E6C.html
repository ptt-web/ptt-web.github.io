我是超超超新手啦， 可是我有聽過皂化的時候， 鹼會先跟“飽和脂肪酸”反應， 等飽和脂肪酸反應完，再和“單元不飽和脂肪酸”反應， 單元的反應完，再和“多元不飽和脂肪酸”反應， 一般來說，脂肪酸好壞是：多元不飽和>單元不飽和>飽和， (這個好壞之前上生活化學課時老師講的，但他是說吃下去的情況下) 所以如果是用減鹼的方式， 可能會剛好一起把好的脂肪酸都作用掉了， 為什麼呢？不是減了鹼嗎？ 因為我們用皂化值所算出來所需的鹼量只是一個大概， 並不一定計算出來的鹼量可以100%完全和油脂反應掉， 有可能剩鹼，或是剩油， 情況一：如果使用超脂的方式， 在一開始反應中剩下的沒反應掉的鹼， 可以繼續和加進去的油脂中的飽和脂肪酸反應， 這樣一來，就會剩下很多好的脂肪酸-->所以滋潤皮膚 情況二：如果第一次反應，也就是未超脂前，剩的不是鹼而是油呢？ 那應該就沒有差別了吧，反正剩的油應該就是多元不飽和脂肪酸了(理論上來說) 超脂只是讓油更多，更滋潤肌膚。 就像甜杏仁油，它的多元和單元不飽和脂訪酸就占了所有脂肪酸的90%以上， 所以這種好的油才會建議拿來超脂用，因為要皂化成皂的脂肪酸， 就讓便宜的飽和脂肪酸去犧牲就好了， 好的脂肪酸等別人犧牲完，再來扮演滋潤的角色就夠了， 而這也正是我們所想要的。 這樣講起來，好像減鹼再超脂是最保險的方式？？ 至於是不是超脂的皂比較厚實， 這可能要用同一批油同一批NaOH，同樣的比例來實驗看看才知道 ※ 引述《SmallPigChan (有人要找我外拍嗎?)》之銘言： : ※ 引述《airstation (自製的唇蜜跟攻唇相似100)》之銘言： : : 因為...超脂可以添加本身皂體以外的額外油脂...<==就是這樣..所以有差別待遇 : : 因為..多餘的鹼用完了~而且還是按照正確的配方的鹼比例.. : : 所以皂體穩定~還可以增加保存的期限... : 對不起喔～又來問了。 : 假設 -5% 的 NaOH ，就好像 95% 的油脂是正常比例皂化， : 剩下5% 的油指，是額外多出來的（一如超脂），唯一的不同 : 是，這些油什麼時候出現。 : 因此不管超脂或是減鹼，不是都有皂體以外的油脂？那還有 : 差別待遇嗎？ : 同樣的，不管超脂或是減鹼都會把多餘的鹼用完， : 而且兩種方法都會剩下一定的油脂。既然兩種方法都把 : 鹼用完了，而且也都剩下油脂，那保存期限應該沒有不 : 一樣阿。 : : 基本上~~有種說法是說~ : : 皂體裡面的鹼量~~是維持皂體持久度的一個關鍵... : : 如果減少的話..酸敗機率可能會跟著增加... : : 其實這些都只是理論.. : : 如果你認為減鹼也OK的話..那其實就不用太擔心了@@a : 不管是減鹼或是超脂，皂體內的鹼量不是都比油少嗎？ : 所以，如果減鹼的持久度低，那超脂應該也一樣阿。 : 只要不去思考什麼時候加入那過多的油脂。 : 其實我來問大家是想確定兩種方法其實是沒有差別的。 : 另外： : CH2OCOR : | 加熱 : CHOCOR + 3NaOH ----> 3R-COONa + CH2OH-CHOH-CH2OH : | (皂) (甘油) : CH2OCOR : 只要 NaOH 都反應，甘油應該都一樣多吧？ -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 59.112.2.231
