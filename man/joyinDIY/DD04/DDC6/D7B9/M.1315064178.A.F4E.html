1.鹼量可能不夠： 一般提供的皂化價為平均值，實際上，皂化價會隨著油的品質改變 例如說：當年的風土條件、產地、等級... 所以每次買新的油，我都會先打少量的皂做測試 尤其是未精製、初榨的油，特別容易有皂化價不一的情形 想要得到品質穩定的皂，還是選用同一來源的精製油最保險 不過未精製油的洗感還有減少製程的環境友善度，都是難以捨去的:) 2.攪皂工具： 有些新手會拿湯匙、筷子、迷你打蛋器打皂，真的是讓自己事倍功半 一定要挑大隻的打蛋器，趕快去食品材料行或大型量販店好好選一支吧 攪拌頭要大到放不進馬克杯，握把和保特瓶的瓶蓋一樣粗，這樣才好握好用 3.配方 橄欖油或是初榨油品多...trace慢很正常 加入米糠油(20%以內)、蓖麻油(10%以內)、椰子油...這類的油放得多，就會快了 4.隔水加熱 到達一定溫度以上，皂體會軟化，反而會看不出來有無trace 攪拌到一定的時間(半小時~一小時)，可以熄火降溫，看看有無變稠？ 如果有變稠，就可以停下來了；沒有的話，就再繼續加熱攪拌。 ※ 引述《elmo777 (神秘密)》之銘言： : 大家好~ : 我是手工皂的新新手 : 爬過了文, 似乎沒看到我類似的問題, : 故請益^^ : 昨天我第一次做皂 : 配方如下 : 90g的椰子油(15%) : 90g的棕櫚油(15%) : 300g的初榨橄欖油(50%) : 120g的甜杏仁油(20%) : 總共油重600g : 86.31g的NaOH : 224.402g 的水 : 製造過程都如同書上所寫(參考格子的書籍) : 不時的一邊隔水加熱一邊攪拌, 約一小時 : 我的皂液確始終不見那"ㄍㄡˊㄍㄡˊ"的美乃滋狀 : (有濃稠了小小一點點, 但是就只到很稀的優酪乳那樣) : 顏色也有從原先的超透明深綠色變成很漂亮的淺淺"乳綠色" : 這是否就是沒有trace呢? : 是不是我哪裡出了問題... :'( : 今天早上看了一下 : 放了一晚的皂, 每一顆上面都浮很明顯的一層油 : (下面則是 淺淺乳綠色不知道是不是皂的東東) : 會不會是我太想要他的保溼, 橄欖油加了這麼多, 還加甜杏仁油 : 才會這樣? 好難過... : 謝謝各位^^ -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 114.34.232.108
