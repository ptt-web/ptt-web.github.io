天然的植物萃取途徑有很多種方式 例如用酒精/丁二醇/中藥水煮熬燉/油浸泡萃取...等等 先思考萃取出來的有效精華成分是"油溶性"還是"水溶性" 以紫草根為例,它的有效成分"紫草素"用油浸泡出來的成分會比水煮的多 所以用浸泡油入皂,他的期待功效絕對會比水煮或是只加入紫草粉來的好 另外不同等級/產區/品種的紫草根,以及浸泡時間長短所出來的顏色也會有所差異 因為懶的拍照片,只好用之前的照片參考參考浸泡油的顏色囉~ http://jjcc68.pixnet.net/blog/post/25994206 建議你到有信用的中藥店, 我的經驗是"軟根"紫草根的效果較好,另外你可以聞氣味知道新鮮度 有誠信優良的出產商他的紫草根,是真的全部只取"根部" 外觀上可見顏色很均勻較深紅或深黑,根型及剝落片完整,整條撥開連內部都是紫根片 有些品質較差的出產商就不是全部用"根", 你會看到有些木頭色雜質在其中,或是只有外層是紫草色,但剝開來內部是木質 氣味上比較有"腐味",這種等級就要避免囉~ 祝打皂順利^^ ※ 引述《misskate (貓娃撲粉蝶)》之銘言： : 話說看了很多人都打紫草皂！ : 我想有一部份是因為它出品的顏色吸引人. : 所以我也是被吸引的人之一~ : 話說第一塊我是想實驗看看，看出品會是如何？ : 在求快之下！我想到也可以用紫草去煮水， : 用來替代水丫！ : 嗯……結果我那鍋皂出品的是抹綠色！ : 我一整個有驚~ : 因為不死心， : 所以乖乖的用紫草泡了油！ : 我想這次和大家的比較相同， : 是紫藍色的那種， : 就是有一點點紫色呈現出來的顏色就是了！ : 但……我真的很好奇？ : 為什麼『顏色』會有差別？ : 因為我用的油完全相同，只差別在於煮水或是浸泡油而以！ : 出來的顏色竟然會有所差？ : http://tpling.pixnet.net/blog/post/27353332#comment-30424028 : 爬了文，這篇文說明了『紫草』及『紫草根』的不同！ : 但……我又不解了！因為我用的是同一種， : 很單純的就是去中藥店買紫草而以，我也沒有理會它根不根的問題？ : 因為，我真的不能了解，是什麼問題會讓成品出來的色澤會不相同， : 盼打皂高手及高手中的高中能告訴我了！ -- 即使流轉千萬載,也不要忘記回家的道路~ ~JC的原心倉庫http://jjcc68.pixnet.net/blog -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 123.192.106.81
