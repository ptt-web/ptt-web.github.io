標題[雪兒] 花水的種類及療效之簡介
時間Mon Mar 15 11:41:00 2010
不是每一種香草(藥草)都有花水，或者說可以買得到它們的花水 以下除了小玉自己用過的，其他是否有這樣的功效？不可而知！ 因為相關的研究資料太少了，或者說我沒找到可提供參考的書籍 如果這些從網路上集結而成的資料有誤，請多多提出指正囉！ 讓我們一起整理出正確又齊全的訊息給joyinDIY板 １．迷迭香花水 ( Rosemary water ) 功效：助益肝臟排毒，具有肌膚防皺，去除皮膚多餘油脂及青春痘。 適用：適合任何肌膚尤其油性、發炎膚質，男性肌膚較宜 　 (注意：不適合高血壓癲癇患者使用，避免在懷孕期間使用 ) ２．絲柏 ( Cyprès ) 功效：靜脈循環系統 (腿部沉重、靜脈曲張、痔瘡，改善酒糟鼻症)。能緊實肌膚，對於 面皰、靜脈曲張、微血管破裂都有幫助。是為很好的保濕、收斂、抗老化之功效 能增加並保持皮膚的水分，對於各種皮膚為很好的保養品，可美白、細胞重生。 適用：中性至乾性、成熟、敏感以及暗沉無生氣的肌膚 ３．玫瑰花水 ( Rose Floral Water ) 功效：安撫、滋潤、供給養份、收斂肌膚、幫助傷口癒合。 適用：適用於任何肌膚 (注意：會引起妊娠，懷孕期間勿使用！) (小玉的心得：冬季濕敷佳，但是皮膚脆弱的時候會很刺激~) ４．橙花花水 ( Neroli / Orange Floral Water ) 功效：鎮定痙攣作用，具收斂效果，平衡油脂分泌，改善粉刺、痘痘；可幫助細胞再生 修補凹洞、疤痕，滋潤肌膚。 適用：各種膚質，特別有益用於老化及敏感肌膚，對油性皮膚亦有平衡之功效。 (小玉的心得：油性皮膚溼敷可明顯感受到出油減少，美白效果不錯，結痂後不易留疤) ５．矢車菊花水 ( Cornflower Water ) 功效：皮膚病、腸道不適、結膜炎。 適用：油性及受損性肌膚，可用於眼部消炎、改善黯沉 ６．洋甘菊花水 ( Camomille water ) 功效：平撫情緒，減緩頭痛，潔淨肌膚，消除脹氣，防發炎及助眠，消退眼睛腫漲、 眼屎。 適用：適合過敏及乾性膚質 (注意：懷孕前四個月勿使用！) (小玉的心得：不管是哪一種洋甘菊都好臭……) ７．薰衣草花水 ( Lavander Floral water ) 功效：傷口癒合，殺菌，風濕性疼痛，蟲咬。 適用：適合所有肌膚 (小玉的心得：薰衣草是入門首選，氣味芳香，而且適用範圍廣，修護力強) ８．天竺葵 ( Geranium ) 功效：止血止痛，抗憂鬱，抗菌怯蟲，收斂，傷口癒合，除臭，降血糖，利尿等。 適用：油/乾/粉刺以及敏感性皮膚皆具有平衡和加強皮膚順應性的效果 ９．薄荷花水 ( Peppermint Water ) 功效：抗痙攣，醒腦，消炎抗菌，消退胃腸脹氣，鎮痛，解消充血腫脹，通經，收斂， 袪痰，退燒，助排汗，收縮血管，驅蠕蟲。 適用：油性肌膚，不適用敏感性肌膚 (注意：懷孕、哺乳期間勿使用！) １０．香桃木 ( Myrte ) 功效 ：安撫，抗菌，除臭，收斂，驅蟲。 適用：油性膚質 11.接骨木花水 ( Elderflower water ) 功效：安撫敏感性肌膚，收斂保濕，治眼疾，消退眼睛浮腫及黑眼圈 適用：脆弱和老化的肌膚 12.椴花(菩提) ( Tilleul ) 功效：溫和順氣助消化，安撫，皮膚曬傷，抗菌，抗氧化。 適用：乾性肌膚 13.茉莉花水 ( Jasmine flower water ) 功效：對乾燥與缺乏水份，太油與敏感性肌膚都具有平衡的作用。 適用：適用於任何肌膚 (注意：高度敏感勿使用！) (小玉的心得：除了香味很舒服，濕敷後沒有感覺…..我果然是中性皮膚？) 14.茶樹花水 ( Tee Tree Floral Water ) 功效：有止癢功效，對淨化皮膚，平衡油脂分泌，對於粉刺，痘痘等效果不錯， 對頭皮屑，香港腳……等也有不錯的幫助。 適用：適用於油性與中性肌 (小玉的心得：幫助痘痘消炎很有效！) 15.金縷梅花水 ( Witch Hazael Floral Water ) 功效：對毛孔粗大，易阻塞，對油性肌膚具有調節皮脂分泌與收斂毛孔的功效。 適用：適合油性問題膚質 (注意：勿服用！) (小玉的心得：減少出油和粉刺生長後，毛孔就會縮回去，但是回不去原狀) 16.香蜂草花水 ( Melissa Floral Water ) 功效：癒合傷口，止血，抗黴菌，防溼疹，鎮定皮膚，增強皮膚彈性，對過敏性膚質 效果很好。 適用：油性青春痘膚質 (注意：懷孕期間勿使用！) 17.依蘭依蘭花水 (Ylang Ylang Floral Water ) 功效：能平衡皮脂分泌，對頭皮也有刺激及補強作用，使新生頭髮具有光澤。 適用：乾性、油性膚質 (注意事項：刺激及敏感皮膚，發炎肌膚避免使用。) 18.佛手柑花水 ( Bergamot Floral Water ) 功效：對溼疹、乾癬、發炎、皮脂漏、瘤、牛皮癬都有療效，對皮膚是很好的抗菌劑， 佛手柑也具有美白作用，但須小心光敏性。 適用：油性、發炎膚質 (注意：因具光敏性，需晚上使用) 19.蘆薈水 ( Aloe Vera Juice ) 功效：含有大量的維生素B群及C、葉酸、胡蘿蔔素(維生素A的前身)，具有滋養皮膚， 保濕，能提供皮膚再所需的營養，廣泛使用在製皂，化妝品及面霜中。 適用：各種膚質，特別有益油性皮膚，有平衡之功效。 20.快樂鼠尾草花水 ( Clary Sage Floral Water ) 功效：具有抗炎抗菌抑汗緊實任何肌膚均可使用，尤其對頭皮出油抑制效果，對頭髮增 生也有幫助。 適用：適合所有肌膚 (注意：鎮定效果強，勿在開車或飲酒前使用，孕婦、低血壓及癌症避免使用) 21.金盞花花水 ( Marigold ) 功效：抗細菌、黴菌及各種真菌感染、對付皮膚上各種細菌感染，撫平舊疤、保濕、抗 老化、活化肌膚；口服則可針對淋巴炎，淋巴系統的排毒，及增強肝功能，提升 人體免疫能力。 適用：受傷的肌膚、乾性肌 (小玉的心得：傷口一定要清乾淨，癒合效果太好會把髒東西包進去。會色素沉澱，不 要長期使用) 22.蘋果花 Apple flower 功效：含有豐富的蘋果多酚，可以改善皮膚過敏、粗糙的現象， 具有消除青春痘、緊實肌膚、抗氧化的功效。 適用：油性肌、敏感肌 (有這種花水嗎？知道哪裡買得到的人告訴我吧？) 23.檸檬花水 ( Lemon water ) 功效：藉著去除老死細胞的使黯沉的膚色明亮，對油膩的膚質有淨化美白的功效。 因屬柑橘類，有光敏性，請避免白天使用。 適用：各種膚質皆可使用，特別有益老化及敏感肌膚，對油性皮膚亦有平衡功效。 (注意：因具光敏性，需晚上使用) 24.杜松果 ( Juniperus comminus ) 功效：強效的利尿劑。它能使組織肌肉收縮，擠壓出細胞間隙的液體，並促進腎功能 適用：適合油性或粉刺型皮膚 (注意：有腎臟問題者請避免使用) 25.菩提花水 ( Lime Flower Water) 功效：具有顯著的鎮靜、安撫和抗壓功效；可用它來對付失眠、焦慮狀態以及神經衰弱 等現象。用於治療乾性濕疹及發疹的痕癢，效果十分顯著 適用：油性肌 (注意：腎臟病患者以及懷孕頭三個月期間的孕婦請避免使用) 26.西洋蓍草 ( Yarrow Hydrosol ) 功效：對於消化系統、生殖系統以及肌肉系統都有抗痙攣的效果，具有輕微的殺菌消毒 效果，對幫助問題皮膚、粉刺與皮膚感染等症狀特別有效。 適用：受損、粉刺肌 27.歐白芷根 ( Angelica ) 功效：對神經系統具有獨特的鎮靜、沉澱以及安撫效果，亦是一種溫和的助消化劑，能 溫暖消化系統來促進食欲以及調理消化機能 適用：通常用於添加食物或芳療 28.大西洋雪松 ( Cedarwood / cedrus atlantica manet ) 功效：平衡而臉部皮膚油脂、牛皮癬、皮膚發炎、過敏紅疹、水痘、發炎粉刺、脫皮、 容易流汗的腳。對極度搔癢的問題，可以搭配西洋嗜草來使用。 適用：油性發炎肌 29.尤加利 ( Eucalyputus Blue / Eucalyptus globules ) 功效：具相當強的抗氧化功能，可運用在精神疲憊與恢復活力，或是感冒季節的自我預防 適用：對皮膚沒有功效
