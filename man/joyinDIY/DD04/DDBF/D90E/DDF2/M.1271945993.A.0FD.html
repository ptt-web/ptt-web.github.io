資料來源：自由時報 <<製作身體防曬乳>> 基本工具： 電子秤1個 / 500元 百靈攪拌器MR400/ 1400元 量杯兩個 / 30元 燒杯兩個 / 100元 溫度計1支 / 50元 攪拌棒1支 / 15元 身體防曬乳液材料： ‧油相材料‧ 防曬劑parsol 1789 3公克 / 約50元 防曬劑parsol SLX 5公克 / 約70元 防曬劑parsol 340 5公克 / 約50元 奈米二氧化鈦 5公克 / 約60元 化粧品級山茶花油 15公克 / 約39元 900型天然橄欖乳化臘 2公克 / 約9元 油包水型冷作乳化劑 4公克 / 約24元 薄荷腦 1公克 / 約4元 防曬增效滑順劑 2公克 / 約8元 ‧水相材料‧ A.防曬劑parsol HS 3公克 / 約36元 B.小黃瓜萃取液 5公克 / 約20元 C.蘆薈萃取液 5公克 / 約15元 D.食鹽1公克 / 約1元 E.純水40公克 F.奈米銀抗菌劑 3公克 / 約12元 G.試藥級氫氧化鈉0.5公克 / 約0.4元 步驟說明： 1.先用電子秤將各種材料依此比例秤出，將油相材料中所有材料逐一倒入量杯中， 用攪拌棒攪拌至融合。 2.將水相材料中A~F材料逐一倒入量杯中，用攪拌棒攪拌均勻，最後加入G，直到呈 現透明色狀液體即可。 ※氫氧化鈉有腐蝕性，建議戴手套進行，攪拌小心勿濺出觸碰皮膚。 3.將油相調和液置於量杯中，隔水加熱至80度並攪拌，至乳化蠟與防曬劑parsol 1789 都溶解即可。 4.將水相調和液置於量杯中，隔水加熱至80度約兩分鐘，並不斷攪拌。 5.不必等降溫，直接將1/5的水相調和液倒入油相調和液中，油相調和液以電動攪拌 器攪拌九十秒，呈現乳霜狀或者豆花碎塊狀即可。 重複此步驟五次，每次將1/5的水相調和液倒入油相調和液中，直到水相調和液完 全與油相調和液融合。 靜置約15分鐘，若無產生分離狀態即大功告成。 奈米二氧化鈦會沉澱，因此使用前需搖勻。 小叮嚀： 油包水型乳液一定要用有底盤的吸底式電動攪拌器，否則容易失敗。 還有1/5的水相調和液倒入油相調和液中，一定要完全調和，才能進行下個步驟。 如何保存： 保存期限可放一年 請放置陰涼處保存，避免讓陽光直射 也要避免放浴室和冰箱，以免因潮濕、溫度不穩定而讓成分產生變化 -- 哈囉，說你好！！ http://www.wretch.cc/blog/demirain -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 122.146.33.63 ※ 編輯: demiris 來自: 122.146.33.63 (08/15 10:08)
