我多事去計算了一下娜娜媽書本中的配方， (因為各種油種與品項都有精確列出，可以知道有沒有超脂或減鹼) 發現以她的配方，是沒有作減鹼的，而且母乳冰塊約略是氫氧化鈉重量的2.5倍。 所以，她書中的母乳皂，幾乎都應該稱呼為2.5倍母乳皂。 按照這本書的二, 三十種母乳皂配方來看(變化使用了20種以上的不同油品)， 一公斤上下的2.5倍母乳皂成品，通常使用了220克到260克的母乳冰塊。 而原po做一公斤的母乳皂成品，通常使用200多克的母乳。 這比例，你的母乳皂成品比較接近所謂2倍母乳皂。 很多代製母乳皂的人，會要求比實際使用量還要多的母乳冰塊， 這是為了預防意外狀況，準備多一點總比臨時不夠用要好。 單憑原po提供的網頁資訊，難以推測他用了幾公克的油， 也不確定他用了哪些品項的油，比例多少，所以難以推測他使用了多少氫氧化鈉。 所以單憑原po給的網頁資訊，難以確定他使用的母乳是氫氧化鈉的幾倍。 所以不知道他做的是幾倍母乳皂。 至於他網頁上寫的'正常母乳量', '加倍母乳量'與'半母乳'， 那是在指定客戶要提供給他的母乳量， 而不是在說他的成品是幾倍幾倍的母乳皂。 ※ 引述《hyochon (事業運旺旺來)》之銘言： : 幫朋友的朋友代製母乳皂 : 但是她聽到&我查 : 外界製作一公斤母乳皂的乳量都會拉到400cc : 請問這就是所謂的四倍母乳皂嗎? : 另外我還聽說 有人做"加倍" 可以拉到700cc : 讓我真的感到很驚訝 : 水份超出那麼多還可以成皂嗎??? : 還是有什麼特別的撇步呢? : 另外 : 400cc的母乳量來做一公斤的皂 : 那油的比例不就大概只能做600出頭 : 否則模應該會容不下吧? : 因為我之前做母乳皂都是習慣做200cc出頭(集乳袋一袋) : 是照花蓮姐的配方 : 但是現在200cc卻被稱為半母乳 好像我偷工減料似的=_= : 還請前輩們幫我解惑一下 -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 123.0.206.48
