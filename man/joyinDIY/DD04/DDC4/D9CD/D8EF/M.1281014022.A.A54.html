我自己做婚禮小物的時候 就是用果凍模(一個模子八個造型) 果凍模一個大約20-30元就有 如果只用簡單的配方(橄欖油，椰子油，棕櫚油)，成本不高 也可以用圓形的紗紗袋去包裝(一個只要3.5元) 我那時只買100個，三百個好像可以壓到一個3元 不過結婚時忙到我忘了幫婚禮小物拍照(哭哭....) 一顆皂約35-40g重，加上成本絕對可以壓在一顆皂35元以下 之前幫朋友做手工皂婚禮小物 一公分高，直徑約4-5公分的愛心皂 150顆左右，成本可以壓在1000元左右 http://missrain.pixnet.net/blog/post/10714197 貴就貴在用的油和包裝材料 當然，打皂花的時間和手工就無價了 800g的油，大約要加250g的水 (大概而已，因為要量鹼量再來算水的重量) 所以大約可以做出一kg左右的皂 我只會打固體皂，不會打鈉皂 所以我不知道你還缺什麼工具 不過以上價格可以供你參考 可是固體皂要放一個月等待皂化唷 ※ 引述《cc8d (欣)》之銘言： : 我是手工皂新人 : 看了唐亞的書籍才對手工皂產生興趣 : 接觸手工皂的經歷很淺 : 只做過液體皂 : 而且只打了四五鍋皂 : 現在為了準備結婚禮物而努力在版上爬文 : 目前屬意的應該是固體手工皂 : 可是我沒有做過固體皂的經驗 : 不不知道份量該怎麼抓 : 想請問版大們 : 如果想做 : 2cm高X5cm四方 : 的固體皂 : 數量300個 : 油品總重大約需要多少克呢[液體皂只要管重量@@] : 如果像以下版友scolary所做的婚禮皂 : 做到像我上面講的尺寸的話 : 含包裝成本有辦法壓到一塊35元內嗎 : http://tw.myblog.yahoo.com/savon520 : 而且我應該還缺一些工具 : 例如矽膠膜之類的 : [因為沒做過也不知道我還缺哪些工具現在想到的只有模具而已] : 請問各位版大 : 以我目前的狀況 : 我該自己做還是委託有在銷售的版友做呢 -- 如果你能遇到天使 你會祈求什麼樣的幸福 一個天使能給予什麼樣的幸福 一個人又能要求什麼幸福？ -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 114.37.154.239
