中藥房買的到的大青葉粉, 青黛粉, 紫草粉, 梔子粉, 分別可做出黃綠色, 藍色, 藍紫色, 暗黃色.... 還有海藻粉, 大麥若葉粉, 也是綠色系天然染料. 以油色較淡的油品浸泡紅椒粉, 綠茶, 紫草, 分別可以得到橘紅, 綠色, 紫紅色的浸泡油 有些油也有天然的色澤, 如棕櫚果油~~橘黃色, 未精緻酪梨油~~綠色,, 還有菠菜汁, 紅蘿蔔汁, 都可以代替水溶鹼以後, 分別做出綠色 或橘色的皂來. 未精緻酪梨油浸泡綠茶後, 綠上加綠, 但還是會退色. 炭粉~ 依使用量不同, 可做出灰色到黑色的皂 2G ~ 2% 最多變化的是紫草浸泡油, (別妄想煮水) 依其浸泡比例及配方的油品比例不同, 可變化出 藍 至深紫, 甚至綠色, 灰色的皂 不是每種原料拿去泡油就會泡出想要的顏色 有效成份也有水溶性和油溶性的分別 但凡是天然的就很容易退色, 所以下文看看就好 beta 胡蘿蔔素 = 黃色 肉桂 = 橘色 ~~ 褐色 紅椒 = 紅色 ~~ 褐色 可可粉 = 褐色 ~~ 巧克力色 葉綠素 = 綠色 鼠尾草 = 灰灰的綠 芙蓉 = 玫瑰紅 ~~~ 紫色 橘皮 = 檸檬皮 = 黃色 茴香 = 綠色 蒔蘿 = 淡綠色 ~~~~ 深綠色 迷迭香 = 綠色 ~~~ 褐色 薑 = 淡褐色的牛奶糖色 黑莓 = 土黃色 藍莓 = 淡土黃色 萵苣 = 淺綠 (非常淡哦 !) 胡蘿蔔 = 淡橘色 野玫瑰果 = 淡粉紅色 ~~~ 深粉紅色 胡荽葉 = 一開始淡綠色會慢慢變褐色 金盞草 = 亮黃色或橘色 蘆薈 = 灰灰的綠 玫瑰花瓣 = 深褐色 蒲公英 = 有點像是褪色的黃會慢慢變褐色斑點樣 小黃瓜 = 灰灰的綠 玉米粉 = 淡淡偏褐的黃 像甜菜根 與火龍果, 多鮮艷的紫紅色啊 遇鹼........ 就像鼻涕一樣, 毫無美色可言 要運用植物性的天然顏色, 要考慮遇鹼褐變 &急速退色 剛做好再怎麼美麗可愛, 也會隨著時間淡去那些顏色 炭粉 & 石泥類除外 先醬. -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 61.227.139.112
