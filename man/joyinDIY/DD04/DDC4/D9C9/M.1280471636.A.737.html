標題[分享]苦茶油三色皂
時間Thu Nov 6 13:56:20 2008
想要手工皂呈現美麗的好氣色，除了材料油本身的顏色外，就是利用添加物來顯色。 只是天然的色彩美則美矣，不是遇鹼變色就是怕光害，令人難以捉摸， 所幸還是有不少的天然色彩對自己的素顏有信心，堅持以原來的面目見人。 此次的苦茶油三色皂就是運用耐鹼又不畏光的素材來顯色。 製造日期：2008/10/30 熟成日期：2008/12/30 總油重：1300g 材料：苦茶油 520g ( 40% ) 未精製乳油木果脂 195g ( 15% ) 米糠油 260g ( 20% ) 椰子油 195g ( 15% ) 棕櫚油 130g ( 10% ) 氫氧化鈉 166g ( 鹼化率90% ) 純水 500g 色染：澳洲天然礦石泥 ( 淡粉紅 ) 2小匙；有機可可粉1/2小匙 精油：花梨木 玫瑰天竺葵 薰衣草 秘魯香脂 維吉尼亞雪松 甜橙 苦橙葉 作法：1.先將164g氫氧化鈉以純水溶成鹼液，等待降溫。 2.乳油木果脂切小塊與椰子油、棕櫚油隔水加熱，融化後再加入其他材料油。 3.油溫36℃、鹼液35℃左右，將鹼液倒入油中並充分攪拌均勻。 4.材料油呈濃稠狀時，加入精油攪拌後取少許皂液以可可粉調色。 5.先倒入1/3原色皂液入模，抹平後再薄薄的鋪上一層可可色皂液。 6.剩下的皂液以淡粉紅礦石泥調色後全部倒入模型(此步驟動作需輕、緩)。 7.皂液連同模型置入保麗龍箱保溫24小時以上。 8.取出後脫模切塊並置於通風處等待熟成，熟成時間約2個月。 後記：苦茶油與米糠油入皂後顏色偏黃，可以自成一色，如此就省掉調色的動作。 淡黃色皂液經過淡粉紅礦石泥調色後，色彩相當素雅。 因為淡黃色與淡粉紅都是屬於輕色調，界面不明顯， 我在兩色中置入咖啡色(可可色)就可清楚的分出兩色； 不過當時皂液還很柔軟，所有的分層動作都必須粉小心，才不會破壞液面。 分層皂做起來頗費功，不過成品是會讓人忘卻辛勞的。 參考圖文連結：http://blog.xuite.net/jessica8945/blog/20582410 -- 　 ～～自由自在　～　來去如風～～ 　▁ ▂ ▃ ▄ ▅ 頑皮俠 ▅ ▄ ▃ ▂ ▁　 http://blog.xuite.net/jessica8945/blog -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 123.204.107.6
